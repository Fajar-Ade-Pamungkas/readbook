HTML5 Drag and Drop

Drag and drop adalah bagian dari standar HTML5.

Drag and drop is a part of the HTML5 standard.

Drag and Drop

Drag (geser) dan drop (jatuhkan) adalah feature yang sangat umum. Ketika kamu "mengangkut" sebuah object dan menyeretnya ke lokasi yang berbeda.

Di HTML5, Drag and Drop adalah bagian dari standar, dan element apapun bisa di seret (draggable).

HTML Drag and Drop Example

Contoh Drag dan Drop HTML

Contoh dibawah ini adalah sebuah contoh sederhana drag dan drop:

Input:

<!DOCTYPE html>
<html>
<head>
<title> Judul Halaman </title>
<style> #div1 {
width:350px;
height:70px;
padding:10px;
border:1px solid #aaaaaa;
}
</style>
<script>
function allowDrop(ev) {    
ev.preventDefault();
}

function drag(ev) {    
ev.dataTransfer.setData("text", ev.target.id);
}

function drop(ev) {
    ev.preventDefault();
    var data = ev.dataTransfer.getData("text");
    ev.target.appendChild(document.getElementById(data));
}
</script>
</head>
<body>

<div id="div1" ondrop="drop(event)" ondragover="allowDrop(event)"></div>
<img id="drag1" src="img_logo.gif" draggable="true" ondragstart="drag(event)" width="336" height="69">

</body>
</html>

Ini mungkin terlihat rumit, tapi ayo pergi semua bagian - bagian dari event drag dan drop.

It might seem complicated, but lets go through all the different parts of a drag and drop event.

Make an Element Draggable

Buatlah sebuah element yang dapat digeser

Pertama - tama: Untuk membuah element draggable, setel attribute draggable menjadi true:

<img draggable="true">

What to Drag - ondragstart and setData()

Bagaimana untuk Menggeser - ondragstart dan setData()

Lalu tentukan apa yang harus terjadi ketika element digeser.

Pada contoh diatas, attribute ondragstart memanggil sebuah fungsi, drag(event), yang menentukan bagaimana data digeser.

Metode dataTransfer.setData() menyetel tipe data dan nilai dsri dragged data (data yang digeser):

function drag(ev) {    
ev.dataTransfer.setData("text", ev.target.id);
}

Pada kasus ini, tipe data adalah "text" dan nilainya adalah id dari dragged element ("drag1").

Where to Drop - ondragover

Kemana harus di taruh - ondragover

Event ondragover menentukan kemana dragged data (data yang digeser) bisa digeser.

Dengan default, data/element tidak bisa didrop (taruh) di element lainnya. Untuk mengizinkan sebuah drop, kita harus mencegah default handling dari element.

Ini dilakukan dengan pemanggilan metode event.preventDefault() untuk event ondragover:

event.preventDefault()

Do the Drop - ondrop

Lakukan Drop - ondrop

Ketika dragged data ditaruh (dropped), sebuah event drop terjadi.

Pada contoh diatas, attribute ondrop memanggil sebuah fungsi, drop(event):

function drop(ev) {
    ev.preventDefault();
    var data = ev.dataTransfer.getData("text");
    ev.target.appendChild(document.getElementById(data));
}

Kode Menjelaskan:

  • Panggillah preventDefault() untuk mencegah browser default handling dari data (default itu terbuka seperti link pada drop)
  • Dapatkan dragged data dengan metode dataTransfer.getData(). Metode ini akan mengembalikan data apapun yang disetel menjadi tipe yang sama dalsm metode setData()
  • Dragged data adalah id dari dragged element ("drag1")
  • Lampirkanlah element yang dapat digeser kedalam element drop
