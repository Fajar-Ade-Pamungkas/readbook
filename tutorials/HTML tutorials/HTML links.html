HTML Links

Link ditemukan hampir di semua halaman web. Link mengizinkan prngguna untuk mengklik mereka, jalan untuk berpindah dari satu halaman ke halaman yang lain.

HTML Links - Hyperlinks

HTML link merupakan hyperlink.

Hyper link adalah teks atau gambar yang kamu bisa klik dan melompat ke dokumen yang lain.

HTML Links - Syntax

Pada HTML, link didefinisikan dngan tag <a>.

<a href="url">Teks link</a>.

Contoh

Input:

<a href="http://www.readbook.co.id/HTML HOME.html"> Visit our HTML tutorial </a>
Visit our HTML tutorial

Attrihute hre menentukan alamat tujuan (http://www.readbook.co.id/HTML HOME.html)

Link text adalah bagian yang terlihat (Visit our HTML tutorial).

Pengklikan pada link text, akan mengirimkan kamu ke alamat tertentu.

Link text tidak harus berupa teks, link text bisa sebuah gambar HTML atau element HTML lainnya.
Tanpa sebuah trailing slash pada alamat subfolder, kamu mungkin menghasilkan dua permintaan kepada server.
Kebanyakan server akan secara otomatis menambahkan sebuah trailing slash menuju address, dan setelah itu membuat permintaan baru.

Local Links

local links adalah sebuah link lokal yang pengalamatannya berada dijangkauan lokal.

Contoh diatas menggunakan sebuah absolute URL (sebuah alamat web penuh).

sebuah link lokal (link menuju web site yang sama) ditentukan dengan URL yang relatif (tanpa http://www....).

Contoh

Input:

<a href="http://www.readbook.co.id/HTML HOME.html"> Global link </a>
<a href="../CSS tutorials/CSS HOME.html"> Local link</a>

HTML Links - Colors

Ketika kamu menggerakan mouse pada sebuah link, dua hal yang normalnya akan terjadi:


  • Arah panah mouse akan berganti menjadi sebuah tangan kecil
  • Warna element link akan berubah

Dengan default, sebuah link akan tampak seperti ini (pada semua browser):


  • sebuah unvisited link (link yang belum dikunjungi) digaris bawahi dan berwarna biru
  • sebuah visited link (link yang sudah dikunjungi) digaris bawahi dan berwarna ungu
  • sebuah active link (link yang aktif) digaris bawahi dan berwarna merah

kamu bisa mengubah warna defaul, dengan memggunakan style:

Contoh

Input:

<!DOCTYPE html>

<html>
<head>
<style>
a:link    {
color:green;
background-color:transparent;
text-decoration:none
}
a:visited {
color:pink;
background-color:transparent;
text-decoration:none
}
a:hover   {
color:red;
background-color:transparent;
text-decoration:underline
}
a:active  {
color:yellow;
background-color:transparent;
text-decoration:underline
}
</style>
</head>
<body>

<a href="HTML CSS.html"> This is a link </a>

</body>
</html>

HTML Links - The target Attribute

Attribute target menentukan kemana untuk membuka dokument link.

Contoh ini akan membuka dokument link dalam sebua browser window atau dalam sebuah new tab:

Contoh

Input:

<a href="http://www.readbook.co.id/HTML HOME.html" target="_blank"> Visit our HTML tutorial </a>
Visit our HTML tutorial
Nilai Target Deskripsi
_blank Membuka dokument link dalam window atau tab baru
_self Membuka dokument link dalam frame (bingkai) yang sama seperti ia diklik (ini default)
_parent Membuka dokument link dalam parent frame (bingkai induk)
_top Membuka dokument link dalam full body dari window
framename Membuka dokument link dalam sebuah named frame (bingkai yang dinamai)

Jika webpage kamu kamu dikunci dalam sebuah frame, kamu busa menggunakna target="_top" untuk menjeda keluar dari frame:

Contoh

Input:

<a href="http://www.readbook.co.id/HTML5.html" target="_top"> HTML5 </a>

HTML Links - Image as Link

Ini sudah umum digunakan untuk gambar sebagai link:

Contoh

Input:

<a href="HTML images.html" target="_blank" >
<img src="../../pictures/Rain Fall.png" alt="Rainly" style="width:42px; height:42px; border:0">
</a>

Klik gambar diatas untuk menuju ke sebuah halaman lainnya.

border:0 ditambahkan untuk mengatasi masalah di IE9 (dan versi sebelumnya) daru penampilan sebuah border sekitar gambar.

HTML Links - Create a Bookmark

HTML bookmark digunakan untuk memperbolekan pembaca untuk melompati ke bagian - bagian yang spesifik dari sebuah Web page.

Bookmark itu praktis jika website kamu memiliki halaman yang panjang.

Untuk membuat sebuah bookmark, kamu pertama-tama harus membuat bookmark, dan kemudian menambahkan sebuah link untuk untuk bookmark tersebut.

Ketika link dikelik, halaman akan mengulung (scroll) ke lokasi dengan bookmark.

Contoh

Pertama, buatlah sebuah bookmark drngan attribute id:

<h2 id="localInk">Local link</h2>

Lalu tambahkan sebuah link menuju bookmark ("Local link"), dari dengan halaman yang sama:

<a href="#localInk">Local link</a>

Atau tambahkan sebuah link menuju bookmark ("localInk"), dari halaman lainnya.

Contoh

Input:

<a href="HTML CSS.html#myStyling" target="_blank" > menuju ke id="myStyling" di halaman lainnya. </a>
menuju ke id="myStyling" di halaman lainnya.

Ringkasan Materi

  • Gunakanlah element <a> HTML untuk mendefinisikan sebuah link
  • Gunakanlah attribute href HTML untuk mendefinisikan alamat link
  • Gunakanlah attribute target HTML untuk mendefinisikan kemana untuk membuka dokument link
  • Gunakanlah element <img> HTML (didalam <a>) untuk menggunakan gambar sebagai link
  • Gunakanlah attribute id HTML (id="nilai") untuk mendefinisikan bookmark dalam halaman
  • Gunakanlah attribute href HTML (href="#value") untuk link yang menuju bookmark
