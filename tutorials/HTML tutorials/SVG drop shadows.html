SVG Drop Shadows

<defs> and <filter>

Semua internet SVG filters didefinisikan dengan sebuah element <defs>. Element <defs> adalah singkatan untuk definitions dan berisikan definisi tentang element spesial (demikian sebagai filters).

Element <filter> digunakan untuk mendefinisikan sebuah filter SVG. Element <filter> memiliki sebuah attribute id yang diharuskan yang mana untuk mengidentifikasi filter. Grafik lalu menitikan pada filter untuk digunakan.

SVG <feOffset>

Example 1

Contoh 1

Element <feOffset> digunakan untuk membuat efek bayangan belakang. Ide untuj membawa sebuah grafik SVG (image atau element) dan menggerakkan itu bit kecil pada xy plane.

Offset contoh pertama sebuah rectangle (dengan <feOffset>), lalu mencampurkan original di atas dari offset image (dengan element <feBlend>):

Disini adalah kode SVG:

Contoh SVG

<svg height="120" width="120">
  <defs>
    <filter id="f1" x="0" y="0" width="200%" height="200%">
      <feOffset result="offOut" in="SourceGraphic" dx="20" dy="20" />
       <feBlend in="SourceGraphic" in2="offOut" mode="normal" />
    </filter>
  </defs>
  <rect width="90" height="90" stroke="green" stroke-width="3"
  fill="yellow" filter="url(#f1)" />
</svg>

Penjelasan kode:

  • Attribute id dari element <filter> mendefinisikan sebuah unique name kepada filter
  • Attribute filter dari element <rect> me-link element menuju "f1" filter

Example 2

Contoh 2

Sekarang, offset image bisa diblurkan (dengan element <feGuassianBlur>):

Disini adalah kode SVG:

Contoh SVG

<svg height="140" width="140">
  <defs>
    <filter id="f2" x="0" y="0" width="200%" height="200%">
      <feOffset result="offOut" in="SourceGraphic" dx="20" dy="20" />
       <feGaussianBlur result="blurOut" in="offOut" stdDeviation="10" />        <feBlend in="SourceGraphic" in2="blurOut" mode="normal" />
    </filter>
  </defs>
  <rect width="90" height="90" stroke="green" stroke-width="3"
  fill="yellow" filter="url(#f2)" />
</svg>

Penjelasan kode:

  • Attribute stdDeviation dari element <feGaussianBlur> mendefinisikan jumlah blur

Example 3

Sekarang, buatlah bayangan hitam:

Disini adalah kode SVG:

Contoh SVG

<svg height="140" width="140">
  <defs>
    <filter id="f3" x="0" y="0" width="200%" height="200%">
      <feOffset result="offOut" in="SourceAlpha" dx="20" dy="20" />
       <feGaussianBlur result="blurOut" in="offOut" stdDeviation="10" />        <feBlend in="SourceGraphic" in2="blurOut" mode="normal" />
    </filter>
  </defs>
  <rect width="90" height="90" stroke="green" stroke-width="3"
  fill="yellow" filter="url(#f3)" />
</svg>

Penjelasan kode:

    >
  • Attribute in dari element <feOffset> diubah menjadi "SourceAlpha" yang mana menggunakan Alpha Channel untuk blur pengganti dari entire RGBA pixel

Example 4

Contoh 4

Sekarang, perlakukan bayangan menjadi berwarna:

Disini adalah kode SVG:

Contoh SVG

<svg height="140" width="140">
  <defs>
    <filter id="f4" x="0" y="0" width="200%" height="200%">
      <feOffset result="offOut" in="SourceGraphic" dx="20" dy="20" />
          <feColorMatrix result="matrixOut" in="offOut" type="matrix"
      values="0.2 0 0 0 0 0 0.2 0 0 0 0 0 0.2 0 0 0 0 0 1 0" />  
<feGaussianBlur result="blurOut" in="matrixOut" stdDeviation="10" />        <feBlend in="SourceGraphic" in2="blurOut" mode="normal" />
    </filter>
  </defs>
  <rect width="90" height="90" stroke="green" stroke-width="3"
  fill="yellow" filter="url(#f4)" />
</svg>

Penjelasan kode:

  • Ffilter <feColorMatrix> digunakan untuk transformasi warna pada offset image closer menjadi hitam. Tiga nilai daru '0.2' pada matrix all get multiplied dengan red, green, dan blue channel. pengurangan nilai - nilai itu membawa color closer menjadi hitam (black is 0)
