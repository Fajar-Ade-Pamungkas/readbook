HTML5 - SVG <rect>

SVG Shapes

Bentuk - bentuk SVG

SVG memiliki beberapa element bentuk yang sudah dikenal yang bisa digunakan oleh developers:

  • Rectangle (persegi) <rect>
  • Circle (Lingkaran) <circle>
  • Ellipse (oval) <ellipse>
  • Line (garis) <line>
  • Polyline (segitiga satu) <polyline>
  • Polygon (segitiga banyak) <polygon>
  • Path (garis edar) <path>

Chapter berikut akan menjelaskan setiap element, Dimulai dengan rect element.

SVG Rectangle - <rect>

Example 1

Contoh 1

Element <rect> digunakan untuk membuat sebuah persegi panjang dan variasi dari bentuk persegi panjang:

Disini adalah kode SVG:

Contoh Kode SVG

<svg width="400" height="110">
  <rect width="300" height="100" style="fill:rgb(0,0,255);stroke-width:3;stroke:rgb(0,0,0)" />
</svg>

Penjelasan Kode:

  • Attribute width dan height dari element <rect> mendefinisikan tingi dan lebar dari rectangle
  • Attribute style digunakan untuk mendefinisikan properti - properti CSS untuk rectangle
  • Properti fill CSS mendefinisikan fill color (warna isi) dari rectangle
  • properti stroke-width CSS mendefinisikan lebar atau tebal border dari rectangle
  • Properti Stroke CSS mendefinisikan warna border dari rectangle

Example 2

Contoh 2

Ayo lihatlah di contoh kainnya yang berisikan beberapa attribute baru:

Disini adalah kode SVG:

Contoh Kode SVG

<svg width="400" height="180">
  <rect x="50" y="20" width="150" height="150"
  style="fill:blue;stroke:pink;stroke-width:5;fill-opacity:0.1;stroke-opacity:0.9" />
</svg>

Penjelasan kode:

  • Attribute x mendefinisikan posisi kiri rectangle (contoh: x="50" menempatkan rectangle 50 px dari margin kiri)
  • Attribute x mendefinisikan posisi atas rectangle (contoh: y="50" menempatkan rectangle 50 px dari margin atas)
  • Properti fill-opacity CSS mendefinisikan opacity (sifat tak tembus cahaya) dari fill color (jangkauan legal: 0 sampai 1)
  • Properti stroke-opacity mendefinisikan opacity dari stroke color (jangkauan legal: 0 sampai 1)

Example 3

Contih 3

Definisikan opacity untuk seluruh element

Disini adalah kode SVG:

Contoh Kode SVG

<svg width="400" height="180">
  <rect x="50" y="20" width="150" height="150"
  style="fill:blue;stroke:pink;stroke-width:5;opacity:0.5" />
</svg>

Penjelasan kode:

  • Properti opacity CSS mendefinisikan nilai opacity untuk seluruh element (jangkauan legal: 0 sampai 1)

Example 4

Contoh terakhir, buatlah sebuah rectangle dengan sudut yang dibulatkan:

Disini adalah kode SVG:

Contoh Kode SVG

<svg width="400" height="180">
  <rect x="50" y="20" rx="20" ry="20" width="150" height="150"
  style="fill:red;stroke:black;stroke-width:5;opacity:0.5" />
</svg>

Penjelasan kode:

  • Attribute rx dan ry membukatkan sudut dari rectangle
