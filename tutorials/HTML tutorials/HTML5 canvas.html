HTML5 Canvas

Element <canvas> HTML digunakan untuk menggambar grafik pada sebuah halaman web.

Grafik disamping kiri diciptakan dengan <canvas>. Ia menampilkan empat element: a red rectangle, a gradient rectangle, a multicolor rectangle, and a multicolor text.

What is HTML Canvas?

Apa itu HTML Canvas ?

Element <canvas> HTML digunakan untuk menggambar grafik, on the fly, via scripting (biasanya JavaScript).

Element <canvas> merupakan sebuah container (tempat) saja. Kamu harus menggunakan script untuk menggambar grafik yang sebenarnya.

Canvas memiliki beberapa metode untuk penggambaran paths, boxes, circles, text, dan penambahan gambar.

Canvas Examples

Contoh Canvas

Canvas merupakan sebuah area persegi panjang pada sebuah halaman HTML. Dengan default, sebuah canvas tidak memiliki border dan tidak memiliki konten.

Markup terlihat seperti ini:

<canvas id="myCanvas" width="200" height="100"></canvas>

Catatan: Selaltu tentukan sebuah attribute id (untuk diarahkan ke dalam sebuah script), dan sebuah attribute width dan height untuk mendefinisikan ukuran dari canvas.

Untuk menambahkan sebuah border, gunakanlah attribute style.

Contoh Dasar Canvas

<canvas id="myCanvas" width="200" height="100" style="border:1px black solid;"></canvas>
