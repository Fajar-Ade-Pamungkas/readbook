Attribute Style HTML

Pengaturan gaya dari sebuah element HTML, bisa dilakukan dengan attribute style.

Attribute style HTML memiliki sintaksis berikut:

style="properti:value;"

Properti sebuah properti (pelengkap) CSS. Value merupakan sebuah nilai CSS.

Kamu akan mempelajari lebih tentsng CSS dilain waktu di tutorial ini.

HTML Text Alignment

Properti text-align mendefinisikan perataan teks secara horizontal (horizontal text alignment) untuk digunakan pada element HTML.

HTML Styles

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<h3 style="text-align:center;" > Ini adalah heading </h3>
<p style="text-align:right;"> Ini adalah paragraf </p>

<p style="text-align:left;"> Ini adalah paragraf </p>

</body>

</html>

Ini adalah heading

Ini adalah paragraf

Ringkasan Materi

  • Gunakanlah attribute style untuk penggayaan element HTML
  • Gunakanlah background-color untuk warna latar belakang
  • Gunakanlah color untuk pewarnaan
  • Gunakanlah font-size untuk ukuran teks
  • Gunakanlah font-family untuk jenis (atau bentuk) teks
  • Gunakanlah text-align untuk perataan teks
