HTML Color

Layar Komputer dan TV menampilkan warna dengan pengkombinasian warna (cerah) Merah, Hijau, dan Biru.

Color Names

Dengan CSS, warna bisa diatur dengan menggunakan nama - nama warna:

Contoh

Color Nama
Red
Orange
Yellow
Cyan
Blue

CSS men-support 140 standar color name

RGB (Red, Green, Blue)

Dengan HTML, nilai warna RGB bisa ditentukan menggunakan formula ini: rgb(red, green, blue)

Setiap parameter (merah, hijau dan biru) mendefinisikan intensitas dari warna antra 0 sampai 255.

Untuk contoh, rgb(255,0,0) ditampilkan sebagai warna merah, karena merah diatur untuk nilai tertingginya (255) dan nilai terendahnya (0). Eksperiment dengan mencampurkan nilai RGB dibawah:

Contoh

Color RGB
rgb(255,0,0)
rgb(255,255,0)
rgb(0,255,0)
rgb(0,255,255)
rgb(0,0,255)

Corak abu-abu seringkali didefinisikan menggunakan nilai yang sama untuk 3 nilai RGB:

Contoh

Color RGB
rgb(0,0,0)
rgb(128,128,128)
rgb(255,255,255)

Hexadecimal Colors

Dengan HTML, nilai RGB bisa ditentukan menggunakan nilai warna hexadecimal dalam formula: #RRGGBB, dimana RR (red), GG (green), dan BB (blue) merupakan nilai hexadecimal antara 00 dan FF (sama seperti decimal 0-255).

Untik contoh,#FF0000 ditampilkan sebagai merah, karena merah diatur menjadi nilai tertinggi (FF) dan yang lainnya diatur menjadi nilai terendah (00).

Contoh

Color HEX
#FF0000
#FFFF00
#00FF00
#00FFFF
#0000FF

Corak abu-abu seringkali didefinisikan menggunakan nilai yang sama untuk 3 nilai RGB:

Contoh

Color Hex
#000000
#808080
#FFFFFF
