SVG Gradients - Radial

SVG Radial Gradient - <radialGradient>

Element <radialGradient> digunakan untuk mendefinisikan sebuah radial gradient.

Element <radialGradient> harus dikurung (nested) dengan sebuah tag <defs>. Tag <defs> merupakan singkatan untuk definition dari element spesial (demikian sebagai gradient).

Example 1

Contoh 1

Definisikan sebuah ellipse dengan radial gradient dsri putih menuju biru:

Disini adalah kode SVG:

Contoh SVG

<svg height="150" width="500">
  <defs>
    <radialGradient id="grad1" cx="50%" cy="50%" r="50%" fx="50%" fy="50%">
      <stop offset="0%" style="stop-color:rgb(255,255,255);stop-opacity:0" />
      <stop offset="100%" style="stop-color:rgb(0,0,255);stop-opacity:1" />
    </radialGradient>
  </defs>
  <ellipse cx="200" cy="70" rx="85" ry="55" fill="url(#grad1)" />
</svg>

Penjelasan kode:

  • Attribute id dari tag <radialGradient> mendefinsikan sebuah unique name kepada gradient
  • Attribute cx, cy, r mendefinisikan outermost circle (lingkaran paling luar) dan fx dan fy innermost circle (lingkaran paling dalam)
  • Jangkauan warna untuk gradient bisa ditenangkan (composed) dari dua warna atau lebih. Setiap warna ditentukan dengan tag <stop>. Attribute offset digunakan untuk mendefinisikan kemana warna gradient dimulai dan diakhiri
  • Attribute fill me-link element ellipse menjadi gradient

Example 2

Contoh 2

Definisikan ellipse yang lain dengan radial gradient dari putih menjadi biru:

Disini adalah kode SVG:

Contoh SVG

<svg height="150" width="500">
  <defs>
    <radialGradient id="grad2" cx="20%" cy="30%" r="30%" fx="50%" fy="50%">
      <stop offset="0%" style="stop-color:rgb(255,255,255);stop-opacity:0" />
      <stop offset="100%" style="stop-color:rgb(0,0,255);stop-opacity:1" />
    </radialGradient>
  </defs>
  <ellipse cx="200" cy="70" rx="85" ry="55" fill="url(#grad2)" />
</svg>
