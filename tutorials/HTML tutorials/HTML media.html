HTML5 Multimedia

Multimedia pada Web, yaitu sound, music, video, movies dan animations.

What is Multimedia?

Apa itu Multimedia ?

Multimedia datang dalam banyak format yang berbeda. Multimedia hampir bisa menjadi apapun yang kamu dengar dan kamu lihat.

Contoh: Pictures, music, sound, videos, records, films, animations, dan banyak lagi.

Web page seringkali berisi element - element multimedia dengan tipe dan format yang berbeda.

Pada chapter ini, kamu akan mempelajari tentang format multimedia yang berbeda - beda.

Multimedia Formats

Element multimedia (seperti sound atau video) disimpan dalam file media.

Cara yang paling umum untuk men-discover (mengetahui) tipe dari sebuah file, adalah untuk melihat file extension. ketika sebuah browser melihat file extension .html atau .htm, browser akan memperlakukan file tersebut sebagai file HTML. ekstensi .xml mengindikasikan sebuah file XML, dan ekstensi .css mengindikasikan sebuah style sheet file. Picture diakui dengan extension seperti .gif, .png, dan .jpg.

File multimedia juga memiliki format mereka sendiri dan extensionyang berbeda - beda seperti: .swf, .wav, .mp3, .mp4, .mpg, .wmv, and .avi.

Common Video Formats

Format Video Umum

MP4 adalah format baru dan format mendatang untuk Internet Video.

MP4 direkomendasikan oleh Youtube.

MP4 disupport oleh Flash Players.

MP4 disupport oleh HTML5.

Format File Deskripsi
MPEG .mpg
.mpeg 		MPEG. dikembangkan oleh Moving Pictures Expert Group. Format video populer pertama pada web. Digunakan untuk disupport oleh semua browser, tapi MPEG tidak disuuport di HTML5 (see MP4).
AVI .avi AVI (Audio Video Interleave). Dikembangkan oleh Microsoft. Secara umum digunakan dalam camera video dan perangkat keras TV. Bermain bagus di komputer Window, tapi tidak di web browser.
WMV .wmv MWV (Windows Media Video). Dikembangkan oleh Microsoft. Secara umum digunakan dalam camera video dan perangkat keras TV. Bermain bagus di komputer Window, tapi tidak di web browser.
QuickTime .mov QuickTime. Dikembangkan oleh Apple. Secara umum digunakan dalam camera video dan perangkat keras TV. Bermain bagus di komputer Apple, tapi tidak di web browser. (See MP4)
RealVideo .rm
.ram 		RealVideo. Dikembangkan oleh Real Media untuk memperbolehkan video streaming dengan low bandwidths. RealVideo masih digunakan untuk video online dan Internet TV, tapi tidak dimainkan didalam web browser.
Flash .swf
.flv 		Flash. Dikembangkan oleh Macromedia. Seringkali memerlukan komponen extra (plug-in) untuk memainkan dalam web browser.
Ogg .ogg Theora Ogg. Dikembangkan oleh Xiph.Org Foundation. Disupport oleh HTML5.
WebM .webm WebM. Dikembangkan oleh web raksaksa (web giants) seperti Mozilla, Opera, Adobe, dan Google. Disupport oleh HTML5.
MPEG-4 atau MP4 .mp4 Dikembangkan oleh Moving Pictures Expert Group. Berdasarkan pada QuickTime. Secara umum digunakan di kamera video dan persngkat keras TV yang lebih baru. Disupport oleh semua browser HTML5. Direkomendasikan oleh Youtube.

Hanya Video MP4, WebM dan Ogg yang disupport oleh standar HTML5 terbaru.

Sound Formats

MP3 adalah format terbaru untuk memapatkan recorded music (rekaman musik). Istilah MP3 telah menjadi synonymous (searti) dengan musik digital.

Jika website kamu merupakan tentang recorded music (rekaman musik), MP3 adalah Pilihannya.

Format File Deskripsi
MIDI .mid atau .midi MIDI (Musical Instrument Digital Interface). Format utama untuk semua peranhkat musik elektronik seperti synthesizers dsn PC sound card. file MIDI tidak berisi sound, tapi notasi atau catatan digital yang bisa dimainkan oleh elektronik. Bermain baik di semua komputer dan perangkat musik, tapi tidak di web browser.
RealAudio .rm atau .ram RealAudio. Dikembangkan oleh Real Media untuk memperbolehkan streaming tentang autio dengan low bandwidths. Tidak bermain baik di web browser
WMA .wma WMA (Window Media audio). Dikembangkan oleh microsoft. Secara umum digunakan pada music players. Bermain baik di komputer Windows, tapi tidak di web browser.
AAC .aac AAC (Advanced Audio Coding). Dikembangkan oleh Apple sebagai format default untuk iTunes. Bermain baik komputer Apple, tapi tidak di web browser.
WAV .wav WAV. Dikembangkan oleh IBM dan Microsoft. Bermain baik di Windows, Macintosh, dan sistem operasi Linux. Disupport oleh HTML5.
Ogg .ogg Ogg. Dikembangkan oleh Xiph.Org Foundation. Disupport oleh HMTL5
MP3 .mp3 File MP3 sebenarnya adalah sound part (bagian suara) dari file MPEG. MP3 merupakan format paling populer. Mengkombinasikan goo compression (small files) dengan kualitas tinggi. Disupport oleh semua browser.
MP4 .mp4 MP4 merupakan sebuah format video, tapi bisa juga digunakan untuk audio. Video MP4 merupakan format video mendatang pada Internet. Ini memimpin support otomatis untuk audio MP4 oleh semua browser.

Hanya audio MP3, WAV dan Ogg yang disupport oleh standar HTML5 terbaru.
