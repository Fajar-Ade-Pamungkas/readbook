Selalu tentukan width (lebar) dan height (tinggi) dari sebuah gambar. Jika width dan height tidak ditentukan, halaman akan menampilkan gambar sebagaimana ukuran aslinya ketika dimuat.

HTML Images Syntax

Pada HTML, gambar didefinisikan dengan tag <img>.

Tag <img> itu kosong, ia diisi oleh attribute saja, dan tidak memiliki closing tag (penutup tag).

Attribute img menentukan URL (web address) dari sebuah gambar:

<img src="disini url" alt="disini alternatif teks">

The alt Attribute

Attribute alt menentukan sebuah teks alternatif untuk sebuah gambar, jika ga,bar tidak bisa ditampilkan.

Attribute alt menyediakan informasi altenatif untuk sebuah gambar, jika pengguna tidak bisa meilhat gambar maka akan ada kalimat penerangan. (sebab koneksi yang lambat, sebuah error dalam attribute srd, atau jika menggunakan sebuah screen reader).

Jika sebuah browser tidak bisa menemukan gambar, browser akan menampilkan teks alt:

Contoh

Input:

<img src="../../../Pictures/Ska~1/ska~1.png" alt="ska-1" style="width:200px; height:200px">

Memories of Schoooling

Attribute alt karena diperlukan, Sebuah halaman web tidak akan mem-validasi secara benar tanpa alternative text.

HTML Screen Readers

Screen reader (pembaca layar) adalah sebuah program software yang bisa membaca apa yang ditampilkan pada layar.

Screen reader sangat berguna untuk orang tuna netra, gangguan penglihatan, atau pembelajaran yang tidak baik.

Screen reader bisa membaca attribute alt.

Image Size - Width and Height

Kamu bisa menggunakan attribute style untuk menentukan width dan height dari gambar.

Nilai ditentukan dalam satuan pixel (menggunakam px setelah nilai):

Contoh

Input:

<img src="../../../Pictures/Ska~1/ska~1.png" alt="ska-1" style="width:200px; height:200px">

Width and Height or Style?

Keduanya attribute width, height dan style diperbolehkan dslam standar HTM5 terakhir.

Kita suggestikan kamu gunakanlah attribute style. Ini mengatasi style sheet dari perubahan ukuran asli dari gambar:

Contoh

Input:

<!DOCTYPE html>

<html>
<head>
<style>
width:100%;
height:auto;
</style>
</head>
<body>

<img src="../../../Pictures/Ska~1/Men in Black.jpg" alt="MiB" style="width:200px; height:200px">
<img src="../../../Pictures/Ska~1/ska~1.png" alt="ska-1" style="width:200px; height:200px">


</body>

</html>

Images in Another Folder

Jika tidak ditentukan, browser mengira untuk mencari gambar dalam folder yang sama seperti web page.

Akan tetapi, ini umum untuk menyimpan gambar dalam sebuah sub-folder. Kamu kemudian harus memasukan nama folder dalam attribute src:

Contoh

Input:

Using an Image as a Link

Untuk menggunakan sebuah gambar sebagai link, cara sederhana ialah menaruh tag <img> didalam tag <a>:

Contoh

Input:

<a href="HTML images.html" target="_blank" >
<img src="../../pictures/Rain Fall.png" alt="Rainly" style="width:42px; height:42px; border:0">
</a>

Klik gambar diatas untuk menuju ke sebuah halaman lainnya.

border:0 ditambahkan untuk mengatasi masalah di IE9 (dan versi sebelumnya) daru penampilan sebuah border sekitar gambar.

Image Floating

Gunakanlah properti float CSS untuk membiarkan gambar mengapun (float).

Gambar bisa mengapung (float) ke kiri atau ke kanan dari sebuah text:

Contoh

Input:

<p>
<img src="../../pictures/Rain Fall.png"" alt="Rainly" style="float:right;width:42px;height:42px;">
The image will float to the right of the text.
</p>

<p>
<img src="../../pictures/Rain Fall.png" alt="Rainly" style="float:left;width:42px;height:42px;">
The image will float to the left of the text.
</p>

The image will float to the right of the text.

The image will float to the left of the text.

Image Maps

Gunakanlah tag <img> untuk mendefinisikan sebuah image-map. Image-map adalah gambar dengan area yang dapat diklik.

Attribute nama dari tag <map> dihubungkan dengan attribute usemap <img> dan membuta sebuah hubungkan antara image (gambar) dan map (peta).

Tag <map> berisi sebuah nomor dari tag <area>, yang mendefinisikan area yang bisa diklik dalam image-map:

Contoh

Input:

<p>Click on the sun or on one of the planets to watch it closer:</p>

<img src="planets.gif" alt="Planets" usemap="#planetmap" style="width:145px;height:126px;">

<map name="planetmap">
<area shape="rect" coords="0,0,82,126" alt="Sun" href="sun.htm">
<area shape="circle" coords="90,58,3" alt="Mercury" href="mercur.htm">
<area shape="circle" coords="124,58,8" alt="Venus" href="venus.htm">
</map>

Ringkasan Materi

  • Gunakanlah element <img> HTML untuk mendefinisikan sebuah gambar
  • Gunakanlah attribute src HTML untuk mendefinisikan URL dari gambar
  • Gunakanlah attribute alt> HTML untuk mendefinisikan alternative text untuk gambar. jika gambar tidak bisa ditampilkan
  • Gunakanlah attribute width dan height HTML untuk mendefinisikan ukuran dari gambar
  • Gunakanlah properti width dan height CSS untuk mendefinisikan ukuran dari gambar (untuk kemungkinan lainnya)
  • Gunakanlah properti float CSS untuk membiarkan gambar mengambang (float)
  • Gunakanlah element <map> HTML untuk mendefinisikan sebuah image-map (gambar-peta)
  • Gunakanlah element <area> HTML untuk mendefinisikan area yang bisa diklik (clickable areas) dalam image-map
  • Gunakanlah attribute usemap element <img> HTML untuk titik (point) ke image-map
Pemuatan (loading) gambar mengambil waktu. Semakin besar suatu gambar maka semakin lambat halaman kamu dimuat. Gunakanlah gambar secara hati - hati.
