HTML5 - Canvas Clock Numbers

Part III - Draw Clock Numbers

Bagian II - Gambarlah Nomor Jam

Jam Membutuhkan nomor - nomor. Buatlah sebuah fungsi untuk menulis atau menggambar clock numbers:

JavaScript

function drawClock() {
    drawFace(ctx, radius);
    drawNumbers(ctx, radius);
}

function drawNumbers(ctx, radius) {
    var ang;
    var num;
    ctx.font = radius*0.15 + "px arial";
    ctx.textBaseline="middle";
    ctx.textAlign="center";
    for(num= 1; num < 13; num++){
        ang = num * Math.PI / 6;
        ctx.rotate(ang);
        ctx.translate(0, -radius*0.85);
        ctx.rotate(-ang);
        ctx.fillText(num.toString(), 0, 0);
        ctx.rotate(ang);
        ctx.translate(0, radius*0.85);
        ctx.rotate(-ang);
    }
}

Example Explained

Contoh Menenrangkan

Aturlah font size (dari object gambar) menjadi 15% dari radius:

ctx.font = radius*0.15 + "px arial";

Aturlah text alignment (perataan teks) menjadi middle (pertengahan) dan center (pusat) dari print position atau posisi cetak:

ctx.textBaseline="middle";
ctx.textAlign="center";

Kalkulasikan rint position (untuk 12 nomor) menjadi 85% dari radius, dirotasi (PI/6) untuk setiap nomor:

for(num= 1; num < 13; num++) {
    ang = num * Math.PI / 6;
    ctx.rotate(ang);
    ctx.translate(0, -radius*0.85);
    ctx.rotate(-ang);
    ctx.fillText(num.toString(), 0, 0);
    ctx.rotate(ang);
    ctx.translate(0, radius*0.85);
    ctx.rotate(-ang);
}
