HTML5 Support

Kamu bisa "mengajarkan" browser lama untuk meng-handle HTML5 secara benar.

HTML5 Browser Support

HTML5 didukung dalam semua browser modern.

Dalam penambahan, semua browser, lama dan baru, secara otomatis meng-handle element - element yang tidak dikenal (recognized) sebagai satu baris element (inline).

Karena ini, kamu bisa "mengajari" browser lama untuk menangani element HTML yang "tidak diketahui".

Kamu bahkan "mengajari" IE6 (Windows XP 2001) bagaimana untuk menangani element HTML yang tidak diketahui.

Define HTML5 Elements as Block Elements

HTML5 mendefinisikan delapan element HTML semantik (arti kata) baru. Semua ini merupakan element block-level.

Untuk keamanan prilaku yang baik dalam browser lama, kamu bisa menyetel properti display CSS menjadi block:

header, section, footer, aside, nav, main, article, figure {    
display: block;
}

Adding New Elements to HTML

Kamu juga bisa menambahkan element baru apapun ke HTML dengan sebuah trik browser.

Contoh ini menambahakan sebuah element baru yang disebut <myHero> ke HTML, dan mendefinisikan sebuah tampilan style untuknya:

Input:

<!DOCTYPE html>
<html>
<head>
<title> Penciptaan Sebuah Element HTML Baru </title>
  <script>document.createElement("myHero")</script>
  <style>
  myHero {      
display: block;
      background-color: #ddd;
      padding: 50px;
      font-size: 30px;
  }
  </style>
</head>

<body>

<h1> My First Heading </h1>

<p> My First Paragraph </p>

<myHero> My First Hero </myHer>
</body>
</html>

My First Heading

My first paragraph.

My First Hero

Pernyataan JavaScript document.createrElement("myHero") ditambahakan, hanya untuk memenuhi IE.

Masalah Dengan Internet Explorer

Kamu bisa gunakan deskripsi solusi diatas, untuk semua element HTML baru, tapi:

Internet Explorer 8 dan versi sebelumnya, jangan biarkan penggayan (styling) element - element yang tidak diketahui.

Terimakasih banyak, Sjoerd Visscher yang menciptakan "HTML Enabling JavaScript", "the shiv":

HTML5 Enabling JavaScript

<!--[if lt IE 9]>  
<script src="http://html5shiv.googlecode.com/svn/trunk/html5.js"></script>
<![endif]--

Kode diatas adalah sebuah comment, tapi versi terdahulu untuk IE9 akan membaca kode (dan mengerti kode itu)

<!DOCTYPE html>
