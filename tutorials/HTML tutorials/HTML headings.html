HTML Heading

Heading itu penting dalam dokumen HTML.

HTML Headings

Heading didefinisikan dengan tag <h1> sampai dengan <h6>.

<h1> heading yang paling penting. <h6> mendefinisikan heading paling kurang penting.

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<h1> This is a heading</h1>
<h2> This is a heading</h2>
<h3> This is a heading</h3>
<h4> This is a heading</h4>
<h5> This is a heading</h5>
<h6> This is a heading</h6>

</body>

</html>

Perhatikanlah: Browser secara otomatis menambahkan beberapa spasi kosong (sebuah margin) sebelum dan sesudah masing-masing heading.

Heading Itu Penting

Gunakanlah Heading untuk heading (bagian kepala) saja. Jangan gunakan heading untuk membuat teks BESAR atau tebal.

Search engine menggunakan heading kamu untuk index struktur dan konten halaman web kamu.

Para pengguna men-skim halaman kamu dengan headingnya. Itu penting untuk penggunaan heading untuk menampilkan struktur dokumen.

Heading h1 harus di heading utama, diikuti dengan heading h2, lalu yang lebih kurang penting h3 dan seterusnya.

HTML Horizontal Rules

Tag <hr membuat sebuah garis secara horizontal pada sebuah halaman HTML.

Element <hr> bisa digunakan untuk memisahkan konten:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<p> This is a paragraph </p>
<hr>
<p> This is a paragraph </p>
<hr>
<p> This is a paragraph </p>

</body>

</html>

HTML <head> Element

Element <head> tidak memiliki apapun untuk dilakukan dengan HTML heading.

Element <head> berisi meta data. Meta data tidak ditampilkan.

Element <head> ditempatkan diantara tag <html> dan tag <body>:

Contoh

Input:

<!DOCTYPE html>

<html>
<head>
<title> My First HTML </title>
<meta charset="UTF-8">

</head>
<body>

<p> This is a paragraph </p>
<hr>
<p> This is a paragraph </p>
<hr>
<p> This is a paragraph </p> </body>

</html>

This is a paragraph

Meta data artinya data tentang data. HTML meta data adalah data tentang dokumen HTML.

HTML <title Element

<title> Element adalah meta data. Ini mendefinisikan judul dokumen HTML.

Titile tidak akan ditampilkan pada dokumen, tapi mungkin ditampilkan pada browser tab.

HTML <meta> Element

Element <meta> juga merupakan meta data.

Ini bisa juga digunakan untuk mendefinisikan character set, dan informasi lainnya tentang dokumentp HTML.

More Meta Elements

Pada chapter tentang HTML style kamu mengetahui (discover) banyak element meta:

HTML <style> element digunakan untuk mendefinisikan internal Css style sheets.

HTML <link> element digunakan untuk mendefinisikan external CSS style sheets.

Saran HTML: Bagaimana Meninjau Sumber HTML

Sudah pernahkah kamu melihat sebuah halaman web dan heran "Hey! Bagaimana mereka melakukan itu?"

Untuk menemukannya, klik kanan pada halaman (webpage) dan pilihlah "View Page Source" (pada Chrome) atau "View Source" (pada IE), atau mirip di browser lainnya. Ini akan membuka sebuah jendela berisikan kode HTML dari halaman.
