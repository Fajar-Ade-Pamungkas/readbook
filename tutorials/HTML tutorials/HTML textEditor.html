HTML Text Editor

Tulislah HTML Menggunakan Notepad atau TextEdit

HTML bisa diedit dengan menggunakan editor HTML propesional seperti:


  • Microsoft WebMatrix
  • Sublime Text

Akan tetapi, untuk pembelajaran HTML kita rekomendasikan sebuah text editor seperti Notepad (PC) atau TextEdit (Mac).

Kita percaya menggunakan sebuah text editor sederhana adalah cara yang baik untuk belajar HTML.

Ikutilah 4 langkah dibawah ini untuk membuat web pertama kamu dengan Notepad.

Langkah 1: Bukalah Notepad

Untuk membuka Notepad pada window 7 atau versi sebelumnya:

Klik start (bawah kiri pada layar). klik All Program. klik Accessories. klik Notepad.


Untuk membuka Notepad pada window 8 atau versi sesudahnya:

Klik start (disimbol window bawah kiri pada layar). lalu Ketik Notepad.

Langkah 2: Tulislah Beberapa HTML

Tulislah atau salinlah HTML ini kedalam Notepad.

<!DOCTYPE html>

<html>
<head>
</head>
<body>

<h1> Ini heading </h1>

<p> Ini paragraf </p>

</body>
</html>

Langkah 3: Simpanlah halaman HTML

Simpanlah file pada komputer kamu.

Pilih file > Save as pada menu Notepad.

Namailah file "index.html" atau kata lainnya dengan akhiran html atau htm.

UTF-8 adalah persandian yang mengarah kepada file HTML.

ANSI encoding hanya meng-cover karakter - karakter US (united State) dan Western European.



Kamu bisa menggunakan akhiran .htm atau .html sebagai file extention. Tidak ada yang berbeda, itu terserah anda.

Langkah 4: Menampilkan Halaman HTML di Browser Kamu

Bukalah file HTML yang disimpan di browser kamu. Hasil akan terlihat nanti saat halaman web berhasil dimuat.

Untuk Membuka file HTML pada browser, double click pada file tersebut, atau klik kanan dan pilih open with.
