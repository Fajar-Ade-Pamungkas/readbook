HTML Forms

The <form> Element

HTML form (formulir) digunakan untuk mengumpulkan input pengguna.

Element <form> mendefinisikan sebuah formulir HTML:

<form>
.
form elements
.
</form>

HTML form berisi form element.

Form element memiliki tipe element input yang berbeda - beda mulai dari checkbox, radio button, submit button dan selebihnya.

The <input> Element

Element <input> adalah element formulir paling penting.

Element <input> memiliki banyak variasi, tergantung pada attribute tipe.

Disini adalah tipe - tipe yang digunakan dalam chapter ini:

Type Deskripsi
text Mendefinisikan input (berupa) teks normal
radio Mendefinisikan input tombol radio (untuk penyeleksian satu dari banyaknya pilihan)
submit Mendefinisikan tombol submit (untuk penyerahan formulir)

Kamu akan mempelajari banyak tentang tipe input nanti di tutorial ini.

Text Input

<input type="text"> mendefinisikan bidang input satu baris untuk input teks:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<form>  
First name:<br>
  <input type="text" name="firstname"><br>
  Last name:<br>
  <input type="text" name="lastname">
</form>
</body>

</html>
  First name:
 
  Last name:
 
Catatan: formulir itu sendiri tidak diperlihatkan. Juga perhatikan bahwa default width (lebar) dari text field (bidang teks) adalah 20 karakter.

The Submit Button

<input type="submit"> mendefinisikan sebuah tombol untuk submitting (penyerahan) formulir ke form-handler.

Form-handler secara khas adalah sebuah halaman server dengan sebuah script (naskah) untuk processing input data.

Form-handler ditentukan dalam attribute action (aksi) formulir:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<form action="action_page.php">  
  First name:<br>
  <input type="text" name="firstname" value="Mickey"><br>
  Last name:<br>
  <input type="text" name="lastname" value="Mouse"><br><br>

  <input type="submit" value="Submit">
</form>
</body>

</html>
  First name:
 
  Last name:
 

 
Catatan: Web ini belum memiliki sebuah web server, jadi bila kamu tombol submit maka akan keluar halaman "not found" atau sebagainya.

The Action Attribute

Attribute action mendefinisikan aksi untuk ditunjukanketika formulir diserahkan.

Carah umu untuk menyerahkan (submit) formulir menuju server, adalah dengan menggunakan sebuah tombol submit.

Secara normal, firmulir diserahkan ke halaman web pada sebuah web server.

Pada contoh diatas, sebuah server-side script ditentukan untuk mengatasi formulir yang diserahkan.

<form action="action_page.php">

Jika attribute actiondi hilangkan, action diatur menjadi halaman pada saat ini.

The Method Attribute

Attribute method menentukan metode HTTP (GET atau POST) untuk digunakan ketika penyerahan formulir:

<form action="action_page.php" method="get">

atau:

<form action="action_page.php" method="post">

Kapan Untuk Menggunakan GET ?

Kamu bisa menggunakan GET (default method):

Jika form submission adalah pasif (seperti search engine query), dan tanpa informasi sensitif.

Ketika kamu menggunakan GET, form data akan diperlihatkan dalam page address (alamat halaman), seperti contoh ini:

action_page.php?firstname=Mickey&lastname=Mouse
GET sangat cocok untuk jumlah singkat (short amounts) data. Keterbatasan ukuran diatur dalam browser kamu.

Kapan Untuk Menggunakan POST ?

Kamu lebih baik gunakan POST:

Jika formulir sedang mengunggah data, atau mencantumkan (include) informasi sensitif (password).

POST menawarkan keamanan yang lebih baik karena data yang diserahkan tidak diperlihatkan dalam page address (alamat halaman).

Grouping Form Data with <fieldset>

Element <fieldset> mengelompokkan data berhubungan pada formulir.

Element <fieldset> mendefinisikan sebuah caption untuk element <fieldset>.

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<form action="action_page.php">  
  <fieldset>   
<legend>Personal information:</legend>
  First name:<br>
  <input type="text" name="firstname" value="Mickey"><br>
  Last name:<br>
  <input type="text" name="lastname" value="Mouse"><br><br>
  <input type="submit" value="Submit">
</fieldset>
</form>
</body>

</html>
 
    Personal information:     First name:
   
    Last name:
   

     

HTML Form Attributes

Sebuah element <form> HTML, dengan semua attribut yang mungkin diatur, akan terlihat seperti ini:

Contoh

Input:

<form action="action_page.php" method="post" target="_blank" accept-charset="UTF-8" enctype="application/x-www-form-urlencoded" autocomplete="off" novalidate>
.
form elements
.
</form>

Attribute - attribute <form>

Disini adalah daftar dari attribute <form>:

Attribute Deskripsi
accept-charset Menentukan charset yang digunakan dalam submitted form (default: the page charset).
action Menentukan sebuah alamat (URL) dimana untuk menyerahkan formulir (default: submitting page).
autocomplete Menentukan jika browser harus autocomplete (lengkapi secara otomatis) formulir (default:on).
enctype Menentukan encoding (penyadian) data yang diserahkan (default: is url-encoded).
method Menentukan metode HTTP yang digunakan ketika penyerahan formulir (default: GET)
name Menentukan sebuah nama yang digunakan untuk mengidentifikasi formulir (for DOM usage: document.forms.name).
novalidate Menentukan bahwa browser tidak harus mem-validasi (mengesahkan) formulir.
target Menentukan taget dari address dalam attribute action (default: _self).

Kamu akan mempelajari lagi tentang attribute dalam chapter selanjutnya.

More Examples

  E-mail:<br>
  <input type="text" name="email" value="your email"><br>
  Comment:<br>
  <input type="text" name="comment" value="your comment" size="50"><br><br>
  <input type="submit" value="Send">   <input type="reset" value="Reset">
</form>
</body>

</html>
