HTML5 Canvas Reference

Description

Penjelasan

Element <canvas> digunakan untuk menggambar grafik, on the fly, via scrtipting (biasanya JavaScript).

Akan tetapi, element <canvas> tidak memiliki kemampuan gambar dari pemiliknya (canvas hanya sebuah container untuk grapik) - Kamu harus menggunakan sebuah script untuk menggambar grafik yang sebenarnya.

Metode getContex() mengembalikan sebuah object yang menyediakan metode dan properti untuk gambar pada canvas.

Referensi ini akan meng-cover properti dan metode dari object getContex("2"), yang mana bisa digunakan untuk menggambar teks, garis, kotak, lingkaran, dan banyak lagu - pada canvas.

Browser Support

Nomor pada tabel menentukan browser versi pertama yang mendukung secara penuh element <canvas>.

BROWSER SUPPORT
Element
<canvas> 4.0 9.0 2.0 3.1 9.0

Internet Explorer 9, Firefox, Opera, Chrome, and Safari support <canvas> and its properties and methods.

Note: Internet Explorer 8 and earlier versions, do not support the <canvas> element.

Colors, Styles, and Shadows

Properti Deskripsi
fillStyle Mengatur warna, gradient, atau pattern (pola) yang digunakan untuk gambar
Nilai:
  • color (atau warna)
  • gradient
  • pattern (atau pola)
strokeStyle Mengatur warna, gradient, atau pola yang digunakan untuk strokes (coretan)
Nilai:
  • color (atau warna)
  • gradient
  • pattern (atau pola)
shadowColor Mengatur warna untuk digunakan pada shadow (bayangan)
Nilai:
  • color (atau warna)
shadowBlur Mengatur blur level (level buram) untuk shadow
Nilai:
  • number (nomor)
shadowOffsetX Mengatur jarak horizontal bayangan dari bentuk
Nilai:
  • number (nomor)
shadowOffsetY Mengatur jarak vertikal bayangan dari bentuk
Nilai:
  • number (nomor)
Metode Deskripsi
createLinearGradient() Membuat sebuah linear gradient (untuk digunakan pada konten canvas)
sintaksis: context.createLinearGradient(x0,y0,x1,y1)
Nilai:
  • x0 - koordinat x titik awal dari gradient
  • y0 - koordinat y titik awal dari gradient
  • x1 - koordinat x titk akhir dari gradient
  • y1 - koordinat y titik akhir dari gradient
createPattern() Mengulangi suatu element tertentu dalam direksi atau intruksi tertentu.
Sintaksis: context.createPattern(image,"repeat|repeat-x|repeat-y|no-repeat");
Nilai:
  • image - menentukan elemen image, canvas, atau video daru pola untuk digunakna
  • repeat - Default. Pola mengulang sedara horixontal dan vertikal
  • repeat-x - Pola mengulang secara horizontal saja
  • repeat-y - Pola mengulang secara vertikal saja
  • no-repeat - Pola akan ditampilkan sekali saja (tidak diulang)
createRadialGradient() Membuat sebuah radial/circular gradient (untuk digunakan pada konten canvas)
sintaksis: context.createRadialGradient(x0,y0,r0,x1,y1,r1)
Nilai:
  • x0 - koordinat x lingkaran awal dari gradient
  • y0 - koordinat y lingkaran awal dari gradient
  • r0 - radius dari lingkaran awal
  • x1 - koordinat x lingkaran akhir dari gradient
  • y1 - koordinat y lingkaran akhir dari gradient
  • r1 - radius dari lingkaran akhir
addColorStop() Menentukan warna dan posisi berhenti dalam sebuah gradient object.
sintaksis: gradient.addColorStop(stop,color)
Nilai:
  • stop - Nilai antara 0.0 sampai 0.1 yang mempresentasikan posisi antara awal dan akhir dalam sebuah gradient
  • color - Nilai CSS color untuk ditampilkan di stop position

Line Styles

Properti Deskripsi
lineCap Mengatur style dari end caps untuk garis
sintaksis: contex.lineCap= "butt|round|square";
Nilai:
  • butt - Default, Sebuah flat edge ditambahkan ke setiap akhri garis
  • round - sebuah cap akhir bulat ditambahkan untuk setiap akhir garis
  • square - sebuah cap akhir kotak ditambahkan untuk setiap akhir garis
lineJoin Mengatur tipe dari corner created, ketika dua garis bertemu
sintaksis: contex.lineJoin= "bevel|round|miter";
Nilai:
  • bevel - membuat sebuah sudut menyerong (bveled corner)
  • round - membuat sudut membulat atau melengkung (rounded corner)
  • miter - Default. membuat sebuah sudut naik (sharp corner)
lineWidth Mengatur lebar atau tebal garis saat ini
sintaksis: contex.lineWidth= number;
Nilai:
  • number - tebal garis saat ini, dalam satuan pixel
miterLimit Mengatur panjang miter maksimum
sintaksis: contex.lineLimit= number;
Nilai:
  • number - sebuah nomor positif yang menentukan panjang miter maksimum melebihi miterLimit, sudut akan ditampilkan seperti lineJoin "bevel"

Rectangles

Metode Deskripsi
rect() Membuat sebuah reactangle (persgi panjang)
sintaksis: contex.rect(x,y,width,height)
Nilai:
  • x - koordinat x sudut kiri atas dari rectangle
  • y - koordinat y sudut kiri atas dari rectangle
  • width - tebal dari rectangle, dalam satuan pixel
  • height - tinggi dari rectangle, dalam satuan pixel
fillRect() Menggambar sebuah persegi panjang "terisi"
sintaksis: contex.fillRect(x,y,width,height)
Nilai:
  • x - koordinat x sudut kiri atas dari rectangle
  • y - koordinat y sudut kiri atas dari rectangle
  • width - tebal dari rectangle, dalam satuan pixel
  • height - tinggi dari rectangle, dalam satuan pixel
strokeRect() Menggambar sebuah persegi panjang (tidak ada isian) sintaksis: contex.strokeRect(x,y,width,height)
Nilai:
  • x - koordinat x sudut kiri atas dari rectangle
  • y - koordinat y sudut kiri atas dari rectangle
  • width - tebal dari rectangle, dalam satuan pixel
  • height - tinggi dari rectangle, dalam satuan pixel
clearRect() Membereskan pixel tertentu dengan sebuah persegi panjang yang diberikan
sintaksis: contex.clearRect(x,y,width,height)
Nilai:
  • x - koordinat x sudut kiri atas dari rectangle
  • y - koordinat y sudut kiri atas dari rectangle
  • width - tebal dari rectangle untuk dibersihkan, dalam satuan pixel
  • height - tinggi dari rectangle untuk dibersikan, dalam satuan pixel

sintaksis: contex.fill()

sintaksis: contex.stroke()
beginPath() Memulai sebuah path, atau me-reset pat saat ini
sintaksis: contex.beginPath()
moveTo() Menggerakan path ke titik tertentu dalam canvas, tanpa pembuatan sebuah garis
sintaksis: contex.moveTo(x,y)

  • y - koordinat y tentang kemana harus menggerakan path
closePath() Membuat sebuah path dari titik balik saat inu untuk titik permulaan
sintaksis: contex.closePath()
lineTo() Menambahkan sebuah titij baru dan membuat sebuah garis untuk titik itu dari titik yang ditentukan terakhir dalam canvas
sintaksis: contex.moveTo(x,y)
Nilai:
  • x - koordinat x tentang kemana harus membuat garis (berpindah)
  • y - koordinat y tentang kemana harus membuat garis (berpindah)
clip() Melipat daerah dari bentuk apapun dan ukuran dari canvas asli
sintaksis: contex.clip()
quadraticCurveTo() Membuat sebuah quadratic Bézier curve
sintaksis: contex.quadraticCurveTo(cpx,cpy,x,y)
Nilai:
  • cpx - koordinat x dari titik kendali Bézier
  • cpy - koordinat y dari titik kendali Bézier
  • x - koordinat x dari titik akhir
  • y - koordinat y dari titik akhir
bezierCurveTo() Membuat sehuah cubic Bézier curve
sintaksis: contex.bezierCurveTo(cp1x,cp1y,cp2x,cp2y,x,y)
Nilai:
  • cp1x - koordinat x dari titik kendali Bézier pertama
  • cp1y - koordinat y dari titik kendali Bézier pertama
  • cp2x - koordinat x dari titik kendali Bézier kedua
  • cp2y - koordinat y dari titik kendali Bézier kedua
  • x - koordinat x dari titik akhir
  • y - koordinat y dari titik akhir
arc() Membuat sebuah arc/curve (kurva) (yang digunkan untuk membuat lingkaran atau bagian dsri lingkaran)
sintaksis: contex.arc(x, y, r, sAngle, eAngle, counterclockwise)
Nilai:
  • x - Koordinat x dari pusat lingkaran
  • y - koordinat y dari pusat lingkaran
  • r - radius (jari - jari) dari lingkaran
  • sAngle - Angle (sudut) awal, pada radians (0 merupakan posisi jam 3 dari arc's circle)
  • eAngle - Angle (sudut) akhir, pada radians
  • counterclockwise - Optional. Menentukan apakah gambar harus di-counterclockwise atau clockwise. False adalah default, dan mengindikasikan clockwise (searah putaran jarum jam). Sedangkan true mengindikasikan counter-clockwise
arcTo() Membuat sebuah arc/curve (kurva) diantara dua tangent
sintaksis: context.arcTo(x1, y1, x2, y2, r)
Nilai:
  • x1 - koordinat x dari tangent (garis singgung) pertama
  • y1 - koordinat y dari tangent (garis singgung) pertama
  • x2 - koordinat x dari tangent (garis singgung) kedua
  • y2 - koordinat y dari tangent (garis singgung) kedua
  • r - radius dari arc
isPointInPath() Mengembalikan true jika titik tertentu berada di path saat ini, jika tidak false
sintaksis: context.isPointInPath(x, y)
Nilai:
  • x - koordinat x untuk tes
  • y - koordinat y untuk tes

Transformations

Metode Deskripsi
scale() Me-skala gambar saat ini menjadi lebih besar atau lebih kecil
sintaksis: context.scale(scalewidth, scaleheight)
Nilai:
  • scalewidth - Me-skala lebar dari gambar saat ini (1=100%, 0.5=50%, 2=200%, dan seterunsya.)
  • scaleheight - Me-skala tinggi dari gambar saat ini (1=100%, 0.5=50%, 2=200%, dan seterunsya.)
ratate() Me-rotasi gambar saat ini
sintaksis: context.rotate(angle)
Nilai:
  • angle - Sudut (angle) rotasi, pada radians.
    Untuk mengkalkulasikan dari drajat menjadi radian: degrees*Math.PI/180.
    Contoh: Untuk rotasi 5 derajat, ditentukan berikut: 5*Math.PI/180
translate() Memetakan kembali posisi (0,0) pada canvas
sintaksis: context.translate(x, y)
Nilai:
  • x - nilai untuk menambahkan manjadi horizontal koordinat (x)
  • y - nilai untuk menambahkan manjadi vertikal koordinat (y)
transform() Mengganti transformation matrix saat ini untuk gambar
sintaksis: context.transform(a, b, c, d, e, f)
Nilai:
  • a - Scales the drawing horizontally (Me-skala gambar secara horizontal)
  • b - Skew the the drawing horizontally (Memiringkan/mencondongkan gambar secara horizontal)
  • c - Skew the the drawing vertically (Memiringkan/mencondongkan gambar secara vertikal)
  • d - Scales the drawing vertically (Me-skala gambar secara vertikal)
  • e - Moves the the drawing horizontally (Menggerakan gambar secara horizontal)
  • f - Moves the the drawing vertically (Menggerakan gambar secara vertikal)
setTransform() Mereset transform saat ini menjadi matrix identitas. Lalu menjalankan metode transform()
sintaksis: context.setTransform(a, b, c, d, e, f)
Nilai:
  • a - Scales the drawing horizontally (Me-skala gambar secara horizontal)
  • b - Skew the the drawing horizontally (Memiringkan/mencondongkan gambar secara horizontal)
  • c - Skew the the drawing vertically (Memiringkan/mencondongkan gambar secara vertikal)
  • d - Scales the drawing vertically (Me-skala gambar secara vertikal)
  • e - Moves the the drawing horizontally (Menggerakan gambar secara horizontal)
  • f - Moves the the drawing vertically (Menggerakan gambar secara vertikal)

Text

Properti Deskripsi
font Mengatur properti font saat ini untuk konten teks
textAlign Mengatur perataan teks saat ini untuk konten teks
textBaseline Mengatur garis dasar atau garis pangkal teks yang digunkan ketika penggambaran teks
Syntax: context.textBaseline="alphabetic|top|hanging|middle|ideographic|bottom";
Nilai:
  • alphabetic - Default. The text baseline is the normal alphabetic baseline
  • top - The text baseline is the top of the em square
  • hanging - The text baseline is the hanging baseline
  • middle - The text baseline is the middle of the em square
  • ideographic - The text baseline is the ideographic baseline
  • bottom - The text baseline is the bottom of the bounding box
Metode Deskripsi
fillText() Menggambar teks "terisi" pada canvas
Syntax: context.fillText(text, x , y, maxWidth);
Nilai:
  • text - Menentukan teks yang akan ditulis pada canvas
  • x - koordinat x kemana pewarnaan (painting) awal teks (relative to the canvas)
  • y - koordinat x kemana pewarnaan (painting) awal teks (relative to the canvas)
  • maxWidth - Optional. Maksimum lebar yang diizinkan dari teks, dalam satuan pixel
strokeText() Menggambar text pada canvas (tidak ada isi)
Syntax: context.strokeText(text, x , y, maxWidth);
Nilai:
  • text - Menentukan teks yang akan ditulis pada canvas
  • x - koordinat x kemana pewarnaan (painting) awal teks (relative to the canvas)
  • y - koordinat x kemana pewarnaan (painting) awal teks (relative to the canvas)
  • maxWidth - Optional. Maksimum lebar yang diizinkan dari teks, dalam satuan pixel
measureText() Mengembalikan sebuah object yang berisikan width daru teks tertentu
Syntax: context.measureText(text);
Nilai:
  • text - teks untuk diperhitungkan (measured)

  • sheight - Optional. Tinggi dari image yang di-clip
  • x - Koordinat x dimana tempat atau letak image pada canvas
  • y - Koordinat y dimana tempat atau letak image pada canvas
  • width - Optional. Lebar dari image untuk digunakan (merenggangkan atau mengurangi (ukuran) image)
  • height - Optional. Tinggi dari image untuk digunakan (merenggangkan atau mengurangi (ukuran) image)

sintaksis: imgData.width

Metode Deksripsi
creatImageData() Membuat suatu (image) baru, blank ImageData object
sintaksis: var imgData= context.createImageData(width,height);
sintaksis: var imgData= context.createImageData(imageData);
Nilai:
  • width - The width of the new ImageData object, in pixels
  • height - The height of the new ImageData object, in pixels
  • imageData - anotherImageData object
getImageData() Mengembalikan ImageData object yang menyalin pixel data untuk rectangle tertentu pada sebuah canvas
sintaksis: context.getImageData(x,y,width,height);
Nilai:
  • x - koordinat x (dalam satuan pixel) dari sudut kiri atas untuk salin awal dari
  • y - koordinat y (dalam satuan pixel) dari sudut kiri atas untuk salin awal dari
  • width - lebar dari area persegi panjang yang akan kamu salin
  • height - Tinggi dari area persegi panjang yang akan kamu salin
putImageData() Menaruh data gambar (dari sebuah object ImageData tertentu) kembali kedalam canvas
Sintaksis: context.putImageData(imgData,x,y,dirtyX,dirtyY,dirtyWidth,dirtyHeight);
Nilai:
  • imgData - Specifies the ImageData object to put back onto the canvas
  • x - The x-coordinate, in pixels, of the upper-left corner of the ImageData object
  • y - The y-coordinate, in pixels, of the upper-left corner of the ImageData object
  • dirtyX - Optional. The horizontal (x) value, in pixels, where to place the image on the canvas
  • dirtyY - Optional. The vertical (y) value, in pixels, where to place the image on the canvas
  • dirtyWidth - Optional. The width to use to draw the image on the canvas
  • dirtyHeight - Optional. The height to use to draw the image on the canvas

Compositing

Properti Deskripsi
globalAlpha Mengatur alphabet saat ini atau nilai tembus pandang (transparency) dari gambar
Sintaksis: context.globalAlpha=Number;
Nilai:
  • Number - Nilai transparency. Harus menjadi sebuah nomor antara 0.0 (fully transparent) dan 1.0 (no transparency)
globalCompositeOperation Mengatur bagaimana sebuah image baru digambar kedalam sebuah image yang telah ada
Sintaksis: context.globalCompositeOperation="source-in";
Nilai:
  • source-over - Default. Menampilkan source image over (image sumber melebihi) the destination iamge
  • source-atop - Menampilkan source image di atas dari destination image. Bagian dari source image yang berads diluar destination image yang tidak ditampilkan
  • source-in - Menampilkan source image dalam destination image. Hanya bagian dari source image yang berada DIDALAM destination image yang ditampilkan dan destinastion image yang transparan
  • source-out - Menampilkan source image dalam destination image. Hanya bagian dari source image yang berada DILUAR destination image yang ditampilkan dan destinastion image yang transparan
  • destination-over - Menampilkan destination image yang melebihi source image
  • destination-atop - Menampilkan destination image di atas dari source image. Bagian dari destination image yang berads diluar destination image yang tidak ditampilkan
  • destination-in - Menampilkan destination image kedalam source image. Hanya bagian dari destination image yang berada DIDALAM source image yang ditampilkan dan source image yang transparan
  • destination-out - Menampilkan destination image kedalam source image. Hanya bagian dari destination image yang berada DILUAR source image yang ditampilkan dan source image yang transparan
  • lighter - Menampilkan source image + destination image
  • copy - Menampilkan source image. Destination image diabaikan
  • xor - Sourcr image dikombinasikan dengan menggunakan sebuah exclusive OR dengan destination image
