Pengenalan HTML

Apa itu HTML ?

HTML adalah sebuah bahasa pembangun konten - konten web page (atau Markup Language) untuk mendeskripsikan dokumen web (webpage).

  • Standar - standar HTML untuk Hyper Text Markup Language
  • Sebuah bahasa markup adalah sebuah pengaturan markup tags
  • Dokumen HTML dideskripsikan oleh tag - tag HTML
  • Setiap tag HTML mendeskripsikan konten dokumen yang berbeda - beda

Contoh menjelaskan

  • Pendekalarasian DOCTYPE mendefinisikan tipe dokumen menjadi HTML
  • Teks diantara <html> dan </html> mendeskripsikan sebuah dokumen HTML
  • Teks diantara <head> dan </head> menyediakan informasi tentang dokumen
  • Teks diantara <title> dan </title> menyediakan sebuah judul (title) untuk dokumen
  • Teks diantara <body> dan </body> mendeskripsikan konten halaman yang terlihat
  • Teks diantara <h1> dan </h1> mendeskripsikan sebuah heading
  • Teks diantara <p> dan </p> mendeskripsikan sebuah paragraf

Gunakanlah pendeksripsian ini, sebauh web browser bisa menampilkan sebuah dokumen dengan sebuah heading dan sebauh paragraf.

HTML Tags

HTML tag adalah kata kunci (tag names) yang dihimpit oleh angle brackets atau kurung segitiga.

<tagname> Kontent </tagname>

  • HTML tag normalnya berpasangan seperti <p> dan <p>
  • Tag pertama pada adalah start tag, dan tag kedua adalah end tag
  • end tag ditulis seperti start tag, namun dengan simbol garis miring sebelum tag nama
  • start tag seringkali disebut dengan opening tag, dan end tag disebut closing tag.

Web Browser

Maksud dari sebuah web browser (seperti Chrome, IE, Firefox, Safari) adalah untuk membaca dokumen HTML dan menampilkannya.

Browser tidak menampilkan HTML tag, tapi menggunakannya untuk memutuskan bagaimana menampilkan dokumen.

struktur Halaman HTML

Dibawah ini adalah visualisasi tentang strultru halaman HTML:

<html>
<head>
<title>Page title</title>
</head>

<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
</body>
</html>
Hanya area <body> (the white area) yang ditampilkan oleh browser.

Pendeklarasian <!DOCTYPE>

Pendeklarasian <!DOCTYPE> membantu browser unthk menampilkan webpage secara benar.

Disini ada tipe dokumen berbeda pada web

Untuk menampilkan sebuah dokumen secara benar, browser harus tahu tipe dan versi.

pendeklarasian doctype tidak termasuk case sensitive (kata yang sensitif). Semua case bisa diterima:

<!DOCTYPE html>

<!DOCTYPE HTML>

<!doctype html>

<!Doctype Html>
