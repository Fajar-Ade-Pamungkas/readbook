SVG <path>

SVG Path - <path>

Element <path> digunakan untuk mendefinisikan sebuah path.

Berikut komando atau perintah yang disediakan untuk path data:

  • M = moveto (gerakan ke)
  • L = lineto (gariskan ke)
  • H = horizontal lineto (garis mendatar ke)
  • V = vertical lineto (garis tegak ke)
  • C = curveto (kurva untuk)
  • S = smooth curveto (kurva lembut untuk)
  • Q = quadratic Bézier curve (kurva Bézier kuadrat)
  • T = smooth quadratic Bézier curveto (kurva Bézier kuadrat halus)
  • A = elliptical Arc (busur oval)
  • Z = closepath (tutup path)

Catatan: semua komando atau perintahs diatas bisa juga diekspresikan dengan huruf kecil. Huruf kapital berarti absolutely positioned, huruf kecil berarti relatively positioned.

Contoh 1

Contih dibawah ini mendefinisikan sebuah path yang dimulai pada posisi 150,0 dengan sebuah garis untuk posisi 75,200 lalu dari sana, sebuah garis menuju 225,200 dan akhirnya penutupan path kembali ke 150,0:

Disini adalah kode SVG:

Contoh Kode SVG

<svg height="210" width="400">
  <path d="M150 0 L75 200 L225 200 Z" />
</svg>

Example 2

Contoh 2

Bézier curves digunakan untuk mode smooth curves (model kurva halus) yang bisa di-skala dengan tidak tentu. Secara umum, pengguna menyeleksi dua titik akhir dan satu atau dua titik kendali. Sebuah kurva Bézier dengan satu titik kendali disebut dengan quadratic Bézier curve dan macam-macam dengan dua titik kendali disebut cubic.

Contoh berikut membuat sebuah QUADRATIC BÉZIER CURVE, dimana A dan C merupakan titik awal dan akhir, B merupakan titik kendali:

Disini adalah kode SVG:

Contoh Kode SVG

<svg height="400" width="450">
  <path id="lineAB" d="M 100 350 l 150 -300" stroke="red"
  stroke-width="3" fill="none" />
  <path id="lineBC" d="M 250 50 l 150 300" stroke="red"
  stroke-width="3" fill="none" />
  <path d="M 175 200 l 150 0" stroke="green" stroke-width="3"
  fill="none" />
  <path d="M 100 350 q 150 -300 300 0" stroke="blue"
  stroke-width="5" fill="none" />
  <!-- Mark relevant points -->
  <g stroke="black" stroke-width="3" fill="black">
    <circle id="pointA" cx="100" cy="350" r="3" />
    <circle id="pointB" cx="250" cy="50" r="3" />
    <circle id="pointC" cx="400" cy="350" r="3" />
  </g>
  <!-- Label the points -->   <g font-size="30" font-family="sans-serif" fill="black" stroke="none"
  text-anchor="middle">
    <text x="100" y="350" dx="-30">A</text>
    <text x="250" y="50" dy="-10">B</text>
    <text x="400" y="350" dx="30">C</text>
  </g>
</svg>

Rumit ? YA !!! Karena dari melibatkan diri dengan kerumitan dalam penggambaran path, itu direkomendasi secara tinggi untuk menggunakan sebuah SVG editor untuk membuat grafik yang rumit.
