HTML dan XHTML

XHTML adalah HTML yang ditulis sebagai XML.

Apa itu XHTML?

  • XHTML didirikan untuk EXtensible HyperText Markup Language.
  • XHTML hampir mirip dengan HTML
  • XHTML lebih teliti (stricter) daripada HTML
  • XHTML adalah HTML yang didefinisikan sebagai XML application
  • XHTML didukung oleh semua mayor browser

Kenapa XHTML?

Banyak pages (halaman) di internet yang berisi "bad" HTML.

Kode HTML ini bekerja baik di banyak browser (bahkan jika ia tidak mengikut aturan HTML)

<html>
<head>  
<title>This is bad HTML</title>

<body>
<h1>Bad HTML
<p>This is a paragraph
</body>

Pemasaran hari ini terdiri dari teknologi browser yang berbeda. Beberapa browser berjalan pada komputer dan beberapa browser berjalan pada mobile phones atau perangkat kecil lainhya. Semakin kecil perangkat seringkali lack resource (kekurangan sumber) atau kekuatan untuk menafsirksn "bad" markup.

XML adalah sebuah markup language dimana dokumen harus ditandai ulang dengan benar (menjadi "formula yang benar").

Jika kamu mau belajar XML, silahkan baca XML tutorial.

Dengan pengkombinasian kekuatan dari HTML dan XML, XHTML berkembang.

XHTML adalah HTML yang didesign ulanh sebagai XML.

The Most Important Differences from HTML:

Document Structure

  • XHTML DOCTYPE adalah mandatory (bersifat perintah)
  • Attribute xmlns pada <html> adalah mandatory
  • <html>, <head>, <title>, dan <body> merupakan mandatory

XHTML Elements

  • Element - element XHTML harus di "properly nested"
  • Element - element XHTML harus selalu ditutup
  • Element - element XHTML harus lowercase (huruf kecil semua)
  • Dokumen - dokumen XHTML harus memili satu root element

XHTML Attributes

  • Nama attribute harus lowercase
  • nilai attribute harus dipakaikan tanda kutip
  • Attribute yang disingkat (Attribute minimization) dilarang

<!DOCTYPE ....> Is Mandatory

Sebuah dokumen XHTML harus memiliki sebuah pendeklarasian DOCTYPE XHTML.

Daftar lengkap tentang semua XHTML Dcotypes dicari dalam referensi tag HTML.

<html>, <head>, <title>, dan <body> juga harus disediakan, dan attribute xmlns dalam <html> harus menentukan xml namespase untuk dokumen.

Contoh ini menampilkan sebuah Dokumen XHTML dengan minal tag yang diperlukan:

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

<html xmlns="http://www.w3.org/1999/xhtml">

<head>  
<title>Title of HTML</title>
</head>
<body>

beberapa Konten

</body>
</html>

Element XHTML Harus di-Properly Nested

Dalam HTML, beberapa element bisa di-"improperly nested" dengan satu sama lain, seperti ini:

<b><i>This text is bold and italic</b></i>

Dalam XHTML, semua element harus di-"properly nested" dengan satu sama lain, seperti ini:

<b><i>This text is bold and italic</i></b>

Element XHTML Harus Selalu Ditutup

Ini salah:

<p>This is a paragraph
<p>This is another paragraph

Ini benar:

<p>This is a paragraph </p>
<p>This is another paragraph </p>

Elements Kosong (Tunggal) juga Harus Ditutup

Ini salah:

A break: <br>
A horizontal rule: <hr>
An image: <img src="happy.gif" alt="Happy face">

Ini benar:

A break: <br />
A horizontal rule: <hr />
An image: <img src="happy.gif" alt="Happy face" />

Element XHTML Harus dalam Lower Case (Huruf kecil)

Ini salah:

<BODY>
<P>This is a paragraph</P>
</BODY>

Ini benar:

<body>
<p>This is a paragraph</p>
</body>

Nama Attribute XHTML Harus dalam Lower Case (Huruf Kecil)

Ini salah:

<table WIDTH="100%">

Ini benar:

<table width="100%">

Nilai Attribute Harus Diberi Tanda Kutip

Ini salah:

<table width=100%>

Ini benar:

<table width="100%">

Attribute Minimization Is Forbidden

Peminimalan attribute itu dilarang


Ini salah:

<input type="checkbox" name="vehicle" value="car" checked />

Ini benar:

<input type="checkbox" name="vehicle" value="car" checked="cheched" />

Ini salah:

<input type="text" name="lastname" disabled />

Ini benar:

<input type="text" name="lastname" disabled="disable" />

Bagaimana Cara Mengubah HTML Menjadi XHTML ?

  • Tambahkanlah <!DOCTYPE> XHTML untuk baris pertama dari setiap halaman
  • Tambahkanlah sebuah attribute xmlns kedalam element <html> pada setiap halaman
  • Ubahlah semua nama element menjadi lowercase (huruf kecil)
  • Tutuplah semua empty elements (element kosong atau tunggal)
  • Ubahlah semua nama attribute menjadi lowercase
  • Berilah tanda kutip di semua nilai attribute
