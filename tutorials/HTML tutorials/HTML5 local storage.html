HTML5 Local Storage

HTML locak storage, lebih baik daripada cookies.

What is HTML Local Storage?

Apa itu Penyimpanan Lokal HTML?

Dengan local storage, applikasi web bisa menyimpan data secara lokal dengan browser pengguna.

Sebelum HTML5, application data harus disimpan dalam cookies, termasuk dalam setiap server request. Local storage lebih aman, dan jumlah besar dari data yang bisa disimpan secara lokal, tanpa mempengaruhi website performance.

Tidak seperti cookies, batas penyimpanan jauh lebih besar (paling rendah 5MB) dan informasi tidak pernah ditranfer ke server.

Local storage ialah per origin (per domain dan protocol). Semua halaman dsri satu sumber (origin), bisa menyimpan dan mengakses data yang sama.

HTML Local Storage Objects

Object Penyimpanan Lokal HTML

HTML local storage menyediakan dua object untuk penyimanan data pada client (pelangan):

  • window.localStorage - menyimpan data dengan tidak ada expiration date (tanggal kadaluwarsa)
  • window.sessionStorage - meyimpan data untuk satu sesi (data dihilangkan ketika browser tab ditutup)

Sebelum penggunaan local storage, periksalah browser support untuk localStorage dan sessionStorage:

if(typeof(Storage) !== "undefined") {
    // Code for localStorage/sessionStorage.
}else{
    // Sorry! No Web Storage support..
}

The localStorage Object

Object localStorage

Object localStorage menyimpan data dengan tanpa ada expiration date (tanggal kadaluwarsa). Data tidaka akan dihapus ketiks browser ditutup, dan akan disediakan pada hari, minggu atau tahun berikutnya.

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<div id="result"></div>

<script>
// Check browser support
if (typeof(Storage) !== "undefined") {
// Store
localStorage.setItem("lastname", "Smith");
// Retrieve
document.getElementById("result").innerHTML = localStorage.getItem("lastname");
} else {
document.getElementById("result").innerHTML = "Sorry, your browser does not support Web Storage...";
}
</script>

</body>

</html>

Example explained:

Contoh menerangkan:

  • Buatlah sepasang nama/nilai localStorage dengan name="lastname" dan value="Smith"
  • Dapatkan kembali nilai dari "lastname" dan sisipkan itu kedalam element dengan id="resul"

    • Contoh diatas bisa juga ditulis seperti ini:

    // Store
    localStorage.lastname = "Smith";
    // Retrieve document.getElementById("result").innerHTML = localStorage.lastname;

    Sintaksis untuk penggantian item localStorage "lastname" sebagai berikut:

    localStorage.removeItem("lastname");

    Catatan: sepasang nama/nilai selalu disimpan sebagai string. Ingatlah untuk mengkonversikan mereka menjadi format lainnya ketika dibutuhkan!

    Contoh berikut menghitung (count) nomor dari waktu seorang pengguna setelah mengklik seguah tombol. Pada kode unu nilai string di konversikan menjadi nomor untuk bisa meningkatkan penghitung:

    Input:

    <!DOCTYPE html>
    <html>
    <head>
    <title> Judul Halaman </title>
    <script>
    function clickCounter() {
    if(typeof(Storage) !== "undefined") {
    if (localStorage.clickcount) {
    localStorage.clickcount = Number(localStorage.clickcount)+1;
    } else {
    localStorage.clickcount = 1;
    } document.getElementById("result").innerHTML = "You have clicked the button " + localStorage.clickcount + " time(s).";
    } else {
    document.getElementById("result").innerHTML = "Sorry, your browser does not support web storage...";
    }
    }
    </script>
    </head>
    <body>

    <p><button onclick="clickCounter()" type="button">Click me!</button></p>
    <div id="resulting"></div>
    <p>Click the button to see the counter increase.</p>
    <p>Close the browser tab (or window), and try again, and the counter will continue to count (is not reset).</p>
    </body>
    </html>

    Click the button to see the counter increase.

    Close the browser tab (or window), and try again, and the counter will continue to count (is not reset).

The sessionStorage Object

Object sessionStorage

Object sessionStorage sama dengan objet localStorage, kecuali bahwa sesstionStorage menyimpan data hanya untuk satu sesi. (Lalu) Data dihapus ketika pengguna menutup specific browser tab (tab browser itu).

Contoh berikut menghitung nomor waktu (times) pengguna setelah mengklik sebuah tombol di sesi sekarang ini:

Input:

<!DOCTYPE html>
<html>
<head>
<title> Judul Halaman </title>
<script>
function clickCounter() {
if(typeof(Storage) !== "undefined") {
if (sessionStorage.clickcount) {
sessionStorage.clickcount = Number(sessionStorage.clickcount)+1;
} else {
sessionStorage.clickcount = 1;
} document.getElementById("result").innerHTML = "You have clicked the button " + sessionStorage.clickcount + " time(s).";
} else {
document.getElementById("result").innerHTML = "Sorry, your browser does not support web storage...";
}
}
</script>
</head>
<body>

<p><button onclick="clickCounter()" type="button">Click me!</button></p>
<div id="resulting"></div>
<p>Click the button to see the counter increase.</p>
<p>Close the browser tab (or window), and try again, and the counter will continue to count (is not reset).</p>
</body>
</html>

Click the button to see the counter increase.

Close the browser tab (or window), and try again, and the counter is reset.
