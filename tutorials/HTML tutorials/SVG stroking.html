SVG Stroke Properties

SVG Stroke Properties

Properti - properti Stroke SVG

SVG menamwarkan jangkauan luas tentang properti stroke. Pada chapter ini kita akan terlihat sebagai berikut:

  • stroke
  • stroke-width
  • stroke-linecap
  • stroke-dasharray

Semua properti stroke bisa diterapkan untuk bermacam - macam garis, teks dan outline dari element seperti sebuah lingkaran.

SVG stroke Property

Properti stroke mendefinisikan warna dari sebuah garis, teks atau outline dari sebuah element:

Disini adalah kode SVG:

Contoh Kode SVG

<svg height="80" width="300">
  <g fill="none">
    <path stroke="red" d="M5 20 l215 0" />
    <path stroke="black" d="M5 40 l215 0" />
    <path stroke="blue" d="M5 60 l215 0" />
  </g>
</svg>

SVG stroke-width Property

Properti stroke-width mendefinisikan ketebalan dari suatu garis, teks, atau outline dari suatu element:

Disini adalah kode SVG:

Contoh Kode SVG

<svg height="80" width="300">
  <g fill="none" stroke="black">
    <path stroke-width="2" d="M5 20 l215 0" />
    <path stroke-width="4" d="M5 40 l215 0" />
    <path stroke-width="6" d="M5 60 l215 0" />
  </g>
</svg>

SVG stroke-linecap Property

Properti stroke-linecap mendefinisikan tipe akhir berbeda untuk sebuah path terbuka:

Disini adalah kode SVG:

Contoh Kode SVG

<svg height="80" width="300">
  <g fill="none" stroke="black" stroke-width="6">
    <path stroke-linecap="butt" d="M5 20 l215 0" />
    <path stroke-linecap="round" d="M5 40 l215 0" />
    <path stroke-linecap="square" d="M5 60 l215 0" />
  </g>
</svg>

SVG stroke-dasharray Property

Properti stroke-dasharray digunakan untuk membuat dashed lines (garis putus - putus):

Disini adalah kode SVG:

Contoh Kode SVG

<svg height="80" width="300">
  <g fill="none" stroke="black" stroke-width="4">
    <path stroke-dasharray="5,5" d="M5 20 l215 0" />
    <path stroke-dasharray="10,10" d="M5 40 l215 0" />
    <path stroke-dasharray="20,10,5,5,5,10" d="M5 60 l215 0" />
  </g>
</svg>
