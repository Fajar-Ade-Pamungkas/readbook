HTML Symbol Entities

HTML entities didescribsikan dalam chapter sebelumnya.

Banyak ilmu Matematika, ilmu teknik dan simbol mata uang, yang tidak dipersembahkan atau disediakan pada keyboard.

Untuk menambahkan simbol - simbol ini ke halaman HTML, kamu bisa menggunakansebuah HTML entity name.

Jika tidak ada entity name yang tersedia, kamu bisa gunakansebuah entity number; referensi Decimal (atau Hexadecimal).

Jika kamu menggunakan sebuah HTML entity name atau hexadecimal number, karaktrr akan selalu menampilkan secara benar.
Ini adalah kebebasan tentang apa halaman character set (encoding) yang kamu gunakan.

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<p>I will display &euro;</p>
<p>I will display &#8364;</p>
<p>I will display &#x20AC;</p>

</body>

</html>

Akan ditampilkan menjadi

I will display €

I will display €

I will display €
