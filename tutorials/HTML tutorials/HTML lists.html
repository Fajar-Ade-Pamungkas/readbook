Unordered HTML Lists

Unordered list dimulai dengan tag <ul>. setiap list item (barang) diawali dengan tag <li>.

List item akan ditandai dengan bullet (lingkaran hitam kecil):

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>
</body>

</html>

Unordered HTML Lists - The Style Attribute

Attribute style bisa ditambahka ke unordered list (daftar tidak berurut), untuk mendefinisikan style untuk penanda:

Style Deskripsi
list-style-type:disc List item akan ditandai dengan bullet atau lingkaran hitam kecil (default)
list-style-type:circle List item akan ditandai dengan lingkaran
list-style-type:square List item akan ditandai dengan kotak hitam kecil
list-style-type:none List item tidak akan ditandai

<ul style="list-style-type:disc">

Ordered HTML Lists - The Type Attribute

Attribute type bisa ditambahkan ke ordered list, untuk mendefinisikan tipe penanda:

Type Deskripsi
type="1" List item akan dinomori atau ditandai dengan angka (default)
type="A" List item akan dinomori atau ditandai dengan huruf besar
type="a" List item akan dinomori atau ditandai dengan huruf kecil
type="I" List item akan dinomori atau ditandai dengan angka romawi besar
type="i" List item akan dinomori atau ditandai dengan angka romawi kecil

HTML Description Lists

HTML juga mendukung description list.

Tag <dl> mendefinisikan description list (daftar penjelas), tag <dt> mengistilahkan (nama) dan tag <dd> mendeskripsikan setiap term (nama):

Angka Romawi Kecil

<dl>
<dt>Coffee</dt>
<dd>- black hot drink</dd>
<dt>Milk</dt>
<dd>- white cold drink</dd>
</dl>

Horizontal Lists

HTML list bisa di-styled (gaya) dalam banyak cara yang berbeda dengan CSS.

Satu cara yang populer adalah untuk style (gaya) sebuah list atau daftar untuk ditampilkan secara horizontal atau mendatar:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<style>
ul#menu li {    
display:inline;
}
</style>
</head>

<body>

<h2> Horizonral List </h2>

<ul id="menu">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>
</body>

</html>

Ringkasan materi

  • Gunakanlah element <ul> HTML untuk mendefinisikan unordered lis (daftar tak tersusun)
  • Gunakanlah attribute style> HTML untuk mendefinisikan bullet style
  • Gunakanlah element <ol> HTML untuk mendefinisikan ordered list (daftar tersusun)
  • Gunakanlah attribute type HTML untuk mendefinisikan tipe penomomoran
  • Gunakanlah element <li> HTML untuk mendefinisikan list item
  • Gunakanlah element <dl> HTML untuk mendefinisikan description list
  • Gunakanlah element <dt> HTML untuk mendefinisikan description term
  • Gunakanlah element <dd> HTML untuk mendefinisikan description data
  • List bisa di-nested (disarangkan) didalam list
  • Gunakanlah properti CSS display:inline; untuk menampilkan list secara horizontal (mendatar)

HTML List Tags

Tag Deskripsi
<ul> Mendefinisikan sebuah unordered list (daftar tak bersusun)
<ol> Mendefinisikan sebuah ordered list (daftar bersusun)
<li> Mendefinisikan sebuah list item (isi dari daftar)
<dl> Mendefinisikan sebuah description list (daftar pendeskripsian)
<dt> Mendefinisikan sebuah term (istilah) dalam sebuah description list
<dd> Mendefinisikan pendeskripsian dalam description list
Contoh diatas ada yang tidak ditampilkan dengan tidak benar, terutama 3 contoh terakhir.
