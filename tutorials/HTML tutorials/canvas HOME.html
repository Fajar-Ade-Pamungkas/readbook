HTML5 - Canvas Tutorial

Element <canvas> HTML digunakan untuk menggambar grafik pada sebuah halaman web.

Grafik disamping kiri diciptakan dengan <canvas>. Ia menampilkan empat element: a red rectangle, a gradient rectangle, a multicolor rectangle, and a multicolor text.

What is HTML Canvas?

Apa itu Canvas?

Element <canvas> digunakan untuk menggambar grafik, on the fly, via scrtipting (biasanya JavaScript).

Element <canvas> juga merupakan container untuk grafik. Kamu harus menggunakan sebuah script (JavaScript) untuk menggambar grafik yang sebenernya.

Canvas memiliki beberapa metode untuk drawing path, boxes, circles, text, and adding images.

HTML Canvas Can Draw Text

HTML Canvas bisa menggambar teks

Canvas bisa menggambar colorful text, dengan atau tanpa animasi.

HTML Canvas Can Draw Graphics

HTML Canvas bisa menggambar grafik

Canvas memiliki feacture yang hebat untuk grsphical data presentation dengan sebuah imagery of graphs dan charts.

HTML Canvas Can be Animated

HTML Canvas bisa dianimasikan

Object Canvas bisa bergerak. Apapun bisa terjadi: dari simple bouncing balls untuk complex animations.

HTML Canvas Can be Interactive

HTML Canvas Bisa Mnejadi Interaktif

Canvas bisa merespon JavaScript Evenet.

Canvas bisa merespon aksi pengguna apa saja (ssperti key clicks, mouse clicks, button clicks, finger movement).

HTML Canvas Can be Used in Games

Metode - metode canvas animasi, menawarkan berbagai kemungkinan untuk HTML gaming applications.

Canvas Example

Contoh Canvas

Dalam HTML, sebuah element <canvas> terlihat seperti ini:

<canvas id="myCanvas" width="200" height="100"></canvas>

Element <canvas> harus memiliki sebuah attribute id jadi ia bisa diarahkan oleh JavaScript.

Attribute width dan height diperlukan untuk mendefinisikan ukursn dsri canvas.

Tip: Kamu bisa memiliki multiple <canvas> elements pada satu HTML page.

Dengan default, element <canvas> tidak memiliki border dan tidak ada konten.

Untuk menambahkan border, gunakanlah attribute style:

Contoh

<canvas id="myCanvas" width="200" height="100"
style="border:1px solid #000000;">
</canvas>
