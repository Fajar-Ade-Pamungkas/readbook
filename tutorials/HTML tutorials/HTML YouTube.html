HTML YouTube Videos

Cara paling mudah untuk memainkan video dalam HTML, adalah menggunakan YouTube.

Struggling with Video Formats?

Tetap Berjuang denganFormat Video?

Versi - versi yang berbeda dari browser support yang berbeda format video yang berbeda.

Di tutorial sebelumnya, kamu telah melihat bahwa kamu mungkin harus mengubah atau mengkonversikan video kamu menjadu format video yang berbeda untuk membuat mereka dimain di semua browser.

Pengubahan video ke format yang berbeda bisa menjadi sulit dan memakan waktu.

Sebuah solusi yang lebih mudah mungkin membiarkan YouTube memainkan video di halaman web kamu.

YouTube Video Id

Id Video YouTube

YouTube akan menampilkan sebuah id (seperti XGSy3_Czz8k), ketika kamu menyimpan (atau memainkan) sebuah video.

Kamu bisa menggunakan id ini dan mengarahkan pada video kamu di HTML.

Playing a YouTube Video in HTML

Pemutaran Video YouTube di HTML

Untuk memutar video kamu di halaman web, lakukanlah langkah - langkah berikut:

  • Unggahlah video ke YouTube
  • Ambillah sebuah catatan id video
  • Definusikan sebuah element <iframe> dalam halaman web kamu
  • Izinkan tujuan attribute src ke URL video
  • Gunakanlah attribute height dan width untuk menentukan dimensi dari player (pemutar)
  • Tambahkan parameter lainnya untuk URL

Contoh - Penggunaan iFrame (metode yang dirrkomendasikan)

<!DOCTYPE html>

<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<iframe width="420" height="315" src="http://www.youtube.com/embed/XGSy3_Czz8k?autoplay=1"> </iframe>

</body>
</html>

YouTube Parameters

Parameter - parameter YouTube.

autohide

Nilai 0: Player controls selalu diperlihatkan.

Nilai 1: Player controls secara otomatis disembunyikan ketika video diputar.

Nilai 2 (default): Jika player (pemutar) memiliki rasio 16:9 atau 4:3, sama seperti 1, kalau tidak sama seperti 0.

autoplay

Nilai 0 (default): Video tidak akan diputar secara otomatis ketika memuat player.

Nilai 1: Video akan dimainkan secara otomatis ketika memuat player.

controls

Nilai 0: Player controls tidak ditampilkan. Video memuat dengan segera.

Nilai 1 (default): Player control ditampilkan. Video memuat dengan segera.

loop

Nilai 0 (default): video hanya dimainkan sekali saja.

Nilai 1: Video akan di-loop atau diulang - ulang (selalu).

playlist

A comma separated list of videos to play (in addition to the original URL).

Sebuah daftar pemisah koma dari video untuk dimainkan (pada penambahan URL asli).

YouTube <object> Embeds

YouTube <object> embeds diprotes dari bulan Januari 2015.

Kamu harus migrasi-kan applikasi kamu untuk menggunakan <iframe> embeds.

Contoh - Penggunaan <object> (diprotes)

<!DOCTYPE html>

<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<object width="420" height="315" data="http://www.youtube.com/embed/XGSy3_Czz8k?autoplay=1"> </object>

</body>
</html>

Contoh - Penggunaan <embed> (diprotes)

<!DOCTYPE html>

<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<embed width="420" height="315" src="http://www.youtube.com/embed/XGSy3_Czz8k?autoplay=1">

</body>
</html>
