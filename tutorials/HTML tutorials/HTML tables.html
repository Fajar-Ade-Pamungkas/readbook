Defining HTML Tables

Contoh

Input:

<table style="width:100%">
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
</table>
                               
JillSmith50
EveJackson94

Penjelasan contoh:

  • Tabel didefinisikan dengan tag <table>.
  • Tabel dibagi kedalam baris tabel (table row) dengan tag <tr>.
  • Baris tabel dibagi kedalam data tabel (table data) dengan tag <td>.
  • Baris tabel juga dibagi kedalam tabel kepala (table heading) dengan tag <th>.
Tabel data <td> adalah data pengisi tabel.
<td> bisa berisi semua urutan element HTML seperti teks, gambar, list (daftar), tabel lainnya, dan sebagainya.

An HTML Table with a Border Attribute

Jika kamu tidak menentukan sebuah border untuk tabel, tabel akan ditampilkan tanpa border.

Border bisa ditambahkan menggunakan attribute border:

Contoh

Input:

<table style="width:100%" border="1">
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
</table>
                               
JillSmith50
EveJackson94
The border attribute is on its way out of the HTML standard! It is better to use CSS.

Untuk menambahkan border, gunakanlah properti border CSS:

Contoh

Input:

table, th, td {
border:1px black solid ;
}
                               
JillSmith50
EveJackson94

Ingatlah untuk mendefinisikan border untuk keduanya tabel dan tabel cell.

An HTML Table with Collapsed Borders

Jika kamu mau border menjadi collapse (saling berimpitan) kedalam satu border, tambahkan border-collapse CSS:

Contoh

Input:

table, th, td {
border:1px black solid;
border-collapse:collapse;
}
                               
JillSmith50
EveJackson94

An HTML Table with Cell Padding

Cell padding menentukan spasi diantara cell content (isi) dan border itu sendiri.

Jika kamu tidak menentukan sebuah padding, tabel cell akan ditampilkan tanpa padding.

Untuk mengatur padding, gunakanlah properti padding CSS:

Contoh

Input:

table, th, td {
border:1px black solid;
border-collapse:collapse;
}
th, td {    
padding: 15px;
}
                               
JillSmith50
EveJackson94

HTML Table Headings

Table Hedings didefinisikan dengan tag <th>.

Dengan default, semua major browser menampilkan table heading seperti teks tebal dan rsts tengah:

Contoh

Input:

<!DOCTYPE html>
<html>
<head>
<style>
table, th, td {
border:1px black solid;
border-collapse:collapse;
}
th, td {    
padding: 15px;
}
</style>
</head>
<body>

<table style="width:100%" border="1">

Untuk teks rata kiri untuk table heading (kepala tabel). Gunakanlah properti text-align CSS:

Kamu bisa tambahkan properti ini ke contoh tag diatas

th {
text-align:left;
}

Table Cells that Span Many Columns

Untuk membuat semua cell span lebih dari satu colom, gunakanlah attribute colspan (merentangkan colom):

Contoh

Input:

<!DOCTYPE html>
<html>
<head>
<style>
table, th, td {
border:1px black solid;
border-spacing: 15px;
border-collapse: separate;
}
th, td {    
padding: 15px;
}
</style>
</head>
<body>

<table style="width:100%" border="1">
<tr>
    <th> Nama </th>
    <th colspan="2"> Telephones </th>
</tr>
<tr>
<td> Fajar </td>
<td> 081288397435 </td>
<td> 083819136838 </td>
</tr>
</table>

</body>
</html>
                           
Name Contact
Fajar Ade Pamungkas 081288397435 083819136838

Ringkasan Materi

  • Gunakanlah element <table> HTML untuk mendefinisikan sebuah tabel
  • Gunakanlah element <tr> HTML untuk mendefinisikan baris tabel
  • Gunakanlah element <td> HTML untuk mendefinisikan data tabel
  • Gunakanlah element <caption> HTML untuk mendefinisikan caption (judul) tabel
  • Gunakanlah properti border CSS untuk mendefinisikan border
  • Gunakanlah properti collapse CSS untuk merapatkan border cell
  • Gunakanlah properti padding CSS untuk menambahkan padding pada cell
  • Gunakanlah properti text-align CSS untuk perataan cell teks
  • Gunakanlah properti border-spacing CSS untuk mengatur spasi antara kolom
  • Gunakanlah attribute colspan untuk membuat (merentangkan) cell ke banyak kolom
  • Gunakanlah attribute rowspan untuk membuat (merentangkan) cell ke banyak baris
  • Gunakanlah attribute id untuk dengan unik mendefinisikan satu tabel
  • Gunakanlah properti vertical-align CSS untuk meratakan teks secara vertical

HTML Table Tags

Tag Deskripsi
<table> Mendefinisikan sebuah tabel
<th> Mendefinisikan header cell (kepala cell atau tabel) dalam sebuah tabel
<tr> Mendefinisikan baris dalam sebuah tabel
<td> Mendefinisikan sebuah cell dalam tabel
<caption> Mendefinisikan sebuah judul tabel
<colgroup> Menentukan sebuah grup daru satu atau lebih colom dalam sebuah tabel untuk formatting
<col> Menentukan properti kolom untuk setiap kolom dengan sebuah element <colgroup>
<thead> Menggrupkan header content (konten kepala tabel) dalam sebuah tabel
<tbody> Menggrupkan body content (konten badan tabel) dalam sebuah tabel
<tfoot> Menggrupkan footer content (konten kaki tabel) dalam sebuah tabel
