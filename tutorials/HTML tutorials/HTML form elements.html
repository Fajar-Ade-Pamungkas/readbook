HTML Form Elements

Chapter ini mendeskripsikan semua element formulir HTML.

The <input> Element

Element formulir yang paling penting adalah element <input>.

Element <input> bisa bervariasi dalam banyak cara, tergantung pada attribute type.

Semua tipe input HTML di-cover dalam chapter selanjutnya.

The <select> Element (Drop-Down List)

Element <select> mendefinisiksn sebuah drop-down list:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<form action="http//:www.w3schools.com/action_page.php">  
<select name="cars">
  <option value="volvo">Volvo</option>
  <option value="saab">Saab</option>
  <option value="fiat">Fiat</option>
  <option value="audi">Audi</option>
</select>
</form>
</body>

</html>

Coba kamu tambahkan attribute selected seperti ini pada salah satu opsi

  <option value="fiat" selected>Fiat</option>

The <textarea> Element

Element <textarea> mendefinisikan multi-line input field (sebuah area teks):

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<form action="http//:www.w3schools.com/action_page.php">  
<textarea name="message" rows="10" cols="30"> The cat was playing in the garden. </textarea> <br>
<input type="submit">
</form>
</body>

</html>

The <button> Element

Element <button> mendefinisikan tombol yang dapat diklik:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<button type="button" onclick="alert('Hello World!')">Click Me!</button>
</body>

</html>

HTML5 Form Elements

HTML5 menambahkan element formulir berikut ini:

  • <datalist>
  • <keygen>
  • <output>
Dengan default, browser tidak menampilkan element yang tak diketahui. Element baru tidak akan menghancurksn halaman kamu.

HTML5 <datalist> Element

Element <datalist> menentukan sebuah daftar dari opsi yang dikenal untuk element <input>.

Para pengguna akan melihat sebuah drop-down list dari opsi yang dikenal sebagai input data.

Attribute list dari element <input>, harus mengarah ke attribute id element <datalist>.

Contoh

Input:

Sebuah element <inout> dengan pre-defined values dalam sebuah <datalist>:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<form action="action_page.php">  
  <input list="browsers">
  <datalist id="browsers">
    <option value="Internet Explorer">
    <option value="Firefox">
    <option value="Chrome">
    <option value="Opera">
    <option value="Safari">
  </datalist>
</form>
</body>

</html>

HTML5 <keygen> Element

Tujuan dari element <keygen> adalah untuk menyediakan cara keamanan untuk authenticate users.

Element <keygen> menentukan sebuah key-pair generator field dalam form.

Ketika formulir diserahkan, dua kunci dihasilkan, satu privasi (pribadi) dan satu publik.

Private key disimpan secara lokal, dan public key dikirim ke server.

Public key bisa digunakan untuk menghasilkan sebuah sertifikat nasabah (client sertificate) untuk authenticate (membuktikan) di masa depan.

Contoh

Input:

Formulir dengan keygen field

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<form action="action_page.php">  
Username: <input type="text" name="user"> <br><br>

Encryption: <keygen name="security"> <br><br>

<input type="submit">
</form>
</body>

</html>

Encryption:

HTML5 <output> Element

Element <output> mempersembahakan hasil dari kalkulasi (sepertu satu yang ditunjukan oleh sebuah script).

Contoh

Input:

Menunjukan kalkulasi dan menampilkan hasil dalam element <output>:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<form action="http://www.w3schools.com/html/action_page.php" oninput="x.value=parseInt(a.value)+parseInt(b.value)">
0
<input type="range" id="a" name="a" value="50">
100 +
<input type="number" id="b" name="b" value="50">
=
<output name="x" for="a b"></output>
<br><br>
<input type="submit">
</form>
</body>

</html>
0 100 + =
