JavaScript membuat halaman HTML lebih dinamis dan interaktif

The HTML <script> Tag

Tag <script> digunakan untuk mendefinisikan sebuah script (JavaScript).

Element <script> salah satu berisi scripting statements (pernyataan penaskahan) atau tujuannya untuk sebuah file script external sepanjang attribute src.

Yang umum digunakan untjk JavaScript adalah manipulasi gambar, validasi form, dan dinamis merubah konten.

Script dibawah ini menuliskan kata "Hello JavaScript!" kedalam element HTML dengan id="demo":

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<p id="demo"></p>

script>
document.getElementByid("demo").innerHTML="Hello JavaScript !";
</script>

</body>

</html>

Untuk mempelajari tentang JavaScript kunjungilah, artikel kami tentang Tutorial JavaScript.

The HTML <noscript> Tag

Tag <niscript> digunakan untuk menyediakan sebuah konten alternatif untuk pengguna yanh telah melumpuhkan atau me-nonaktifkan (disable) script pada browser mereka atau memiliki browser yang tidak mendukung client-side scripting.

Element <noscript> bisa berisi semua element yang bisa kamu temukan didalam element <body> dari sebuah halaman HTML normal.

Konten didalam element <noscript> hanya akan ditampilkan jika suatu script tidak disupport, atau dilumpuhkan dalam browser pengguna:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<p id="demo"></p>

script>
document.getElementByid("demo").innerHTML="Hello JavaScript !";
</script>


<noscript>
Sorry, your browser do not support for JavaScript !
</noscript>

<p> A browser without support for JavaScript will show the text written inside the noscript element. </p>

</body>

</html>

JavaScript bisa mengubah style HTML

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<p id="demo"> Helo world</p>

script>
x = document.getElementByid("demo");
x.style.color= "red";
x.style.fontSize= "20px";
</script>

</body>

</html>

Hello World !

JavaScript bisa mengubah gambar HTML

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<img onclick="changeImage()" src="../../hey.jpg"width="150px" height="auto" id="myImage">

script>
function changeImage() {
var image = document.getElementByid("myImage");
if (image.src.match("bulbon")){
image.src="../../pictures/coolImage.jpg";
} else{
image.src="../../pictures/Rain Fall.png";
}
}
</script>

</body>

</html>
