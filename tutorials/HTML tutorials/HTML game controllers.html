Dapatkan Kontrol

Sekarang kita ingin mengontrol red square (persegi merah).

Tambahkan empat button (tombol) up, down, left, dan right.

Tulislah sebuah fungsi untuk setiap button untuk menggerakan komponen pada arah tertentu (selected direction).

Buatlah dua properti baru pada component constructor dan sebut mereka speedX dan speedY. Properti-properti ini sedang digunakan sebagai indikator kecepatan:

Tambahkanlah sebuah fungsi dalam component constructor, yang disebut newPos(), yang mana menggunakan properti speedX dan speedY untuk mengubah posisi komponen.

Fungsi newpost dipanggil dari fungsi updateGaneArea sebelum penggambaran (atau gambar) komponen:

Keyboard as Controller

Keyboard sebagai Controller (Pengontrol)

Kita juga bisa mengontrol red square dengan menggunakan arrow key (kunci tanda panah) pada keyboard.

Buatlah sebuah metode yang mengecek jika sebuah kunci ditekan, dan mengatur properti key dari myGameArea object untuk key code. Ketika kunci ditekan, aturlah properti key menjadi false:

Contoh:

var myGameArea = {
canvas : document.createElement("canvas"), start : function() {
this.canvas.width = 480;
this.canvas.height = 270;
this.context = this.canvas.getContext("2d");
document.body.insertBefore(this.canvas, document.body.childNodes[0]);
this.interval = setInterval(updateGameArea, 20);
window.addEventListener('keydown', function (e) {
myGameArea.key = e.keyCode;
})
window.addEventListener('keyup', function (e) {
myGameArea.key = false;
})
},
clear : function(){
this.context.clearRect(0, 0, this.canvas.width, this.canvas.height);
}
}

Lalu kita bisa menggerakan red square jika salah satu dari arrow key ditekan:

Contoh:

function updateGameArea() {

Kunci Lebih dari satu ditekan (secara bersamaan)

Apa (yang terjadi) jika lebih dsri satu kunci ditekan pada waktu yang sama?

Pada contoh diatas, komponen hanya bisa bergerak secara horizontal data secara vertikal saja. Sekarang kita ingin komponen untuk juga bergerak secara diagonal.

Buatlah keys array untuk myGameArea object, dan sisipkan satu element untuk setisp kunci yang ditekan, dan berilah ia nilai true, nilai yang sama yaitu true sampai kunci tidak lebih lama ditekan, nilai menjadi false dalam fungsi event listener keyup:

Contoh:

Using The Mouse Cursor as a Controller

Penggunaan Mouse Cursor sebagai Pengendali

Jika kamu ingin mengontrol red square dengan menggunakan mouse cursor, tambahkanlah sebuah metode dalam myGameArea object yang memperbaharui koordinat x dan y dari mouse cursor:

Contoh:

var myGameArea = {
canvas : document.createElement("canvas"),
start : function() {
this.canvas.width = 480;
this.canvas.height = 270;
this.canvas.style.cursor = "none"; //hide the original cursor
this.context = this.canvas.getContext("2d");
document.body.insertBefore(this.canvas, document.body.childNodes[0]);
this.interval = setInterval(updateGameArea, 20);
window.addEventListener('mousemove', function (e) {
myGameArea.x = e.pageX;
myGameArea.y = e.pageY;
})
},
clear : function(){
this.context.clearRect(0, 0, this.canvas.width, this.canvas.height);
}
}

Kemudian kita bisa menggerakan red square menggunakan mouse cursor:

Contoh:

function updateGameArea() {
myGameArea.clear();
if (myGameArea.x && myGameArea.y) {
myGamePiece.x = myGameArea.x;
myGamePiece.y = myGameArea.y;
}
myGamePiece.update();
}

Touch The Screen to Control The Game

Menyentuh Layar untuk Mengontrol Game

Kita juga bisa mengendalikan red square dalam sebuah touch screen (layar sentuh).

Tambahkanlah sebuah metode dalam myGameArea object yang menggunakan koordinat x dan y tentang dimana layar disentuh:

Contoh:

var myGameArea = {
canvas : document.createElement("canvas"),
start : function() {
this.canvas.width = 480;
this.canvas.height = 270;
this.context = this.canvas.getContext("2d");
document.body.insertBefore(this.canvas, document.body.childNodes[0]);
this.interval = setInterval(updateGameArea, 20);
window.addEventListener('touchmove', function (e) {
myGameArea.x = e.touches[0].screenX;
myGameArea.y = e.touches[0].screenY;
})
},
clear : function(){
this.context.clearRect(0, 0, this.canvas.width, this.canvas.height);
}
}

Lalu kita bisa menggerakan red square jika pengguna menyentuh layar, dengan menggunakan kode yang sama seperti yang kita lakukan untuk mouse cursor:

Contoh:

function updateGameArea() {
myGameArea.clear();
if (myGameArea.touchX && myGameArea.touchY) {
myGamePiece.x = myGameArea.x;
myGamePiece.y = myGameArea.y;
}
myGamePiece.update();
}

Controllers on The Canvas

Pengendali pada Canvas

Kita juga bisa menggambar button milik kita pada canvas, dan menggunakannya sebagai pengendali:

Contoh:

function startGame() {
myGamePiece = new component(30, 30, "red", 10, 120);
myUpBtn = new component(30, 30, "blue", 50, 10);
myDownBtn = new component(30, 30, "blue", 50, 70);
myLeftBtn = new component(30, 30, "blue", 20, 40);
myRightBtn = new component(30, 30, "blue", 80, 40);
myGameArea.start();
}

Tambahkanlah sebuah fungsi baru yang menggambarkan (jika) sebuah komponen, pada kasus ini ialah button di klik.

Mulailah dengan menambahkan event listener untuk mengecek jika tombol mouse diklik (mousedown dan mouseup). Untuk menyetujui dengan touch screen, juga tambahkanlah event listener untuk emngecek jika layar diclik (touchstart dan touchend):

Contoh:

var myGameArea = {
canvas : document.createElement("canvas"),
start : function() {
this.canvas.width = 480;
this.canvas.height = 270;
this.context = this.canvas.getContext("2d");
document.body.insertBefore(this.canvas, document.body.childNodes[0]);
this.interval = setInterval(updateGameArea, 20);
window.addEventListener('mousedown', function (e) {
myGameArea.x = e.pageX;
myGameArea.y = e.pageY;
})
window.addEventListener('mouseup', function (e) {
myGameArea.x = false;
myGameArea.y = false;
})
window.addEventListener('touchstart', function (e) {
myGameArea.x = e.pageX;
myGameArea.y = e.pageY;
})
window.addEventListener('touchend', function (e) {
myGameArea.x = false;
myGameArea.y = false;
})
},
clear : function(){
this.context.clearRect(0, 0, this.canvas.width, this.canvas.height);
}
}

Sekarang myGameArea object memiliki properti yang memberitahukan kita koordinat x dan y tentang suatu klik. Kita gunakan properti-properti ini untuk mengecek jika klik dimainkan pada salah satu tombol biru kita.

Metode baru disebut dengan clicked, merupakan sebuah metode tentang component constructor dan checked mengecek jika komponen sedang diklik.

Pada fungsi updateGameArea, kita menambil aksi yang diperlukan jika salah satu tombol biru diklik:

Contoh:

function component(width, height, color, x, y) {
this.width = width;
this.height = height;
this.speedX = 0;
this.speedY = 0;
this.x = x;
this.y = y;
this.update = function() {
ctx = myGameArea.context;
ctx.fillStyle = color;
ctx.fillRect(this.x, this.y, this.width, this.height);
}
this.clicked = function() {
var myleft = this.x;
var myright = this.x + (this.width);
var mytop = this.y;
var mybottom = this.y + (this.height);
var clicked = true;
if ((mybottom < myGameArea.y) || (mytop > myGameArea.y) || (myright < myGameArea.x) || (myleft > myGameArea.x)) {
clicked = false;
}
return clicked;
}
}

function updateGameArea() {
myGameArea.clear();
if (myGameArea.x && myGameArea.y) {
if (myUpBtn.clicked()) {
myGamePiece.y -= 1;
}
if (myDownBtn.clicked()) {
myGamePiece.y += 1;
}
if (myLeftBtn.clicked()) {
myGamePiece.x += -1;
}
if (myRightBtn.clicked()) {
myGamePiece.x += 1;
}
}
myUpBtn.update();
myDownBtn.update();
myLeftBtn.update();
myRightBtn.update();
myGamePiece.update();
}
