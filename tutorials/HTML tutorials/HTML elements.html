HTML Elements

Dokumen HTML dibuat oleh element HTML.

Elemen HTML

Element HTML ditulis dengan sebuah start tag dan end tag, dengan konten yang berada diantara kedua tag tersebut:

<tagname> Kontent </tagname>

Element HTML adalah apapun dari start tag sampai end tag:

<p>My first HTML paragraph.</p>

Start tag Element content End tag
<h1> Heading pertamaku </h1>
<p> paragraph pertamaku </p>
<br>
Beberapa elemen HTML tidak memiliki end tag (penutupan tag).

Nested HTML Elements

elemen HTML bisa disarangkan (elemen bisa berisikan elemen lain).

Semua dokumen HTML terdiri dari elemen HTML yang bersarang.

Contoh ini berisikan 4 elemen HTML:

Contoh

Input:

<!DOCTYPE html>
<html>
<body>

<h1> Ini heading pertamaku </h1>
<p> Ini paragraf pertamaku</p>

</body>
</html>

Ini Heading pertamaku

Ini paragraf pertamaku

Penjelasan Contoh HTML

Elemen <html> mendefinisikan keseluruhan dokumen.

Ia memiliki sebuah start tag <html>dan sebuah end tag </html>.

Elemen berisi elemen HTML yang lainnya (elemen <body>).

<html>
<body>

<h1> Ini heading pertamaku </h1>
<p> Ini paragraf pertamaku</p>

</body>
</html>

elemen <body> mendefinisikn badan dokumen atau isi dokumen.

Ia memiliki sebuah start tag <body>dan sebuah end tag </body>.

elemen ini berisi 2 elemen HTML yang lainnya (<h1> dan <p>).

<body>

<h1> Ini heading pertamaku </h1>
<p> Ini paragraf pertamaku</p>

</body>

elemen <h1> mendefinisikan Kepala HTML.

Ia memiliki sebuah start tag <h1>dan sebuah end tag </h1>.

elemen ini berisi: Ini heading pertamaku.

<h1> Ini heading pertamaku </h1>

elemen <p> mendefinisikan sebuah paragraf HTML.

Ia memiliki sebuah start tag <p>dan sebuah end tag </p>.

elemen ini berisi: Ini paragraf pertamaku.

<p> Ini paragraf pertamaku</p>

Jangan Lupa Menutup dengan End Tag

Beberapa elemen HTML akan menampilkan secara benar, bahkan jika kamu lupa menutupnya dengan end tag:

Contoh

Input:

<!DOCTYPE html>
<html>
<body>

<h1> Ini heading pertamaku
<p> Ini paragraf pertamaku

</body>
</html>

Ini Heading pertamaku

Ini paragraf pertamaku

Contoh diatas mengerjakan pada semua browser, karena penutupan tag adalah opsi ditentukan.

Jangan pernah mempercayakan cara ini. Ini mungkin mengproduksi hasil yang keliru dan/atau error jika kamu lupa menutup dengan end tag.

Empty HTML Elements

Elemen HTML dengan tidak adanya konten disebut dengan empty elements.

<br> adalah sebuah empty element tanpa penutup tag (<br> tag mendefinisikan sebuah line break atau baris baru).

Empty element bisa di "tutup" pada tag pembuka seperti <b />.

HTML5 tidak memerlukan empty element untuk ditutup. Tapi jika kamu mau validasi yang lebih akurat, atau kamu butuh untuk membuat dokument yang mudah dibaca oleh XML parsers, kamu harus menutup semua elemen HTML.

Saran HTML: Gunakanlah Tag Lowercase

HTML tag bukan merupakan case sensitive: <p> artinya sama seperti <P>.

Standar HTML5 tidak perlu lowercase tag (tag dengan huruf kecil), tapi W3S menyarankan lowercase pada HTML4, dan menuntut lowercase untuk tipe dokumen yang lebih akurat seperti XHTML.

Di Readbook kita selalu gunkan lowercase tag.
