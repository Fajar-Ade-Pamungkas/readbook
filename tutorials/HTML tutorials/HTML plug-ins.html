HTML Plug-Ins

Maksud dari sebuah plug-in, adalah untuk memperpanjang fungsionalitas dari browser HTML.

HTML Helpers (Plug-ins)

Pembantu HTML (Plug-Ins)

Helper applications merupakan program komputer yang memperluas atau memperpanjang fungsionalitas standar tentang sebuah web browser.

Helper applications juga disebut plug-ins.

Contoh plug-ins yang sudah diketahui secara baik adalah Java applets.

Plug-ins bisa ditambahkan ke halaman web dengan tag <object> atau tag <embed.

Plug-ins bisa digunakan untuk banyak tujuan: menampilkan map, men-skan virus, verifikasi id bank kamu, dan lain sebagainya.

Untuk menampilkan video dan audio: Gunakanlah tag <video> dan <audio>.

The <object> Element

Element <object> disupport di semua browser.

Element <object> mendefinisikan sebuah object tertanam (embedded object) dengsn sebuah dokumen HTML.

Ia berguna untuk menanamkan plug-ins (seperti Java applet, PDF readers, Flash Players) dalam web pages.

Input:

<!DOCTYPE html>

<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<object width="400" height="50" data="bookmark.swf"></object>

</body>
</html>

Element <object> juga bisa digunakan untuk memasukan HTML dalam HTML.

Input:

<!DOCTYPE html>

<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<object width="100" height="500" data="HTML snippet.html"> </object>

</body>
</html>

Atau gambar jika kamu suka:

Input:

<!DOCTYPE html>

<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<object width="128" height="auto" data="Rain Fall.png"> </object>

</body>
</html>

The <embed> Element

Element <embed> disupport di semua mayor browser.

Element <embed> juga mendefinsikan sebuah objet tertanam dengan sebuah dokumen HTML.

Web browser telah mensupport element <embed> untuk waktu yang lama. Akan tetapi, <embed> belum menjadi sebuah bagian dari spesifikasi HTML sebelum HTML5. Element akaj memvalid di halaman HTML5, tapi tidak di halaman HTML4.

Input:

<!DOCTYPE html>

<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<embed width="400" height="50" src="bookmark.swf">

</body>
</html>
Catatlah bahwa element <embed> tidak memiliki sebuah closing tag (seperti </embed>). Dan tidak bisa berisi alternative text.

Element <embed> juga bisa digunakan untuk memasukan HTML dalam HTML.

Input:

<!DOCTYPE html>

<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<embed width="100%" height="500" src="audio.html">

</body>
</html>

Atau gambar jika kamu suka:

Input:

<!DOCTYPE html>

<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<embed width="128" height="128" src="Rain Fall.png">

</body>
</html>
