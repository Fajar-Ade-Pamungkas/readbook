HTML Game Components

Tambahkan sebuah kotak merah kedalam game area:

Add a Component

Menambahkan Komponen

Buatlah component constructor (pembangun komponen), yang mana mengizinkan kamu untuk menambahkan komponen-komponen kedalam gamearea.

Object constructor disebut dengan component, dan kita membuat komponen pertama kita, yang disebut myGamePiece:

Contoh:

var myGamePiece;

function startGame() {
myGameArea.start();
myGamePiece = new component(30, 30, "red", 10, 120);
}

function component(width, height, color, x, y) {
this.width = width;
this.height = height;
this.x = x;
this.y = y;
ctx = myGameArea.context;
ctx.fillStyle = color;
ctx.fillRect(this.x, this.y, this.width, this.height);
}

Frames

Bingkai

Untuk membuat gane siap untuk beraksi, kita akan memperbaharui menampilkan (dalam waktu) 50 per detik, yang mana merupakan banyak seperti frame dalam sebuah movie.

Pertama, buatlah sebuah fungsi baru yang bernama updateGameArea().

Pada myGameArea object, tambahakan sebuah interval yang mana akan menjalankan fungsi updateGameArea() setiap 20 milidetik (50 times per second). Tambahkan juga fungsi bernama clear(), untuk membersihkan seluruh (konten dalam) canvas.

Pada component constructor, tambahkanlah fungsi bernama update(), untuk meng-handle gambar dari komponen.

Fungsi updateGameArea() memanggil metode clear() dan update().

Hasil yang komponen digambar dan dibersihkan (dalam) 50 time per second:

Contoh:

var myGameArea = {
canvas : document.createElement("canvas"),
start : function() {
this.canvas.width = 480;
this.canvas.height = 270;
this.context = this.canvas.getContext("2d");
document.body.insertBefore(this.canvas, document.body.childNodes[0]);
this.interval = setInterval(updateGameArea, 20);
},
clear : function() {
this.context.clearRect(0, 0, this.canvas.width, this.canvas.height);
}
}

function component(width, height, color, x, y) {
this.width = width;
this.height = height;
this.x = x;
this.y = y;
this.update = function(){
ctx = myGameArea.context;
ctx.fillStyle = color;
ctx.fillRect(this.x, this.y, this.width, this.height);
}
}

function updateGameArea() {
myGameArea.clear();
myGamePiece.update();
}

Why Clear The Game Area?

Kenapa harus membersihkan Area Permainan?

Ini mungkin terlihat tidak diperlukan untuk membersihkan game area pada setiap update. Akan tetapi, Jika kita meninggalkan (game) metode clear(), semua pergerakan dari komponen akan meninggalkan sebuah trail tentang dimana komponen diposisikan pada frame terakhir:

Contoh:

function updateGameArea() {
// myGameArea.clear();
myGamePiece.x += 1;
myGamePiece.update();
}

Many Components
