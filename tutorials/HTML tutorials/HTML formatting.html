HTML Formatting Elements

Pada chapter sebelumnya, kamu memepelajari tentang penggayaan HTML, menggunakan attribute style.

HTML juga mendefinisikan element special untuk pendefinisian teks dengan sebuah arti istimewa.

HTML menggunakan elemen seperti <b> dan <i> untuk formatting input, seperti teks tebal atau miring.

Formatting elements didesign untuk menampilkan tipe istimewa teks:

  • Bold text
  • Important text
  • Italic text
  • Emphasized text
  • Marked text
    • Small text
  • Deleted text
  • Inserted text
  • Subscripts
  • Superscripts

HTML Bold dan Strong Formatting

Element <b> mendefinisikan teks tebal, tanpa penambahan ekstra.

HTML Formatting

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<p>> Ini teks biasa </p>
<p> <b>Ini teks bold </b></p>

</body>

</html>

Ini teks biasa

Ini teks bold

Ini teks emphasized

Browser menampilkan <strong> sebagai <b> dan <em> sebagai <i>.

Akan tetapi, Disini ada sebuah perbedaan dalam arti tag ini: <b> dan <i> mendefinisikan teks bold dan italic, tapi <strong> dan <em> mengartikan bahwa teks itu "Penting".

HTML Text Formatting Elements

Tag Deskripsi
<b> Mendefinisikan teks bold (tebal)
<em> Mendefinisikan teks emphasized (miring khusus)
<i> Mendefinisikan teks italic (miring)
<strong> Mendefinisikan sebuah teks strong (tebal khusus)
<small> Mendefinisikan sebuah teks kecil
<sub> Mendefinisikan sebuah teks subscripted (diturunkan)
<sup> Mendefinisikan sebuah teks superscripted (dinaikan)
<ins> Mendefinisikan sebuah teks inserted (disisipkan garis bawah)
<del> Mendefinisikan sebuah teks delete (coret teks)
<mark> Mendefinisikan sebuah teks yang ditandai/disoroti
