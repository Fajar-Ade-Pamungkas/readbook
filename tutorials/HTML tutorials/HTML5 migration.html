Migration from HTML4 to HTML5

Chapter ini adalah seutuhnya tentsng bagaimana berpindah (bermigrasi) dari tipikal halaman HTML4 ke tipikal halaman HTML5.

Chapter ini mempertunjukan bagaimana untuk mengubah halaman HTML4 kedalam halaman HTML5, tanpa menghancurkan apapun tentsng konten atau struktur aslinya.

Kamu bisa bermingrasi ke HTML5 dari HTML4 atau XHTML menggunakan resep atau cara yang sama.

Change to HTML5 Doctype

Ubahlah doctype, dari doctype HTML4:

HTML4

<!DOCTYPE html HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

Menjadi doctype HTML5:

HTML5

<!DOCTYPE html>

Change to HTML5 Encoding

Ubahlah informasi encoding, dari HTML4:

HTML4

<meta http-equiv="Content-Type" content="text/html;charset=utf-8">

Menjadi HTML5:

HTML5

<meta charset="UTF-8">

Add The Shiv

Element semantik HTML5 didukung pada semua browser modern.

Pada penambahannya, kamu bisa "mengajarkan" browser lama bagaimana untuk menangani "element - element yang tidak diketahui".

Tambahkan shiv untuk Internet Explorer support:

HTML5 Enabling JavaScript

<!--[if lt IE 9]>  
<script src="http://html5shiv.googlecode.com/svn/trunk/html5.js"></script>
<![endif]--

Bacalah tentang shiv di HTML Browser Support.

Perbedaan Antara article> <section>

dan div>

Ini perbedaan cukup membingungkan (lack of) dalam standar HTML5, antara <article>, <section> dan <div>.

Pada standar HTML5, <seftion> didefinisikan sebagai sebuah blok dari element - elemen yang terhubung (block of related elements).

Element <article> didefinisikan sebagai sebuah kelengkapan, blok berisikan dirinya dari element - element yang terhubung (self-contained block of related elements).

Element <div> didefinisikan sebagai sebuah blok dari element - element turunan (blok of children elements).

Bagaimana untuk menterjemahkannya?

Pada contoh diatas, keta telah menggunakan <section> sebagai sebuah container (tempat pengisian) untuk <article> terhubung.

Tapi, kita bisa menggunakan <article> sebagai contsiner untuk article sama baiknya.

Disini adalah beberapa contoh perbedaan:

<article>
<h2>Famous Cities</h2>

<article>

</article>
