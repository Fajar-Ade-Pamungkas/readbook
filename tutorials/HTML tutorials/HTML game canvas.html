HTML - Game Canvas

Element <canvas> HTML ditampilkan sebagai rectangular object (object persegi) pada halaman web:

HTML Canvas

Element <canvas> sempurna untuk pembuatan game dalam HTML.

Element <canvas> menamwarkan semua fungsionalitas yang kamu perlukan untuk membuat game.

Gunakan JavaScript utuk menggambar, menulis, menyisipkan gambar, dan banyak lagi. kedalam <canvas>.

.getContext("2d")

Element <canvas> memiliki suatu built-in object, yang disebut getContext("2d") object, dengan metode dan properti untuk menggambar.

Kamu bisa mempelajari banyak tentang element <canvas> dan getContext("2d") object, pada Tutorial Canvas kita.

Get Started

Ayo Mulai

Untuk membuat game, mulailah dengan pembuatan gaming area (area permainan) dan membuatnya siap untuk menggambar:

Contoh:

function startGame() {
myGameArea.start();
}

var myGameArea = {
canvas : document.createElement("canvas"),
start : function() {
this.canvas.width = 480;
this.canvas.height = 270;
this.context = this.canvas.getContext("2d");
document.body.insertBefore(this.canvas, document.body.childNodes[0]);
}
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
canvas {
border: 1px solid #d3d3d3;
background-color: #f1f1f1;
}
</style>
</head>
<body onload="startGame()">

<script>
function startGame() {
myGameArea.start();
}

var myGameArea = {
canvas : document.createElement("canvas"),
start : function() {
this.canvas.width = 480;
this.canvas.height = 270;
this.context = this.canvas.getContext("2d");
document.body.insertBefore(this.canvas, document.body.childNodes[0]);
}
}
</script>

</body>
</html>

Tripy it!

Object myGameArea akan memiliki banyak properti dsn metode nantinya di tutorial ini.

Fungsi startGame() memanggil metode start() dari myGameArea object.

Metode start() membuat sebuah element <canvas> dan menyisipkannya sebagai first childnode dari element <body>.
