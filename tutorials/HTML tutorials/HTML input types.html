HTML Input Types

Input Types

Chapter ini mendefinisikan tipe - tipe input dari element <input>.

Input Type: text

<input type="text"> mendefinisikan sebuah ole-line input field untuk input berupa teks:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<form>  
First name:<br>
  <input type="text" name="firstname"><br>
  Last name:<br>
  <input type="text" name="lastname">
</form>
</body>

</html>
  First name:
 
  Last name:

Input type: submit

<input type="submit"> mendefinisikan sebuah tombol untuk submitting (penyerahan) formulir ke form-handler.

Form-handler secara khas adalah sebuah halaman server dengan sebuah script (naskah) untuk processing input data.

Form-handler ditentukan dalam attribute action (aksi) formulir:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<form>  
  First name:<br>
  <input type="text" name="firstname" value="Mickey"><br>
  Last name:<br>
  <input type="text" name="lastname" value="Mouse"><br><br>
  <input type="submit" value="Submit">
</form>
</body>

</html>
  First name:
 
  Last name:
 

 

Jika kamu menghilangkan attribute value tombol submit, tombol akan mendapat default text:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<form>  
  First name:<br>
  <input type="text" name="firstname" value="Mickey"><br>
  Last name:<br>
  <input type="text" name="lastname" value="Mouse"><br><br>
  <input type="submit">
</form>
</body>

</html>
  First name:
 
  Last name:
 

 

Input Type: radio

<input type="radio"> mendefinisikan sebuah radio button.

Radio button membiarkan seorsng pengguna memilih HANYA SATU dari nomor terbatas dari pilihan:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<form action="http://www.w3schools.com/html/action_page.php"  
  <input type="radio" name="gender" value="male" checked><br>
  <input type="radio" name="gender" value="female"><br>
  <input type="radio" name="gender" value="other"><br>
</form>
</body>

</html>
  Male
  Female
  Other

</html>

I have a car

HTML5 Input Types

HTML5 menambahkan beberapa tipe input baru:

  • color
  • date
  • datetime
  • datetime-local
  • email
  • month
  • number
  • range
  • search
  • tel
  • time
  • url
  • week
Tipe - tipe input diatas mungkin tidak didukung oleh web browser versi lama, akan diperlakukan sebagai input type text.

Input Type: number

<input type="number"> digunakan untuk input field yang harus berisi sebuah nilai numeric.

Kamu bisa menyetel restrictions (pembatasan) pada nomor.

Tergantung pada browser pendukung, pembatassn bisa menerapkan untuk input field.

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<p>
Depending on browser support:<br>
Numeric restrictions will apply in the input field.
</p>
<form action="http://www.w3schools.com/html/action_page.php">
Quantity (between 1 and 5):
<input type="number" name="quantity" min="1" max="5">
<input type="submit">
</form>

<p><b>Note:</b> type="number" is not supported in IE9 and earlier.</p>

</body>

</html>

Depending on browser support:
Numeric restrictions will apply in the input field.

Quantity (between 1 and 5):

Note: type="number" is not supported in IE9 and earlier.

Input Restrictions

Disini adalah sebuah daftar dari beberapa input resrictions umum (beberapa yang baru dalam HTML5):

Attribute Deskripsi
disabled Menentukan bahwa sebuah input field harus dilumpuhkan atau dinonaktifkan
max Menentukan nilai maksimum untuk sebuah input field
maxlength Menentukan (panjang) nomor maksimum karakter untuk input field
min Menentukan nilai minimum untuk sebuah inout field
pattern Menentukan sebuah regular expression untuk mengecek nilai input berlawanan (against)
readonly Menentukan bahwa sebuah input field hanya dibaca (tidak bisa diubah)
required Menentukan bahwa sebuah input field diperlukan (atau harus diisi)
size Menentukan width atau lebar (dalam karakter) dari sebuah input field
step Menentukan legal number intervals untuk sebuah input field
value Menentukan nilai default untuk sebuah input field

Kamu akan mempelajari lebih tentang input restrictions dalam chapter selanjutnya.

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<p>
Depending on browser support:<br>
Fixed steps will apply in the input field.
</p>
<form action="http://www.w3schools.com/html/action_page.php">
Quantity
<input type="number" name="points" min="0" max="100" value="30">
<input type="submit">
</form>

<p><b>Note:</b> type="number" is not supported in IE9 and earlier.</p>

</body>

</html>

Depending on browser support:
Fixed steps will apply in the input field.

Quantity:

Note:type="number" is not supported in IE9 and earlier.
