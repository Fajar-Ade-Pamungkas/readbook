Dokumen HTML

Semua dokumen HTML harus dimulai dengan sebuah deklarasi tipe: <!DOCTYPE html>.

Dokumen HTML itu sendiri dimulai dengan <html dan diakhiri dengan </html>.

Bagian yang terlihat daru dokumen HTML adalah diantara <body> dan </body>.

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<h1> Ini heading pertamaku </h1>

<p> Ini paragraf pertamaku</p>

</body>

</html>

Ini Heading pertamaku

Ini paragraf pertamaku

HTML Heading

Heading didefinisikan dengan tag <h1> sampai <h6>

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<h1> This is a heading</h1>
<h2> This is a heading</h2>
<h3> This is a heading</h3>
<h4> This is a heading</h4>
<h5> This is a heading</h5>
<h6> This is a heading</h6>

</body>

</html>

Perhatikanlah!

Contoh diatas untuk tag <h1> seharusnya lebih besar daripada yang lainnya. Ini karena tag <h1> sudah digunakan pada halaman ini jadi apabila digunakan dua kali atau lebih akan menjadi kecil daripada tag yang pertama.

HTML Paragraf

HTML paragraf didefinisikan oleh tag <p>:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<p> This is a paragraph </p>
<p> This is another paragraph </p>

</body>

</html>

This is a paragraph

This is a paragraph

HTML link

HTML link didefinisikan dengan tag <a>:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<a href="HTML HOME.html"> This is a link </a>

</body>

</html>
This is a link

Destinasi link ditentukan di attribute href.

Attribut digunakan untuk menyediakan informasi tambahan tentang elemen HTML.

HTML Images

HTML images didefinisikan dengan tag <img>.

Source file (src), alternative text (alt), dan size (width dan height) disediakan sebagi attribute:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<img src="../../pictures/Rain Fall.png" alt="Rainly" width="100" height="100">

</body>

</html>

Kamu akan memperlajari lagi tentang attribute di lain chapter.
