SVG <text>

SVG Text - <text>

Element <text> digunakan untuk mendefinisikan sebuah teks.

Example 1

Contoh 1

Tulislah sebuah teks:

Disini adalah kode SVG:

Contoh Kode SVG

<svg height="30" width="200">
  <text x="0" y="15" fill="red">I love SVG!</text>
</svg>

Example 2

Contoh 2

Rotasikan teks:

Disini adalah kode SVG:

Contoh Kode SVG

<svg height="60" width="200">
  <text x="0" y="15" fill="red" transform="rotate(30 20,40)">I love SVG</text>
</svg>

Example 3

Contoh 3

Element <text> bisa dirangkai dalam nomor apa saja dari sub-groups dengan element <tspan>. Setiap element <tspan> bisa berisi format dan posisi yang berbeda.

Teks di beberapa garis (dengan element <tspan>):

Disini adalah kode SVG:

Contoh Kode SVG

<svg height="90" width="200">
  <text x="10" y="20" style="fill:red;">Several lines:
    <tspan x="10" y="45">First line.</tspan>
    <tspan x="10" y="70">Second line.</tspan>
  </text>
</svg>

Example 4

Contoh 4

Teks sebagai link (dengan element <a>):

Disini adalah kode SVG:

Contoh Kode SVG

<svg height="30" width="200">
  <a xlink:href="http://www.w3schools.com/svg/" target="_blank">
    <text x="0" y="15" fill="red">I love SVG!</text>
  </a>
</svg>
