HTML Encoding (Character Set)

Untuk menampilkan sebuah halaman HTML secara benar, sebuah web browser harus mengetahui character set (character encoding) tuk digunakan.

What is Character Encoding?

ASCII adalah Standar penyandian karakter pertama (dipanggil juga character set). ASCII mendefinisikan 127 karakter alphanumeric berbeda yang bisa digunakan di internet.

ASCII mendukung nomor (0-9), english letters (A-Z)dan beberapa karakter special seperti ! $ + - ( ) @ < > .

ISO-8859-1 adalah default character set untuk HTML 4. Ia juga mendukung 256 kode karakter berbeda.

Karena ANSI dan ISO itu terbatas, default character encoding berubah menjadi UTF-8 dalam HTML5.

UTF-8 (Unicode) meng-cover hampir semua karakter dan simbol di dunia.

Semua processor HTML 4 juga mendukung UTF-8.

The HTML charset Attribute

Untuk menampilkan sebuah halaman HTML dengan benar, web browser harus mengetahui character set yang digunakan pada halaman.

Ini ditentukan dalam teg <meta>:

Untuk HTML4:

<meta http-equiv="Content-Type" content="text/html;charset=ISO-8859-1">

Untuk HTML5:

<meta charset="UTF-8">
Jika browser mendeteksi ISO-8859-1 dalam sebuah web page, ia men-default menjadi ANSI, karena ANSI adalah indetik (mirip) dengan ISO-8859-1 kecuali bahwa ANSI memiliki 32 karakter ekstra.

The ASCII Character Set

ASCII menggunakan nilai dsri 0 - 31 (dan 127) untuk kontrol karakter.

ASCII menggunakan nilai dari 32 - 126 untuk letter (huruf), digit (angka) dan simbol.

ASCII tidak menggunakan nilai dari 128 - 255.

The ANSI Character Set (Windows-1252)

ANSI adalah serupa dengan ASCII untuk nilai dari 0 - 127.

ANSI memiliki sebuah priorietary set (penyetel pemilik) karakter untuk nilai dari 128 - 159.

ANSI serupa dengan UTF-8 untuk nilai dari 160 - 255.

The ISO-8859-1 Character Set

8859-1 adalah serupa dengan ASCII untuk nilai dari 0 - 127.

8859-1 tidak menggunakan nilai dari 128 - 159.

8859-1 serupa dengan UTF-8 untuk nilai dari 160 - 255.

The UTF-8 Character Set

UTF-8 adalah serupa dengan ASCII untuk nilai dari 0 - 127.

UTF-8 tidak menggunakan nilai dari 128 - 159.

UTF-8 serupa dengan kedua ANSI dan 8859-1 untuk niali dari 160 - 225.

UTF-8 melanjutkan dari nilai 256 dengan lebih dari 10.000 karakter berbeda.
