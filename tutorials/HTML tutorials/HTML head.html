HTML Head

The HTML <head> Element

Element <head> adalah sebuah container (tempat) untuk untuk metadata (data about data).

Metadata HTML adalah data tentang dokumen HTML. Metaata tidak ditampilkan.

Metadata secara tipikal (secara khas) mendefinisikan document, styles, links, scripts dan infomasi meta lainnya.

Tag - tag berikut mendefinisikan meta data: <title> <style>, <meta>, <script>, dan <base>.

Omitting <html> and <body>?

Dalam standar HTML5, tag <html>, tag <body> dan tag <head> bisa dihilangkan (omitted).

Kode berikut akan memvalidasi sebagai HTML5:

Contoh

Input:

<!DOCTYPE html>

<title> Judul Halaman </title>

<h1> This is a Heading</h1>
<p> This is a Paragraph</p>

This is a Heading

This is a Paragraph

Kita tidak merekomendasikan penghapusan tag <html> dan <body>.

element <html> adalah document root. ia direkomendasikan sebagai tempat untuk penentuan page language (bahasa halaman):

<!DOCTYPE html>
<html lang="en-US">

Pendeklarasian bahasa penting untuk accessibility application (screen readers) dan search engines (mesin pencari).

Penghilangan tag <html> dana <body> bisa merusak software DOM/XML yang tertulis dengan buruk.

Akhirnya, penghapusan atau penghilangan tag <body> bisa memproduksi error dalam browser lama (IE9).

Omitting <head>

Dalam standar HTML5, tag <head> bisa juga dihilangkan.

Dengan default, browser akan menambahkan semua element sebelum <body>, untuk element <head> dafault (pengaturan yang sudsh diatur).

Kamu bisa mengurangi kerumitan HTML, dengan menghilangkan tag <head>:

Contoh

Input:

<!DOCTYPE html>

<html>
<title> Judul Halaman </title>

<body>
<h1> This is a Heading</h1>
<p> This is a Paragraph</p>
</body>
</html>

This is a Heading

This is a Paragraph

Penghilangan tag tidak familiar (tidak lazim digunakan) untuk para pengembang web. Penghilangan itu butuh waktu untuk memantapkan sebagai garis pedoman (guideline).

Element <style> digunakan untuk mendefinisikan informasi style kepada dokumen HTML.

Didalam element <style> kamu bisa menentukan bagaiaman element - element HTML harus diterjemahkan atau diubah dalam sebuah browser:

Input:

The HTML <link> Element

Element <link> mendefinisikan hubungan halaman (page relationship) dengan file dari sumber diluar dokumen HTML saat ini (external resource).

Element <link> paling sering digunakan untuk link ke style sheet (CSS):

Contoh

Input:

<!DOCTYPE html>

<html>
<head>

<title> Judul Halaman </title>
<style rel="stylesheet" href="style.css">

</head>
<body>

<h1> This is a Heading</h1>
<p> This is a Paragraph</p>

</body>
</html>

The HTML <meta> Element

Element <meta> digunakan untuk menentukan page description, keywords, author (nama penulis), dan metadata lainnya.

Metadata digunakan oleh browser (bagaimana tuk menampilkan konten), dengan search engines (keyword), dan layanan web lainnya.

mendefinisikan keyword kepada search engines:

<meta name="keywords" content="HTML, CSS, XML, XHTML, JavaScript">

mendefinisikan description halaman web kamu:

<meta name="description" content="Media Belajar Otodidak Website">

mendefinisikan character set yang digunakan:

<meta charset"UTF-8">

mendefinisikan author (penulis) halaman:

<meta name="author" content="Fajar Ade Pamungkas">

Me-refresh dokumen setiap 30 detik:

<meta http-equiv="refresh" content="30">

Contoh kumpulan meta data

Input:

<!DOCTYPE html>

<html>
<head>

<meta name="keywords" content="HTML, CSS, XML, XHTML, JavaScript">
<meta name="description" content="Media Belajar Otodidak Website">
<meta charset"UTF-8">
<meta name="author" content="Fajar Ade Pamungkas">
<meta http-equiv="refresh" content="30">
<title> Judul Halaman </title>
<style rel="stylesheet" href="style.css">

</head>
<body>

<h1> This is a Heading</h1>
<p> Metadata tidak akan ditampilkan karena merupakan sebuah informasi halaman</p>

</body>
</html>

The HTML <script> Element

Element <script> digunakan untuk mendefinisikan client-side JavaScript.

Script dibawah ini menuliskan kata Hello JavaScript! kedalam sebuah element HTML dengan id="demo":

Contoh

Input:

<!DOCTYPE html>

<html>
<head>
<title> Judul Halaman </title>
script>
document.getElementByid("demo").innerHTML="Hello JavaScript !";
</script>
</head>
<body>

<p id="demo"></p>

</body>
</html>

Untuk mempelajari tentang JavaScript kunjungilah, artikel kami tentang Tutorial JavaScript.

The HTML <base> Element

Element <base> menentukan base URL (URL dasar) dan base target (target dasar) untuk semua URL relatif dalam sebuah halaman:

Contoh

Input:

<!DOCTYPE html>

<html>
<head>
<title> Judul Halaman </title>
<base target="_blank">
script>
document.getElementByid("demo").innerHTML="Hello JavaScript !";
</script>
</head>
<body>

<img src="../images/html5.gif">
<p>
Since we have specified a base URL, the browser will look for the image "html5.gif" at "http://www.w3schools.com/images/html5.gif"
</p>

<p><a href="../index.html">Readbooks</a>
</p>

<p>
The link above opens in a new window. This is because the base target is set to "_blank".
</p>

</body>
</html>

Sejak kita telah menentukan sebuah base URL, browser akan melihat image atau gambar "Rain Falk.png" di "http:www.readbooks.co.id/images/Rain Fall.png"
