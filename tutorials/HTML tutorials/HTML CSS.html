Styling HTML with CSS

CSS berbidang untuk Cascading Style Sheets

Penggayaan bisa ditambahkan ke element HTML dalam 3 cara:


  • Inline - penggunaan attribute style dalam element HMTL
  • Internal - penggunaan element <style> di bagian <head>
  • External - penggunaan satu atau lebih file CSS external

Cara yang paling umum untuk menambahkan styling atau penggayaan, ialah selalu style dalam file CSS terpisah. Tapi, pada tutorial ini, kita gunakan internal styling, karena ini mudah untuk ditinjau dan lebih mudah untuk kamu coba sendiri.

Kamu akan mempelajari banyak tentang CSS di Tutorials CSS.

Inline Styling (Inline CSS)

Inline styling (dalam element) digunakan untuk menerapkan style unik untuk satu element HTML:

Inline styling menggunakan attribute style.

Contoh jni mengubah warna teks dari element <h1> menjadi biru:

Contoh

Input:

<h1 style="color:blue;"> Ini adalah Heading Biru </p>

Ini adalah Heading Blue

Internal Styling (Internal CSS)

Internal styling (dalam file HTML) digunakan untuk mendefinisikan sebuah style (gaya) untuk satu halaman HTML.

Internal styling didefinisikan dalam bagian <head> dari sebuah halaman HTML, dengan sebuah element <style>:

Contoh

Input:

<!DOCTYPE html>

<html>
<head>
<style>
body {background-color:lightgrey;}
h1   {color:blue;}
p    {color:green;}
</style>
</head>
<body>

<h1> This is a heading </h1>
<p> This is another paragraph </p>

</body>

</html>

This is a heading

This is a paragraph.

External Styling (External CSS)

Sebuah External style (diluar file HTML) digunakan untuk mendefinisikan style untuk banyak halaman.

Dengan sebuah external style sheet, kamu bisa mengubah penampilan dari keselurahan web site dengan mengubah satu file.

Untuk menggunakan sebuah external style sheet, tambahkan sebuah link menuju file CSS tersebut di bagian <head> dari halaman HTML:

Contoh

Input:

<!DOCTYPE html>

<html>
<head>
<link rel="stylesheet" type="text/css" href="style.css">
</head>
<body>

<h1> This is a heading </h1>
<p> This is another paragraph </p>

</body>

</html>

This is a heading

This is a paragraph.

Sebuah external styel sheet bisa ditulis dalam text editor apapun. File tidak harus berisi tag HTML. style sheet file harus disimpan dengan extension .css

Disini adalah bagaikana "style.css" terlihat:

body {
background-color:lightgrey;
}
h1   {
color:blue;
}
p    {
color:green;
}

CSS Fonts

Properti color CSS mendefinisikan warna teks untuk digunakan kepada element HTML.

Properti font-family CSS mendefinisikan font untuk digunakan kepada elment HTML.

Properti font-size CSS mendefinisikan ukuran teks untuk digunsksn kepada element HTML.

Contoh

Input:

<!DOCTYPE html>

<html>
<head>
<style>
h1 {    
color: blue;
    font-family: verdana;
    font-size: 300%;
}
p  {    
color: red;
    font-family: courier;
    font-size: 160%;
}
</style>
</head>
<body>

<h1> This is a heading </h1>
<p> This is another paragraph </p>

</body>

</html>

The id Attribute

Semua contoh diats menggunakan CSS untuk style (gaya) element HTML dengan cara umum.

Untuk mendefinisikan special style kepada satu element spesial, pertama tambahakan attribute id kepada element:

<p id="p01">I am different</p>

Lalu definisikan sebuah style kepada element dengan spesifikasi id:

Contoh

Input:

<!DOCTYPE html>

<html>
<head>
<style>
#p01  {    
color:blue;
}
</style>
</head>
<body>

<p> This is a paragraph </p>
<p> This is a paragraph </p>
<p id="p01"> I am diferrent</p>

</body>

</html>

The class Attribute

Untuk mendefinisikan sebuah style kepada type special (class) dari element, tambahkan sebuah attribute class kepada element:

<p class="error">I am different</p>

Sekarang kamu bisa mendefinisikan style yang berbeda kepada element dengan spesifikasi class:

Contoh

Input:

<!DOCTYPE html>

<html>
<head>
<style>
.error  {    
color:blue;
}
</style>
</head>
<body>

<p> This is a paragraph </p>
<p> This is a paragraph </p>
<p class="error"> I am diferrent</p> p> This is a paragraph </p>
<p class="error"> I am diferrent too</p>

</body>

</html>

Gunakanlah id untuk styling element tunggal. Gunakanlah class untuk element yang ingin styling sama.

Ringkasan Materi

  • Gunakanlah attribute style HTML untuk inline styling (styling dalam sebuah element)
  • Gunakanlah element <style> HTML untuk mendefinisikan internal CSS (styling dalam sebuah dokumen HTML)
  • Gunakanlah element <link> HTML untuk mengarah ke sebuah external CSS file (styling diluar dokumen HTML)
  • Gunakanlah element <head> HTML untuk menyimpan element <style> dan <link>
  • Gunakanlah properti color CSS untuk pewarnaan teks
  • Gunakanlah properti font-family CSS untuk jenis bentuk huruf
  • Gunakanlah properti font-size CSS untuk mangatur ukuran teks
  • Gunakanlah properti border CSS untuk mendefinisikan border
  • Gunakanlah properti padding CSS untuk spasi didalam border
  • Gunakanlah properti margin CSS untuk spasi diluar border
