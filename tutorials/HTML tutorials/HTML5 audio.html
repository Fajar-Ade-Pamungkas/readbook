HTML5 Audio

HTML5 menyediakan sebuah stabdar untuk memainkan file audio.

Audio on the Web

Audio pada Web

Sebelum HTML5, Tidak ada standar untuk memainkan file audio pada sebuah web page.

Sebelum HTML5, file audio hanya bisa dimainkan dengan sebuah plug-in (seperti flash).

Element <video> HTML5 menentukan sebuah cara standar untuk menanamkan sebuah audio dalam web page.

The HTML <audio> Element

Untuk menampilkan sebuah audio dalam HTML, gunakanlah element <audio>:

Contoh

Input:

<!DOCTYPE html>
<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<audip controls>
  <source src="audio.mp3" type="audio/mpep">
  <source src="audio.ogg" type="audio/ogg">
Your browser does not support the audio tag.
</audio>

</body>
</html>

How it Works

Bagaimana Audio Bekerja.

Attribute controls menambahakan kontrol audio, seperti play, pause, volume.

Teks diantara tag <audio> dan </audio> hanya akan ditampilkan dalam browser yang tidak men-support element <video>.

Element <source> bisa me-link ke file audio yang berbeda. Browser akan menggunakan format yang diakui pertama (first recognized format).

HTML <audio> Autoplay

Untuk memulai sebuah audio secara otomatis gunakanlah attribute autoplay:

Contoh

Input:

<!DOCTYPE html>
<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<audio autoplay>
  <source src="lagu.mp3" type="audio/mpeg">
  <source src="lagu.ogg" type="audio/ogg">
Your browser does not support the video tag.
</audio>

</body>
</html>

Attribute autoplay tidak bekerja di perangkat mobile seperti iPad dan iPhone.

HTML Audio - Methods, Properties, and Events

HTML5 mendefinisikan metode DOM, dan event untuk element <audio>.

Ini mengizinkan kamu untuk load, play, dan pause audio, sebagus penyetingan durasi dan volume.

Ada juga DOM events yang bisa memberitahukan kamu ketika sebuah audio mulai dimainkan, dipause dan sebagainya.

Untuk referensi DOM penuh, silahkan kunjungi HTML5 Audio/Video DOM Reference.
