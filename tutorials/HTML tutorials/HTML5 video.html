HTML5 Video

Playing Videos in HTML

Memainkan Video di HTML

Sebelum HTML5, Tidak ada standar untuk menampilkan video pada sebuah web page.

Sebelum HTML5, video hanya bisa dimainkan dengan sebuah plug-in (seperti flash).

Element <video> HTML5 menentukan sebuah cara standar untuk menanamkan sebuah video daoam web page.

Browser Support

Nomor pada tabel menentukan browser versi pertams yang mendukung secara penuh element <video>.

BROWSER SUPPORT
Element
<video> 4.0 9.0 3.5 4.0 10.5

The HTML <video> Element

Untuk menampilkan sebuah video dalam Html, gunakanlah element <video>:

Contoh

Input:

<!DOCTYPE html>
<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
Your browser does not support the video tag.
</video>

</body>
</html>

How it Works

Bagaimana Video Bekerja.

Attribute controls menambahakan kontrol video, seperti play, pause, volume.

Ide yang bagus untuk selalu memasukan attribute width dan height.

Jika height dan width tidak diatur, browser tidak akan mengetahui ukuran video. Efeknya akan menjadi halaman akan berubah (atau flicker) selama video dimuat.

Teks diantara tag <video> dan </video> hanya akan ditampilkan dalam browser yang tidak men-support element <video>.

Element <source> bisa me-link ke file video yang berbeda. Browser akan menggunakan format yang diakui pertama (first recognized format).

HTML <video> Autoplay

Untuk memulai sebuah video secara otomatis gunakanlah attribute autoplay:

Contoh

Input:

<!DOCTYPE html>
<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<video width="320" height="240" autoplay>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
Your browser does not support the video tag.
</video>

</body>
</html>

Attribute autoplay tidak bekerja di perangkat mobile seperti iPad dan iPhone.

HTML Video - Methods, Properties, and Events

HTML5 mendefinisikan metode DOM, dan event untuk element <video>.

Ini mengizinkan kamu untuk load, play, dan pause video, sebagus penyetingan durasi dan volume.

Ada juga DOM events yang bisa memberitahukan kamu ketika sebuah video mulai dimainkan, dipause dan sebagainya.

Contoh

Input:

<!DOCTYPE html>
<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<div style="text-align:center">
<button onclick="playPause()">Play/Pause</button>
<button onclick="makeBig()">Big</button>
<button onclick="makeSmall()">Small</button>
<button onclick="makeNormal()">Normal</button>
<br><br>
<video width="320" height="240" autoplay>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
Your browser does not support the video tag.
</video>
</div>
<script>
var myVideo = document.getElementById("video1");

function playPause() {
if (myVideo.paused)
myVideo.play();
else
myVideo.pause();
}

function makeBig() {
myVideo.width = 560;
}

function makeSmall() {
myVideo.width = 320;
}

function makeNormal() {
myVideo.width = 420;
}
</script>

</body>
</html>
