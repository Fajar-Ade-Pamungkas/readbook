Contoh ini menggunakan empat element <div> untuk menciptakan multiple column layout (tata ruang kolom lebih dari satu):

Website Layout Using HTML5

HTML5 menawarkan element semantik baruyang mendefinisikan bagian - bagian beda dsri halaman web:

<header>
<nav>
<section> <aside>
<article>
<footer>
  • <header> - Mendefinisikan sebuah header untuk sebuah dokumen atau sebuah sesi
  • <nav> - Mendefinisikan suatu container untuk navigasi link
  • <section> - Mendefinisikan sebuah sesi (bagian) dalam sebuah dokumen
  • <article> - Mendefinisikan sebuah independent self-contained article
  • <aside> - Mendefinisikan konten samping (content aside) dari kontent (seperti sidebar)
  • <footer> - Mendefinisikan sebuah footer (kaki) dokumen atau sesi
  • <details> - Mendefinisikan penambahan details (rincian)
  • <summary> - Mendefinisikan heading untuk element <detail>

HTML Layout Using Tablesn

Element <table> tidak didesign untuk menjadi sebuah layout tool.
Tujuan element <table adalah untuk menampilkan tabular data.

Layout bisa dicapai menggunakan element <table>, karena element tabel bisa di-styled dengan CSS:

Contoh

HTML

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<table class="lamp">
<tr>
  <th>
    <img src="/images/lamp.jpg" alt="Note" style="height:32px;width:32px">
  </th>
  <td>
    The table element was not designed to be a layout tool.   </td>
</tr>
</table>

</body>

</html>

CSS

<style>
table.lamp {    
width:100%;     border:1px solid #d4d4d4;
}
table.lamp th, td {    
padding:10px;
}
table.lamp th {    
width:40px;
}
</style>
   
      Element tabel tidak didesign untuk menjadi sebuah layout tool (peralatan rancangan).  

Warning !

Pembuatan layout (tata ruang) dengan tabel tidaklah salah, tapi itu tidak direkomendasikan !
Jadi hindarilah tabel untuk pembuatan layout.
