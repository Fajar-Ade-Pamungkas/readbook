Attribute menyediakan informasi tambahan tentang elemen HTML.

HTML Attributes

  • Attribut HTML bisa memiliki attribute
  • Attribute menyediakan informasi tambahan tentang sebauh elemen
  • Attribute selalu ditentapkan pada start tag
  • Attribute datang dalam sepasang nama/nilai seperti: nama="nilai"

lang Attribute

Bahasa dokumen bisa dideklarasikan dalam tag <html>.

Bahasa dideklarasikan dalam attribute lang.

Pendeklarasian sebuah bahasa itu penting untuk accessibility application (pembaca layar) dan search engines (mesin pencari):

Contoh

Input:

<!DOCTYPE html>

<html lang="en-US">

<head>

<title> Judul Halaman </title>

</head>

<body>

<h1> Ini heading </h1>

<p> Ini paragraf </p>

</body>

</html>

Kata pertama dari kedua kata menentukan bahasa (en). Jika ada dialek, gunakan kata kedua (US).

title Attribute

Paragraf HTML didefinisikan dengan tag <p>.

Pada contoh ini, elemen <p> memiliki attribute tiitle. Nilai daru attribute adalah "About Readbook"

Contoh

Input:

<!DOCTYPE html>

<html lang="en-US">

<head>

<title> Judul Halaman </title>

</head>

<body>

<p title="About W3Schools">
W3Schools is a web developer's site.
It provides tutorials and references covering
many aspects of web programming,
including HTML, CSS, JavaScript, XML, SQL, PHP, ASP, etc.
</p>

</body>

</html>

W3Schools is a web developer's site. It provides tutorials and references covering many aspects of web programming, including HTML, CSS, JavaScript, XML, SQL, PHP, ASP, etc.

Ketika kamu mengerakkan mouse diatas element, title akan ditamoilkan sepeeti sebuah tooltip (alat tip).

href Attribute

Link HTML didefinisikan dengan tag <a>. Alamat link ditentukan dalam attribute href:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<a href="HTML HOME.html"> Readbook.co.id </a>

</body>

</html>
Readbook.co.id

Kamu akan mempelajari link dan rag <a> pada lain chapter.

Size Attribute

Gambar HTML didefinsikan dengan tag <img>.

nama file dari sumber atau source (src), dan ukuran gambar (widt dan height) semua disediakan sebagai attrbite:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<img src="../../pictures/Rain Fall.png" alt="Rainly" width="100" height="100">

</body>

</html>

Ukuran gambar ditentukan dalam satuan pixel: width="100" berarti 100 pixel lebar layar.

Kamu akan mempelajari lebih banyak tentang gambar dan tag <img> di tutorial ini.

alt Attribute

Attribute alt menentukan sebuah teks alternatif untuk digunakan, ketika sebuah elemen HTML tidak bisa ditampilkan.

Nilai dari attribute bisa disediakan oleh "screen reader". Cara jni, sesorang "mendengarkan" webpage, sebagai contoh seorang tunanetra, bisa "mendengar" element.

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<img src="../../pictures/Rain Fall.png" alt="Rainly" width="100" height="100">

</body>

</html>

Kita Sarankan: Gunakanlah Selalu Attribute Lowercase

Standar HTML5 tidak mengharuskan attribute nama dengan lowercase (huruf kecil).

Attribute title bisa ditulis dengan upper atau lower case seperti Title dan atau TITLE.

Kita rekomendasikan lowercase di HTML4, dan dituntut lowercase untuk tipe dokumen akurat seperti XHTML.

Lowercase paling umum digunakan. Lowercase lebih mudah untuk tipe.

Di Readbook kita selalu gunakan nama attribut lowercase.

Kita Sarankan: Selalu Kutipkan Nilai Attribute

Standar HTML5 tidak mengharuskan menaruh nilai attribute diantara tanda kutp.

Attrivute href, ditunjukan keatas, bisa ditulis seperti:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<a href="HTML HOME.html"> Readbook.co.id </a>

</body>

</html>
Readbook.co.id

Kita rekomendasikan Kutipkan pada HTML4 dan dituntut untuk tipe dokumen yang lebih akurat seperti XHTML.

Terkadang ini dibutuhkan untuk menggunakan kutip. Ini tidak akan menampilkan secara benar, karena ini berisi sebuah spasi:

Contoh

Input:

<!DOCTYPE html>

<html lang="en-US">

<head>

<title> Judul Halaman </title>

</head>

<body>

<p title=About W3Schools>
W3Schools is a web developer's site.
It provides tutorials and references covering
many aspects of web programming,
including HTML, CSS, JavaScript, XML, SQL, PHP, ASP, etc.
</p>

</body>

</html>

W3Schools is a web developer's site. It provides tutorials and references covering many aspects of web programming, including HTML, CSS, JavaScript, XML, SQL, PHP, ASP, etc.

Selalu sarankan selalu gunakan kutip untuk nilai attibute. Menghilangkan kutip akan mengeluarkan sebuah error.

Single atau Double Qoutes ?

Doubelquote merupakan yang paling umum pada HTML, tapi single quote bisa juga digunakan.

Pada beberapa situasi, ketika nilai attribute itu sendiri berisikan double quotes, ini diperlukan untuk menggunakan single quotes:

<p title='John "Menembak" seekor Angsa'>

Atau

<p title="John 'Menembak' seekor Angsa">

Chapter Summary atau Ringkasan Bab

  • Semua elemen HTML bisa memiliki Attribute
  • Attribute title HTML menyediakan Informasi "tooltip" tambahan
  • Attribute href menyediakan informasi alamat untuk link
  • Attribute width dan height HTML menyediakan informasi ukuran untuk Gambar
  • Attribute alt HTML menyediakan teks untuk screen reader
  • Di Readbook kita selalu gunakan lowercase untuk nama attribute HTML
  • Di Readbook kita selalu gunakan quote untuk nilai attribute dengan doubel quotes

HTML Attributes

Dibawah jnj adalah sebuah daftar alfabet dari beberapa attribute yang serinhkali digunakan pada HTML:

Attribute Description
alt Menentukan sebuah alternative text untuk sebuah gambar
disabled Menentukan sebauh element input harus dilumpuhkan atau dinonaktifkan
href Menentukan URL (alamat web) untuk sebuah link
id Menentukan sebuah id unik untuk sebuah element
src Menentukan URL (alamat web) untuk sebuah gambar
style Menentukan sebuah inline CSS stule untuk sebuah element
title Menentukan informasi extra tentang sebuah element
