HTML Comments

Tag comment <!-- dan --> digunakan untuk menyisipkan komentar pada HTML.

HTML Comment Tags

Kamu bisa menambahkan komentar untuk sumber HTML kamu dengan menggunakan sintaksis berikut:

<!-- Tulislah comment kamu disini -->

Perhatian: Disana da sebuah tanda seru atau exclamation point (!) pada pembukaan tag, tapi tidak di penutupan tag.

Comment tidak ditamoilkan oleh browser, tapi mereka bisa membantu dokument HTML kamu.

Dengan comment kamu bisa menempatkan pemberitahuan atau keterangan dan peringatan pada HTML kamu.

Contoh

Input:

<!-- This is a comment -->

<p>This is a paragraph.</p>

<!-- Remember to add more information here -->

This is a paragraph.

Comment juga hebat untuk debugging HTML, karena kamu bisa comment out garis HTML kode, satu waktu, untuk mencari bagian yang error:

Contoh

Input:

<!-- Jangan tampilkan ini
<img border="0" src="pic_mountain.jpg" alt="Mountain">
-->

Conditional Comments

Kamu mungkin secara tidak sengaja "conditional comments" pada HTML.

<!--[if IE 8]>
    .... some HTML here ....
>

Conditional comment mendefinisikan tag HTML untuk dieksekusi hanya dengan Internet Explorer.

Software Program Tags

Tag comment HTML bisa juga dihasilkan dengan berbagai cara (various) program software HTML.

Untuk contoh tag <webbot bot--> dilapisi atau dibalut didalam comment HTML dengan FrontPage dan Expression Web.

Sebagai suatu aturan, membiarkan tag ini stay (tetap ditempat), untuk membantu mendukung software yang dibuat mereka.
