HTML5 Web Worker

Web worker adalah sebuah JavaScript yang berjalan dalam background, tanpa mempengaruhi kinerja dari halaman.

What is a Web Worker?

Apa itu Web Worker ?

Ketika pengeksekusian script dalam sebuah halaman HTML, Halaman menjadi tidak merespon sampai script selesai.

Web worker adalah JavaScript yang berjalan dalam background, secara bebas tentang script lain, tanpa mempengaruhi kinerja dari halaman. Kamu bisa melanjutkan tuk melakukan apapun yang kamu mau: clicking, selecting things, etc., saat web worker berjalan di background.

HTML Web Workers Example

Contoh Web Worker HTML

Contoh dibawah ini membuat sebuah web worker sederhana yang menghitung nomor dalam background:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

<script>
var w;

function startWorker() {
if(typeof(Worker) !== "undefined") {
if(typeof(w) == "undefined") {
w = new Worker("demo_workers.js");
}
w.onmessage = function(event) {
document.getElementById("result").innerHTML = event.data;
};
} else {
document.getElementById("result").innerHTML = "Sorry, your browser does not support Web Workers...";
}
}

function stopWorker() {
w.terminate();
w = undefined;
}
</script>
</head>

<body>

<p>Count numbers: <output id="result"></output></p>
<button onclick="startWorker()">Start Worker</button>
<button onclick="stopWorker()">Stop Worker</button>

<p><strong>Note:</strong> Internet Explorer 9 and earlier versions do not support Web Workers.</p>

</body>

</html>

Check Web Worker Support

Periksalahlah Web Worker Support

Sebelum pembuatan sebuah web worker, periksalah apakah browser pengguna mendukung web worker:

if(typeof(Worker) !== "undefined") {
    // Yes! Web worker support!
    // Some code.....
} else {
    // Sorry! No Web Worker support.. }

Create a Web Worker File

Buatlah sebuah File Web Worker

Sekarang, ayo buat web worker kita dalam sebuah external JavaScript.

Disini, kita membuat sebuah script yang menghitung. Script disimpan dalamfile "demo_workers.js":

var i = 0;

function timedCount() {
    i = i + 1;
    postMessage(i);
    setTimeout("timedCount()",500);
}

timedCount();

Bagian penting dari code diatas adalah metode postMessage() - yang mana digunakan untuk me-post pesan balik ke HTML page.

Note: Secara normal web worker digunakan untuk script sederhana seperti itu, tapi untuk lebih CPU intensive taks.

Create a Web Worker Object

Membuat Object Web Worker

Sekarang kita memiliki file web worker, lalu kita perlu memanggilnya dari sebuah HTML page.

Baris berikut mengecek jika worker sudsh tersedia, Jika tidak - ia membuat sebuah object web worker baru dan menjalankan code di "demo_workers.js":

if(typeof(w) == "undefined") {    
w = new Worker("demo_workers.js");
}

Lalu kita bisa mengirim dan menerima pesan dari web worker.

Tambahkan event listener "onmessage" ke web worker.

w.onmessage = function(event){
    document.getElementById("result").innerHTML = event.data;
};

Ketika web worker memposting sebuah pesan, kode dengan event listener dieksekusi. Data dari web worker disimpan di event.data.

Terminate a Web Worker

Mengakhiri sebuah Web Worker

Ketika object web worker dibuat, ia akan meneruskan ke listen for message (bahkan setelah external script selesai) sampau ia diakhiri.

Untuk mengakhiri web worker, dan free browser/computer resource, gunakanlah metode terminate():

w.terminate();

Reuse the Web Worker

Gunakan kembali Web Worker

Jika kamu menyetel variabel worker menjadi undefined, setelah ia sudah diakhiri, kamu bisa menggunakn kembali kode:

w = undefined;
