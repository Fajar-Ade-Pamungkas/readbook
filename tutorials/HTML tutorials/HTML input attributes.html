The autocomplete Attribute

Attrihute autocomplete menentukan yang mana form atau input field yang harus sudah autocomplete on atau off.

Ketika autocomplete adalah on, browser secara otomatis melengkapi nilai berdasarkan nilai - niali yang pengguna telah masukan sebelumnya.

tip: Ini memungkinkan untuk memiliki autocomple "on" untuk form, dan "off" untuk spesific input field, atau sebaliknya (vice versa).

Attribute autocomplete bekerja dengan <form> dsn tipe <input> berikut: text, search, url, tel, email, password, datepicker, range dan color.

Contoh

Input:

Contoh HTML form dengan attribute autocomplete "on" dan "off" untuk salah satu input

<!DOCTYPE html>

<html>

<head>

<title> Judul Halaman </title>

</head>

<body>

<form action="action_page.php" autocomplete="on">  
First name:<br>
  <input type="text" name="firstname" value="Ade" maxlength="10"><br>
  Last name:<br>
  <input type="text" name="lastname"> <br>
  E-mail:<br>
  <input type="email" name="email" autocomplete="off"> <br>
<input type="submit">
</form>

<p>Fill in and submit the form, then reload the page to see how autocomplete works.</p>
<p>Notice that autocomplete is "on" for the form, but "off" for the e-mail field.</p>
</body>

</html>
