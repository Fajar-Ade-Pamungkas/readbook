HTML Computer Code Elements

Computer Code Element

<code>
var x = 5;
var y = 6;
document.getElementById("demo").innerHTML = x + y;
</code>

HTML Computer Code Formatting

HTML normalnya menggunakan kata variabel untuk ukuran dan spasi.

Ini tidak dibutuhkan ketika menampilkan contoh tentang computer code

Element <kbd>, <samp> dan <code> semuanya mendukung fixed letter size and spacing.

HTML <kbd> For Keyboard Input

Element <kbd> mendefinisikan input keyboard (masukan keyboard):

Contoh

Input:

<kbd>File | Open...</kbd>
File | Open...

HTML <samp> For Computer Output

Element <samp> mendefinisikan sample output (sampel keluaran) dari sebuah program komputer:

Contoh

Input:

<samp>
demo.example.com login: Apr 12 09:10:17
Linux 2.6.10-grsec+gg3+e+fhs6b+nfs+gr0501+++p3+c4a+gr2b-reslog-v6.189
</samp>
demo.example.com login: Apr 12 09:10:17 Linux 2.6.10-grsec+gg3+e+fhs6b+nfs+gr0501+++p3+c4a+gr2b-reslog-v6.189

HTML <code> For Computer Code

Element <code> mendefinisikan sebuah potongan dari kode pemrograman.

Contoh

Input:

<code>
var x = 5;
var y = 6;
document.getElementById("demo").innerHTML = x + y;
</code>
var x = 5; var y = 6; document.getElementById("demo").innerHTML = x + y;

Perhatikanlah, element <code > tidak mengeyedua ekstra spasi dan garis baru.

Untuk memperbaiki ini, Kamu bisa menaruh element <code> didalam element <pre>:

Contoh

Input:

<pre>
<code>
var x = 5;
var y = 6;
document.getElementById("demo").innerHTML = x + y;
</code>
</pre> 

var x = 5;
var y = 6;
document.getElementById("demo").innerHTML = x + y;

HTML <var> For Variables

Variabel bisa menjadi variabel pada sebuah expresi matematika atau sebuah variabel pada pemrogram yang konteks:

Contoh

Input:

Einstein wrote: <var>E</var> = <var>m</var><var>c</var><sup>2</sup>.
Einstein wrote: E = mc2.
