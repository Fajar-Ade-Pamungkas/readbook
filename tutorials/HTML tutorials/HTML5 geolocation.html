HTML Geolocation APU digunakan untuk (mengetahui) lokasi posisi pengguna.

Locate the User's Position

Lokasi Posisi Pengguna

HTML Geolocation API digunakan untuk mendapatkan posisi geografis dari seorang pengguna.

Sejak itu, kita bisa berkompromi secara rahasia, posisi tidak disediakan tanpa pengguna yang mensetujuinya.

Using HTML Geolocation

Metode getCurrentPosition() digunakan untuk mendapatkan posisi pengguna.

Contoh dibawah ini mengembalikan latitude (garis lintang) dan longitude (garis bujur) tentang posisi pengguna.

Contoh

Input:

<!DOCTYPE html>
<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<p>Click the button to get your coordinates.</p>
<button onclick="getLocation()">Try It</button>
<p id="demo"></p>
<script>
var x = document.getElementById("demo");
function getLocation() {
    if (navigator.geolocation) {
        navigator.geolocation.getCurrentPosition(showPosition);
    } else {
        x.innerHTML = "Geolocation is not supported by this browser.";
    }
}
function showPosition(position) {
    x.innerHTML = "Latitude: " + position.coords.latitude +
    "<br>Longitude: " + position.coords.longitude;
}
</script>

</body>
</html>

Click the button to get your coordinates.

Contoh menjelaskan:

  • Periksalah jika Geolocation didukung oleh web browser kamu
  • Jika didukung, jalankan metode getCurrentPosition(), Jika tidak didukung, tampilkan pesan pada user.
  • Jika metode getCurrentPosition() berhasil, ia akan mengembalikan sebuah koordinat object ke fungsi yang ditentukan dalam parameter (showPosition).
  • Fungsi showPosition() mengeluarkan Latitude (garis lintang) dan Longitude (garis bujur).

Contoh diatas merupkan setiap dasar Geolocation script, dengan tidak ada error handling (penanganan error).

Handling Errors and Rejections

Penanganan Error dan Penolakan

Parameter kedua dari metode getCurrentPosition() digunakan untuk handle error. Parameter kedua menentukan sebuah fungsi untuk dijalankan jika ia gagal mendapatkan lokasi pengguna.

Contoh

Input:

<!DOCTYPE html>
<html>
<head>
<title> Judul Halaman </title>
</head>
<body>

<p>Click the button to get your coordinates.</p>
<button onclick="getLocation()">Try It</button>
<p id="demo"></p>
<script>
var x = document.getElementById("demo");
function getLocation() {
    if (navigator.geolocation) {
        navigator.geolocation.getCurrentPosition(showPosition, showError());
    } else {
        x.innerHTML = "Geolocation is not supported by this browser.";
    }
}
function showPosition(position) {
    x.innerHTML = "Latitude: " + position.coords.latitude +
    "<br>Longitude: " + position.coords.longitude;
}

function showError(error) {
    switch(error.code) {
        case error.PERMISSION_DENIED:
            x.innerHTML = "User denied the request for Geolocation."
            break;
        case error.POSITION_UNAVAILABLE:
            x.innerHTML = "Location information is unavailable."
            break;
        case error.TIMEOUT:
            x.innerHTML = "The request to get user location timed out."
            break;
        case error.UNKNOWN_ERROR:
            x.innerHTML = "An unknown error occurred."
            break;
    }
}
</script>

</body>
</html>

Location-specific Information

Informasi Lokasi Spesifik

Halaman ini menampilkan bagaimana menunjukan posisi user pada sebuah map.

Geolocation juga sangat berguna untuk informasi lokasi yang lebih spesifik, seperti:

  • Up-to-date local information
  • Penunjukan points-of-interest near the user
  • Turn-by-turn navigation (GPS)

The getCurrentPosition() Method - Return Data

Metode getCurrentPosition() - Mengembalikan Data

Metode getCurrentPosition() mengembalikan sebuah object jika ia successful (berhasil). Properti latitude, Longitude dan accuracy selalu dikembalikan. Properti lain dikembalikan jika tersedia:

Properti Mengembalikan (returns)
coords.latitude Latitude (garis lintang) sebagai nomor desimal
coords.longitude Longitude (garis bujur) sebagai nomor desimal
coords.accuracy Akurasi dari posisi
coords.altitude altitude (ketinggian) dalam (satuan / Mdpl) meter diatas permukaan laut (meters above of mean sea level)
coords.altitudeAccuracy Akurasi ketinggian dari posisi
coords.heading Heading (kepala) derajat degrees clockwise (derajat searah jarum jam) dari Utara
coords.speed Kecepatan dalam meter per detik
timestamp tanggal/waktu dari respon

Geolocation object - Other interesting Methods
