HTML Block and Inline Elements

Setiap element HTML memiliki suatu nilai tampilan default, tergantung pada tipe element itu, Nilai tampilan default untuk kebanyakan element adalah blok dn inline.

Block-level Elements

Element block-level selalu diawali pada sebuah garis baru dan mengambil penuh luas yang tersedia (merentangkan ke kiri dan ke kanan sejauh luas bidang tersebut).

Element<div> adalah element block-level.

Contoh - contoh dari element block-level:

  • <div>
  • <h1> sampai <h6>
  • <p>
  • <form>

Inline Elements

Satu element tidak diawali pada baris baru dan hanya mengambil luas sebanyak yang diperlukan.

Ini adalah sebuah element <spn> inline didalam sebuah paragraf.

Contoh - contoh dari element inline :

  • <span>
  • <a>
  • <img>

The <div> Element

element <div> adalah element block-level yang seringkali digunakan sebagai container untik element HTML lainnya.

Element <div> tidak memiliki attribute yang diperlukan, taypi style dan class umum digunakan untuk membantu men-style nya.

Ketika digunakan dengan CSS, element <div> bisa digunakan untuk style blok - blok dari konten:

Contoh

Input:

<!DOCTYPE html>
<html>
<body>

<div style="background-color:black; color:white; padding:20px;">

<h2>London</h2>

<p>London is the capital city of England. It is the most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitants.</p>
<p>Standing on the River Thames, London has been a major settlement for two millennia, its history going back to its founding by the Romans, who named it Londinium.</p>

</div>

</body>
</html>

London

London is the capital city of England. It is the most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitants.

The <span> Element

Element <span> adalah sebuah element inline yang seringkali digunakan sebagai container untuk beberapa teks saja.

Element <span> tidak memiliki attribute yang diperlukan, tapi style dan class umum digunakan untuk membantu men-style nya.

Ketika digunakan dengan CSS, element <div> bisa digunakan untuk style bagian - bagian dari teks:

Contoh

Input:

<!DOCTYPE html>

<html>

<head>

</head>

<body>

<h2> My <span style="color:red">Importan</span> Heading</h2>

</body>

</html>

My Important Heading
