JavaScript HTML DOM - Pengubahan HTML

HTML DOM memperbolehkan JavaSdript untuk mengubah konten dari elemen HTML.

Pengubahan HTML Output Stream

JavaScript bisa meenciptakan konten HTML dinamis:

Waktu :

Pada JavaScript, document.write() bisa digunakan untuk menulis secara langsung HTML output stream:

Contoh

Input

<!DOCTYPE html>
<html>
<body>

<script>
document.write(Date());
</script>

</body>
</html>

Output

Jangan pernah gunakan document.write() setelah document dimuat. Ini akan menjadi overwrite (menulis lebih) pada dokumen.

Pengubahan Konten HTML

Cara termudah untuk memodifikasi konten dari sebuah elemen HTML adalah dengan menggunakan properti innerHTML.

Untuk mengubah konten dari elemen HTML, gunakanlah sintaksis ini:

document.getElementById(id).innerHTML = HTML baru;

Contoh ini mengubah konten daru sebuah elemen <p>:

Input

<html>
<body>

<p id="demo"> Hello guys</p>

<script>
document.getElementById("demo").innerHTML = "My name is Fajar";
</script>

<p> Teks diatas mengubah dengan sebuah naskah (script). </p>

</body>
</html>

Output

Teks diatas mengubah dengan sebuah naskah (script).

Penggunaan Event

HTML DOM memperbolehkan kamu untuk mengeksekusi kode ketika sebuah event (peristiwa) terjadi.

Event dihasilkan oleh browser ketika "suatu hal terjadi" pada elemen HTML:


  • Sebuah elemen diklik on
  • Halaman telah dimuat
  • Bidang input diubah

Kamu akan mempelajari banyak tentang event pada chapter selanjutnya dari tutorial ini.

Contoh ini mengubqh style dari elemen HTML dengan id="id1", ketika pengguna mengklik sebuah button (tombol):

Input

<html>
<body>

<p id='id1'> Hello World!</p>

<script>
document.getElementById("id1").innerHTML = "My name is Fajar";
</script>

</body>
</html>

Output

Halo Dunia!

Input

<html>
<body>

<h1 id='header'>My Heading</h1>

<script>
document.getElementById("header").innerHTML = " New Heading";
</script>

</body>
</html>

Output

My Heading

Penjelasan contoh:

  • documen HTML diatas berisikan sebuah elemen<h1> dengan id="header"
  • Kita gunakan HTML DOM untuk mendapatkan elemen dengan id="header"
  • JavaScript mengubah konten (innerHTML) dari elemen tersebut.

Pengubahan Nilai dari sebuah Attribute

Untuk mengubah nilai dari sebuah attribute HTML, gunakanlah sintaksis ini:

document.getElementById(id).attribute=new value

Contoh ini mengubah nilai daru sebuah attribute src dari elemen <img>:

Input

<html>
<body>

<img id='myImage' src="cool image.jpg">

<script>
document.getElementById("myImage").src = "hey.jpg";
</script>

</body>
</html>

Output


Penjelasan contoh:

  • Dkumen HTML diatas beriian sebuah elemen <img> dengan id="myImage"
  • Kita gunakan HTML DOM untuk mendapatkan element dengan id="myImage"
  • JavaScript mengubah attribute src dari elemen tersebut dari "cool image.jpg" menjadi "Hey.jpg"
