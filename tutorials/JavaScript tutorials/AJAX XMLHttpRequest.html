AJAX - XMLHttpRequest Object

Dasar (keystone) dari AJAX adalah object XMLHttpRequest.

XMLHttpRequest Object

Semua browser modern mendukung object XMLHttpRequest.

Object XMLHttpRequest bisa digunakan untuk pertukaran data dengan web server.

Membuat sebuah XMLHttpRequest Object

Semua browser modern (Chrome, Firefox, IE7+, Edge, Safari dan Opera) memiliki built-in XMLHttpRequest object.

Sintaksis untuk pembuatan sebuah object XMLHttpRequest:

variable = new XMLHttpRequest();

Contoh

Input:

<!DOCTYPE html>
<html>
<body>

<h2>The XMLHttpRequest Objecth2>

<p id="demo">Let AJAX change this text.</p>

<button type="button" onclick="loadDoc()">Change Content</button>

<script>
function loadDoc() {
var xhttp = new XMLHttpRequest();
xhttp.onreadystatechange = function() {
if (this.readyState == 4 && this.status == 200) {
document.getElementById("demo").innerHTML = this.responseText;
}
};
xhttp.open("GET", "ajax_info.txt", true);
xhttp.send();
}
</script>
</body>
</html>

Output:

Access Across Domains

Untuk alasan keamanan, browser modern tidak mengizinkan access across domains (akses domain dari satu sisi ke sisi lain).

Ini berarti kedua halaman web dan file XML mencoba untuk memuat, harus dilokasikan pada server yang sama.

Contoh - contoh yang ada di W3Schools (web source it) semua membuka file XML yang dilokasikan pada W3Schools domain.

Jika kamu ingin menggunakan contoh diatas pada salah satu web page milik kamu sendiri, file XML yang kamu muat harus dilokasikan pada server milik kamu.

Older Browser (IE5 dan IE6)

Versi dulu dari Internet Explorer (5/6) menggunakan ActiveX object alternatif dari XMLHttpRequest object:

variable = new ActiveXObject("Microsoft.XMLHTTP");

Untuk menangani IE5 dan IE6, mengecek jika browser mendukung XMLHttpRequest, atau lainnya membuat sebuah ActiveX object:

Contoh

Input:

page ajax 2 contoh 2 <!DOCTYPE html>
<html>
<body>

<h2>The XMLHttpRequest Object</h2>

<p id="demo">Let AJAX change this text.</p>

<button type="button" onclick="loadDoc()">Change Content</button>

<script>
function loadDoc() {
var xhttp; if (window.XMLHttpRequest) {
// code for modern browsers
xhttp = new XMLHttpRequest();
} else {
// code for IE6, IE5
xhttp = new ActiveXObject("Microsoft.XMLHTTP");
} xhttp.onreadystatechange = function() {
if (this.readyState == 4 && this.status == 200) {
document.getElementById("demo").innerHTML = this.responseText;
}
};
xhttp.open("GET", "ajax_info.txt", true);
xhttp.send();
}
</script>
</body>
</html>

Metode - metode Object XMLHttpRequest

Method Description
new XMLHttpRequest() Membuat sebuah object XMLHttpRequest baru
abort() Membatalkan request saat ini
getAllResponseHeaders() Mengembalikan informasi header
getResponseHeader() Mengembalikan spesifik informasi header
open(method, url, async, user, psw) Menentukan request
  • method: the request type GET or POST
  • url: the file location
  • async: true (asynchronous) or false (synchronous)
  • user: optional user name
  • psw: optional password
send() Mengirimkan request kr server
Digunakan untuk request GET
send(string) Mengirim request ke server.
Digunakan untuk request POST
setRequestHeader() Menambahkaj sepasang label / nilai kepada header untuk dikirim

XMLHttpRequest Object Properties

Returns the response data as XML data
Property Description
onreadystatechange Mendefinisikan sebuag fungsi untuk dipanggil ketika properti readyState berubah
readyState Menahan status dari XMLHttpRequest.
  • 0: request not initialized
  • 1: server connection established
  • 2: request received
  • 3: processing request
  • 4: request finished and response is ready
responseText Mengembalikan data respon sebagi string
responseXML Mengembalikan response data sebagai dat XML
status Mengembalikan status-nomor dari sebuah request
  • 200: "OK"
  • 403: "Forbidden"
  • 404: "Not Found"
Untuk sebuah daftar lengkap silahkan kunjungi Http Messages Reference
statusText Mengembalikan status-text (e.g. "OK" or "Not Found")
