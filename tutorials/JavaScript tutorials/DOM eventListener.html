JavaScript HTML DOM - EventListener

Metode EventListener

Contoh

Tambahkanlah sebuah event listener yang menembak (fire) ketika pengguna mengklik tombol:
document.getElementById("myBtn").addEventListener("click", displayDate);

Metode addEventListener() menempelkan sebuah event handler untuk elemen tertentu.

Metode addEventListener() menempelkan sebuah event handler untuk sebuah elemen tanpa penulisan lebih event handler yang ada.

Kamu bisa menambahkan banyak event handler untuk satu elemen.

Kamu bisa menambahkan banyak event handler untuk yipe yang sama pada satu elemen, i.e two "click" events.

Kamu bisa menambahkan event listener untuk object DOM apapun bukan hanya elemen HTML. i.e window object.

Metode addEventListener() membuatnya lebih mudah untuk mengontrol bagaimana event menjangkau ke bubbling.

Ketika menggunakan addEventListener(), JavaScript dipisahkan dari HTML markup, untuk lebih mudah dibaca dsn memperbolehkan kamu tuk menambahkan event listener bahlan ketika kamu tidak mengpntrol HTML markup.

Kamu bisa denga mudah mengganti sebuah event listener dengan menggunakan metode removeEventListener.

Sintaksis

element.addEventListener(event, function, useCapture);

Parameter pertama adalah tipe dari event (seperti "click" atau"mousedown").

Parameter kedua adalah fungsi kita yang ingin memanggil ketika event terjadi.

Parameter ketiga adalah sebuah nilai boolean penentuan apakah menggunakan event bubbling atau event capturing. Parameter ini merupakan opsi.

Catatlah bahwa kamu jangan gunakan prefix "on" untuk event: Gunakanlah "click" pengganti dari "onclick".

Menambahkan sebuah Event Handler untuk sebuah Element

Contoh

Sinyal "Hello world!" ketika pengguna mengklik pada sebuah elemen:
element.addEventListener("click", function(){ alert("Hello World!"); });

Kamu bisa juga mengarah pada sebuah fungsi "nama" eksternal:

Contoh

Sinyal "Hello world!" ketika pengguna mengklik pada sebuah elemen:
element.addEventListener("click", myFunction);

function myFunction(){
alert("Hello World!");
}

Menambahkan Banyak Event Handler untuk Elemen yang Sama

Metode addEventListener() memperbolehkan kamu untuk menambah banyak event untuk elemen yang sama, tanpa penulisan lebih (overwriting) event yang suda ada:

Contoh

Sinyal "Hello world!" ketika pengguna mengklik pada sebuah elemen:
element.addEventListener( "click", myFunction);
element.addEventListener("click", mySecondFunction);

Kamu bisa menambahkan event yang berbeda tipa untuk elemen yang sama:

Contoh

Sinyal "Hello world!" ketika pengguna mengklik pada sebuah elemen:
element.addEventListener("mouseover", myFunction);
element.addEventListener("click", mySecondFunction);
element.addEventListener("mouseout", myThirdFunction);

Menambahkan sebuah Event Handler untuk Window Object

Metode addEventListener memperbolehkan kamu untuk menambahkan event listener pada object DOM HTML apapun sebagi element HTML, dokumen HTML, object window, atau object yang event pendukung , seperti xmlHttpRequest object.

Contoh

window.addEventListener("resize", function(){
    document.getElementById("demo").innerHTML = sometext;
});

Passing Parameters

Ketika melewati nilai - nilai parameter, gunakanlah sebuah "anonymous function" yang memanggil fungsi tertentu dengan parameter - parameter:

Contoh

element.addEventListener("click", function(){ myFunction(p1, p2); });

Event Bubling atau Event Capturing?

Ada dua cara Event Propagation pada HTML DOM, bubbling (penggelembunagn) dan capturing (menangkap).

Event Propagation (peristiwa perambatan / perkembangbiakan) adalah sebuah cara pendefinisian susunan elemen (element prder) ketika sebuah event terjadi. Jika kamu memiliki sebuah elemen <p> didalam sebuah <div>, dan pengguna mengklik pada elemen <p>, yang mana event "click" elemen harus ditangani pertama kali?

Pada bubbling inner (bagian dalam) kebanyakan event element ditangani pertama kali, dan lalu outer (sebelah luar): event klik elemen <p> ditangani pertama kali, lalu event klik elemen <div>.

Pada capturinyg outer (bagian luar) kebanyakan event element ditangani pertama kali, dan lalu inner (sebelah dalam): event klik elemen <div> akan ditangani pertama kali, lalu event klik elemen <p>.

Dengan metode addEventListener() kamu bisa menentukan tipe propagation (perkembangbiakan) dengan menggunakan parameter "useCapture":

addEventListener(event, function, useCapture);

Nilai default adalah false (salah), yang mana aoan menggunakan bubbking propagation, ketika nilai diset menjadi benar, event menggunakan capturing propagantion.

Contoh

document.getElementById("myP").addEventListener("click", myFunction, true);
document.getElementById("myDiv").addEventListener("click", myFunction, true);

Metode removeEventListener()

Metode removeEventListener() me-remove (mencabut) event handler yang telah dilampirkan dengan metode addEventListener():

Contoh

element.removeEventListener("mousemove", myFunction);

Browser Support

Nomor pada table menenetukan browser versi pertama yang mendukung secara penuh metode ini .

BROWSER SUPPORT
Metode
addEventListener() 1.0 9.0 1.0 1.0 7.0
removeEventListener() 1.0 9.0 1.0 1.0 7.0

Catatan: Metode addEventListener() dan removeEventListener() tidak didukung pada IE 8 dan versi sebelumnya, dan Opera 6.0 dan versi sebelumnya Bagaimanapun, untuk versi borwser tertentu, kamu bisa menggunakan metode attachEvent() untukmelampirkan sebuah event handler untuk element dan metode detachEvent() untuk mencopotnya:

element.attachEvent(event, function);
element.detachEvent(event, function);

Contoh

Cross-browser solution:
var x = document.getElementById("myBtn");
if (x.addEventListener) {                // For all major browsers, except IE 8 and earlier  

x.addEventListener("click", myFunction);

} else if (x.attachEvent) {              // For IE 8 and earlier versions
   

x.attachEvent("onclick", myFunction);

}
