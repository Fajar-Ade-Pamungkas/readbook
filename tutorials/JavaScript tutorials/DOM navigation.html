JavaScript HTML DOM - Navigation

Dengan HTML DOM, kamu bisa menavigasi node tree menggunakan node relationsips.

DOM Nodes

Pencatatan untuk W3C HTML DOM Standard, segalanya pada sebuah dokumen HTML adalah node:

  • Seluruh dokumen adalah dokumen node
  • Setiap element HTML adalah elemen node
  • Teks didalam elemen HTML adalah teks node
  • Setiap attribute HTML adalah attribut node
  • Semua comment adalah comment node

Dengan HTML DOM, semua node didalam node tree bisa diakses oleh JavaScript.

Node baru bisa dibuat, dan semua node bisa dimodifikasi atau dihapus.

Node Relationships

Node - node pada node tree memiliki hierarchical relationship (hubungan hirarki) untunk masing - masing lainnya.

Istilah parent, child dan sibling digunakan untuk mendeksripsikan hubungan.

  • Pada sebuah node tree, node yang berada diatas disebut dengan root (atau root node).
  • Setiap node memiliki persis satu parent, kecuali root (yang mana tidak memilikk parent)
  • Sebuah node bisa memiliki sebuah nomor dari children.
  • Sibling (brother and sister) adalah node dengan parent yang sama.
<html>

  <head>      
<title>DOM Tutorial</title>
  </head>

  <body>
      <h1>DOM Lesson one</h1>
      <p>Hello world!</p>
  /body>


</html>

Dari HTML diatas kamu bisa membacanya:

  • html adalah root node
  • html tidak memiliki parents
  • html adalah parent dari head dan body
  • head adalah child pertama dari html
  • body adalah child terakhir dari html

dan:

  • head memiliki child: title
  • title memiliki satu child (sebuah teks node): "DOM Tutorial"
  • body has two children: h1 dan p
  • h1 has one child: "DOM Lesson one"
  • p has one child: "Hello world!"
  • h1 and p are siblings

Navigating Between Nodes

Kamu bisa menggunakan properti node berikut untuk menavigasi diantara node dengan JavaScript:

  • parentNode
  • childNodes[nomornode]
  • firstChild
  • lastChild
  • nextSibling
  • previousSibling

Warning !

Error umum pada pemrosesan DOM ialah untuk memperkirakan sebuah elemen node ke teks konten.

Pada contoh ini: <title>Dom Tutoria</titl>, elemen node <titil> tidak berisi teks. ia berisi sebuah teks node dengan nilai "DOM Tutorial".

Nilai dari teks node bisa diakses oleh properti innerHTML node atau nodeValue.

Child Nodes dan Node Values

Pada penambahan pada properti innerHTML, kamu bisa juga menggunakan metode childNode dan NodeValue untuk mendapatkan konten dari sebuah elemen.

Contoh berikut mengoleksi nilai node dari elemen <h1> dan salinan itu dimasukan kedalam elemen <p>:

Contoh

<html>
<body>

<h1 id="intro">My First Page</h1>

<p id="demo">Hello!</p>

<script>
var myText = document.getElementById("intro").childNodes[0].nodeValue;
document.getElementById("demo").innerHTML = myText;
</script>

</body>
</html>

Pada contoh diatas, getElementById adalah sebuah metode, sedangkan childNodes dan nodeValues adalah properti.

Pada tutorial ini kita gunakan properti innerHTML. akan tetapi, pembelajaran metode diatas sangat berguna untuk mengerti struktur tree dan navigasi dari DOM.

Penggunaan properti firstChild itu sama menggunakan childNodes[0]:

Contoh

<html>
<body>

<h1 id="intro">My First Page</h1>

<p id="demo">Hello!</p>

<script>
var myText = document.getElementById("intro").firstChild.nodeValue;
document.getElementById("demo").innerHTML = myText;
</script>

</body>
</html>

DOM Root Nodes

Ada dua properti special yang memgizinkan mengakses full dokumen:

  • document.body - body dari dokumen
  • document.documentElement - full dokumen

Contoh

<html>
<body>

<p> Hello World!</h1>

<div>
<p> DOM sangat berguna !</p>
<p> Contoh ini akan mempertontonkan properti <ib> document.body</b>.</p>
</div>

<script>
alert(document.body.innerHTML);
</script>

</body>
</html>

Contoh

<html>
<body>

<p> Hello World!</h1>

<div>
<p> DOM sangat berguna !</p>
<p> Contoh ini akan mempertontonkan properti <ib> document.documentElement</b>.</p>
</div>

<script>
alert(document.documentElement.innerHTML);
</script>

</body>
</html>

Properti nodeName

Properti nodeName menentukan nama daei sebuah node.

  • nodeName merupakan read-only
  • nodeName dari sebuah element node itu sama seperti tag name
  • nodeName daei attribute node merupakan nama attribute
  • nodeName dari sebuah teks node itu selalu #teks
  • nodeName dari dokumen node itu selalu #document

node nodeName selalu berisi uppercase (huruf kapital) tag name dari sebuah elemen HTML.

Properti nodeValue

Properti nodeValue menentukan nilai dari sebuah node.

  • nodeValue untuk node - node elemen adalah undefined (tak didefinisi)
  • nodeValue untuk node - node teks adalah teks itu sendiri
  • nodeValue untuk node - node attribute adalah nilai attribut

Properti nodeType

Properti nodeType mengembalikan tipe dari node. nodeType merupakan read-only.

Tipe - tipe node paling penting adalah:

Tipe Element NodeType
Element 1
Attribute 2
Text 3
Comment 8
Document 9
