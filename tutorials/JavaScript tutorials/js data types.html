Tipe Data JavaScript

String, Number, Boolean, Array, Object.

Tipe - Tipe Data JavaScript

Variabel JavaScript dapat memegang (hold) banyak tipe data : Number, String, Object, Array, dan sebagainya.

Contoh

input

var Panjang = 155 ; //Number

var lastName = "Pamungkas" ; // String

var carName = ["BMW", "Ferarri", "RX-7", "Nissan Skyline"] ; //Array

var orang = {firstName :"Fajar", lastName:"Ade Pamungkas"} ; //Object

Konsep Dari Tipe - Tipe Data

Dalam pemrograman, tipe data merupakan sebuah konsep penting.

Untuk dapat mengoperasikan pada variabel. Ini penting untuk mengetahui sesuatu tentang tipe.

Tanpa tipe data, computer tidak dapat memecahkan masalah dengan benar.

Contoh

input

var x = 6 + "Halo";

Apa yang akan terjadi jika nilai 6 ditambah dengan nilai "Halo". Apa akan menjadi error atau akan menjadi hasil ?

JavaScript akan memperlakukan contoh diatas seperti :

Contoh

input

var x = "6" + "Halo";

!

 Ketika kamu menambahkan Number dengan String, maka JavaScript akan memperlakukan Number menjadi String.

Contoh

input

var x = 6 + "Halo";

output

JavaScript Memiliki Tipe Dinamis

JavaScript memiliki tipe dinamis (berubah). Maksudnya adalah variabel yang sama bisa digunakan sebagai tipe - tipe yang berbeda.

Contoh

input

var x; //Undefinined (belum terdefinisi)

var x = "Good luck"; // String

var x = 10; //Number

String JavaScript

String (atau teks) adalah rangkaian karakter seperti "David Beckham".

String ditulis dengan menggunakan tanda petik (quotes). Kamu bisa gunakan petik satu (') ataupun petik dua (").

Contoh

input

var x = "Halo! Apa kabar ?"; // menggunalan petik dua (double quotes)

var x = 'Halo! Apa kabar ?'; // menggunakan petik satu (single quotes)

Kamu juga bisa menggunakan tanda petik pada String. Selama tanda petik tersebut tersusun terampit secara benar.

Contoh

input

var x = "Halo! Apa kabar ? 'Kamu disana' "; // menggunakan petik satu (single quotes) didalam petik dua (double quotes)

var x = 'Halo! Apa kabar ? "Kamu disana" '; // menggunakan petik dua (double quotes) didalam petik satu (single quotes)

Hasil

Readbook

Penggunakan kedua petik ( single dan double quotes) dalam satu String.

Kamu akan mempelajari lebih lagi di lain chapter

Number JavaScript

JavaScript hanya memiliki satu tipe Number.

Number bisa ditulis dengan atau tanpa desimal (bilangan yang memiliki titik)

Contoh

input

var x = 32.00; // dengan decimal

var y = 32; // tanpa desimal

ekstra besar atau ekstra kecil bisa ditulis dengan ilmu pengetahuan (exponential atau ilmu aljabar).

Contoh

input

var x = 123e5; // 12300000

var y = 123e-5; // 0.00123

Kamu akan belajar lebih tentang number di lain chapter

Boolean JavaScript

Boolean hanya bisa memiliki 2 (dua) nilai antara true (benar) atau salah (false).

Contoh

input

var x = true;
var y = false;

Boolean sering digunakan dalam pengetesan Kondisi.

Kamu akan belajar lebih tentang Boolean di lain chapter.

Array JavaScript

Array ditulis dengan tanda kurung persegi [] (square bracket).

Item (barang atau isi) array dipisahkan dengan tanda koma.

Berikut adalah kode yang mendeklarasikan (menciptakan) sebuah array yang disebut cars, yang berisikan tiga item (nama mobil).

Contoh

input

var cars =["corvette", "BMW", "RX-8", "Nissan Skyline", "Lamborghini", "GTO Twin-Turbo"];

document.getElementById("dataTypes06").innerHTML=cars[2];

}

/script

/body

/html

Daftar nama array (array indexes) adalah 0 untuk pertama. Yang mana artinya item pertama nilainya adalah 0, item kedua nilainya adalah 1 dan seterusnya.

Operator typeof

Kamu bisa menggunakan typeof untuk mencari tipe dari sebuah variabel JavaSdript :

Contoh

input

var typeof 123; // mengembalikan Number

var typeof "Ade"; // mengembalikan String

var typeof false; // mengembalikan Boolean

var typeof {nama:"ade", kelas:"3", warna:"biru"}; // mengembalikan Object

var typeof ["merah", "biru", "kuning"]; // mengembalikan Object

output

Di JavaScript, Array merupakan tipe special dari Object. Oleh karena itu, typeof ["merah", "biru", "kuning"] akan mengembalikan ke Object.

Null

Di JavaScript, null artinya "nothing" (bukan apa apa). Andaikata (suppose) ini menjadi sesuatu yang tidak ada (doesn't exist).

Sayangnya (unfortunately), di JavaScript. tipe data null adalah sebuah object.

!

 Kamu bisa mempertimbangkannya (consider) suatu bug dalam JavaScript yang tipe null adalah sebuah object. Seharusnya atau lebih baik null
