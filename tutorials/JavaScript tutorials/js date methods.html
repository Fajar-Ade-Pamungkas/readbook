Metode Date JavaScript

Metode Date mengizinkan kamu mendapatkan dan menyeting nilai tanggal (Tahun, Bulan, Hari, Jam, Menit, Detik, dan Milidetik)

Metode Get Date

Metode Get digunakan untuk mendapatkan bagian dari sebuah Date (tanggal). Disini adalah metode get yang paling umum (secara alpabet):

Tabel Metode Get Date
Metode Deskripsi
getDate() Mendapatkan tanggal sebagai number (1-30)
getDay() Mendapatkan hari sebagai number (0-6)
getFullYear() Mendapatkan tahun dengan 4 digit number (YYYY)
getHours() mendapatkan jam (0-23)
getMilliSeconds() Mendapatkan milidetik (0-999)
getMinutes() Mendapatkan menit (0-59)
getMonth() Mendapatkan bulan (0-11)
getSeconds() Mendapatkan detik (0-59)
getTime() Mendapatkan waktu (milidetik sejak Januari 1, 1970)

Metode getTime()

metode getTime() mengembalikan number milidetik sejak 1 Januari 1970:

Contoh

input

p id="demo"/p

script

var x = new Date();
document.getElementById("demo").innerHTML=x.getTime();

/script

output

Metode getFullYear()

metode getFullYear() mengembalikan Tahun dari tanggal saat ini seperti 4 digit number:

Contoh

input

p id="demo"/p

script

var x = new Date();
document.getElementById("demo").innerHTML=x.getFullYear();

/script

output

Metode getDay()

metode getDay() mengembalikan hari dalam seminggu sebagai number (0-6)

Contoh

input

p id="demo"/p

script

var x = new Date();
document.getElementById("demo").innerHTML=x.getDay();

/script

output

!

 Di JavaScript Hari pertama (0) dalam satu pekan adalah hari minggu, Bahkan jika dibeberapa negara didunia menghitung hari dalam sepekan menjadi hari Senin

Kamu juga bisa menggunakan array dengan nama hari dan getDay() untuk mengembalikan hari menjadi sebuah nama hari.

Contoh

input

p id="demo"/p

script

var x = new Date();
var hari = ["Minggu", "Senin", "Selasa", "Rabu", "Kamis", "Jum'at", "Sabtu"];
document.getElementById("demo").innerHTML=hari[x.getDay()];

/script

output

Metode Set Date

Metode Get digunakan untuk menyeting (mengatur) bagian dari sebuah Date (tanggal). Disini adalah metode get yang paling umum (secara alpabet):

Tabel Metode Set Date
Metode Deskripsi
setDate() Menyeting tanggal sebagai number (1-30)
setDay() Menyeting hari sebagai number (0-6)
setFullYear() Menyeting tahun dengan 4 digit number (YYYY)
setHours() Menyeting jam (0-23)
setMilliSeconds() Menyeting milidetik (0-999)
setMinutes() Menyeting menit (0-59)
setMonth() Menyeting bulan (0-11)
setSeconds() Menyeting detik (0-59)
setTime() Menyeting waktu (milidetik sejak Januari 1, 1970)

Metode setFullYear()

metode setFullYear() Menyeting sebuah object date menjadi tanggal tertentu, contoh dibawah mengubah tanggal menjadi 14 januari 2020

Contoh

input

p id="demo"/p

script

var x = new Date();
x.setFullYear(2020, 0, 14);
document.getElementById("demo").innerHTML=x;

/script

output

Metode setDate()

metode setDate() Menyeting suatu tanggal dalam satu bulan (1-31)

Contoh

input

p id="demo"/p

script

var x = new Date();
x.setDate(27);
document.getElementById("demo").innerHTML=x;

/script

output

metode setDate() bisa digunakan untuk menambahkan hari atau memajukan dan memundurkan tanggal untuk sebuah tanggal.

Contoh

input

p id="demo"/p

script

var x = new Date();
x.setDate(x.getDate()+50);
document.getElementById("demo").innerHTML=x;

/script

output

!

 Jika penambahan hari, maka terjadi pergeseran bulan atau tahun, mengubahnya secara otomatis oleh object Date.

Input Dates dan Parsing Date

Jika kamu memiliki String date valid, kamu bisa menggunakan metode Date.parse() untuk mengubah tanggal menjadi milidetik:

Metode Date.parse() mengembalikan number milidetik antara tanggal saat ini dan 1 januari 1970:

Contoh

input

p id="demo"/p

script

var msec = new Date.parse("29 Feb, 2010");
document.getElementById("demo").innerHTML=msec;

/script

output

Lalu kamu bisa menggunakan number milidetik untuk mengkonversikanya menjadi Object Date.

Contoh

input

p id="demo"/p

script

var msec = Date.parse("29 Feb, 2010");
var d = new Date(msec);
document.getElementById("demo").innerHTML=d;

/script

output

Compare Dates (Tanggal Banding)

Date bisa dengan mudah dibandingkan.

Contoh berikut membandingkan tanggal sekarang dengan 14 Januari 2100:

Contoh

input

p id="demo"/p

script

var today, someday, text;
today = new Date();
someday = new Date();
someday.setFullYear(2100, 0, 14);
if(someday > today){
text="Hari ini belum 14 Januari 2100";
} else {
text="Hari ini sudah 14 Januari 2100";
}
document.getElementById("demo").innerHTML=d;

/script

output

!

 JavaScript mengihitung bulan dari 0 s/d 11. Januari adalah 0 dan Desember adalah 11.
