JavaScript Const

ECMAScript 2015

ES2015 mempernkenalkan dua kata kunci JavaScript penting: let dan const.

Vairabel - variabel yang didefinisi dengan prilaki const seperti variabel let, kecuali mereka tidak dapat di berhentikan:

Contoh

Input:

const PI = 3.141592653589793;
PI = 3.14; // Ini akan memberikan sebuah error
PI = PI + 10; // Ini juga akan memberikan sebuah error

Output:

Block Scope

Pendeklarasian sebuah variabel dengan const mirip dengan let ketika ia datang ke Block Scope.

D dideklarasikan dalam blok, pada contoh ini, tidak sama seperti x dideklarasi diluar blok:

Contoh

Input:

var x = 10; // Disini x adalah 10
{
const x = 2 // Disini x adalah
}
// Disini x adalah 10

Output:

Kamu bisa mempelajari banyak tentanh Block Scope pada chapter sebelumnya: Javaascript let.

Ditugaskan ketika Dideklarasi

Variabel const JavaScript harus ditugaskan atau diberikan nilai sebuah nilai ketika mereka dideklarasikan:

Salah

const PI;
PI = 3.14159265359;

Benar

const PI = 3.14159265359;

Bukan Konstanta Asli

Kata kunci const merupakan sedikit menyesatkan ( a little misleading).

Ini TIDAK mendefinisikan sebuah nilai konstanta. Ini mendefinsikan sebuah referensi konstanta untuk sebuah nilai.

Karena itu, kita tidak busa mengubah nilai primitif konstanta, tapi kita bisa mengubah properti - properti dari object konstanta.

Nilai Primitif

Jika kita memberikan sebuah nilai primitif (sederhana) untuk sebuah konstanta, kita tidak bisa mengubah nilai primitif tersebut:

Contoh

Input:

const PI = 3.141592653589793;
PI = 3.14; // Ini akan memberikan sebuah error
PI = PI + 10; // Ini juga akan memberikan sebuah error

Output:

Object Konstanta bisa Berubah

Kamu bisa merubah properti dari object konstanta:

Contoh

Input:

// Kamu bisa membuat sebuah object const:
const car = {type:"Fiat", model:"500", color:"white"};

// Kamu bisa merubah sebuah properti:
car.color = "red";
// Dan kamu bisa menambahkan sebuah properti:
car.owner = "Dave";

Output:

Tapi kamu TIDAK bisa menugaskan kembali sebuah sebuah object constant:

Contoh

Input:

const car = {type:"Fiat", model:"500", color:"white"};
car = {type:"Volvo", model:"EX60", color:"red"}; // ERROR

Output:

Array Konstanta bisa Berubah

Kamu busa merubah elemen - elemen dari array konstanta:

Contoh

Input:

// Kamu bisa membuat sebuah array const:
const cars = ["Saab", "Volvo", "BMW"];

// Kamu bisa merubah sebuah elemen array const:
cars.[0] = "Toyota";
// Dan kamu bisa menambahkan sebuah properti:
cars.push("Audi");

Output:

Kamu TIDAK bisa menugaskan kembali sebuah array konstanta:

Contoh

Input:

const cars = ["Saab", "Volvo", "BMW"];
cars = ["Toyota", "Volvo", "Audi"]; // ERROR

Output:

Pendeklarasian Ulang

Pendeklarasian ulang sebuah variabel var JavaScript diizikan dimanapun pada sebuah program:

Contoh

var x = 2; // Diizinkan
var x = 3; // Diizinkan
x = 4; // Diizinkan

Pendeklarasian ulang atau penugasan ulang sebuah variabel var yang sudah ada atau variabel let untuk const, pada scope yang sama atau pada blok yang sama, tidaklah diizinkan:

Contoh

var x = 2; // Diizinkan
const x = 2; // Tidak diizinkan
{
let x = 2;// Diizinkan
const x = 2; // Tidak diizinkan
}

Pendeklarasian ulang atau penugasan ulang sebuah variabel const yang sudah ada pada scope yang sama atau pada blok yang sama, tidaklah diizinkan:

Contoh

const x = 2; // Diizinkan
const x = 3; // Tidak diizinkan
x = 3;// Tidak diizinkan
var x = 2;// Tidak diizinkan
let x = 2; // Tidak diizinkan

Pendeklarasiam sebuah variabel tanpa const, di sebuah scope lainnya atau di sebuah blok lainnya, adalah diizinkan:

Contoh

const x = 2; // Diizinkan
{
const x = 3; // Diizinkan
}
{
const x = 4; // Diizinkan
}

Hoisting

Variabel yang dideklarasikan dengan var bisa di-hoist (dinaikan) keatas. (Hoisting)

Kamu bisa menggunakan variabel var sebelhm variabel itu dideklarasi:

Contoh

carName = "Volvo"; // Kamu BISA menggunakan carName disini
var carName;

Variabel yang didefinisi dengan const tidak bisa di-hoist (dinaikan) ke atas.

Variabel const tidak bisa digunakan sebelum variabel itu dideklarasikan:

Contoh

carName = "Volvo"; // Kamu TIDAK BISA menggunakan carName disini
const carName = "Volvo";
