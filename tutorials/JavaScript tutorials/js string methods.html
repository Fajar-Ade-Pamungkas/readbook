Metode String JavaScript

Metode string bisa membantu kamu dalam berkerja dengan String.

Penemuan String Dalam String

Contoh Sederhana

Apakah Kamu Memperhatikan ?

Kedua metode search() dan indexof() adalah sama.

mereka menerima atau meyetujui (accept) argumen yang sama maka merekapun aka mengeluarkan hasil yang sama.

Kedua metode itu sama, tapi untuk metode search() bisa mengambil lebih banyak kekuatan nilai cari.

Kamu akan mempelajari lebih banyak tentang powerful search values di chapter tentang regular expression.

Pemisahan Bagian dalam String

Disini ada 3 metode pemisahan suatu bagian dari sebuah string:

  • slice(start,end)
  • substring(start,end)
  • substr(start,length)

Metode slice()

slice() mengekstrak suatu bagian dari sebuah string dan mengembalikan bagian pisahan dalam sebuah string baru.

Metode ini membawa 2 parameter: index awal (posisi) dan index akhir (posisi).

Contoh ini memotong suatu porsi string dari posisi 6 sampai posisi 12.

Contoh Sederhana

buah ="Apel, Mangga, Jeruk, Anggur";
posisi = buah.slice(6,12);

Pengaksesan String Sebagai Sebuah Array Itu Tidak Aman

Kamu mungkin sudah pernah melihat kode sepertu ini, pengaksesan string sebagai sebuah Array.

Contoh Sederhana

kata1="Hello Javascript";
kata1[0];

Penggunaan tersebut tidak aman dan tidak dapat diramal

  • Cara itu tidak bekerja disemua browser (tida di IE 5, IE 6, IE7).
  • Membuat string terlihat seperti array (tetapi mereka bukan array).
  • kata1[0] = "H" tidak memberikan error (tapi dapat berkerja).
