String JavaScript

String secara sederhana menyimpan sebuah susunan karakter sepertu "Deja vu of Devil".

String bisa menjadi teks apa saja didalam tanda petik (quotes). kamu bisa gunakan single atau double quotes.

Contoh Sederhana

carName = "Gallardo 3000";
carName = 'Gallardo 3000';

Kamu bisa menggunakan tanda petik (quotes) didalam string. selama mereka tidak berantakan atau saling melingkupi.

Contoh Sederhana

carName = "Gallardo '3000' ";
carName = 'Gallardo "3000" ';

String Length

Panjang dari sebuah string bisa ditemukan dengan properti length.

Contoh Sederhana

txt = "AHAKSMDBDKSLSNSJS";
lght= txt.lenght;

Karakter Special

Karena String harus ditulis dengan menggunakan tanda petik, JavaScript akan salah mengerti (misunderstand) String ini:

Contoh Sederhana

txt = "Ibu berkata "Kamu harus menjadi menjadi orang sukses, nak!" kepada ku.";

String bisa dipotong menjadi "we are the so-called"

. Solusi untuk menghindari masalah ini ialah dengan menggunakan \ karakter escape.

Contoh Sederhana

x = "ini \"benar\" !" txt = "Ibu berkata \"Kamu harus menjadi menjadi orang sukses, nak!\" kepada ku.";

Karakter escape \ bisa digunakan untuk menyisipkan karakter special lain didalam String.

Ini adalah daftar karakter special yang bisa ditambahkan menggunakan karakter escape \.

Tabel Event HTML
karakter Deskripsi
\' Single quote (petik satu)
\" Double quote (petik dua)
\\ Blackslash (garis miring terbalik)
\n New line (garis baru)
\r Carriage return
\t Tab (label)
\b Backspace (spasi)
\f form feed (formulir)

String Bisa Menjadi Object

Normalnya, String adalah nilai sederhana, dibuat dari Literal: var firstName="Jigol";

Tapi string bisa juga didefinisikan object dengan kata kunci new: var firstName=new String("Jigol")

Contoh Sederhana

var x = "Jigol"; //typeof x akan mengembalikan string

var y = new String("Jigol"); //typeof y akan mengembalikan object

!

 Jangan buat String menjadi Object, cara itu akan memperlambat kecepatan pengeksekusian.
Kata kunci new mempersulit kode. Cara ini dapat memproduksi hasil yang tidak terduga (unexpected).

Penggunaan operator == untuk melihat kesamaaan nilai

Contoh Sederhana

var x = "Jigol";
var y = new String("Jigol");
// (x==y) adalah benar, karena x dan y memiliki nilai yang sama

Properti Dan Metode JavaScript

Nilai sederhana, seperti "Fajar Ade Pamungkas" tidak bisa memiliki properti atau metode (karena mereka bukan merupakan object).

Tapi dengan JavaScript, mereka memiliki properti dan metode juga disediakan untuk nilai sederhana (primitive values), karena JavaScript memperlakukan nilai sederhana sebagai object, ketika pengeksekusian properti dan metode.

Metode String disampulkan di chapter selanjutnya
