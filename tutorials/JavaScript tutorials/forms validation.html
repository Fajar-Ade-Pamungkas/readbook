Formulir JavaScript

Pengesahan Formulir JavaScript

Pengesahan formulir HTML bisa dilakukan dengan JavaScript.
Jika sebuah bidang formulir (fname) kosong, fungsi ini menandakan (alert) sebuah pesan, dan mengembalikan false, untuk mengatasi formulir dari yang di-submit (diserahkan):

Contoh JavaScript

function validateForm() {
    var x = document.forms["myForm"]["fname"].value;
    if (x == null || x == "") {
        alert("Name must be filled out");
        return false;
    } }

Fungsi bisa dipanggil ketika formulir sudah diserahkan (submit):

Contoh Formulir HTML

form name="myForm" action="demo_form.asp" onsubmit="return validateForm()" method="post"
Name: input type="text" name="fname"
input type="submit" value="Submit"
/form

Output

Name:

Pengesahan formulir HTML

Pengesahan formulir HTML bisa dilakukan secara otomatis oleh browser: Jika sebuah bidang formulir (fname) kosong, attribute required mengatasi formulir ini dari yang diserahkan (submit):

Contoh Formulir HTML

form action="demo_form.asp" method="post"
input type="text" name="fname" required
input type="submit" value="Submit"
/form

Output

Name:

!

 Pengesahan formulir HTML tidak bekerja pada Internet Explorer 9 dan versi terdahulunya.

Data Validation

Pengesahan data diproses untuk memastikan (ensuring) bahwa komputer input itu bersih, benar, dan berguna.
Tugas - tugas pengesahan pengetikan adalah:

  • memiliki isian pengguna pada bidang formulir yanh diperlukan ?
  • memiliki catatan pengguna sebuah tanggal yang valid ?
  • memiliki teks catatan pengguna dalam sebuah bidang numeric ?

Paling sering, tujuan dari pengesahan data adalah memastikan input yang benar untuk sebuah pengaplikasian komputer,
Pengesahan bisa didefinisikan dengan banyak metode yang berbeda, dan disebar (deployed) dengan cara yang berbeda - beda. Server side validation dikerjakan oleh sebuah web server, setelah input sudah dikirim ke server.
Client side validation dikerjakan oleh web browser, sebwlum input dikirim menuju web server.

Pengesahan Batasan HTML (HTML Constraint Validation)

HTML5 memperkenalkan konsep pengesahan HTML baru yang disebut dengan constraint validation.
Pengesahan larangan HTML didasari pada:

  • Larangan pengesahan HTML Input Attributes
  • Larangan pengesahan CSS Pseudo Selectors
  • Larangan pengesahan DOM Properties and Methods

Larangan pengesahan HTML Input Attributes

Attribute Description
disabled Menentukan bahwa element input harus dilumpukan (disabled)
max Menentukan nilai maksimum dari sebuah element input
min Menentukan nilai minimal dari sebuah element input
pattern menentukan pola nilai (value pattern) dari sebuah element input
required menentukan bahwa bidang input memerlukan sebuah elemen
type Menentukan tipe dari sebuah element input

Untuk daftar lengkap, silahkan kunjungi: HTML Input Attribute.

Larangan Pengesahan CSS Pseudo Selectors

Selector Description
:disabled Menyeleksi element - element input dengan attribut "disable" yang ditentukan
:invalid Menyeleksi elemen - elemen input denga nilai - nilai yang tidak valid
:optional Menyeleksi elemen - elemen input dengan tanpa attribute "required" yang ditentukan
:required Menyeleksi elemen - elemen input dengan attribute "required" yang ditentukan
:valid Menyeleksi elemen - elemen input dengan nilai yang valid

Untuk daftar lengkap, silahkan kunjungi CSS Pseudo Selector
