JavaScript Sorting Arrays

Sorting an Array

Metode sort() bisa digunakan untuk mengurutkan array secara alpabet:

Contoh

input

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.sort(); // Mengurutkan elemen - elemen fruits

output

Reversing an Array

Metode reverse() membalikan urutan elemen - elemen dalam array.
Kamu bisa menggunakan untuk mengurutkan array secara terbalik.

Contoh

input

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.sort(); // Pertama mengurutkan elemen - elemen fruits fruits.reverse(); // Kemudian membalikkan susunan elemen - elemen fruits (secara terbalik)

output

Numeric Sort

Dengan default, fungsi sort() mengurutkan array sebagai String.
Ini berkerja dengan baik untuk String ("Apple" urutan sebelum "Banana").
Biarpun, jika Number diurutkan sebagai String , "25" lebih besar daripada "100", karena "2" lebih besar daripada "1".
Oleh karena itu, metode sort() akan mengeluarkan hasil yang salah ketika pengurutan Number.
Kamu bisa memperbaiki ini dengan penyediaan sebuah fungsi perbandingan:

Contoh

input

var points = [40, 100, 1, 5, 25, 10];
points.sort(function(a, b){return a - b})

output

Gunakanlah trik yang sama untuk mengurutkan sebuah array menurun (Array descending):

Contoh

input

var points = [40, 100, 1, 5, 25, 10];
points.sort(function(a, b){return b - a})

output

Fungsi Banding

Tujuan dari fungsi banding adalah untuk mendefinisikan sebuah alternatif pengurutan suatu susunan.
Fungsi banding harus mengembalikan sebuah nilai negatif, nol, atau positif tergantung pada pernyataan:

Rumus Fungsi Banding

points.sort(function(a, b){return a - b})

Ketika fungsi sort() membandingkan dua nilai, ia akan mengirimkan nilai - nilai kepada fungsi banding dan mengurutkan nilai - nilai berdasarkan kepada pengembalian nilai (negatif, nol, positif).
Contoh:
Ketika perbandingan dan 100, metode sort() memanggil fungsi perbandingan function(40,100).
Fungsi mengkalkulasikan 40 - 100, dan mengembalikan -60 (sebuah nilai negatif).
Fungsi sort akan mengurutkan 40 sebagai sebuah nilai yang lebih kecil daripada 100.
Kamu bisa menggunakan potongan (snippet) kode inj untuk eksperimen dengan pengurutan secara numeric (angka) dan secara alpabet (huruf):

Contoh

input

button onclick="myFunction1()" Sort Alphabetically/button
button onclick="myFunction2()" Sort Numerically/button

p id="demo"/p

script
var points = [40, 100, 1, 5, 25, 10];
document.getElementById("demo").innerHTML = points;

function myFunction1() {
points.sort();
document.getElementById("demo").innerHTML = points;
}
function myFunction2() {
points.sort(function(a, b){return a - b});
document.getElementById("demo").innerHTML = points;
}
/script

output

Pengurutan sebuah Array dalam Susunan Acak

Contoh

input

var points = [40, 100, 1, 5, 25, 10];
points.sort(function(a, b){return 0.5 - Math.random()})

output

Mencari Nilai Array Terbesar (atau Terkecl)

Tidak ada built-in functions untuk menemukan nilai maksimal atau minimal pada sebuah Array.
Biarpun, setelah kamu menyusun sebuah array, kamu bisa menggunakan index (daftar isi) untuk memperoleh (obtain) nilai - nilai tertinggi dan terendah.
Pengurutan susunan naik (ascending):

Contoh

input

var points = [40, 100, 1, 5, 25, 10];
points.sort(function(a, b){return a - b}); // sekarang nilai dari points[0] berisi nilai terendah
// dan nilai dari points[points.length-1] berisi nilai tertinggi

output

Pengurutan susunan turun (descending):

Contoh

input

var points = [40, 100, 1, 5, 25, 10];
points.sort(function(a, b){return b - a}); // sekarang nilai dari points[0] berisi nilai tertinggi
// dan nilai dari points[points.length-1] berisi nilai terendah

output

Pengurutan sebuah array secara keseluruhan merupakan suatu metode yang tidak efisien jika hanya ingin mencari nilai tertinggi (atau terendah).

Penggunaan Math.max() Pada sebuah Array

Kamu bisa menggunakan penerapan Math.max() untuk mencari nilai tertinggi pada sebuah array:

Contoh

input

// document.getElementById("demo").innerHTML= myArrMax(40, 100, 1, 5, 25, 10);

functionmyArrMax(Arr){
return Math.max.apply(null, Arr)
};

output

Math.max.apply([1, 2, 3]) adalah equivalent (padankata atau kata yang sama) untuk Math.max(1, 2, 3).

Penggunaan Math.min() Pada sebuah Array

Kamu bisa menggunakan penerapan Math.min() untuk mencari nilai terendah pada sebuah array:

Contoh

input

// document.getElementById("demo").innerHTML= myArrMax(40, 100, 1, 5, 25, 10);

functionmyArrMax(Arr){
return Math.min.apply(null, Arr)
};

output

Math.min.apply([1, 2, 3]) adalah equivalent (padankata atau kata yang sama) untuk Math.min(1, 2, 3).

Metode JavaScript Max dan Min Ku

Solusi tercepat yaitu dengan menggunakan metode "home made" atau metode buatan sendiri.
Fungsi ini me-loop (mengakali) sepanjang sebuah Array pembandingan setiap nilai dengan nilai tertinggi yang ditemukan:

Contoh (Mencari Nilai Tertinggi)

input

function myArrayMax(arr) {
var len = arr.length
var max = -Infinity;
while (len--) {
if (arr[len] > max) {
min = arr[len];
}
}
return max;
}

output

Fungsi ini me-loop (mengakali) sepanjang sebuah Array pembandingan setiap nilai dengan nilai terendah yang ditemukan:

Contoh (Mencari Nilai Terendah)

input

function myArrayMax(arr) {
var len = arr.length
var min = Infinity;
while (len--) {
if (arr[len] < min) {
min = arr[len];
}
}
return min;
}

output

Pengurutan Array Object

Array JavaScript sering kali berisikan object - object:

Contoh

var cars = [
{type:"Volvo", year:2016},
{type:"Saab", year:2001},
{type:"BMW", year:2010}
];

Bahkan jika objet memiliki properti - properti dari tipe - tipe data yang berbeda, metode sort() bisa digunakan untuk mengurutkan array.
Solusinya adalah dengan menulis fungsi banding untuk membandingkan nilai - nilai properti:

Contoh

input

cars.sort(function(a, b){return a.year - b.year)

output

Perbandingan properti - properti String sedikit lebih rumit:

Contoh

input

cars.sort(function(a, b){
var x = a.type.toLowerCase();
var y = b.type.toLowerCase();
if (x < y) return -1;}
if (x > y) { return 1;}
return 0;
});

output
