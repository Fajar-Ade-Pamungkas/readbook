JavaScript Object Accessors

JavaScript Accessors (Getters and Setters)

ECMAScript 5 (2009) memperkenalkan (Penyerap dan Pengatur).

Getter dan Setter mengizinkan kamu untuk mendefinisikan Object Accessor (Computed Properties).

JavaScript Getter (Kata kunci get)

Contoh ini menggunakan sebuah properti lang untuk get (mendapatkan) nilai dari properti language.

Contoh

Input:

// Buatlah sebuah Object
var person = {
firstName : "David",
lastName : "John",
language : "en",
get lang() {
return this.language;
}
};
// Tampilkan data dari object menggunakan sebuah penyerap (getter)
document.getElementById("demo").innerHTML = person.lang;

Output:

JavaScript Setter (Kata kunci set)

Contoh ini menggunakan sebuah properti lang untuk set (mengatur) nilai dari properti language.

Contoh

Input:

var person = {
firstName : "David",
lastName : "John",
language : "",
set lang(lang) {
this.language = lang;
}
};
// atur sebuah properti objet dengan menggunakan setter:
person.lang = "en";
// Tampilkan data dari object:
document.getElementById("demo").innerHTML = person.language;

Output:

JavaScript Fungsi atau Getter?

Apa perbedaan antara kedua contoh berikut?

Contoh1

Input:

var person = {
firstName : "David",
lastName : "John",
fullName :
function() {
return this.firstName + " " + this.lastName;
}
};
// Tampilkan data dari object dengan menggunakan metode:
document.getElementById("demo").innerHTML = person.fullName();

Output:

Contoh 2

Input:

var person = {
firstName : "David",
lastName : "John",
get fullName() {
return this.firstName + " " + this.lastName;
}
};
// Tampilkan data dari object dengan menggunakan geter (penyerap):
document.getElementById("demo").innerHTML = person.fullName;

Output:

Contoh 1 mengakses fullName sebagai sebuah fungsi: person.fullName().

Contoh 1 mengakses fullName sebagai sebuah properti: person.fullName.

Contoh kedua menyediakan sintaksis yang lebih sederhana.

Data Quality

JavaScript bisa mengamankan lebih baik kualitas data ketika menggunakan getter dan setter.

Penggunaan properti lang, pada contoh ini, mengembalikan nilai dari properti language dalam uppercase (huruf kapital):

Contoh

Input:

// Buatlah sebuah Object
var person = {
firstName : "David",
lastName : "John",
language : "en",
get lang() {
return this.language.toUpperCase();
}
};
// Tampilkan data dari object menggunakan sebuah penyerap (getter)
document.getElementById("demo").innerHTML = person.lang;

Output:

Penggunaan properti lang, pada contoh ini, menyimpan sebuah nilai upper case dalam properti language:

Contoh

Input:

var person = {
firstName : "David",
lastName : "John",
language : "",
set lang(lang) {
this.language = lang.toUpperCase();
}
};
// atur sebuah properti object dengan menggunakan setter:
person.lang = "en";
// Tampilkan data dari object:
document.getElementById("demo").innerHTML = person.language;

Output:

Kenapa Menggunakan Getter dan Setter

  • Karena getter dan setter memberikan sintaksis yang lebih sederhana
  • Karena getter dan setter mengizinkan sintaksis yang sama untuk properti dan metode
  • Karena getter dan setter bia mengamankan lebih baik kualitas data
  • Karena getter dan setter berguna untuk melakukan hal - hal behind-the-scenes (dibalik layar)

Contoh Counter (penghitung)

Contoh

Input:

var myObject = {
counter : 0,
get reset() {
this.counter = 0;
},
get increment() {
this.counter++;
},
get decrement() {
this.counter--;
},
set add(value) {
this.counter += value;
},
set substract(value) {
this.counter -= value;
}
};
// Mainkan dengan counter:
myObject.reset;
myObject.add = 5;
myObject.substract = 1;
myObject.increment;
myObject.decrement;

Output:

Object.defineProperty()

Metode Object.defineProperty() bisa juga digunakan untuk menambahkan Getter dan Setter:

Contoh

Input:

//Definisikan Object
var obj = {counter:0};

//Definisikan Setter
Object.defineProperty(obj, "reset" {
get : function () {this.counter = 0;}
});

Object.defineProperty(obj, "increment", {
get : function () {this.counter++;}
});

Object.defineProperty(obj, "decrement", {
get : function () {this.counter--;}
});

Object.defineProperty(obj, "add", {
get : function () {this.counter += value;}
});

Object.defineProperty(obj, "substract", {
get : function () {this.counter -= value;}
});

// Mainkan dengan counter:
obj.reset;
obj.add = 5;
obj.substract = 1;
obj.increment;
obj.decrement;
