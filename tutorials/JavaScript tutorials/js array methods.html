Metode Array JavaScript

Kekuatan dari Array JavaScript kepalsuan di metode array.

Pengubahan Array Menjadi String

Metode toString() JavaScript mengubah Array menjadi String (koma sebagai pemisah) dari nilai array.

Contoh

input

p id="demo"/p

script

var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon"];
document.getElementById("demo").innerHTML= animes.toString();

/script

output

Methode join() juga menggabungkan semua element array menjadi String.

kegunaanya seperti toString(), tapi kamu bisa menentukan tanda pemisahnya.

Contoh

input

p id="demo"/p

script

var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon"];
document.getElementById("demo").innerHTML= animes.join("~");

/script

output

Poping dan Pushing

Ketika kamu bekerja dengan Array, ini memudahkan untuk me-remove (mencopot) element dan menambahkan element baru.

Ini adalah penjelasa apa itu poping dan pushing.

Poping berguna untuk me-remove (mencopot) element Array, dan Pushing berguna untuk menambahkan element baru pada Array.

Poping

Metode pop() berguna untuk me-remove (mencopot) element terakhir pada Array.

Contoh

input

var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon"];
animes.pop(); // me-remove element terakhir (Doraemon) dari animes

output

Metode pop() mengembalikan nilai yang di-popped out.

Contoh

input

var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon"];
x = animes.pop(); // Nilai dari x adalah Doraemon

output

Pushing

Metode push() berguna untuk menambahkan element baru pada suatu Array (diposisi terakhir):

Contoh

input

var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon"];
animes.push("Boruto"); // Menambahkan element baru (Boruro) kedalam variabel animes

output

Metode push() mengembalikan panjang atau jumlah element dalam variabel animes.

Contoh

input

var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon"];
animes.push("Boruto"); // Menambahkan element baru (Boruro) kedalam variabel animes

output

Shifting Element (Pergeseran Elemen)

shifting sama dengan popping, perbedaanya shifting me-remove element pertama.

Metode shift() me-remove element pertama dalam Array dan menggeser posisi element sesusahnya maju kedepan.

Contoh

input

var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon"];
animes.shift(); // Me-remove element pertama (Boruro) dari variabel animes

output

Metode unshift() menambahkan element baru pada variabel animes (diposisi pertama)

Contoh

input

var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon"];
animes.unshift("Boruto", "Saint Saiyans"); // Menambahkan element baru (Boruro) kedalam variabel animes diposisi pertama

output

Metode shift() mengembalikan nama element yang telah di re-move .

Metode unshift() mengembalikan panjang atau jumlah element yang berada pada variabel tersebut.

Changing Element (Perubahan Elemen)

Element Array dapat diakses menggunakan index number.

!

 Index Array dimulai dengan 0, [0] untuk element urutan pertama, [1] untuk element urutan kedua, [2] untuk element urutan ketiga.

Contoh

input

var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon"];
animes[0] = "One Piece";
// Mengubah element pertama dari variabel animes menjadi "One Pice"

output

Properti length menyediakan satu cara mudah untuk melampirkan sebuah element baru pada Array.

Contoh

input

var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon"];
animes[animes.length] = "Boruto"; // Melampirkan "Boruto" pada variabel animes

output

Dengan menggunakan properti length kamu dapat melakukan penambahan tanpa me-remove element lainnya.

Deleting Element (Penghapusan Elemen)

Sejak Array JavaScript adalah Object. Element dapat dihapus menggunakan operator JavaScript delete.

Contoh

input

var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon"];
delete animes[0] ;
// Mengubah element pertama dari variabel animes menjadi tidak terdefinisi

output

!

 Penggunaan operator delete pada meninggalkan undefined holes (lubang tak definisi) pada array. Gunakanlah metode pop() dan shift()

Splicing an Array (Penyambungan sebuah Array)

Metode splice() bisa digunakan untuk menambahkan item baru pada sebuah Array.

Contoh

input

var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon"];
animes.splice( 2, 0, "Boruto", "Saint Saiyans") ; // Melampirkan "Boruto" pada variabel animes

output

Parameter pertama (2) mendefinisikan posisi dimana element baru harus ditambahkan (spliced in).

Parameter kedua (0) mendefinisikan berapa banyak element yang harus dicopot. Sisi lain dari parameter ("Boruto", "Saint Saiyans") mendefinisikan element yang ditambahkan.

Penggunaan splice() untuk me-Remove Element

Dengan terampil penyetingan parameter kamu bisa menggunakan splice() untuk me-remove element tanpa meninggalkan lubang atau jejak yang membekas (holes) dalam Array.

Contoh

input

var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon"];
animes.splice(0, 1) ;

output

Parameter pertama (0) mendefinisikan posisi dimana element baru harus ditambahkan.

Parameter kedua (1) mendefinisikan berapa banyak element yang harus dicopot. Sisi lain dari parameter dihilangkan berarti tidak ada element baru yang akan ditambahkan.

Sorting an Array (Penyusunan Array)

Metode sort() menyusun Array secara Alphabet.

Contoh

input

var animes = [ "Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon", "Boruto", "Bleach", "One Piece", "Fairy Tall" ];
animes.sort(); // menyusun element dari variabel animes

output

Reversing an Array (Penyusunan secara terbalik Array)

Metode reverse() mmembalikan element dalam Array.

Kamu bisa menggunakanya untuk meyusun array dalam susunan menurun dari element pertama ke element terakhir.

Contoh

input

var animes = [ "Naruto", "Dragon Ball", "Yu-Gi-Oh!", "Doraemon", "Boruto", "Bleach", "One Piece", "Fairy Tall", "Saint Saiyans" ];
animes.reverse(); // menyusun element dari variabel animes secara menurun

output

Numeric Sort (Menyusun angka)

Dengan default, fungsi sort() memilah nilai seperti String.

Ini bekerja dengan baik untuk String ("Boruto" datang setelah "Naruto").

Biarpun, jika number dipilah sebagai String, "25" lebih besar daripada "100", karena 1 lebih besar dari pada 2.

Oleh karena itu, metode sort() akan menghasilkan keluaran yang salah penyusunan Number.

Kamu bisa memperbaiki ini dengan penyediaan sebuah compare function (fungsi banding).

Contoh

input

!DOCTYPE html

html

head
titleReadbook/title
/head

body

p Klik tombol dibawah untuk menyusun Number /p

button onclick="myFunction()"Klik disini/button

script

var points = [0.1, 7, 100, 55, 70,-400] document.getElementById("demo").innerHTML=points;

function myFunction() {
points.sort(function(a,b) {return a-b}); document.getElementById("demo").innerHTML=points;
}

// menyusun number dari variabel points
/script

/body

/html

output

Atau

Contoh a > b

var points = [0.1, 7, 100, 55, 70,-400];
points.sort(function(a,b) {return a > b});

Gunakan trick yang sama untuk menyusun array secara menurun (descending)

Contoh b - a

var points = [0.1, 7, 100, 55, 70,-400];
points.sort(function(a,b) {return b - a});

Atau

Contoh b > a

var points = [0.1, 7, 100, 55, 70,-400];
points.sort(function(a,b) {return b > a});

Fungsi Banding

Tujuan dari fungsi banding untuk mendefinisikan susunan urutan alternatif.

Fungsi banding akan mengembalikan bilangan negatif, nol dan positif sepanjang argument (pernyataan):

points.sort(function(a,b) {return b - a});

Ketika fungsi sort() membandingkan dua nilai, ia mengirimkan nilai pada fungsi banding dan menyusunnya menurut nilai dan mengembalikan nilai (negatif, nol dan positif).

Contoh

Ketika pembandingan 40 dan 100, metode sort() memanggil fungsi banding function(40, 100).

Lalu function mengkalkulalsikan 40 - 100 dan mengembalikan -60 (nilai negatif).

Fungsi sort akan memilah 40 sebagai nilai yang lebih kecil dibandingkan 100.

Slicing an Array (Pemotongan Array)

Metode slice() memotong potongan dari array dan memasukannya kedalam Array baru.

Ini adalah contoh memotong bagian daru array pengawalanya dari array element 1 (MU).

Contoh

input

var championShip = [
"Man City",
"MU",
"Liverpool",
"Barcelona",
"Real Madrid",
"Bayern Munchen",
"Dotchmund"
];
championsShip.slice(1);

output

Club yang mengikuti Championship yaitu

Metode slice() dapat mengambil 2 element dari Array (1, 3).

Methode lalu menyeleksi element depan dan element memotong element belakang dari parameter (1, 3).

Contoh

input

var championShip = [
"Man City",
"MU",
"Liverpool",
"Barcelona",
"Real Madrid",
"Bayern Munchen",
"Dotchmund"
];
championsShip.slice(1,3);

output

Club yang mengikuti Championship yaitu

Jika kamu menghilangkan parameter kedua, maka akan menghapus element pada baris depan saja. seperti contoh pertama.
