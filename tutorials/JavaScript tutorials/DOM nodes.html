JavaScript HTML DOM Element (Nodes)

Penambahan dan pencopotan Node (elemen - elemen HTML)

Penciptaan Elemen HTML Baru (Nodes)

Untuk menambahkan sebuah elemen baru ke HTML DOM, kamu harus membuat elemen (elemen node) pertama, dan melampirkannya ke elemen yang sudah ada.

Contoh

<div id="div1">
<p id="p1"> Ini adalah paragraf </p id="p1">
<p id="p2"> Ini adalah paragraf kedua</p>
<div id="div1">

<script>

var para = document.createElement("p");
var node = document.createTextNode("This is new.");
para.appendChild(node);

var element = document.getElementById("div1");
element.appendChild(para);

</script>

Penjelasan Contoh

Kode ini membuat elemen <p> baru:

var para = document.createElement("p");

Untuk menambahkaj teks untuk elemen <p>, kamu harus membuat sebuah teks node terlebih dahulu. Kode ini membuat sebuah teks node:

var node = document.createTextNode("This is a new paragraph.");

Lalu kamu harus melampirkan teks node untuk elemen <p>:

para.appendChild(node);

Terakhir kamu harus melampirkan elmen baru untuk elemen yang ada.

Kode ini menemukan sebuah elemen yang ada:

var element = document.getElementById("div1");

Kode ini melampirkan elemen baru untuk elemen yang ada:

element.appendChild(para);

Penciptaan Elemen HTML baru - inserBefore()

Metode appendChild pada contoh sebelumnya, melampirkan elemen baru sebagai last child of parent (turunan terahir dari induk).

Jika kamu tidak ingin, kamu bisa menggunakan metode inserBefore():

Contoh

<div id="div1">
<p id="p1"> Ini adalah paragraf </p id="p1">
<p id="p2"> Ini adalah paragraf kedua</p>
<div id="div1">

<script>

var para = document.createElement("p");
var node = document.createTextNode("This is new.");
para.appendChild(node);

var element = document.getElementById("div1");
var child = document.getElementById("p1");
element.insertBefore(para,child);

</script>

Pencopotan Elemen HTML Yang Ada

Untuk mencopot sebuah elemen HTML, kamu harus tahu parent dari elemen:

Contoh

<div id="div1">
<p id="p1"> Ini adalah paragraf </p id="p1">
<p id="p2"> Ini adalah paragraf kedua</p>
<div id="div1">

<script>

var parent = document.getElementById("div1");
var child = document.getElementById("p1");
parent.removeChild(child);

</script>

Metode node.remove() dilaksanakan atau diterapkan pada DOM 4 specification.
Tapi karena kecilnya browser support, kamu lebih baik tidak menggunakannya.

Penjelasan Contoh

Dokumen HTML ini berisi sebuah elemen <div> dengan dua child node (dua elemen <p>):

<div id="div1"> <p id="p1">This is a paragraph.

<p id="p2">This is another paragraph.

</div>

Mencari elemen dengan id="div1":

var parent = document.getElementById("div1");

Mencari elemen dengan id="p1":

var child = document.getElementById("p1");

Mencopot child dari parent :

parent.removeChild(child);

Ini akan bagus untukdapat mencopot sebuah elemen tanoa mengarah pada parent.
Tapi maaf. DOM perlu mengetahui kedua elemen yang ingin kamu copot, dan parent nya.

Disini adalah sebuah workaround umum: Mencaru child yang ingin kamu copot dan memggunakan properti parentNode-nya untuk mencari parent:

var child = document.getElementById("p1");
child.parentNode.removeChild(child);

Pencopotan Elemen HTML

Untuk mencopot sebuah elemen pada HTML DOM, kita gunakan metode replaceChild:

Contoh

<div id="div1">
<p id="p1"> Ini adalah paragraf </p id="p1">
<p id="p2"> Ini adalah paragraf kedua</p>
<div id="div1">

<script>

var para = document.createElement("p");
var node = document.createTextNode("This is new.");
para.appendChild(node);

var element = document.getElementById("div1");
var child = document.getElementById("p1");
element.replaceChild(para,child);

</script>
