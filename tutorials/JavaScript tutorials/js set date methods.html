JavaScript Set Date Methods

Metode set date memperbolehkan kamu untuk mengatur nilai waktu (seperti: tahun, bulan, hari, jam, menit, detik bahkan millidetik) untuk sebuah Object Date.

Tabel Metode Set Date
Metode Deskripsi
setDate() Menyeting tanggal sebagai number (1-30)
setDay() Menyeting hari sebagai number (0-6)
setFullYear() Menyeting tahun dengan 4 digit number (YYYY)
setHours() Menyeting jam (0-23)
setMilliSeconds() Menyeting milidetik (0-999)
setMinutes() Menyeting menit (0-59)
setMonth() Menyeting bulan (0-11)
setSeconds() Menyeting detik (0-59)
setTime() Menyeting waktu (milidetik sejak Januari 1, 1970)

Metode setFullYear()

metode setFullYear() Menyeting sebuah object date menjadi tanggal tertentu, contoh dibawah mengubah tanggal menjadi 14 januari 2020

Contoh

input

p id="demo"/p

script

var x = new Date();
x.setFullYear(2020);
document.getElementById("demo").innerHTML=x;

/script

output

Tahun sudah diatur menjadi ...

Metode setFullYears() juga bisa optionally (secara pilihanmu ) mengatur bulan dan hari

Contoh

input

p id="demo"/p

script

var x = new Date();
x.setFullYear(2020, 0, 14);
document.getElementById("demo").innerHTML=x;

/script

output

Tahun, bulan dan hari sudah diatur menjadi ...

Metode setMonth()

metode setMonth() mengatur bulan dari object (date) waktu saat ini sebagai number (0 - 11):

Contoh

input

p id="demo"/p

script

var x = new Date();
document.getElementById("demo").innerHTML=x.setMonth(11);

/script

output

bulan sekarang telah diatur menjadi ...

metode setDate() mengatur tanggal dari object (date) waktu saat ini sebagai number (1 - 31):

metode setDate() bisa juga digunakan untuk menambahkan hari ke dalam tanggal (waktu):

Contoh

input

p id="demo"/p

script

var x = new Date();
document.getElementById("demo").innerHTML=x.setDate(d.getDate() + 13);

/script

output

Penambahan hari pada tanggal dapat mengeserkan bulan dan tahun sekarang, merubah ini secara otomatis ditangani oleh object date.

Perbandingan Tanggal (Date)

Date bisa secara mudah dibandingkan.
Contoh berikut membandingkan tanggal sekarang dengan 14 Januari, 2100:

Contoh

input

p id="demo"/p

script

var today, someday, text;
today = new Date();
someday = new Date();
someday.setFullYear(2100, 0, 14);
if(someday > today){
text="Hari ini belum 14 Januari 2100";
} else {
text="Hari ini sudah 14 Januari 2100";
}
document.getElementById("demo").innerHTML=d;

/script

output

JavaScript menghitung bulan dari 0 - 11. Januari dilambangkan 0 dan Desember dilambangkan 11.
