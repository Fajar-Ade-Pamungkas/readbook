Array JavaScript

Array digunakan untuk menyimpan nilai lebih dari satu dalam satu variabel.

Tampilan Array

Pada contoh ini, kita menggunakan sebuah script untuk menampilkan array dalam element paragraf dengan id="demo"


Contoh

p id="demo" /p

script
var cars =["Gallardo", "Lamborghini", "RX-8"];
document.getElementById("demo").innerHTML= cars;
/script

Baris pertama (didalam script) menciptakan sebuah array yang dinamai cars.

Baris kedua "mencari" element dengan id="demo" dan "menampilkan" array dengan "innerHTML".

Cobalah oleh Kamu Sendiri

Ciptakanlah sebuah Array dan berikan nilai untuk itu

Contoh

Input

p id="demo" /p

script
var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", 'Doraemon'];
document.getElementById("demo").innerHTML= animes[0];
/script

Output

Nilai dari penyeleksian animes[0] adalah

Spasi atau baris baru tidaklah penting, pendeklarasian bisa dirangkai menurun seperti:

Contoh

Input

p id="demo" /p

script
var animes = [
"Naruto",
"Dragon Ball",
"Yu-Gi-Oh!",
'Doraemon'
];
document.getElementById("demo").innerHTML= animes[2];
/script

Output

Nilai dari penyeleksian animes[2] adalah

!

 Jangan pernah Menaruh menaruh tanda koma pada element terakhir ("doraemon",), efeknya adalah browser tidak konsekuenan antara satu sisi dan sisi lain. (browser akan mengira bahwa ada element lain setelahnya)

Apa itu Array ?

Array adalah variabel special, dimana kamu bisa menaruh nilai lebih dari satu dalam satu waktu.

Jika kamu memiliki daftar item (daftar nama mobil, seperti contoh), penyusunan itu dalam satu variabel bisa dilihat seperti ini:

var car1 = "Gallardo"; var car2 = "Nissan"; var car3 ='Audi';

Biarpun, apakah jika kamu mau me-loop sepanjang cars dan mencari satu yang spesifik. dan apakah jika kamu tidak hanya memiliki 3 item tapi 300 item?

Solusinya adalah Array.

Sebuah array dapat menaruh banyak nilai dibawah satu nama, dan kamu bisa mengaksesnya dengan pemicu untuk nomor urut (index number)

Pembuatan Array

Penggunaan literal Array adalah cara paling mudah untuk membuat Array JavaScript.

Sintaksis:

var namaArray= ["item1", "item2", ....];

contoh:

var animals = ["Tiger", "Panda", "Koala", "sheep"];

Penggunaan Kata Kunci new JavaScript

Contoh berikut juga membuat sebuah array dan memberikan nilai pada array:

Contoh

Input

p id="demo" /p

script
var cars = new Array("Gallardo", "Nissan", 'Audi', "RX-8");
document.getElementById("demo").innerHTML= cars[3];
/script

Output

Nilai dari penyeleksian cars[3] adalah

!

 Kedua contoh diatas melakukan hal yang sama, ada yang tidak perlu menggunakan new Array().
Untuk kesederhanaan, keahlian pembacaan dan kecepatan pengeksekusian, gunakanlah satu yang utama (Metode literal Array)

Pengaksesan Elemen - elemen Array

Kamu mengarah pada element Array dengan pemicu index number (nomor urut).

Pernyataan ini mengakses element pertama dari array bernama cars.

var name = cars[0];

Pernyataan ini memodifikasi element pertama dalam cars

cars[0] = "Ferrari";

!

 [0] untuk element pertama, [1] untuk element kedua. Index number dimulai dari angka 0.

Kamu bisa memiliki Object - object Berbeda dalam Satu Array

Variabel - variabel JavaScript bisa menjadi Object. Array adalah macam - macam object special.

Karena itu, Kamu bisa memiliki variabel - variabel dengan tipe yang berbeda dalam Array yang sama.

Kamu bisa memiliki object dalam array, kamu bisa memiliki fungsi dalam array, kamu bisa memiliki array dalam array.

myArray[0] = Date.now;
myArray[1] = function;
myArray[2] = myCars;

Array Adalah Object

Array adalah tipe JavaScript yang Special, operator typeof dalam JavaScript akan mengembalikan "object" untuk Array.

Tapi, Array JavaScript dideskripsikan terbaik yaitu sebagai Array.

Array menggunakan number untuk mengakses elementnya. Pada contoh ini, person[0] mengembalikan "Fajar".

Contoh

Input

p id="demo" /p

script
var person = ["Fajar", "Ade", "Pamungkas"];
document.getElementById("demo").innerHTML= person[0];
/script

Output

Nilai dari penyeleksian person[0] adalah

Object menggunakan nama untuk mengakses "anggotanya", pada contoh ini person.middleName mengembalikan "Ade".

Contoh

Input

p id="demo" /p

script
var person = {firstName:"Fajar", middleName:"Ade", lastName:"Pamungkas"};
document.getElementById("demo").innerHTML= person.middleName;
/script

Output

Nilai dari penyeleksian person.middleName adalah

Meode dan Properti Array

Kekuatan sesungguhnya dari Array JavaScript adalah pembangunan properti dan metode Array.

var x = cars.length; //properti length mengembalikan berapa banyak anggota dalam array
var x = cars.sort(); //metode sort() menyusun array dalam susunan alpabet

Metode array di cover di chapter selanjutnya

Properti length

Properti length dari sebuah Array mengembalikan panjang sebuah array.

Contoh

Input

var saudara = [
"Andi Oka Mahendra ",
"Iqbal Abdillah ",
"Rizki Ade Pangestu ",
"Fajar Ade Pamungkas "
];
jumlahsaudara.length;

Output

Properti length mengembalikan berapa jumlah item dalam sebuah array

!

 Properti length selalu lebih satu daripada index number array tertinggi.

Penambahan Element Array

Cara paling mudah untuk menambahkan element Array yaitu menggunakan metode push():

Contoh

Input

var saudara = [
"Andi Oka Mahendra ",
"Iqbal Abdillah ",
"Rizki Ade Pangestu ",
"Fajar Ade Pamungkas "
];
tambahkan = saudara.push("Ella");

Output

Metode push() menambahkan data baru dalam sebuah array

Element baru juga bisa ditambahkan ke sebuah Array, menggunakan properti length:

Contoh

Input

var saudara = [
"Andi Oka Mahendra ",
"Iqbal Abdillah ",
"Rizki Ade Pangestu ",
"Fajar Ade Pamungkas "
];
saudara[saudara.length] = "Ella";

Output

Metode push() menambahkan data baru dalam sebuah array

Penambahan Element baru dengan index tinggi akan membuat undefined "holes" (lubang dalam array).

Contoh

Input

var saudara = [
"Andi Oka Mahendra ",
"Iqbal Abdillah ",
"Rizki Ade Pangestu ",
"Fajar Ade Pamungkas "
];
saudara[10] = "Ella";

Output

Metode push() menambahkan data baru dalam sebuah array

Pengakalan Element Array

Cara terbaik untuk me-loop sepanjang array adalah menggunakan loop for

Contoh

Input

function $arrays10(){
var index,text;
var text = ""; var fruits = ["Apel", 'Mangga', "Jeruk", "Anggur"];
for(index = 0; index < fruits.length; index++){
text += fruits[index];
}
document.getElementById("$arrays10").innerHTML=text;
}

Output

Metode push() menambahkan data baru dalam sebuah array

Associative Arrays

Banyak bahasa pemrograman yang men-support dengan index nama (named indexes).

Array dengan index nama disebut dengan associative Array (atau hashes).

JavaScript TIDAK mendukung array dengan index nama.

Dalam JavaScript, array selalu menggunakan index nomor.

Contoh

Input

var player = [];
player[0] = "Draxler";
player[1] = "De Bruyne";
player[2] = "Özil";
var x = player.length; // player.length akan mengembalikan banyak item
var y = player[2]; // player[2] akan mengembalikan item "Özil"

Output

!

Warning !!!

Jika kamu menggunakan index nama, JavaScript akan mengartikan lain menjadi Object Standar.
Setelah itu, semua metode dan properti array akan menghasilkan hasil yang keliru.

Contoh

Input

var player = [];
player["asRoma"] = "Draxler";
player["manCty"] = "De Bruyne";
player["arsenal"] = "Özil";
var x = player.length; // player.length akan mengembalikan 0
var y = player[2]; // player[2] akan mengembalikan undefined

Output

Perbedaan Antara Object dan Array

Dalam JavaScript, Array menggunakan index nomor
Dalam JavaScript, Object menggunakan index nama

!

 Array adalah jenis Object yang special, dengan index nomor.

Kapan untuk Menggunakan Array, Kapan untuk Menggunakan Object

  1. JavaScript tidak mendukung associative array
  2. Kamu lebih baik gunakan object jika kamu mau element nama menjadi string (text)
  3. kamu lebih baik gunakan Array jika kamu mau element nama menjadi b0 nomor

Hindarilah new Array()

Ada yang tidak diperlukan untuk menggunakan JavaScript's built-in Array constructor new Array().

Gunakanlah alternative []

Ini adalah dua pernyataan yang berbeda, keduanya menciptakan array baru yang kosong yang diberi nama point.

var points = new Array(); //buruk
var points = []; //baik

Ini adalah dua pernyataan yang berbeda, keduanya menciptakan array yang berisikan nilai 6 nomor.

var points = new Array(100, 22, 33, 40, 58, 96); //buruk
var points = [100, 22, 33, 40, 58, 96]; //baik

Kata kunci new hanya akan mempersulit suatu kode, dan bisa juga mengeluarkan hasil yang tidak pasti.

var points = new Array(100, 96); //membuat sebuah array dengan dua element

Apa yang akan terjadi, jika saya menghapus satu element?

Contoh

Input

var points = new Array(100); //menghapus satu element array

Output

How to Recognize an Array

Pertanyaan umum yang biasa ditanyakan: "Bagai saya tahu jika sebuah variabel adalah Array?"

Masalahnya adalah jika kita gunakan operator typeof akan mengembalikan "object".

Contoh

Input

var points = [440, 200, 10, 40, 90];
typeof points // hanya akan mengembalikan object

Output

Operator typeof akan mengembalikan object, karena Array JavaScript merupakan sebuah object.

Solusi ke 1

Untuk mengatasi masalah ini, ECMAScript 5 mendefinisikan sebuah metode baru yaitu Array.isArray().

Contoh

Input

var points = [440, 200, 10, 40, 90];
Array.isArray(points); // akan mengembalikan true (benar)

Output

Masalah dengan metode ECMAScript 5 ini yaitu metode ini tidak didukung oleh browser versi terdahulu.

Solusi ke 2

Untuk mengatasi masalah ini, kamu bisa membuat fungsi isArray() kamu sendiri.

Contoh

Input

p Ini adalah fungsi "buatan sendiri", ketika digunakan pada Array maka akan mengembalikan "true"./p

p id="demo" /p

script
var animes = ["Naruto", "Dragon Ball", "Yu-Gi-Oh!", 'Doraemon'];
document.getElementById("demo").innerHTML= isArray(animes);

function isArray(x) {
return x.constructor.String().indexOf("Array") > -1
}
/script

Output

Ini adalah fungsi "buatan sendiri", ketika digunakan pada Array maka akan mengembalikan "true".

Fungai diatas selalu mengembalikan true (benar), jika pernyataan adalah Array.

Atau lebih tepatnya, akan mengembalikan true jika object protoype nya berisikan kata "Array".

Solusi ke 3

Pengunaan operator instanceof mengembalikan true jika object dibuat dengan diberi contructor.

Contoh

Input

var points = [440, 200, 10, 40, 90];
points instanceof Array; // akan mengembalikan true (benar)

Output
