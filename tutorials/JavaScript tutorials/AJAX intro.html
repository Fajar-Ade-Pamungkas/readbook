AJAX Introduction

AJAX adalah sebuah mimpi pengembang, karena kamu bisa:

  • Membaca data dari sebuah web server - setelah halaman sufsh dimuat
  • Meng-update halaman web tanpa memuat ulang halaman
  • Mengirimkan data dari sebuah web server - dalam background

Contoh

Input:

<!DOCTYPE html>
<html>
<body>

<div id="demo">
<h2>Let AJAX change this text</h2>
<button type="button" onclick="loadDoc()">Change Content</button>
</div>

<script>
function loadDoc() {
var xhttp = new XMLHttpRequest();
xhttp.onreadystatechange = function() {
if (this.readyState == 4 && this.status == 200) {
document.getElementById("demo").innerHTML =
this.responseText;
}
}; xhttp.open("GET", "ajax_info.txt", true);
xhttp.send();
} </script>
</body>
</html>

Output:

Fungsi diatas tidak bisa ditampilkan, karena AJAX bisa diakses bila kamu memiliki sebuah web server.

Penjelasan Contoh AJAX

Halaman HTML

Apa itu AJAX ?

AJAX = Asynchronous JavaScript And XML.

AJAX bukanlah sebuah bahasa pemrograman.

AJAX hanya menggunakan sebuah kombinasi tantang:


  • Sebuah browser built-in XMLHttpRequest object (untuk meminta data dari sebuah web server)
  • JavaScript dan HTMLDOM (untuk menampilkan atau menggunakan data)
AJAX adalah sebuah nama yang menyesatkan. Aplikasi AJAX mungkin menggunakan data dengan sebuah web server, tapi ini adalah dengan kata lain biasa untuk men-tranport data seperti penjelasan teks atau teks JSON.

AJAX mempngizinkan halaman web untuk di-update secara tidak selaras oleh pertukaran data dengan web server dibelakang layar. Ini mengaetikan bahwa AJAX memungkinkan untuk meng-update bagian bagian dari halaman web, tanpa memuat ulang seluruh halaman.

Bagaimana AJAX Bekerja

Alur Request AJAX

Browser

sebuah Event terjadi:
  • Membuat sebuah object XMLHttpRequest
  • Mengirim HttpRequest
→

Internet

→

Web Server

  • Memproses HttpRequest
  • Membuat respon dan mengirim data kembalik ke browser

Browser

  • Memproses data yang dikembalikan dengan JavaScript
  • Meng-update konten halaman
←

Internet

↵
1. Sebuah event terjadi pada sebuah halaman web (halaman dimuat, sebuah button diklik)
2. Sebuah object XMLHttpRequest dibuat oleh JavaScript
3. Object XMLHttpRequest mengirim sebuah permintaan kepada web server
4. Server memproses permintaaan
5. Server mengirim sebuah respon balik kepada halaman web
6. Respon dibaca oleh JavaScript
7. Proper Action (seperti page update) dipertunjukan oleh JavaScript

Perhatian !!!

Semua contoh pada turorial AJAX ini tidak bisa menampilkan apapun, karena AJAX hanya bisa dieksekusi apabila kamu memiliki sebuah web server.

Tutorial ini hanya untuk gambaran "Apa dan bagaimana cara mengoperasikan AJAX".
