JavaScript Versions

JavaScript diciptakanoleh Brendan Eich pada tahun 1995, dan menjadi sebuah standar ECMA pada tahun 1997.
ECMAScript adalah nama Official Bahasa.
Dari 2015 ECMAScript dinamai dengan tahun (ECMAScript 2015)

ECMAScript Editions

Versi Nama Official Deskripsi
1 ECMAScript 1 (1997) Edisi Pertama.
2 ECMAScript 2 (1998) Editorial changes only.
3 ECMAScript 3 (1999) Menambahkan Regular Expressions.
Menambahkan try/catch.
4 ECMAScript 4 Tidak pernah dirilis.
5 ECMAScript 5 (2009)
Bacalah ES5 		Menambahkan "strict mode".
Menambahkan JSON support.
Menambahkan String.trim().
Menambahkan Array.isArray().
Menambahkan Array Iteration Methods.
5.1 ECMAScript 5.1 (2011) Editorial berubah.
6 ECMAScript 2015
Bacalah ES6 		Menambahkan let and const.
Menambahkan nilai - nilai parameter default
Menambahkan Array.find().
Menambahkan Array.findIndex().
7 ECMAScript 2016 Menambahkan exponential operator (**).
Menambahkan Array.prototype.includes.
8 ECMAScript 2017 Menambahkan string padding.
Menambahkan new Object properties.
Menambahkan Async functions.
Menambahkan Shared Memory.
9 ECMAScript 2018 Menambahkan rest / spread properties.
Menambahkan Asynchronous iteration.
Menambahkan Promise.finally().
Additions untuk RegExp.

ECMAScript sering kali disingkat menjadi ES

JavaScript / ECMAScript

JavaScript dikembangkan untuk NetScape. NetScape 2 adalah browser pertamauntuk menajalankan JavaScript,
Setelah Netscape, Mozilla foundation melanjutkan untuk mengembangkan JavaScript untuk browser Firefox.
Versi terakhir JavaScript ialah 1.8.5 (mirip ECMAScript 5).
ECMAScript dikembangkan oleh ECMA Internatioanl setelah Organisasi diadopsi JavaScript.
Edisi pertama dari ECMAScript dirilis pada tahun 1997.
