JavaScript HTML DOM Events

HTML DOM memperbolehkan JavaScript untuk menjangkau HTML event.

Jangkauan Menuju Event

JavaScript bisa dieksekusi ketikasebuah event terjadi, seperti ketika seorang pengguna pada sebuah elemen HTML.

Untuk mengeksekusi kode ketika seorang pengguna mengklik sebuah elemen, menambahkan kode JavaScript pada sebuah attribute event HTML:

onclick=JavaScript

Contoh dari event HTML:


  • Ketika penggunak mengklik mouse
  • Ketika sebuah halaman web telah dimuat
  • Ketika sebuah gambar sudah dimuat
  • Ketika mouse bergerak melebihi sebuah elemen
  • Ketika sebuah bidang input diubah
  • Ketika sebuah formulir HTML di-submit (dikirim)
  • Ketika seorang pengguna mencoba sebuah kode

Pada contoh ini, konten dari elemen <h1> diubah ketika penguna mengkliknya:

Input

<html>
<body>

<h1 onclick="this.innerHTML= 'Ooops!'"> Hello guys</p>

</body>
</html>

Output

HELLO WORLD!

Pada contoh ini, sebuah fungsi dipanggil dari sebuah event handler:

Input

<html>
<body>

<h3 id="changeText(this)"> Hello guys</h3>

<script>
function changeText(id){
id.innerHTML= "Oooops!";
}
</script>

<p> </p>

</body>
</html>

Output

KLIKLAH TEKS INI

HTML Event Attributes

Untuk menugaskan atau memberi event kepada elemen HTML kamu bisa menggunakan attribute (pelengkap) event:

Input

<html>
<body>

<button onclick="changeDate()"> Klik disini</button>

<p id="timeNow"> Hello guys</p>

<script>
function changeDate(){
document.getElementById("timeNow").innerHTML= Date();
}
</script>

<p> </p>

</body>
</html>

Output

Klik tombol diatas untuk melihat waktu sekarang

Pada contoh diatas, sebuah fungsi yang dinamai changeDate dieksekusi ketiks button diklik.

Pemberian Event Menggunakan HTML DOM

HTML DOM memperbolehlan kamu untuk menugaskan event untuk elemen HTML menggunakan JavaScrip:

Contoh

Input

<html>
<body>

<button id="myBtn"> Klik disini</button>

<p> Fungsi dibawah membuat button menampilkan tanggal tanpa menyertakan "onclick" pada button. </p>

<script>
var displayDate = Date();
document.getElementById("myBtn").onclick= displayDate;
</script>

<p> </p>

</body>
</html>

Output

Fungsi dibawah membuat button menampilkan tanggal tanpa menyertakan "onclick" pada button.

Pada contoh diatas, sebuah fungsi diberi nama "displayDate" yang ditugaskan untuk element HTML dengan id="myBtn".

Fungsi akan dieksekusi ketika button diklik.

Event onload dan onunload

Event onload dan onunload dicetus (trigger) ketika pengguna memasukan (enters) atau meninggalkan halaman.

Event onload bisa digunakan untuk mengecek pengunjung browser.

Event onload dan unonload bisa digunakan pada persetujuan cookies.

Contoh

Input

<html>
<body onload="checkCookies()">

<button> Klik disini</button>

<p> Fungsi dibawah membuat button menampilkan tanggal tanpa menyertakan "onclick" pada button. </p>

<script>
var displayDate = Date();
document.getElementById("myBtn").onclick= displayDate;
</script>

<p> </p>

</body>
</html>

Output

Fungsi dibawah membuat button menampilkan tanggal tanpa menyertakan "onclick" pada button.

Event onchange

Event onchange seringkali digunakan pada kombinasi dan validasi dari bidang input.

Dibawah adalah sebuah contoh bagaimana untuk menggunakan onchange. Fungsi upperCase() akan dipanggil ketika pengguna mengubah konten dari sebuah bidang input.

Contoh

Input

<html>
<body>

<input type="text" onchange="upperCase()" id="myInput" >

<button> Klik disini</button>

<script>
var displayDate = Date();
</script>

<p> </p>

</body>
</html>

Output

Event onmouseover dan onmouseout

Event onmouseover dan onmouseout bisa digunakan untuk pencetus (trigger) sebuah fungsi ketika pengguna mouse over atau out dari elemen 0 HTML:

Event onmousedown, onmouseup dan onclick

Event onmouse, onmouseup dan onclick adalah semua bagian dari a mouse-click. Pertama ketika mouse-button diklik, event onmousedown dicetuskan, lalu ketika mouse-button dirilis, event onmouseup dicetuskan, pada akhirnya ketika mouse-click telah selesai, event onclick dicetuskan. The onmousedown, onmouseup and onclick Events The onmousedown, onmouseup, and onclick events are all parts of a mouse-click. First when a mouse-button is clicked, the onmousedown event is triggered, then, when the mouse-button is released, the onmouseup event is triggered, finally, when the mouse-click is completed, the onclick event is triggered.

Contoh Selengkapnya

onmouse and onmouseup

Mengubah sebuah gambar ketika penggunak menekan tombol mouse.

onload

Menampilkan sebuah kotak sinyal ketika halaman telah selesai memuat

onfocus

Mengubah background-color dari sebuahbidang input ketika diakses.

Mouse Event

Mengubah warna dari sebuah element ketika cursor bergerak melebihnya.
