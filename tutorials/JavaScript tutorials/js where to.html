Dimana JavaScript diletakan ?

JavaScript bisa diletakkan pada bagian sesi <head> dan <body>

Tanda script

Pada HTML. JavaScript harus disisipkan diantara <script> dan </script>.

<script>

function myFunction() { <
//Pernyataan
}

</script

!

 Pada contoh terdahulu, penulisan script harus seperti <script type="JavaScript">.
Namun, Sekarang itu sudah tidak diperlukan.
karena <script> sudah menjadi default (setelan pabrik).

Fungsi dan kejadian JavaScript

Fungsi JavaScript merupakan sebuah blok dari kode javaScript, yang bisa dieksekusi dengan cara meminta nya untuk melakukan sesuatu.

Untuk contoh, sebuah fungsi dapat dieksekusi ketika sebuah event (peristiwa) terjadi.
seperti ketika pengguna mengklik sebuah tombol.

Kamu akan belajar tentang fungsi dan event di lain chapter.

JavaScript di head atau body

Kamu bisa menempatkan nomor apapun dari JavaScript dalam sebuah dokumen HTML.

script (naskah javaScript) dapat ditempatkan di head dan body. Atau di keduanya

!

 Tetaplah tempatkan script pada satu tempat. Agar dapat bekerja dengan baik.

JavaScript pada head

Pada contoh ini, fungsi javaScript di tempatkan di bagian head pada halaman HTML.

Fungsi ini dipanggil (diaktifkan) ketika sebuah button (tombol) di klik.

Contoh

input

<!DOCTYPE html>

<html>

<head>

<script>
function myFunction() {

document.getElementById(" par ").innerHTML="Hello JavaScriptor !"";

}
</script>

</head>

<body>

<h1> Readbook </h1>

<p id="demo"> Ini paragraf </p>

<button onclick="myFunction()"> Submit </button>

</body>

</html>

output

Readbook

Ini paragraf

Fungsi ini dipanggil (diaktifkan) ketika sebuah button (tombol) di klik.

input

Dimana script disimpan ?

!

 Cara diatas adalah ide yang bagus untuk menyimpan script diantara tanda body.
Cara ini dapat meningkatkan performa ketika sedang memuat halaman.
Dan kumpulan kode - kode bisa memperlambat halaman saat ditampilkan.

JavaScript dapat ditempatkan di luar file HTML

Cara ini dapat dipraktikan sebuah kode yang sama digunakan pada banyak web page yang berbeda.

File JavaScript harus memiliki format file .js

Untuk menggunakan external script (naskah diluar HTML).
Taruhlah nama file tersebut pada src (sumber) attribute didalam script tag.

!

 External script tidak berisi script.

Keuntungan JavaScript External

Penempatan JavaScript dalam External file (luar file HTML) memiliki beberapa keuntungan, yaitu:

  • Memisahkan HTML dan kode.
  • membuat HTML dan JavaScript mudah dibaca dan mengurus / memelihara.
  • menyembunyikan file JavaScript bisa mempercepat dalam memuat halaman.
