Properti Object JavaScript

Properti merupakan bagian yang sangat penting dari object JavaScript.

Sorting Array

Properti adalah nilai - nilai yang berhububgan dengan sebuah object JavaScript.
Sebuah object JavaScript adalah sebuah koleksi dari properti - properti tanpa susunan (acak).
Properti biasanya bisa dirubah, ditambahkan, dan dihapus, tapi beberapa hanya dibaca.

Pengaksesan Properti JavaScript

Sintaksisi untuk pengaksesan properti dari sebuah object adalah:

objectName.property          // person.age

atau

objectName["property"]       // person["age"]

atau

objectName[expression]       // x = "age"; person[x]

Expression harus mengevaluasi menjadi sebuah properti nama:

Contoh 1

Input

var person = {firstName : "Fajar Ade", lastName : "Pamungkas", age : 20, eyeColor : "brown"}
var x = person.firstName + " is " + person.age + " years old";

Output

Contoh 2

Input

var person = {firstName : "Fajar Ade", lastName : "Pamungkas", age : 20, eyeColor : "brown"}
var x = person["firstName"] + " is " + person["age"] + " years old";

Output

Loop for ... in JavaScript

Pernyataan for...in JavaScript menyimpulkan sepanjang properti - properti daei sebuah object:

Sintaksis
for (variable in object) {
    // kode untuk dieksekusi
}

Blok kode didalam simpul for ... in akan dieksekusi sekali untuk setiap properti.
Penyimpulan sepanjang properti dari sebuah object:

Contoh

Input

<p id="demo"> </p>
<script>

var person = {firstName : "Fajar Ade"; lastName : "Pamungkas"; age : 20; eyeColor : "brown";}
var teks = "";
var x;
for (x in person) {
teks += person[x] + " ";
}
document.getElementById("demo").innerHTML= teks;

</script>

Output

Penambahan Properti Baru Pada Object JavaScript

Kamu bisa menambahkan properti baru untuk object yang sudah ada dengan secara sederhana memberikannya sebagai sebuah nilai.
Menerima itu object person sudah ada - kamu bisa memberikannya properti baru: You can add new properties to an existing object by simply giving it a value.

Contoh

Input

var person = {firstName : "Fajar Ade"; lastName : "Pamungkas"; age : 20; eyeColor : "brown";}
var person.Nasionality = "Indonesia";

Output

Kamu tidak bisa menggunakan kata reserved words untuk penamaan properti (atau metode).

Penghapusan Properti

Kata kunci delete digunakan untuk menghapus sebuah properti dari sebuah object

Contoh

Input

var person = {firstName : "Fajar Ade"; lastName : "Pamungkas"; age : 20; eyeColor : "brown";}
delete person.age; // atau bisa delete person["age"]
var x = person["firstName"] + " is " + person["age"] + " years old";

Output

Kata kunci delete menghapus kedua nilai dari properti dan properti itu sendiri.
Setelah penghapusan, properti tidak bisa digunakan sebelum ia ditambahkan kembali.
operator delete didesign untuk digunakan pada properti - prperti object. Delete tidak memiliki efek pada variabel atau fungsi.
Operator delete tidak harus digunakan pada properti object JavaScript yang sudah dikenal. Operator delete bisa merusak aplikasi kamu.

Property Attrubutes

Semua properti memiliki sebuah nama. Pada penambahannya juga memiliki sebuah nilai.
Nilai adalah salah satu dari attribut - attribut properti.
attribut - attribute lainnya adalah enumerable, configurable, dan writable.
Attribut - attribut ini mendefinisikan bagaimana properti bisa diakses (apakah ia mudah dibaca?, apakah ia mudah ditulis?)
Pada JavaScript, semua attribute bisa dibaca, tapi hanya nilai attribute yang bisa diubah (dan hanya jika properti itu mudah ditulis).
( ECMAScropt 5 memiliki metode - metode untuk kedua menjadi dan pengaturan semua attribute properti).

Properti Prototype

Object JavaScript mewarisi (inherit) properti - properti dari bentuk dasarnya atau purwarupa (prototype) mereka.
Kata kunci delete tidak menghapus properti - properti yang diwarisi, tapi jika kamu menghapus sebuah properti prototypi (purwarupa), ia akan mengefek pada semua object yang diwarisi daei prototype.
