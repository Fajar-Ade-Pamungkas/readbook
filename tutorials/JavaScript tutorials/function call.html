JavaScript Function Call

Metode Reuse

Dengan metode call(), kamu bisa menulis sebuah metode yang bisa digunakan pada objet - object yang berbeda.

Semua Fungsi Adalah Metode

Pada JavaScript semua fungsi adalah metode object.

Jika sebuah fungsi bukan sebuah metode dari object JavaScript, itu adalah sebuah fungsi dari object global (lihatlah chapter sebelumnya).

Contoh dibawah ini membuat sebuah objet dengan 3 properti, firstName, lastName, fullName.

Contoh

Input:

var person = {
firstName:"John",
lastName: "Doe",
fullName: function () {
return this.firstName + " " + this.lastName;
}
}
person.fullName(); // Will return "John Doe"

Output:

Kata kunci this

Pada sebuah definisi fungsi, this mengarah kepada "owner" dari fungsi.

Pada contoh dibawah ini, this adalah object person yang "memiliki" fungsi fullName.

Dengan kata lain, this.firstName mengartikan properti firstName dari object this.

Bacalah tentang kata kunci this di JS this keyword.

Metode call() JavaScript

Metode call() adalah sebuah metode JavaScript yang sudah dikenali.

This bisa digunakan untuk memanggil (invoked) sebuah metode dengan sebuah object pemilik (owner) seperti sebuah argument (parameter).

Dengan call() sebuah object bisa menggunakan sebuah metode termasuk (belonging) untuk sebuah bject lainnya.

Contoh ini memanggil metode fullName dari person, menggunakannya pada person1:

Contoh

Input:

var person = {
fullName: function() {
return this.firstName + " " + this.lastName;
}
}

var person1 = {
firstName:"John",
lastName: "Doe",
}

var person2 = {
firstName:"Mary",
lastName: "Doe",
}
person.fullName.call(person1); // Akan mengembalikan "John Doe"

Output:

Contoh ini memeanggil metode fullName dari person, menggunakannya pada person2:

Contoh

Input:

var person = {
fullName: function() {
return this.firstName + " " + this.lastName;
}
}

var person1 = {
firstName:"John", lastName: "Doe",
}

var person2 = {
firstName:"Mary", lastName: "Doe",
}

person.fullName.call(person2); // Akan mengembalikan "Mary Doe"

Output:

Metode call() dengan Argument (pernyataan)

Metode call() bisa menerima argument atau pernyataan

Contoh

Input:

var person = {
fullName: function(city, country) {
return this.firstName + " " + this.lastName + "," + city + "," + country;
}
}

var person1 = {
firstName:"John",
lastName: "Doe",
}
person.fullName.call(person1, "Oslo", "Norway");
