JavaScript Type Conversion

Number() mengubah menjadi sebuah Number, String() mengubah menjadi sebuah String, Boolean() mengubah menjadi sebuah Boolean.

Tipe - tipe Data JavaScript

Di JavaScript ada 5 tipe data berbeda yang bisa berisi nilai:

  • String
  • Number
  • Boolean
  • Object
  • function

Ada 3 tipe Object:

  • Object
  • Date
  • Array

Ada 2 tipe data yang tidak bisa berisi nilai:

  • null
  • undefined

Operator typeof

Kamu bisa menggunakan operator typeof untuk menemukan tipe data dari variabel JavaScript.

Kamu bisa menulis seperti ini

input

!DOCTYPE html

html

head

/head

body

h1 Readbook /h1

p JavaScript akan mengeluarkan nama hari ini /p

p id="demo" /p

script
document.getElementById("demo").innerHTML=
typeof "John"     + "br" +           // Mengembalikan string
typeof 3.14     + "br" +              // Mengembalikan number
typeof NaN     + "br" +               // Mengembalikan number
typeof false      + "br" +            // Mengembalikan boolean
typeof [1,2,3,4]    + "br" +          // Mengembalikan object
typeof {name:'John', age:34} + "br" +  // Mengembalikan object
typeof new Date()    + "br" +         // Mengembalikan object
typeof function () {}  + "br" +       // Mengembalikan function
typeof myCar   + "br" +  // Mengembalikan undefined
typeof null;                   // Returns object
/script

/body

/html

output

Amatilah

  • Tipe data dari NaN adalah Number
  • Tipe data dari Array adalah Object
  • Tipe data dari date (tanggal) adalah Object
  • Tipe data dari null adalah Object
  • Tipe data dari variabel undefined adalah undefined

!

 Kamu tidak bisa menggunakan untuk menentukan (determine) jika sebuah object JavaScript adalah array (atau date)

Tipe data dari typeof

typeof bukan merupakan tipe data tapi melainkan sebuah operator seperti +-*/ tidak memiliki tipe data apapun.

Tapi operator typeof selalu mengembalikan string berisikan tipe operand.

Metode global String() bisa mengubah Number menjadi String.

Metode ini bisa digunakan pada tipe apa saja Number, Literal, Variabel ataupun Expression.

var x = 123;
document.getElementById("demo").innerHTML=
String(x)  + "br" +  // returns a string from a number variable x

String(100 + 23); // returns a string from a number from an expression

Pengubahan Boolean menjadi String

Dengan metode global String() dapat mengubah nilai Boolean menjasi String:

String(false)  // Mengembalikan "false"
String(true)     // Mengembalikan "true"

Metode Boolean toString() melakukan hal yang sama:

false.toString()  // Mengembalikan "false"
true.toString()     // Mengembalikan "true"

Pengubahan Date menjadi String

Dengan metode global String() dapat mengubah Date menjadi String:

String(Date())  // Mengembalikan

Metode Boolean toString() melakukan hal yang sama:

Date.toString()  //mengembalikan

Pengubahan String menjadi Number

Dengan metode global Number() dapat mengubah String menjadi Number.

String berisikan ("3.14") diubah menjadi Number seperti (3.14).

String Kosong menjadi 0.

Apapun merubah lainnya menjadi NaN (Not a Number).

Number("3.14")  // Mengembalikan 3.14
Number("")  // Mengembalikan 0
Number(" ")  // Mengembalikan 0
Number("99 88")  // Mengembalikan NaN

Di chapter Metode Number kamu akan menemukan banyak metode yang bisa digunakan untuk mengubah String menjadi Number:

Tabel Metode Global
Metode Deskripsi
parseFloat() Mengurai (parse) argumen itu sendiri dan mengembalikan sebuah floating point number.
parseInt() Mengurai argumen dan mengembalikan sebuah integer (bilangan bulat).

Operator Unary +

Operator Unary + dapat digunakan untuk mengubah variabel menjadi Number

var y = "5"  // y Mengembalikan String
var x = + y // x Mengembalikan Number

output

Operator Unary + bisa mengubah String menjadi Number.

Jika itu tidak dapat diubah maka ia akan menjadi Number dengan nilai NaN (Not a Number).

var y = "Dave"  // y Mengembalikan String
var x = + y // x Mengembalikan NaN

output

Operator Unary + bisa mengubah String menjadi Number.

Mengubah Boolean menjadi Number

Dengan metode global Number() dapat mengubah Boolean menjadi Number.

Number(false) // Mengembalikan 0
Number(true)  // Mengembalikan 1

Mengubah Date menjadi Number

Dengan metode global Number() dapat mengubah Date menjadi Number.

d = new Date();
Number(d) // Mwngembalikan

Metode date getTime() melakukan hal yang sama.

d = new Date();
d.getTime() // Mengembalikan

Perubahan Tipe Otomatis

Ketika JavaScript mnecoba mengoperasikan pada tipe data yang salah, JavaScript akan mencoba untuk mengubah nilai menjadi tipe yang benar.

Hasil tidak selalu mengeluarkan apa yang kamu bayangkan.

Kamu bisa menulis seperti ini

input

!DOCTYPE html

html

head

/head

body

h1 Readbook /h1

p JavaScript akan mengeluarkan nama hari ini /p

p id="demo" /p

script
var x = 123;
document.getElementById("demo").innerHTML=
5 + null   + "br" +  //Mengembalikan 5, karena null diubah menjadi 0
"5" + null  + "br" +  // Mengembalikan "5null", karena null diubah menjadi "null"
"5" + 2    + "br" +  // Menjadi 52, karena 2 diubah menjadi "2"
"5" - 2  + "br" +   // Mengembalikan 3, karena "5" diubah menjadi 5
"5" * "2"  + "br" +  // Mengembalikan 10, karena "5" dan "2" diubah menjadi 5 dan 2
/script

/body

/html

output

Perubahan String Otomatis

JavaScript secara otomatis memanggil fungsi toString() variabel untuk mencoba meng-output sebuah object atau sebuah variabel.

document.getElementById("demo").innerHTML= myVar;
// jika myVar = {name:"Fjohn"}  // toString mengubah menjadi "[object Object]"
// jika myVar = [1,2,3,4]       // toString mengubah menjadi "1,2,3,4"
// jika myVar = new Date()      // toString mengubah "Fri Jul 18 2014 09:08:55 GMT+0200"

Number dan Boolean juga diubah, tapi ini sangat tidak visibel (terlihat).

document.getElementById("demo").innerHTML= myVar;
// jika myVar = 123 // toString mengubah menjadi "123"
// jika myVar = true       // toString mengubah menjadi "true"
// jika myVar = false      // toString mengubah menjadi "false"
