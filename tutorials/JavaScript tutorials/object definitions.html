JavaScript Object

JavaScript Object

!

 Dalam JavaScript, Object adalah Raja, jika kamu mengerti object, kamu mengerti JavaScript.

Di JavaScript, hampir "segalanya" adalah sebuah object.

  • Boolean bisa menjadi object (atau data primitif diperlakukan sebagai object).
  • Number bisa menjadi object (atau data primitif diperlakukan sebagai object).
  • String bisa menjadi object (atau data primitif diperlakukan sebagai object).
  • Date selalu merupakan object .
  • Math selalu merupakan object.
  • Regular Expression selalu merupakan object
  • Array selalu merupakan object
  • Function selalu merupakan object

Di JvaScript, semua nilai, kecuali nilai primitif adalah object.
Nilai primitif yaitu Steing ("John Doe"), Number (3.14), true, false, null, dan undefined

Objects Adalah Variables Yang Berisikan Variables

Variabel - variabel bisa berisi nilai tunggal:

Contoh Nilai Primitif

var people = "John Doe";

Object adalah variabel juga. Tapi object bisa berisikan banyak nilai.
Nilai yang ditulsi seperti pasangan nama : nilai (nama dan nilai dipisahkan dengan sebuah tanda titik dua).

Contoh Object

var people = {firstName : "Fajar Ade", lastName : "Pamungkas", age : "20", eyeColor : "brown"};

!

 Sebuah object JavaScript merupakan sebuah pengoleksian nilai - nilai bernama.

Properti - Properti Object

Nilai - nilai bernama pada object JavaScript disebut properti:

Properti Nilai
firstName Fajar Ade
lastName Pamungkas
age 20
eyeColor brown

Object - object ditulis sebagai sepasang nama dan nilai yang mirip dengan:

  • Associative arrays pada PHP
  • Dictionaries pada Python
  • Hash tables pada C
  • Hash maps pada Java
  • Hashes pada Ruby dan Perl

Metode - Metode Object

Metode merupakan aksi yang bisa dikerjakan pada Object.
Properti - properti object bisa menjadi nilai primitif, object - object lain dan fungsi.
Sebuah metode object adalah sebuah properti yang berisikan sebuah function definition:

Properti Nilai
firstName Fajar Ade
lastName Pamungkas
age 20
eyeColor brown
fullName function() {return this.firstName + " " + this.lastName;}

Object JavaScript adalah pengisi untuk nilai bernama yang disebut property dan method

Kamu akan mempelajari tentang metode pada chapter selanjutnya.

Pemciptaan Sebuah Object JavaScript

Dengan JavaScript, kamu bisa mendefinisikan dan menciptakan object kamu sendiri.

Ada cara - cara yang berbeda untuk menciptakan object baru:

  • Definisikan dan ciptakan object tunggal, menggunakan sebuah object yang sesungguhnya.
  • Definisikan dan ciptakan sebuah object tunggal, dengan kata kunci new
  • Definisikanlah sebuah object constructor (pembangun object) dan kemudian ciptakan object - object dari tipe yang dibangun

Pada ECMAScript 5, sebuah object juga bisa diciptakan dengan function Object.create().

Penggunaan Sebuah Object Yang Sesungguhnya

Ini merupakan cara termudah untuk menciptakan sebuah object JavaScript.
Penggunaan sebuah object literal, kamu keduanya mendefini dan menciptakan sebuah object dalam satu pernyataan.
Sebuah object literal adalah daftar dari pasangan nama:nilai (seperti age:20) dalam curly braces atau kurung kurawal {}.
Contoh berikut menciptakan sebuah object JavaScript baru dengan empat properti:

Contoh Penciptaan Object Literal

Input

var people = {firstName : "Fajar Ade", lastName : "Pamungkas", age : 20, eyeColor : "brown"};
var nama = people.firstName;

Output

Spasi dan baris baru tidaklah penting. Sebuah object definisi bisa dijangka dengan baris ganda:

Contoh Penciptaan Object Literal

Input

var people = {
firstName : "Fajar Ade",
lastName : "Pamungkas",
age : 20,
eyeColor : "brown"
};
var nama = people.firstName;

Output

Penggunaan Kata Kunci new JavaScript

Contoh berikut juga menciptakan sebuah Object JavaScript baru dengan empat Properti:

Contoh Penciptaan Object Literal

Input

var people = new Object();
person.firstName = "Fajar Ade";
lastName = "Pamungkas";
age = 20;
eyeColor = "brown"; var nama = people.firstName;

Output

Kedua contoh diatas melakukan sama persis (exactly). Tidak ada yang diperlukan untuk menggunakan new Object.
Untuk kesederhanaan, keahlian pembacaan dan kecepatan pengeksekusian, gunakanlah salah satu (metode literal object).

Penggunaan Sebuah Pembangun Object (Object Constructor)

Contoh diatas dibatasi pada banyak situasi. Mereka hanya menciptakan sebuah object tunggal.
Terkadang kita suka memiliki sebuah "tipe object" yang bisa digunakan untuk menciptakan banyak object dari satu tipe.
Cara standar untuk menciptakan sebuah "tipe object" adalah menggunakan sebuah fungsi pembangun object:

Contoh

Input

function person(first, last, age, eye) {
    this.firstName = first;
    this.lastName = last;
    this.age = age;
    this.eyeColor = eye;
}
var myFather = new person("John", "Doe", 50, "blue");
var myMother = new person("Sally", "Rally", 48, "green");

Output

Fungsi (person) diatas adalah sebuah Constructor Object.
Sekali kamu memiliki sebuah object constructor, kamu bisa menciptakan object baru dari tipe yang sama:

var myFather = new person("John", "Doe", 50, "blue");
var myMother = new person("Sally", "Rally", 48, "green");

Kata kunci this

Pada JavaScript, suatu hal disebut this, adalah object yang "owns" (sendiri) kode JavaScript.
Nilai dari this, ketika digunakan pada sebuah fungsi, adalah object yang "owns" fungsi.
Nilai dari this, ketika digunakan pada sebuah object, adalah object itu sendiri.
Kata kunci this pada sebuah object constructor tidak memiliki nilai. Ia hanya sebuah cadangan (substitute) untuk object baru.
Nilai dari this akan menjadi object baru ketika constructor digunakan untuk menciptakan sebuah object.

!

 Catatan bahwa this bukan merupakan variabel, this adalah kata kunci. Kamu tidak bisa mengubah nilai dari this.

Built-in JavaScript Constructor

JavaScript memiliki built-in constructor untuk obejct - object asal (native)

Contoh

var x1 = new Object();    // sebuah Object object bar
var x2 = new String();    // sebuah object String baru
var x3 = new Number();    // sebuah object Number baru
var x4 = new Boolean()    // sebuah object Boolean baru
var x5 = new Array();     // sebuah object Array baru
var x6 = new RegExp();    // sebuah object RegExp baru
var x7 = new Function();  // sebuah object Function baru
var x8 = new Date();      // sebuah object Date baru

Output

Object Math() tidak ada di daftar. Math adalah sebuah object global. kata kunci new tidak bisa digunakan pada Math.

Apakah Kamu Tahu ?

Seperti yang kamu lihat, JavaScript memiliki versi - versi object dari tipe data primitif String, Number dan Boolean.

Tidak ada alasan untuk membuat object complex (rumit). Nilai primitif mengeksekusi banyak yang lebih cepat.

Dan tidak ada alasan untuk menggunakan new Array(). Gunakanlah array sesungguhnya (array literal) dengan alternatif : []

Dan tidak ada alasan untuk menggunakan new RegExp(). Gunakanlah pola sesungguhnya dengan dengan alternatif: /()/

Dan tidak ada alasan untuk menggunakan new Function(). Gunakanlah ekspresi fungsi dengan alternatif: function () {}

Dan tidak ada alasan untuk menggunakan new Object(). Gunakanlah object sesungguhnya dengan alternatif: {}.

Object JavaScript Merupakan Mutable (Tidak Tetap)

Object tidak tetap: Mereka bisa dialamatkan dengan referensi, bukan nilai.
Jika y adalah sebuah object, pernyataan berikut tidak akan memciptakan sebuah salinan dari y.

var x = y;  // ini tidak akan menciptakan sebuah salinan dari y.

Object x bukan sebuah salinan y, x adalah y, keduanya z dan y bertujuan untuk object yang sama.
Segalanya mengubah menjadi y juga akan mengubah x, karena x dan y adalah object yang sama.

Contoh

Input

var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"}

var x = person;
x.age = 10;     // Ini akan mengubah keduanya x.age dan person.age

Output
