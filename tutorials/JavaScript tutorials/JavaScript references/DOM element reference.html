The HTML DOM Element Object

HTML DOM Nodes

Di HTML DOM (Document Object Model), segalanya merupakan sebuah node:

  • Dokumen itu sendiri adalah sebuah node dokumen
  • Semua element HTML merupakan node element
  • Semua attribute HTML merupakan node attribute
  • Teks didalam element HTML merupakan node teks
  • Comment (komentar) merupakan node comment

The Element Object

Di HTML DOM, Element object mempersembahkan sebuah element HTML.

Element object bisa memiliki child node (node anak) dari element node, text node, atau comment node.

Sebuah Nodelist object mempersembahkan sebuah daftar node - node, sepertu sekumpulan element HTML tentang child node.

Element bisa juga memiliki attribute. Attibute merupakan node attibute (Lihatlah chapter selanjutnya).
