The HTML DOM Style Object

Style object

Style object mempersembahkan sebuah pernyataan individual style.

Access a Style Object

Style object bisa diakses dari bagian head dokumen, atau dari ketentuan HTML element(s).

Pengaksesan style object(s) dari bagian head dokumen:

var x = document.getElementsByTagName("STYLE");

Pengaksesan suatu object style element tertentu:

var x = document.getElementById("myH1").style;

Create a Style Object

Kamu bisa membuat sebuah element <style> dengan penggunaan metode document.createElement():

var x = document.createElement("STYLE");

Kamu juga bisa mengatur properti style dsri sebuah element yangnsudsh ada:

document.getElementById("myH1").style.color = "red";
