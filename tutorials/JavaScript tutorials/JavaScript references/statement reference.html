JavaScript Statement Identifiers

Pernyatan JavaScript seringkali dimulai dengan sebuah statement identifier untuk mengidentifikasi aksi JavaScript untuk dikerjakan.

Statement identifier merupakan reserved word (kata yang dipesan) dan tidak bisa digunakan sebagai nama variabek (ataupun nama lainnya).

Daftar tabel berikut ini adalah semua statement JavaScript:

Tabel Kata kunci JavaScript
Kata kunci Deskripsi
break untuk mengakhiri (terminate) sebuah switch atau sebuah loop.
continue Melompati sebuah loop dan memulainya dari atas.
debugger Menghentikan pengeksekusian JavaScript, dan memanggil (if yang tersedia) fungsi debugging atau penelusuran
do ... while Mengeksekusi sebuah blok dari statement dan mengulang blok saat kondisi itu benar
for Menandai sebuah blok dari statement untuk dieksekusi, sepanjang suatu kondisi itu benar
for in Menandai sebuah blok dari statement untuk dieksekusi kepada setiap element, dari sebuah object (atau array)
function Menyatakan sebuah fungsi
if ... else Menandai sebuah blok dari statement untuk dieksekusi, tergantung pada suatu kondisi
return Menghentikan pengeksekusian sebuah fungsi dsn mengembalikan nilai dsri fungsi tersebut
switch Menandai sebuah blok dari statement untuk dieksekusi, tergantung pada perbedaan persoalan (case)
throw Melemparkan (atau menghasilkan) sebuah error
try ... catch Menerapkan penanganan error untuk blok dari statement
try ... catch ... finally Menandai blok pernyataan untuk dieksekusi ketika sebuah error terjadi di suatu try block, dsnmenerspkan error handling.
var Menyatakan sebuah variabel
while Menandai sebuah blok pernyataan untuk dieksekusi saat kondisi itu benar

!

 Kata kunci JavaScript adalah reserved word, reserved word tidak bisa digunakan untuk penamaan fungsi dan variabel.
