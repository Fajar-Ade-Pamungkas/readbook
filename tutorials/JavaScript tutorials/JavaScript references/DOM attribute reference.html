The HTML DOM Attribute Object

HTML DOM Nodes

Di HTML DOM (Document Object Model), segalanya merupakan sebuah node:

  • Dokumen itu sendiri adalah sebuah node dokumen
  • Semua element HTML merupakan node element
  • Semua attribute HTML merupakan node attribute
  • Teks didalam element HTML merupakan node teks
  • Comment (komentar) merupakan node comment

The Attr Object

Dalam HTML DOM, Attr object mempersembahkan sebuah attribute HTML.

Suatu attribute HTML selalu anggota dari element HTML.

The NamedNodeMap Object

Dalam HTML DOM, NamedNodeMap object mempersembahkan sebuah kumpulan tak berurutan dari suatu node - node attribute element.

Node - node dalam suatu NamedNodeMap bisa diakses dengan nama atau dengan index (nomor).

DOM 4 Warning !!!

Dalam W3C DOM Core, Attr (attribute) object mewariskan (inherit) semua properti dan metode dari Node object.

Dalam DOM 4, Attr object tidak ada waris yang lebih panjang dari Node.

Untuk kualitas kode masa yang datang, kamu harus menghindari penggunaan semua properti dan metode node object di attribute object:
