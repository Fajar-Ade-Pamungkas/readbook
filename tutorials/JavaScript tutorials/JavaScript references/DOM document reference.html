The HTML DOM Document Object

HTML DOM Nodes

Di HTML DOM (Document Object Model), segalanya merupakan sebuah node:

  • Dokumen itu sendiri adalah sebuah node dokumen
  • Semua element HTML merupakan node element
  • Semua attribute HTML merupakan node attribute
  • Teks didalam element HTML merupakan node teks
  • Comment (komentar) merupakan node comment

The Dokument Object

Ketika sebuah dokumen HTML dimuat kedalam web browser, dokumen menjadi sebuah object dokumen.

Document object adalah root node dari dokumen HTML dan "pemilik" dari semua node lainnya:

(element nodes, text nodes, attribute nodes, and comment nodes).

Document object menyediakan properti dan metode untuk mengakses semua node object dari dengan JavaScript.

Tip: Dokument merupakan sebuah bagian dari Window object dan bisa diakses seperti window.document.

Document Object Properties and Methods

Metode dan properti berikut bisa digunakan pada dokumen HTML:

Properti/Metode Deskripsi

document.adoptNode() Mengambil (mengadopsi) sebuah node dari (sebuah) dokumen lain

document.applets Mengembalikan sebuah koleksi dari semua element <applet> dalam dokumen
document.baseURI Mengembalikan absolute base URI (URI dasar yang mutlak) dari dokumen
document.body Menyetel atau mengembalikan document's body (badan dokumen) (element <body>)
document.close() Menutup output stream previously yang dibuka dengan document.open()
document.cookie Mengembalikan semua pasangan nama/nilai dari cookies dalam dokumen
document.createAttribute() Membuat sebuah node attribute
document.createComment() Membuat sebuah node comment dengan teks yang ditentukan
document.createDocumentFragment() Membuat sebuah node DocumentFragment kosong
document.createElement() Membuah sebuah node Element
document.createTextNode() Membuah sebuah node Teks
document.doctype Mengembalikan Document Type Declaration yang berhubungan dengan dokumen
document.documentElement Mengembalikan Document Element dari dokumen (element <html>)
document.documentMode Mengembalikan doe yang digunakan oleh browser untuk menyumbang atau memberikan kepada dokumen
document.documentURI Menyetel atau mengembalikan lokasi dari dokumen
document.domain Mengembalikan nama domain dari server yang memuat dokumen
document.domConfig sudah tidak dipakai. Mengembalikan konfigurasi DOM tentang dokumen
document.embeds Sebuah koleksi dari semua element <embed> dokumen
document.forms Sebuah koleksi dari semua element <form> dokumen
document.getElementById() Mengembalikan element yang memikiki attrihute ID dengan nilai tertentu
document.getElementsByClassName() Mengembalikan sebuah Nodelist (daftar node) yang berisikan semua elemen dengan nama class tertentu
document.getElementsByName() Mengembalikan sebuah Nodelist (daftar node) yang berisikan semua elemen dengan nama tertentu
document.getElementsByTagName() Mengembalikan sebuah Nodelist (daftar node) yang berisikan semua elemen dengan nama tag tertentu
document.hasFocus() Mengembalikan sebuah nilai Boolean yang mengindikasikan apakah dokumen telah fokus (sedang aktif)
document.head Mengembalikan element <head> dari dokumen
document.images Mengembalikan sebuha koleksi dsri semua element <img> dalam dokumen
document.implementation Mengembalikan DOMImplementation object yang mengatasi dokumen ini
document.importNode() Mengimpor (mendatangkan) sebuah node dari sebuah dokumen lain
document.inputEncoding Mengembalikan penyandian (encoding), setel karakter (character set) yang digunakan kepada dokumen
document.lastModified Mengembalikan date (tanggal) dan time (waktu) dokumen yang dimodifikasi (diubah) terakhir
document.links Mengembalikan sebuah koleksi dari semua elemen <a> dan <area> dalam dokumen yang memiliki sebuah attribute href
document.normalize() Mengganti node Teks kosong dan menggabungkan node - node yang berdekatan (adjacent)
document.open() Membuka sebuah HTML output stream untuk mengumpulkan output dari document.write()
document.querySelector() Mengembalikan element pertama yang mencocokan sebuah specified CSS selector(s) dalam dokumen
document.querySelectorAll() Mengembalikan sebuah Nodelist statis yang berisikan semua element yang mencocokan sebuah specified CSS selector(s) dalam dokumen
document.readyState Mengembalikan status (loading) dari dokumen
document.referrer Mengembalikan URL dari dokumen yang dimuat dokumen saat ini
document.removeEventListener() Menggantikan sebuah event handler (penangan kejadian) dari dokumen (yang telah dilampirkan dengan metode addEventListener())
document.renameNode() Menamai ulang node tertentu
document.scripts Mengembalikan sekumpulan dari element <script> dalam dokumen
document.strictErrorChecking Menyetel atau mengembalikan apakah error-checking dipaksakan (diharuskan) atau tidak
document.title Menyetel atau mengembalikan title (judul) dari dokumen
document.URL Mengembalikan URL penuh dari dokumen HTML
document.write() Menulis HTML expression atau kode JavaScript untuk sebuah dokumen
document.writeln() Sama seperti write(), tapi menambahkan sebuah karakter (newline) garis baru setelah setiap pernyataan (statement)

Warning !!!

Dalam W3C DOM Core, Document object mewariskan (inherit) semua properti dan metode dari Node object.

Kebanyakan dari semua parameter dan metode ini tidak membuat ungkapan yang digunakan pada dokumen.

Hindarilah penggunaan semua properti dan metode node object ini pada object dokumen HTML:

Properti / Metode Alasan Untuk Menghindari
document.attributes Dokumen - dokumen tidak memiliki attribute
document.hasAttributes() Dokumen - dokumen tidak memiliki attribute
document.nextSibling Dokumen - dokumen tidak memiliki sibling (saudara)
document.nodeName Ini selalu #document
document.nodeType Ini selalu 9 (DOCUMENT_NODE)
document.nodeValue Dokumen - dokumen tidak memiliki node value (nilai node)
document.ownerDocument Dokumen - dokumen tidak memiliki pemilik (owner) dokumen
document.ownerElement Dokumen - dokumen tidak memiliki pemilik (owner) element
document.parentNode Dokumen - dokumen tidak memiliki node induk (parent node)
document.previousSibling Dokumen - dokumen tidak memiliki sibling (saudara)
document.testContent Dokumen - dokumen tidak memiliki sebuah konten teks
