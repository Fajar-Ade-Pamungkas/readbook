JavaScript String Reference

JavaScript Strings

JavaScript string menyimpan sebuah seri dari karakter seperti "John Doe".

String bisa menjadi teks apapun didalam double atau single quotes:

var carname = "Volvo XC60";
var carname = 'Volvo XC60';

Index - index string merupakan zero-based: karakter pertma berada pada posisi 0, kedua di 1, dan seterusnya.

Untuk tutorial tentang string, bacalah JavaScript String Tutorial.

String Properties and Methods

Metode dan Properti String

Nilai primitif (sederhana) seperti "John Doe", tidak bisa memiliki properti ataupun metode (karena mereka bukan merupakan object).

Tapi dengan JavaScript, metode dan properti juga tersedia untuk nilai primitif, karena JavaScript memperlakukan nilai primitif sebagai object ketika pengeksekusian metode dan properti.

Properti String

Tabel Properti String
Properti Deskripsi
constuctor Mengembalikan fungsi fungsi pembangun String
length Mengembalikan Seberapa panjang sebuah string
prototype Memperbolehkan kamu menambahkan properti dan metode ke sebuah object.

Metode String

Tabel Metode String
Metode Deskripsi

endsWith() Mengecek apakah sebuah string diakhiri dengan string/karakter yang ditentukan
fromCharCode() Mengubah nilai kode tunggal menjadi karakter.
includes() Mengecek apakah sebuah string berisikan string/karakter yang ditentukan.
indexOf() Mengembalikan posisi dari occurence (kejadian) temuan pertama dari nilai tertentu dalam sebuah string.
lastIndexOf() Mengembalikan posisi dari occurence (kejadian) temuan terakhir dari nilai tertentu dalam sebuah string.
localeCompare() Membandingkan dua nilai tertentu pada string saat ini.
match() Mencari sebuah string pada sebuah kecocokan yang berlawanan (match against) sebuah ekpresi teratur, mengembalikan kecocokan tersebut.
repeat() Mengembalikan sebuah string baru dengan sebuah nomor tertentu dari salinan - salinan dari sebuah string yang sudah ada
replace() Mencari sebuah string pada sebuah nilai dan mengembalikan sebuah string baru dengan nilai yang digantikan.
seacrh() Mencari sebuah string pada sebuah nilai dan mengembalikan posisi dari kecocokan.
slice() Mengekstrak (mengambil) sebuah bagian dari sebuah string dan mengembalikan sebuah string.
split() Membelah (split) sebuah string memasukannya kedalam sebuah array dari substring (string pengganti)
tartsWith() Mengecek apakah sebuah string memulai dengan karakter tertentu
substr() Mengekstrak bagian dari sebuah string dari sepanjang (through) posisi mulai sebuah nomor dari karakter.
substring() Mengekstrak bagian dari sebuah string antara dua posisi tertentu
toLocaleLowerCase() mengubah sebuah string menjadi huruf kecil, berdasarkan (according) host's local (tempat terjadinya peristiwa tersebut)
toLocaleUpperCase() mengubah sebuah string menjadi huruf besar, berdasarkan (according) host's local (tempat terjadinya peristiwa tersebut)
toLowerCase mengubah sebuah string menjadi huruf kecil
toString() mengembalikan nilai dari sebuah object String
toUpperCase mengubah sebuah string menjadi huruf kecil
trim() Memindahkan whitespace (spasi) dari akhiran kedua dari sebuah string.
valueOf mengembalikan nilai sederhana dari sebuah object String

String HTML Wrapper Methods

Metode wrapper HTML mengembalikan string wrapped didalam tag HTML yang sewajarnya.

Ini bukan merupakan metode standar, dan mungkin tidak bekerja seperti yang diinginkan dalam semua browser.

Metode Deskripsi
anchor() Membuat sebuah anchor (jangkar)
big() Menampilkan string dengan font besar
blink() Menampilkan sebuah blinking string (string kelap - kelip)
bold() Menampilkan string dalam huruf tebal
fixed() Menampilkan string menggunakan font fixed-pitch
fontcolor() Menampilkan string menggunakan warna tertentu
fontsize() Menampilkan string dengan ukuran tertentu
italics() menampilkan string dalam bentuk miring
link() Menampilkan string sebagaihyperlink
small() Menampilkan string menggunakan font kecil (small)
strike() Menampilkan string dengan sebuah strikethrough (garis delete)
sub() Menampilkan string sebagai teks subscipt
sup() Menampilkan string sebagai teks superscipt
