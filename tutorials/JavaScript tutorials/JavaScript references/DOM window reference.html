The HTML DOM Document Object

HTML DOM Events

HTML DOM event membolehkan JavaScirpt untuk register event handlers yang berbeda pada element di sebuah dokumen HTML.

Event normalnya digunakan pada kombinasi dengan fungsi dan fungsi tidak akan dieksekusi sebelum event terjadi (seperti ketika user mengklik button).

animationend Event terjadi ketika CSS animation telah complete atau selesai AnimationEvent

animationstart Event terjadi ketika CSS animation telah dimulai AnimationEvent
beforeprint Event terjadi ketika halaman (page) hampir dicetak Event
beforeunload Event terjadi ketika dokumen hampir di-unload (dibongkar) UiEvent, Event
blur Event terjadi ketika sebuah element kehilangan fokus FovusEvent
canplay Event terjadi ketika browser bisa mulai memainkan media (ketika ia telah buffered cukup untuk dimulai) Event
canplaythrough Event terjadi ketika browser bisa mulai memainkan sepanjang media tanpa menghentikan buffering Event
change Event terjadi ketika konten dari elemen form, selection, atau keadaan yang dicek telah berubah (untuk <input>, <select>, dan <textarea>) Event
click Event terjadi ketika user mengklik pads sebuah element MouseEvent

invalid Event terjadi ketika sebuah element tidak valid Event

keypress Event terjadi ketika user menekan sebuah kunci (key) KeyboardEvent

show Event terjadi ketika sebuah element <menu> ditampilkan sebagai sebuah context menu Event

storage Event terjadi ketika sebuah Web Storage area diperbaharui StorageEvent
submit Event terjadi ketika sebuah form (formulir) diserahkan Event
suspend Event terjadi ketika browser dengan sengaja tidak sedang mendapatkan media data Event
timeupdate Event terjadi ketika posisi bermain telah diubah (seperti ketika user cepat kedepan untuk sebuah titik berbeda dalam media) Event
toggle Event terjadi ketika user membuka atau menutup element <details> Event
touchcancel Event terjadi ketika touch (sentuhan) is interrupted (disela) TouchEvent

touchstart Event terjadi ketika jari (finger) ditempatkan pada touch screen (layar sentuh) TouchScreen
transitionend Event terjadi ketika sebuah CSS transition telah lengkap atau selesai TransitionEvent
unload Event terjadi sekali halaman telah membongkar (untuk <body>) UiEvent, Event
volumechange Event terjadi ketika volume dari media telah berubah (termasuk pengaturan volume untuk "mute") Event
waiting Event terjadi ketika media telah dijeda (paused) tapi diharapkan untuk resume atau dimulai lagi (seperti ketika media mem-pause untuk memperlancar (buffer) lebih data) Event
wheel Event terjadi ketika mouse berputar menggulung keatas atau kebawah diatas sebuah element WheelEvent
