JavaScript RegExp Reference

RegExp Object

Regular expression adalah sebuah object yang mendeskripsikan sebuah pola (pattern) dari karakter - karakter.

Regular expression digunakan untuk mengerjakan pattern-matching (pola kecocokan) dari karakter - karakter.

Syntax

Contoh

/pattern/modifiers;

Contoh menjelaskan:

  • /w3schools/i  merupakan sebuah regular expression
  • w3schools  merupakan sebuah pola (untuk digunakan dalam sebuah pencarian)
  • i  merupakan sebuah modifier atau pengubah (modifies the search to be case-insensitive)

Untuk tutorial tentang Regular Expression, bacalah JavaScript RegExp Tutorial.

Modifiers

Modifier digunakan untuk melakukan case-sensitive (keadaan sensitif) dan global seacrhes (pencarian global):

Modifier Deskripsi
i Melakukan kecocokan keadaan tidak sensitif (case-insensitive matching)
g Melakukan suatu pencarian global (mencari semua kecocokan daripada penghentian setelah kecocokan pertama)
m Melakukan multiline matching

Brackets

Bracket (kurung persegi) digunakan untuk mencari (atau menemukan) suatu jarak (range) dari karakter:

Expression Deskripsi
[abc] Mencari karakter apa saja yang ada diantara tanda kurung
[^abc] Mencari karakter apa saja yang TIDAK ada diantara tanda kurung
[0-9] Mencari digit apa saja yang ada diantara tanda kurung
[^0-9] Mencari digit apa saja yang ada TIDAK diantara tanda kurung
(x|y) Mencari apapun dari alternatif - alternatif yang ditentukan

Metacharacters

MetaCharacter adalah karakter - akrakter dengan arti special:

Metacharacter Deskripsi
. Mengungkapkan sebuah karakter tunggal, kecuali newline (baris baru) atau line terminator (baris pangakhiran)
\w Mengungkapkan sebuah karakter kata
\W Mengungkapkan sebuah karakter bukan kata (non-word)
\d Mengungkapkan sebuah digit
\D Mengungkapkan sebuah karakter bukan digit
\s Mengungkapkan sebuah karakter whitespace (spasi)
\S Mengungkapkan sebuah karakter bukan whitespace (bukan spasi)
\b Mengungkapkan sebuah kesesuaian pada permulaan/akhiran dari sebuah kata
\B Mengungkapkan sebuah bukan kesesuaian pada permulaan/akhiran dari sebuah kata
\O Mengungkapkan karakter NUL
\n Mengungkapkan sebuah karakter baris baru
\f Mengungkapkan sebuah karakter form feed
\r Mengungkapkan sebuah karakter carriage return
\t Mengungkapkan sebuah karakter tab
\xxx Mengungkapkan karakter tertentu dengan sebuah xxx nomor octal
\xdd Mengungkapkan karakter unicode tertentu dengan dd nomor hexadecimal
\uxxxx Mengungkapkan karakter unicode tertentu dengan sebuah xxxx nomor hexadecimal
  • Metacharacter pertama pada tabel adalah sebuah titik (.)
  • Author (saya) kurang mengerti arti kata "find" jadi saya berinisitif menggantinya dengan kata "Mengungkapkan"

Quantifiers

Quantifier Deskripsi
n+ Mencocokan string apapun yang berisikan (pada) paling sedikit satu huruf n
n* Mencocokan String apapun yang berisikan nol atau lebih kejadian dari n
n? Mencocokan String apapun yang berisikan nol atau satu kejadian dari n
n{X} Mencocokan string apapun yang berisikan suatu rangkaian dari X n's
n{X,Y} Mencocokan string apapun yang berisikan suatu rangkaian dari X menjadi Y n's
n{X, } Mencocokan string apapun yang berisikan suatu rangkaian dari paling sedikit X n's
n$ Mencocokan string apapun dengan n di akhiran dari ia
^n Mencocokan string apapun dengan n di awalan dari ia
?=n Mencocokan string apapun yang diikuti dengan sebuah string n
?!n Mencocokan string apapun yang tidak diikuti dengan string n

RegExp Object Properties

Properti Deskripsi
constructor Mengembalikan fungsi yang dibuat RegExp object's prototype (object Regexp awal)
global Mengecek apakah modifier "g" diatur
ignoreCase Mengecek apakah modifier "i" diatur
lastIndex Menentukan index pada yang mana untuk memulai kecocokan selanjutnya
multiline Mengecek apakah modifier "m" diatur
source Mengembalikan teks dari pola RegExp

RegExp Object Methods

Metode Deskripsi
compile() Diprotes di versi 1.5. Mengumpulkan sebuah regular expression
exec() Mengetes untuk suatu kecocokan di sebuah string. Mengembalikan kecocokan pertama
test() Mengetes untuk suatu kecocokan di sebuah string. Mengembalikan true atau false
toString() Mengembalikan nilai string dari regular expression
