Variabel JavaScript

variabel JavaScript adalah Isian untuk menyimpan (storing) nilai data.

Pada contoh ini, x, y dan z merupakan variabel.

Contoh

input

var x= 2;
var y= 8;
var z= 16;
var a= x + y + z;

output

Readbook

Nilai dari a adalah :

Dari contoh diatas kita dapat mengetahui bahwa :

  • variabel x memiliki nilai 2
  • variabel y memiliki nilai 8
  • variabel z memiliki nilai 16
  • Dan variabel a memiliki nilai 26 dari penjumlahan x + y + z.

    • Banyak Yang Menyukai Algebra

    Algebra (aljabar) merupakan pemberian nama variabel dengan nama yang mendefinisikan suatu variabel.
    Contoh price1, price2, price3 dan total (untuk menentukan hasil penjumlahan).

    Contoh

    input

    var price1 = 1000;
    var price2 = 2000;
    var price3 = 3000;
    var total = price1 + price2 + price3;

    output

    Readbook

    Harga semuanya adalah

    Dalam pemrograman, hanya seperti dalam algebra, kita menggunakan variabel (seperti price1) pemberian nama variabel.

    Dalam pemrograman, hanya seperti dalam algebra, kita menggunakan variabel untuk mengekpresikan (total = price1+price2+price3).

    Dari contoh diatas kita bisa menghitung total menjadi 10600.

    Pengidentifikasi JavaScript

    Semua variabel JavaScript harus diidentifikasi dengan nama - nama yang unik

    Nama - nama yang unik ini disebut identifiers

    Identifier bisa berisi singkatan (seperti x dan y), atau dengan nama yang mengidentifikasikan sebuah variabel (seperti umur, nilaiVolume, nilaiPhi). Aturan umum pembangunan nama untuk variabel (unique identifiers) adalah :

    • Nama variabel bisa berisi kata, number, simbol dolar, garis bawah.
    • Nama variabel harus diawali dengan sebuah kata.
  • Nama variabel bisa juga diawali oleh $ dan _ (kita jarang menggunakan cara ini).
  • Nama sangat sensitif (antara y dan Y merupakan nama berbeda).
  • Reserved word atau nama sudah ditetapkan oleh JavaScript menjadi sebuah kata kunci tidak boleh dipakai sebagai nama variabel.

    • !

    		 JavaScript Identifier merupakan huruf sensitif (case sensitive) apabila terjadi perbedaan sedikit saja pada kata akan terjadi error

    Operator Kepemilikan

    Di JavaScript, tanda (=) merupakan assignment operator (operator kepemilikan) dan bukan tanda sama dengan.

    Berikut ini perbedaan dari algebra. Berikut tidak membuat pengertian kedalam aljabar (algebra).

    Contoh

    input

    var x = x+7;

    output

    Readbook

    Nilai dari x = x+7 adalah :

    Di JavaScript. Bagaimanapun, Operator kepemilikan membuat pengertian yang sempurna. cara itu menandakan x + 7 menjadi nilai x;

    (cara itu menjumlahkan nilai x + 7 dan dimasukan kedalam nilai x. sehingga nilai x ditambah 5 )

    !

    		 Tanda sama dengan ditulis dengan tanda == dalam JavaSdript

    Tipe - tipe Data JavaScript

    Variabel JavaScript bisa memasangkan nomor seperti 100 dan nilai text seperti "David Beckham".

    Di JavaScript nilai berupa teks disebut dengan string.
    JavaScript dapat menangani banyak type data. Namun untuk sekarang, hanya berfikir tentang NUMBER dan STRING

    String ditulis didalam tanda petik (") dan ('), sedangkan number ditulis tanpa tanda petik.

    Jika kamu menaruh tanda petik diantara Number maka itu akan memperlakukannya menjadi data type String (nilai teks) bukan Number (nilai angka).

    Contoh

    input

    var phi = 3.14; // data type Number
    var nama = "Fajar";// data type String
    var hari = "10";// data type String

    Pengumuman (Menciptakan) Variabel JavaScript

    Pembuatan atau penciptaan sebuah variabel JavaScript disebut dengan Pengumuman sebuah variabel.

    Kamu mengumumkan sebuah variabel JavaScript dengan kata kunci var.

    Contoh

    input

    var nama;

    Contoh diatas mengartikan bahwa variabel nama belum memiliki nilai. (Secara teknik cara ini digunakan untuk variabel yang tidak terdeifinisi atau belum memiliki nilai)

    Untuk memberikan (assign) nilai ke Variabel gunakan tanda (=)

    Contoh

    input

    nama = "Fajar";

    Kamu juga boleh memberikan nilai saat menggunakan kata kunci var.

    Contoh

    input

    var nama = "Fajar";
