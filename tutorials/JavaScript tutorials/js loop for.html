JavaScript For Loop

Loop bisa mengeksekusi suatu blok kode a number of times

Loop JavaScript

Loop artinya cekatan (handy), jika kamu mau menjalankan kode yang sama lebih dan lebih lagi, Masing - masing waktu dengan nilai yang berbeda.

Pada banyak kasus ini adalah kasus ketika seseorang menggunakan Array:

Alternatif Penulisan

teks += cars[0] + "br";
teks += cars[1] + "br";
teks += cars[2] + "br";
teks += cars[3] + "br";
teks += cars[4] + "br";
teks += cars[5] + "br";

Loop For

Loop for seringkali kamu akan menggunakannya ketika kamu mau membuat sebuah loop.

Sintaksis:

for (pernyataan1, pernyataan2, pernyataan3) {
blok kode untuk dieksekusi
}

Pernyataan1 dieksekusi sebelum loop (blok kode) dimulai.

Pernyataan2 mendefinisikan kondisi untuk menjalankan loop (blok kode).

Pernyataan3 dieksekusi diwaktu masing - masing setelah loop (blok kode) telah dieksekusi.

Dari contoh diatas, kita dapat membacanya seperti:

Pernyataan1 menyeting sebuah variabel sebelum loop (blok kode) dimulai.

Pernyataan3 meningkatkan nilai i (i++) di masing - masing waktu blok kode dalam loop setelah dieksekusi.

Normalnya, pernyataan 1 untuk mengenalkan variabel yang digunakan dalam loop (i = 0).

Tidak selalu situasi, JavaScript tidak perduli. Pernyataan 1 adalah pilihan.

Kamu bisa menginisiatif banyak nilai dalam Pernyataan 1(Dipisahkan oleh tanda koma).

Pernyataan 2

Seringkali pernyataan 2 digunakan untuk mengevaluasi kondisi dari variabel kata pertama (initial variable).

Tidak selalu situasi, JavaScript tidak perduli. Pernyataan 2 juga merupakan pilihan.

Jika Pernyataan 2 mengeluarkan true maka loop dimulai kembali, Jika pernyataan 2 mengeluarkan false maka loop akan berhenti.

!

 Jika kamu menghilangkan pernyataan 2, maka kamu harus menyediakan break didalam loop. Harapan lainnya lop tidak akan pernah berhenti. Ini akan menghancurkan Browser kamu. Bacalah tentang break di chapter selanjutnya di tutorial ini.

Pernyataan 3

Seringkali pernyataan 3 digunakan untuk meningkatkan nilai dari variabel kata pertama (initial variable).

Tidak selalu situasi, JavaScript tidak perduli. Pernyataan 3 merupakan pilihan.

Jika Pernyataan 3 bisa melakukan apapun seperti kenaikan negatif (i--), kenaikan positif (i = i +15) atau yang lainnya.

Pernyataan 3 bisa juga dihilangkan (seperti ketika kamu menaikan nilai kamu didalam loop).
