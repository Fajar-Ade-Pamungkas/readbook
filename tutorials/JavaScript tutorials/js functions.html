Fungsi - fungsi JavaScript

Fungsi JavaScript adalah sebuah blok kode yang didesign untuk menampilkan sebuah keterangan tugas (a particular task).

Sebuah fungsi JavaScript dieksekusi ketika ada yang memicunya (memanggilnya).

 Contoh diatas digunakan ketika blok fungsi tidak menggunakan button sebagai pemicunya.

Sintaksis Fungsi JavaScript

Fungsi JavaScript didefinisikan dengan kata kunci function diikuti dengan nama dan diikuti simbol kurung ().

Nama fungsi bisa berisi kata atau huruf, angka, simbol dolar, garis bawa (aturan yang sama seperti nama variabel).

simbol kurung (parentheses) boleh berisi nama parameter yang dipisahkan dengan koma:
(paramater1, parameter2, ...)

Kode yang dieksekusi, dengan function, ditempatkan didalam simbol kurung kurawal {}

Contoh

function nama(parameter1, parameter2,...) {

Kode yang dieksekusi

}

Parameter fungsi adalah daftar nama - nama dalam definisi fungsi.

Argumen fungsi adalah nilai sesungguhnya yang diterima oleh fungsi ketika di panggil

Didalam fungsi, argument bertindak (behave) sebagai variabel lokal.

!

 Fungsi banyak kesamaan seperti prosedur atau subroutine di bahasa pemrograman yang lainnya.

JavaScritp Invocation

Kode dalam sebuah fungsi akan dieksekusi ketika sesuatu memanggil sebuah fungsi.


  • Ketika peristiwa terjadi (ketika pengguna mengklik sebuah button atau tombol).
  • Ketika fungsi dipanggil oleh kode JavaScript.
  • Secara otomatis (memanggil dirinya sendiri).

Kamu akan belajar banyak tentang invocation JavaScript di lain Chapter.

Fungsi Return

Ketika JavaScript menjangkau return statement, fungsi akan menghentikan pengeksekusian.

Jika fungsi memanggil pernyataan, JavaScript akan "mengembalikan" eksekusi kode setelah pernyataan pemanggilan atau permohonan.

Fungsi sering memperhitungkan nilai return. Nilai return "dikembalikan" lagi menjadi pemanggil.

Contoh

input

var x = pageFunction2(a,b); // fungsi dipanggil, nilai return akan berakhir kembali di x

function pageFunctions1(a,b) {

return a * b

// fungsi mengembalikan hasil dari a dan b

}

Kenapa Fungsi

Kamu bisa menggunakan kembali kode : menetapkan sekali lagi (once) kode, dan menggunakannya lagi.

Kamu bisa menggunakan kode yang sama dibanyak waktu dengan pernyataan yang berbeda, untuk menghasilkan keluaran yang berbeda.
