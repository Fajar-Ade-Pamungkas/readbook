JavaScript Reserved Words

Dalam JavaScript, beberapa pengidentifikasi seperti reserved words (kata - kata yang ditetapkan oleh javascript sebagai kata kunci, metode dan properti) tidak bisa digunakan untuk penamaan variabel dan fungsi.

Standar - standar JavaScript

ECMAScript 3 (ES3) dirilis pada bulan December 1999.

ECMAScript 4 (ES4) dihentikan (ditelantarkan).

ECMAScript 5 (ES5) dirilis pada bulan December 2009.

ECMAScript 6 (ES6) dirilis pada bulan June 2015, dan merupakan versi resmi terakhir dari JavaScript.

Waktu berlalu dan kita sekarang sedang mulai melihat kelengkapan yang mendukung ES5/ ES6 di semua browser modern.

JavaScript Reserved Words

Dalam JavaScript, kamu tidak bisa menggunakan "kata - kata yang dipesan" sebagai nama variabel, label, ataupun fungsi.

Tabel Kata - kata yang Dipesan
abstract arguments boolean break byte
case catch char class* const
continue debugger default delete do
double else enum* eval export*
extends* false final finally float
for function goto if implements
import* in instanceof int interface
let long native new null
package private protected public return
short static super* switch synchronized
this throw throws transient true
try typeof var void volatile
while with yield

Kata yang ditandai * adalah kata baru dalam ECMAScript5.

Object, Properti dan Metode JavaScript

Kamu juga harus menghindari penggunaan nama pembangun Object, Properti dan Metode JavaScript:

Tabel Kata Pembangun Object, Properti dan Metode
Array Date eval function hasOwnProperty
Infinity isFinite isNaN isPrototypeOf length
Math NaN name Number Object
prototype String toString undefined valueOf

Java Reserved Words

JavaScript sering kali digunakan bersama Java, Kamu lebih baik menghindari penggunaan beberapa Object dan Properti Java sebagai pengidentifikasi JavaScript.

Tabel Kata - kata yang Dipesan Java
getClass java JavaArray javaClass JavaObject JavaPackage

Windows Reserved Words

JavaScript bisa digunakan diluar HTML. Bisa digunakan sebagai bahasa pemrograman di banyak aplikasi lainnya.
Di HTML kamu harus (untuk portabilitas kamu) hindari penggunaan nama dari object dab properti HTML dan Windows.

Tabel Kata - kata yang Dipesan Windows
alert all anchor anchors area
assign blur button checkbox clearInterval
clearTimeout clientInformation close closed confirm
constructor crypto decodeURI decodeURIComponent defaultStatus
document element elements embed embeds
encodeURI encodeURIComponent escape event fileUpload
focus form forms frame innerHeight
innerWidth layer layers link location
mimeTypes navigate navigator frames frameRate
hidden history image images offscreenBuffering
open opener option outerHeight outerWidth
packages pageXOffset pageYOffset parent parseFloat
parseInt password pkcs11 plugin prompt
propertyIsEnum radio reset screenX screenY
scroll secure select self setInterval
setTimeout status submit taint text
textarea top unescape untaint window

HTML Event Handlers

HTML Event Handlers adalah pengatur - pengatur kejadian HTML
Pada tambahan kamu lebih baik menghindari penggunaan nama semua HTML Event Handlers.

Contoh:

Tabel HTML Event Handlers
onblur onclick onerror onfocus
onkeydown onkeypress onkeyup onmouseover
onload onmouseup onmousedown onsubmit
