JavaScript Array Iteration Methods

Metode Iteration (perkataan berulang - ulang ) Array mengoperasi atau mengeksploitasi setiap item array.

Array.forEach()

Metode forEach() memanggil sebuah fungsi (fungsi yang telah dipanggil) suatu kejadian untuk setiap elemen array:

Contoh

input

var txt = "";
var numbers = [45, 4, 9, 16, 25];
numbers.forEach(myFunction);

function myFunction(value, index, array) {
txt = txt + value + "br";
}

output

Catatan fungsi itu mengambil 3 pendapat:

  • Nilai item
  • Index item
  • Array itu sendiri

Contoh dibawah hanya menggunakan parameter nilai. Contoh itu bisa ditulis kembali menjadi:

Contoh

input

var txt = "";
var numbers = [45, 4, 9, 16, 25];
numbers.forEach(myFunction);

function myFunction(value) {
txt = txt + value + "br";
}

output

Array.forEach() didukung di semua browser kecuali Internet Explorer atau versi sebelumnya.

BROWSER SUPPORT
Yes 9.0 Yes Yes Yes

Array.map()

Metode map() menciptakan sebuah array baru dengan pengerjaan sebuah fungsi pada setiap elemen array.
Metode map() tidak mengeksekusi fungsi untuk elemen - elemen array tanpa nilai - nilai.
Metode map() tidak mengubah array asli.
Contoh ini menggandakan setiap nilai array dengan 2:

Contoh

input

var numbers1 = [45, 4, 9, 16, 25];
var numbers2 = numbers1.map(myFunction);

function myFunction(value, index, array) {
return value * 2;
}

output

Catatan fungsi itu mengambil 3 pendapat:

  • Nilai item
  • Index item
  • Array itu sendiri

Ketika fungsi callback hanya menggunakan parameter nilai, index dan parameter array bisa dihilangkan:

Contoh

input

var numbers1 = [45, 4, 9, 16, 25];
var numbers2 = numbers1.map(myFunction);

function myFunction(value) {
return value * 2;
}

output

Array.map() didukung di semua browser kecuali Internet Explorer atau versi sebelumnya.

BROWSER SUPPORT
Yes 9.0 Yes Yes Yes

Array.filter()

Metode filter() menciptakan sebuah array baru dengan elemen - elemen array yang melewati sebuah tes.
Contoh ini menciptakan sebuah array baru dari elemen - elemen dengan nilai lebih besar daripada 18:

Contoh

input

var numbers1 = [45, 4, 9, 16, 25];
var numbers2 = numbers1.filter(myFunction);

function myFunction(value, index, array) {
return value > 18;
}

output

Catatan fungsi itu mengambil 3 pendapat:

  • Nilai item
  • Index item
  • Array itu sendiri

Pada contoh dibawah, callback function tidak menggunakan parameter index dan array, jadi itu bisa dihilangkan:

Contoh

input

var numbers1 = [45, 4, 9, 16, 25];
var numbers2 = numbers1.filter(myFunction);

function myFunction(value) {
return value > 18;
}

output

Array.filter() didukung disemua browser kecual Internet Explorer 8 atau versi sebelumnya.

BROWSER SUPPORT
Yes 9.0 Yes Yes Yes

Array.reduce()

Metode reduce() menjalankan sebuah fungsi pada setiap element array untuk mengeluarkan (menguranginya menjadi) sebuah nilai tunggal.
Metode reduce() mengerjakan dari kiri-ke-kanan dalam array. Lihatlah juga reduceRight().

Metode reduce() tidak mengurangi array asli

Contoh ini mencari jumlah (sum) dari semua number pada sebuah array:

Contoh

input

var numbers1 = [45, 4, 9, 16, 25];
var sum = numbers1.reduce(myFunction);

function myFunction(total, value, index, array) {
return total + value;
}

output

Catatan fungsi itu mengambil 4 pendapat:

  • total ( nilai awal / nilai yang dikembalikan sebelumnya )
  • Nilai item
  • Index item
  • Array itu sendiri

Contoh dibawah tidak menggunakan parameter index dan array, bisa ditulis ulang menjadi:

Contoh

input

var numbers1 = [45, 4, 9, 16, 25];
var sum = numbers1.reduce(myFunction);

function myFunction(total, value) {
return total + value;
}

output

Metode reduce() bisa menerima atau menyetujui sebuah nilai initial (awal)

Contoh

input

var numbers1 = [45, 4, 9, 16, 25];
var sum = numbers1.reduce(myFunction, 100);

function myFunction(total, value) {
return total + value;
}

output

Array.reduce() didukung disemua browser kecual Internet Explorer 8 atau versi sebelumnya.

BROWSER SUPPORT
Yes 9.0 Yes Yes Yes

Array.every()

Metode every() mengecek jika semua nilai array melewati suatu tes.
Contoh init mengecek jika semua nilai array lebih besar daripada 18

Contoh

input

var numbers1 = [45, 4, 9, 16, 25];
var allOver18 = numbers1.every(myFunction);

return value > 18;
}

output

Catatan fungsi itu mengambil 3 pendapat:

  • Nilai item
  • Index item
  • Array itu sendiri

Pada contoh dibawah, callback function tidak menggunakan parameter index dan array (hanya value), jadi itu bisa dihilangkan:

Contoh

input

var numbers1 = [45, 4, 9, 16, 25];
var allOver18 = numbers1.every(myFunction);

function myFunction(value) {
return value > 18;
}

output

Array.every() didukung disemua browser kecual Internet Explorer 8 atau versi sebelumnya.

Array.indexOf()

Mencari sebuah array untuk sebuah nilai element dan mengembalikkan pada posisinya.
Catatan: item pertama memiliki posisi 0, item kedua posisi ke 1 dan seterusnya.

Contoh

input

var fruits = ["Apple", "Orange", "Apple", "Mango"];
var a = fruits.indexOf("Apple");

output

Array.indexOf() didukung disemua browser kecual Internet Explorer 8 atau versi sebelumnya.

BROWSER SUPPORT
Yes 9.0 Yes Yes Yes
Sintaksis:
Array.indexOf(item, start)
item Diperlukan. Item untuk mencari element yang dimaksud
start Pilihan. Dimana untuk mulai mencari.Nilai negatif akan memulainya penghitungan posisi dari belakang lalu mencari sampai ke akhir.

Array.indexOf mengembalikan -1 jika item tidak ditemukan.
Jika item item mempersembahkan lebih dari yang semestinya, ia alan mengembalikan posisi dari kejadian pertama.

Array.lastIndexOf()

Array.lastIndexOf() sama seperti Array.indexOf(), tetapi mencari dari akhir array.

Contoh

input

var fruits = ["Apple", "Orange", "Apple", "Mango"];
var a = fruits.lastIndexOf("Apple");

output

Array.lastIndexOf() didukung disemua browser kecual Internet Explorer 8 atau versi sebelumnya.

BROWSER SUPPORT
Yes 9.0 Yes Yes Yes
Sintaksis:
Array.lastIndexOf(item, start)
item Diperlukan. Item untuk mencari element yang dimaksud
start Pilihan. Dimana untuk mulai mencari.Nilai negatif akan memulainya penghitungan posisi dari belakang lalu mencari sampai ke awal.

Array.find()

Metode find() mengembalikan nilai dari element array pertama yang melewati sebuah fungsi tes.
Contoh ini menemukan (mengembalikan nilai dari) element pertama dengan nilai lebih besar daripada 18:

Contoh

input

var numbers = [4, 9, 16, 25, 29];
var first = numbers.find(myFunction);

function myFunction(value, index, array) {
return value > 18;
}

output

Catatan fungsi itu mengambil 3 pendapat:

  • Nilai item
  • Index item
  • Array itu sendiri

Array.find() tidak didukung di browser terdahulu, Versi - versi browser pertama dengan dukungan penuh didaftar bawah ini.

BROWSER SUPPORT
45 12 25 8 32
