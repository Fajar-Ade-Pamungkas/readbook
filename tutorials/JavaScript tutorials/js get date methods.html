JavaScript Get Date Methods

Metode - metode ini bisa digunakan untuk mendapatkah informasi tentang suatu object tanggal (date):

Tabel Metode Get Date
Metode Deskripsi
getDate() Mendapatkan tanggal sebagai number (1-30)
getDay() Mendapatkan hari sebagai number (0-6)
getFullYear() Mendapatkan tahun dengan 4 digit number (YYYY)
getHours() mendapatkan jam (0-23)
getMilliSeconds() Mendapatkan milidetik (0-999)
getMinutes() Mendapatkan menit (0-59)
getMonth() Mendapatkan bulan (0-11)
getSeconds() Mendapatkan detik (0-59)
getTime() Mendapatkan waktu (milidetik sejak Januari 1, 1970)
Date.now() Mendapatkan date (tanggal), ECMAScript 5.

Metode getTime()

metode getTime() mengembalikan number milidetik sejak 1 Januari 1970:

Contoh

input

p id="demo"/p

script

var x = new Date();
document.getElementById("demo").innerHTML=x.getTime();

/script

output

Tunjukan milidetik

Metode getFullYear()

metode getFullYear() mengembalikan Tahun dari waktu saat ini seperti 4 digit number:

Contoh

input

p id="demo"/p

script

var x = new Date();
document.getElementById("demo").innerHTML=x.getFullYear();

/script

output

Sekarang tahun ...

Metode getMonth()

metode getMonth() mengembalikan bulan dari waktu saat ini sebagai number (0 - 11):

Contoh

input

p id="demo"/p

/script

output

Di JavaScript, bulan pertama (Januari) disimbolkan menjadi nomor 0 dan Desember disimbolkan menjadi 11

Kamu bisa menggunakan array nama dan getDate() untuk mendapatkan hari dengan nama - nam hari.

Contoh

input

p id="demo"/p

script

var x = new Date();
var namemonths = [
"January",
"February",
"March",
"April",
"May",
"June",
"July",
"August",
"September",
"October",
"November",
"December"
];
document.getElementById("demo").innerHTML=x.getDate();

/script

output

Sekarang bulan ...

Metode getDay()

metode getDay() mengembalikan hari dalam seminggu sebagai number (0-6)

Contoh

input

p id="demo"/p

script

var x = new Date();
document.getElementById("demo").innerHTML=x.getDay();

/script

output

Di JavaScript, hari pertama dalam seminggu (0) diartikan sebagai hari minggu, bahkan jika dalam beberapa negara didunia memperhitungkan (consider) hari pertama dalam seminggu diartikan sebagai hari Senin.

Kamu juga bisa menggunakan array dengan nama hari dan getDay() untuk mengembalikan hari menjadi sebuah nama hari.

Contoh

input

p id="demo"/p

script

var x = new Date();
var hari = ["Minggu", "Senin", "Selasa", "Rabu", "Kamis", "Jum'at", "Sabtu"];
document.getElementById("demo").innerHTML=hari[x.getDay()];

/script

output

Metode Date UTC

Metode Date UTC digunakan untuk pengerjaan dengan Date UTC (Universal Time Zone dates):

Metode Set Date

Metode Get digunakan untuk menyeting (mengatur) bagian dari sebuah Date (tanggal). Disini adalah metode get yang paling umum (secara alpabet):

Tabel Metode UTC
Metode Deskripsi
getUTCDate() Sama seperti getDate(), tapi dalam zona waktu UTC
getUTCDay() Sama seperti getDay(), tapi dalam zona waktu UTC
getUTCFullYear() Sama seperti getFullYear(), tapi dalam zona waktu UTC
getUTCHours() Sama seperti getHours(), tapi dalam zona waktu UTC
getUTCMilliSeconds() Sama seperti getMilliSeconds(), tapi dalam zona waktu UTC
getUTCMinutes() Sama seperti getMinutes(), tapi dalam zona waktu UTC
getUTCMonth() Sama seperti getMonth(), tapi dalam zona waktu UTC
getUTCSeconds() Sama seperti getSeconds(), tapi dalam zona waktu UTC
