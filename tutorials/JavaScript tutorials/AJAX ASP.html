AJAX ASP

AJAX bisa digunakan untuk membuat lebih aplikasi interaktif.

Contoh ASP AJAX

Contoh berikut akan menunjukan bagaiaman sebuah halaman web busa berkomunikasi dengan sebuah web server saat pengguna mengetik karakter dalam bidang input:

Contoh

Start typing a name in the input field below:

First name:

Suggestions:

Contoh Menjelaskan

Pada contoh diatas, ketika pengguna mengetik sebuah karakter pada bidang inout, sebuah fungsi yang disebut "showHint()" dieksekusi.

Contoh dicetuskan oleh event onkeyup.

Disini adalah kode HTML:

Contoh

Input:

<html>
<head>
<script>
function showHint(str) {
if (str.length == 0) {
document.getElementById("txtHint").innerHTML = "";
return;
} else {
var xmlhttp = new XMLHttpRequest();
xmlhttp.onreadystatechange = function() {
if (this.readyState == 4 && this.status == 200) {
document.getElementById("txtHint").innerHTML = this.responseText;
}
};
xmlhttp.open("GET", "gethint.asp?q=" + str, true);
xmlhttp.send();
}
}
</script>
</head>
<body>

<p><b>Start typing a name in the input field below:</b></p>
<form>
First name: <input type="text" onkeyup="showHint(this.value)">
</form>
<p>Suggestions: <span id="txtHint"></span></p>
</body>
</html>

Output:

Penjelasan kode:

Pertama, periksalah jika input field itu kosong (str.length === 0). Jika itu benar, bersihkan konten placeholder "txtHint" dan keluarkan fungsi.

Akan tetapi, jika bidang input tidak kosong, lakukanlah hal berikut:


  • Buatlah sebuah object XMLHttpRequest
  • Buatlah fungsi untuk dieksekusi ketika respon server siap
  • Kirimlah request off (permintaan mati) ke file ASP (gethint.asp) pada server
  • Perhatikanlah parameter q ditambahkan ke gethint.php?q="+str
  • variabel str menggenggam konten dari input field atau bidang masukan
