JavaScript Object Prototypes

Setiap JavaScript memiliki sebuah prototype. Prototype juga merupakan sebuah object.
Semua Object JavaScript mewarisi properti dan metode mereka dari prototype mereka.

JavaScript Prototypes

Semua object JavaScript mewariskan propertj dan metode untuk prototipe mereke.

Object yang tercipta mengggunakan sebuah object literal, atau dengan new Object(), inherit dari sebuah prototype disebut Object.prototype.

Object yang tercipta dengan new Date() mewariskan Date.prototype. Object.prototype berada diatas sebuah rantai purwarupa (prototype chain).

Semua object JavaScript (Date, Array, RegExp, Function, ...) inherit (mewariskan) dari Object.prototype.

Penciptaan Sebuah Prototype

Cara standar untuk menciptakan sebuah Object prototype adalah menggunakan sebuah fungsi pembangun object: The standard way to create an object prototype is to use an object constructor function:

Contoh

function (first, last, age, eyeColor) {
this.firstName = first;
this.lastName = last;
this.age = age;
this.eyeColor = eyeColor;
}

Dengan sebuah fungsi pembangun, kamu bisa menggunakan kata kunci new untuk menciptakan object baru dari prototype yang sama:

Contoh

var myBrother1 = new person(Andi, Oka, 30, brown);
var myBrother2 = new person(Iqbal, Abdillah, 28, brown);
var myBrother3 = new person(Rizki, Oka, 23, brown);

Output

Fungsi pembangun merupakan prototype (purwarupa) untuk object person kamu.

Penambahan Properti dan Metode untuk Object

Terkadang kamu ingih menambahkan propeeti baru (atau metode baru) untuk sebuah object yang sudah ada.
Terkadang kamu ingih menambahkan propeeti baru (atau metode baru) untuk semua object yang sudah ada dari sebuah tipe yang diberikan.
Terkadang kamu ingih menambahkan propeeti baru (atau metode baru) untuk sebuah object prototype.

Penambahan sebuah Properti pada sebuah Object

penambahan sebuah properti pada sebuah object yang sudah ada itu mudah:

Contoh 2

Input

var myBrother1 = new person(Andi, Oka, 30, brown);
var myBrother2 = new person(Iqbal, Abdillah, 28, brown);
var myBrother3 = new person(Rizki, Oka, 23, brown); var myBrother1.Nasionality = "Indonesia";

Output

Properti baru akan ditambahkan pada var myBrother1 bukan untuk var myBrother2 atau myBrother3 dan bukan juga untuk variabel lainnya.

Penambahan sebuah Metode pada sebuah Object

penambahan sebuah Metode pada sebuah object yang sudah ada itu mudah:

Contoh 2

Input

var myBrother1 = new person(Andi, Oka, 30, brown);
var myBrother2 = new person(Iqbal, Abdillah, 28, brown);
var myBrother3 = new person(Rizki, Oka, 23, brown);
myBrother2 = function() {
return this.firstName + " " + this.lastName;
}

Output

Metode baru akan ditambahkan pada var myBrother1 bukan untuk var myBrother2 atau myBrother3 dan bukan juga untuk variabel lainnya.

Penambahan Properti pada Prototype

Kamu tidak bisa menambahkan sebuah propeeri baru untuk sebuah prototype dengan cara yang sama seperti kamu menambahkan properti baru untuk object yang sudah ada, karena prototype bukan sebuah object yang sudah ada.

Contoh 2

Input

var person = {firstName : "Fajar Ade"; lastName : "Pamungkas"; age : 20; eyeColor : "brown";}
var person.Nasionality = "Indonesia";

Output

Untuk menambahkan sebuah properti baru untuk sebuah pembangun, kamu harus menambahkannya ke fungsi pembangun:

Contoh

function (first, last, age, eyeColor) {
this.firstName = first;
this.lastName = last;
this.age = age;
this.eyeColor = eyeColor;
this.Nationality = "Indonesia";
}

Output

Properti prototype bisa memiliki sebuah nilai prototype (nilai default)

Penambahan Metode pada Prototype

Fungsi pembangun kamu bisa juga mendefinisikan sebuah metode:

Contoh

function (first, last, age, eyeColor) {
this.firstName = first;
this.lastName = last;
this.age = age;
this.eyeColor = eyeColor;
this.name = function() {
return this.firstName + " " + lastName ;
}
}

Output

Penggunaan Propertu Prototype

Properti JavaScript mengizinkan kamu untuk menambahkan properti baru untuk sebuah prototype yang telah ada:

Contoh

function (first, last, age, eyeColor) {
this.firstName = first;
this.lastName = last;
this.age = age;
this.eyeColor = eyeColor;
}
person.prototype.Nationality = "Indonesian";

Output

Properti prototype JavaScript juga memperbolehkan kamu untuk menambahakan metode baru untuk prototype yang sudan ada: The JavaScript prototype property also allows you to add new methods to an existing prototype:

Contoh

function (first, last, age, eyeColor) {
this.firstName = first;
this.lastName = last;
this.age = age;
this.eyeColor = eyeColor;
}
person.prototype.name = function() {
return this.firstName + " " + lastName ;
}

Output

Hanya memodifikasi prototype milik kamu sendiri. tidak pernah memodifikasi prototype dari object JavaScript standar.
