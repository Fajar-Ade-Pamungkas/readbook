JavaScript Timing Events

JavaScript bisa dieksekusi dalam time-interval.

Ini disebut dengan timing events.

Timing Eventa

Window object mengizinkan pengeksekusian dari kode pada interval waktu tertentu.

Interval waktu ini disebut dengan timing event.

Dua metode kunci untuk menggunakan dengan JavaScript adalah:


  • setTimeout(function, milliseconds)
    • Mengeksekusi sebuah fungsi, setelah menunggu sebuah nomor tertentu dari milidetik.
  • setInterval(function, milliseconds)
    • Sama seperti setTimeout(), tapi membolak balik pengeksekusian dari fungsi dengan terus - menerus

setTimeout() dan setInterval() kedua metode dari object HTML DOM Window.

Metode setTimeout()

window.setTimeout(function, milliseconds);

window.setTimeout bisa ditulis tanpa window prefix.

Parameter pertama adalah fungsu untuk dieksekusi.

Parameter kedua mengindikasikan nomor dari milli detik sebelum pengeksekusian.

Contoh

Klik tombol. tunggu 3 menit, dan halaman akan mengerluarkan alert "Hello":

Input:

<button onclick="setTimeout(myFunction, 3000)">Klik disini</button>

<script>
function myFunction() {
alert("Hello");
}
<script>

Bagaimana untuk Menghentikan Pengeksekusian?

Metode clearTimeout menghentikan pengeksekusian dsri fungsi trrtentu pada seTimeout().

window.clearTimeout(timeoutVariable)

Metode window.clearTimeout() bisa ditulis tanpa window prefix.

Metode clearTimeout() menggunakan variabel yang dikembalikan dari setTimeout():

myVar = setTimeout(function, milliseconds);
clearTimeout(myVar);

Jika fungsi belum dieksekusi, kamu bisa menghentikan pengeksekusian dengan pemanggilan metode clearTimeout():

Contoh

Contoh yang sama seperti diatas, tapi dengan ditambahkan sebuah tombol "stop":

Input:

<button onclick="myVar = setTimeout(myFunction, 3000)">Klik disini</button>
<button onclick="clearTimeout(myVar)"">Klik disini</button>

<script>
function myFunction() {
alert("Hello");
}
<script>

Metode setInterval()

Metode setInterval() mengulang-ulang fungsi yang diberikan di setiap time-interval yang diberi.

window.setInterval(function, milliseconds);

Metode window.setInterval() bisa ditulis tanpa window prefix.

Parameter pertama asalah fungsi untuk dieksekusi.

Parameter kedua mengindikasikan length (panjang) dari time-interval antara setiap pengeksekusian.

Contoh mengeksekusi sebuah fungsi yang dinamai "myTimer" sekilas (once) setiap detik (seperti jam digital).

Contoh

Contoh yang sama seperti diatas, tapi dengan ditambahkan sebuah tombol "stop":

Input:

<script>

var myVar = setInterval(myTimer, 1000);
function myTimer() {
var d = new Date();
document.getElementById("demo").innerHTML =
d.toLocaleTimeString();
}
<script>

Output:

Klik tombol dibawah untuk menampilkan waktu digital

1000 milidetik (millisecond) sama dengan 1 detik.
