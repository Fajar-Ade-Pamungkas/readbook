Object JavaScript

Real Life Objects, Properties And Methods

Di dunia nyata, sebuah mobil adalah object

Sebuah mobil memiliki properti seperti berat mobil dan warna mobil. Dan metode seperti mulai dan berhenti.

Object Propertis Methods
car.name = lamborghini car.start()
car.model = 300 car.drive()
car.weight = 450 kg car.brake()
car.color=gold car.stop()

Semua mobil memiliki properti yang sama, namun nilai properti berbeda dari mobil yang satu dan yang lainnya.

Semua mobil memiliki metode yang sama, namun metode yang ditunjukan pada waktu - waktu yang berbeda.

Object JavaScript

Kamu telah mempelajari bahwa variabel Javascript adalah isian dari tipe data.

Kode ini memberikan nilai sederhana (Ferrari) pada variabel yang bernama car.

Contoh

input

var car = "Ferrari";

document.getElementById("demo").innerHTML=car ;

Object adalah variabel juga, namun object dapat berisi banyak nilai.

Kode ini memberikan banyak nilai {name:"Nissan", model:2000, color:"darkblue"} pada variabel yang bernama car.

Contoh

input

var car = {name:"Nissan", model:2000, color:"darkblue"};

document.getElementById("demo").innerHTML=car.name ;

Nilai ditulis seperti pasangan nama:nilai (nama dan nilai dipisahkan dengan titk dua (:) )

!

 Object JavaScript adalah isian untuk nilai yang dinamai

Properti Object

pasangan (pair) nama:nilai (dalam object javaScript) disebut properti object.

Contoh

input

var me = {nama:"Jigol", alamat:"Rain City", favorColor:"darkblue", state:"single"};

document.getElementById("demo").innerHTML= me.state;

Nama Properti Nilai Properti
nama Jigol
alamat Rain City
favorColor darkblue
state single

Metode Object

Metode adalah aksi yang bisa dipertunjukan pada object

Metode disimpan didalam properti sebagai definisi fungsi.

Nama Properti Nilai Properti
nama Jigol
alamat Rain City
favorColor darkblue
state single
fullSet function() {return this.nama + " " + this.state}

!

 Object JavaScript adalah isian untuk nilai sudah dinamai (disebut properti) dan metode.

Pendefinisan Object

Kamu mendefinisikan (menciptakan) object JavaScript dengan object literal (object tetap).

Contoh

input

var me = {nama:"Jigol", alamat:"Rain City", favorColor:"darkblue", state:"single"};

Spasi dan garis baris tidak penting, pendefinisian object bisa mempersempit baris kedalam baris baru.

Contoh

input

var me = {nama:"Jigol",
alamat:"Rain City",
favorColor:"darkblue",
state:"single";
}

Accessing Object Properties

Kamu bisa mengakses properti object dalam dua cara :

Akses properti 1

Contoh

input

<!DOCTYPE html>

<html>

<head>
</head>

<body>

<p> Biodata diri </p>

<button onclick="myObjects()">Klik disini</button>

var me = {
nama:"Jigol",
alamat :"Rain City",
favorColor:"darkblue",
state:"single"

document.getElementById("myObjects01").innerHTML=
"Nama saya" + " " + me.nama + " <br>" +
"Kota" + " " + me.alamat + "<br>" +
"Warna favorit" + " " + me.favorColor + "<br>" +
"Status" + " " + me.state;
}

</script>

</body>

<>/html>

output

Biodata diri saya

Pengaksesan Metode Object

Kamu bisa mengakses metode object dengan menggunakan sintaksis berikut :

objectName.methodName

Contoh

input

<!DOCTYPE html>

<html>

<head>
</head>

<body>

<p> Biodata diri </p>

<button onclick="myObjects3()">Klik disini</button>

<p id="myObjects03"></p>

<script>

function myObjects3() {
var me = {
nama:"Jigol",
alamat :"Rain City",
favorColor:"darkblue",
state:"single"
fullnama: function () {
return this.nama + " " + this.alamat;
}
};

document.getElementById("myObjects03").innerHTML= me.fullNama();
}

</script>

</body>

<>/html>

output

Jangan Deklarasikan Kata String, Number, Dan Boolean Sebagai Sebuah Object

Ketika Variabel JavaScript dideklarasikan dengan kata kunci new. Variabel diciptakan sebagai sebuah string.

Contoh

var x = new String(); // mendeklarasikan x sebagai sebuah string.

var y = new Number(); // mendeklarasikan y sebagai sebuah Number.

var x = new Boolean(); // mendeklarasikan z sebagai sebuah Boolean.

Hindari object String, Number dan Boolean, karena dapat mengkomplikasi kode kamu dan memperlambat kecepatan pengeksekusian.
