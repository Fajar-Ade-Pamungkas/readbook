JavaScript Cookies

Cookies membiarkan kamu untuk menyusun atau menyimpan informasi pengguna pada web page.

Apa itu Cookies?

Cookies adalah ada, yang disimpan atau disusun di small text files, pada komputer kamu.

Ketika sebuah web server telah mengirim sebuah web page ke browser, Koneksi dalam keadaan mati (shut down), dan server melupakan segalanya tentang pengguna.

Cookies diciptakan untuk mencegah masalah "Bagaimana untuk mengingat informasi (tentang) pengguna":

  • Ketika seorang pengguna mengunjungi sebuah web page, nama si pengguna akan disimpan dalam cookie.
  • Dilain waktu pengguna mengunjungihalaman (page), cookie "mengingat" nama pengguna tersebut.

Cookies disimpan dalam sepasang nama=nilai seperti:

username = John Doe

Ketika sebuah browser meminta sebuah web page dari server, cookie termasuk ke page yang dimuat untuk permintaan. Cara ini server mendapatkan data yang diperlukan untuk "mengingat" informasi (tentang) pengguna.

Memciptakan sebuah Cookie dengan JavaScript

JavaScript bisa membuat, membacw dan menghapus cookies dengan properti document.cookie.

Dengan JavaScript, cookie bisa dibuat atau diciptakan seperti ini:

document.cookie=username=Fajar Ade;

Kamu bisa juga menambahkam sebuah waktu terakhir (pada UTC time). Dengan default, cookie dihaous ketika browser ditutup:

document.cookie=username=Fajar Ade; expires=Thu, 18 Dec 2013 12:00:00 UTC;

Dengan sebuah path parameter, kamu bisa memberitahu browser apa path (jalan setapak) cookie semestinya. Dehgan default, cookie beserta ke halaman saat ini.

document.cookie=username=Fajar Ade; expires=Thu, 18 Dec 2013 12:00:00 UTC; path=/;

Membaca sebuah Cookie dengan JavaScript

Dengan JavaScript, cookies bisa dibaca seperti ini:

var x = document.cookie;

document.cookie akan mengembalikan semua cookies pada satu string seperti: cookie1=nilai; cookie2=nilai; cookie3=nilai;

Mengubah sebuah Cookie dengan JavaScript

Dengan JavaScript, kamu bisa mengubah sebuah cookie dengan cara yang sama seperti membuat cookie:

document.cookie=username=Fajar Ade pamungkas; expires=Thu, 18 Dec 2013 12:00:00 UTC; path=/;

Cookie yang lama ditulis terlalu banyak (overwritten).

contoh diatas ditambahkan nama belakang "pamungkas".

Penghapusam sebuah Cookie dengan JavaScript

Penghapusam sebuah cookie sangat sederhana. Hanya mengatur expires parameter ke sebuah tanggal dilewati:

document.cookie=username=; expires=Thu, 18 Dec 2013 12:00:00 UTC; path=/;

Note: Bahwa kamu tidak harus menentukan sebuah nilai cookie ketika kamu menghapus sebuah cookie.

String Cookie

Properti document.cookie terlihat seperti teks string normal. Tapi padahal tidak.

Bahkan jika kamu menulis sejumlah string cookie untuk document.cookie, ketika kamu menulisnya diluar (lagi), kamu hanya bis melihat pasangan name-value dari itu.

Jika kamu me-setting cookie baru, cookie yang lebih lama tidak ditulis lebih (overwritten). Cookie baru ditambahkan ke document.cookie, jadi jika kamu membaca document.cookie lagi kamu akan mendapatkan sesuatu seperti:

cookie1=nilai; cookie2=nilai;

Jika kamu mau mencari nilai dari satu cookie tertentu, kamu harus menulis sebuah fungsi JavaScript yang mencari nilai cookie dalam string cookie.

Contoh Cookie JavaScript

Pada contoh untuk mengikuti, kita akan membuat sebuah cookie yang menyimpan atau menyusun nama dari seorang pengunjung.

Waktu pertama pengunjung datang di halaman yang sama, pengunjung akan diminta untuk mengisi namanya. Nama kemudian disimpan pada sebuah cookie.

Waktu selanjutnya pengunjung datang pada halaman yang sama, pengunjung akan mendapatkan sebuah pesan selamat datang.

Untuk contoh kita akan membuat 3 fungsi JavaScript:

  • Sebuah fungsi untuk mengatur sebuah nilai cookie
  • Sebuah fungsi untuk mendapatkan sebuah nilai cookie
  • Sebuah fungsi untuk mengecek sebuah nilai cookie

Fungsi untuk Mengatur Cookie

Pertama - tama, kita membuat sebuah fungsi yang menyimpan nama dari pengunjung di sebuah variabel cookie:

Contoh

Input:

function setCookie(cname, cvalue, exdays) {
var d = new Date();
d.setTime(d.getTime() + (exdays*24*60*60*1000));
var expires = "expires=" + d.toUTCString();
document.cookie = cname + "=" + cvalue + "; " + expires;
}

Output:

Penjelasan Contoh:

Parameter dari fungsi diatas adalah nama dari cookie (cname), nilai dari cookie (cname), dan nomor dari hari sampai cookie harus berakhir atau expires (exdays).

Fungsi mengatur sebuah cookie dengan penambahan bersama cookie name, cookie value dan expires string.

Fungsi untuk Mendapatkan Cookie

Lalu, kita buat sebuah fungsi yang mengembalikan nilai dari sebuah cookie tertentu:

Contoh

Input:

function getCookie(cname) {
var name = cname + "=";
var ca = document.cookie.split(';');
for (var i = 0; i = <ca.length; i++) {
var c = ca[i];
while (c.charAt(0) == ' ') c = c.substring(1);
if (c.indexOf(name) == 0) return c.substring(name.length, c.length);
}
return "";
}

Output:

Fungsi Menjelaskan

Membawa cookiename sebagai parameter (cname).

Buatlah sebuah variabel (name) dengan teks untuk mencari (cname + "=").

Splite atau robeklah document.cookie pada semicolon masuk kedalam sebuah array dengan nama ca (ca = document.cookie.split(";")).

Loop through atau simpul sepanjang array ca (i = 0; i < ca.length; i++), dan bacalah setiap nilai c = ca[i]).

Jika cookie ditemukan (c.indexOf(name) == 0), kembalikan nilai dari cookie (c.substring(name.length, c.length, c.length)).

Jika cookie ditemukan, kembalikan "".

Fungsi untuk Mengecek sebuah Cookie

Terakhir, kita buat fungsi yang mengekcek jikasebuah cookie diatur.

Jika cookie diatur, cookie akan menampilakn sebuah sambutan.

Jika cookie tidak diatur, cookie akan menampilkan sebuah prompt box, permintaan untuk nama dari pengguna, dan menyimpan username cookie untuk 365 hari atau selama 1 tahun, dengan pemanggilan fungsi setCookie:

Contoh

Input:

function checkCookie(){
var username = getCookie("username");
if (username != "") {
alert("Welcome again " + username);
} else{
username = prompt("Please enter your name:", "");
if (username != "" && username != null) {
setCookie("username", username, 365);
}
}
}

Output:
