CSS Image Sprites

Image Sprites

Sebuah image sprite merupkaan sekumpulan image yang ditaruh kedalam single image (image tunggal).

Web page dengan banyak image bisa memakan banyak waktu untuk memuat dan menghasilkan multiple server requests.

Penggunaan image sprites akan mengurangi nomor (atau muatan) daei server requests dan menyimpan bandwidth.

Image Sprites - Simple Example

Image Sprites - Contoh Sederhana

Sebagai pengganti dari penggunaan 3 separate image, kita menggunakan single image ini ("img_navspirates.gif"):

Dengan CSS, kita bisa hanya menampilkan bagian dari iamge yang kita perlukan.

Pada contoh berikut CSS menentukan bagian yang mana dari image "img_navsprites.gif" untuk ditampilkan:

Contoh:

#home {
width: 46px;
height: 44px;
background: url(img_navsprites.gif) 0 0;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
#home {
width: 46px;
height: 44px;
background: url(img_navsprites.gif) 0 0;
}

#next {
width: 43px;
height: 44px;
background: url(img_navsprites.gif) -91px 0;
}
</style>
</head>
<body>

<img id="home" src="img_trans.gif"><br><br>

<img id="next" src="img_trans.gif">

</body>
</html>


Contoh Menerangkan:

  • <img id="home" sec="img_trans.gif"> - Hanya mendefinisikan sebuah image kecil transparan karena atttribute src tidak boleh kosong. Image yang ditampilkan akan menjadi background image yang kita tentukan dalam CSS
  • width: 46px; height: 44px; - Mendefinisikan bagian dari iange yang kita ingin lihat
  • background: url(img_navsprites.gif) 0 0; - Mendefinisikan background image dan posisinya (left: 0px;, top: 0px;)
  • Ini merupakan cara paling mudah untuk menggunakan image sprites, sekarang kita ingin memperluasnya dengan menggunakan link dan efek hover.

Image Sprites - Create a Navigation List

Image Sprires - Membuat sebuah Navigation List

Kita ingin menggunakan sprite image ("img_navsprite.gif") untuk membuat sebuah navigarion list.

Kita akan menggunakan sebuah HTML list, karena ini bisa menjadi sebuah link dam juga mendukung sebuah background image:

Contoh:

#navlist {
position: relative;
}

#navlist li {
margin: 0;
padding: 0;
list-style: none;
position: absolute;
top: 0;
}

#navlist li, #navlist a {
height: 44px;
display: block;
}

#home {
left: 0px;
width: 46px;
background: url('img_navsprites.gif') 0 0;
}

#prev {
left: 63px;
width: 43px;
background: url('img_navsprites.gif') -47px 0;
}

#next {
left: 129px;
width: 43px;
background: url('img_navsprites.gif') -91px 0;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
#navlist {
position: relative;
}

#navlist li {
margin: 0;
padding: 0;
list-style: none;
position: absolute;
top: 0;
}

#navlist li, #navlist a {
height: 44px;
display: block;
}

#home {
left: 0px;
width: 46px;
background: url('img_navsprites.gif') 0 0;
}

#prev {
left: 63px;
width: 43px;
background: url('img_navsprites.gif') -47px 0;
}

#next {
left: 129px;
width: 43px;
background: url('img_navsprites.gif') -91px 0;
}
</style>
</head>
<body>

<ul id="navlist">
<li id="home"><a href="default.html"></a></li>
<li id="prev"><a href="css_intro.html"></a></li>
<li id="next"><a href="css_syntax.html"></a></li>
</ul>

</body>
</html>

Contoh Menjelaskan:

  • #navlist {position:relative;} - Posisi diatur menjadi relative untuk memperbolehkan penempatan absolute kedalamnya
  • #navlist li {margin:0;padding:0;list-style:none;position:absolute;top:0;} - margin dan padding diatur menjadi 0, list-style dihapus, dan semua list item diposisikan sebagai absolute
  • #navlist li, #navlist a {height:44px;display:block;} - Tinggi dari semua image adalah 44px

Sekarang mulailah untuk memposisikan dan me-style setiap bagian yang spesifik:

  • #home {left:0px;width:46px;} - Menempatkan semuanya ke kiri dan lebar image adalah 46px
  • #home {background:url(img_navsprites.gif) 0 0;} - Mendefinisikan background image dan memposisikannya (left 0px, top 0px)
  • #prev {left:63px;width:43px;} - Memposisikan 63px ke kanan (#home lebar 46px + beberapa extra space antara items), dan lebar adalah 43px.
  • #prev {background:url('img_navsprites.gif') -47px 0;} - Mendedinsikan background image 47px ke kanan (#home lebar 46px + 1px line divider)
  • #next {left:129px;width:43px;} - Memposisikan 129px ke kanan (dimulai dari #prev adalah 63px + #prev lebar 43px + extra space), dan lebar adalah 43px.
  • #next {background:url('img_navsprites.gif') -91px 0;} - Mendefinisikan background image 91px ke kanan (#home lebar 46px + 1px line divider + #prev lebar 43px + 1px line divider )

Image Sprites - Hover Effect

Sekarang kita ingin menggunakan efek hover untjk navigation list kita.

Catatan: selector :hover bisa digunakan pada semua element, tidak hanya pada link.

Karena ini merupakan one single image, dan bukan six separate files, akan ada yang tidak penundaan (delay) dimuat saat user meng-hover image.

Kita hanya menambahkan tiga baris dari kode untuk menambahkan efek hover:

Contoh:

#home a:hover {
background: url('img_navsprites_hover.gif') 0 -45px;
}

#prev a:hover {
background: url('img_navsprites_hover.gif') -47px -45px;
}

#next a:hover {
background: url('img_navsprites_hover.gif') -91px -45px;
}

padding: 0;

position: absolute;
top: 0;
}

#navlist li, #navlist a {
height: 44px;
display: block;
}

#home {
left: 0px;
width: 46px; background: url('img_navsprites_hover.gif') 0 0;
}

#prev {
left: 63px;
width: 43px;
background: url('img_navsprites_hover.gif') -47px 0;
}

#next {

width: 43px;

background: url('img_navsprites_hover.gif') 0 -45px;

</ul>

</body>

Contoh menerangkan:

#home a:hover {background: transparent url('img_navsprites_hover.gif') 0 -45px;} - Untuk semua (3) hover image yang kita tentukan posisi background yang sama, hanya 45 further down
