CSS3 Resizing

CSS3 Pengukuran Ulang

Properti resize menentukan apakah (iya) atau tidak sebuah element harus dapat diukur ulang oleh pengguna.

Element <div> ini dapat diukur ulang oleh pengguna (bekerja di Chrome, Firefox, Safari dan Opera).

Contoh berikut membiarkan pengguna mengukur ulang lebar dari element <div> saja:

Contoh:

div {
resize: horizontal;
overflow: auto;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div {
border: 2px solid;
padding: 20px;
width: 300px;
resize: horizontal;
overflow: auto;
}
</style>
</head>
<body>

<p><b>Note:</b> Internet Explorer does not support the resize property.</p>

<div>Let the user resize the width of this div element.</div>

</body>
</html>

CSS3 Outline Offset

CSS3 Penyeimbang Garis Luar

Properti outline-offset menambahkan ruang diantara sebuah garis luar dan tepi atau border dari element.

Outline berbeda dari border dalam 3 cara:

  • Sebuah Outline (garis luar) adalah sebuah garis yang digambar disekitar element, diluar dari tepi border
  • Sebuah outline (garis luar) tidak mengambil ruang
  • Sebuah outline (garis luar) mungkin tidak persegi
Div ini memiliki sebuah outline 15px garis luar tepi border.

Contoh berikut menggunakan properti outline-offset untuk menambahkan sebuah ruang 15px diantara border dan outline:

Contoh:

div {
border: 1px solid black;
outline: 1px solid red;
outline-offset: 15px;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div {
margin: 20px;
padding: 10px;
width: 300px;
height: 100px;
border: 1px solid black;
outline: 1px solid red;
/* Move the outline 15px away from the border */
outline-offset: 15px;
}
</style>
</head>
<body>

<p><b>Note:</b> Internet Explorer does not support the outline-offset property.</p>

<div>This div has an outline 15px outside the border edge.</div>

</body>
</html>

Note: Internet Explorer does not support the outline-offset property.

This div has an outline 15px outside the border edge.

CSS3 User Interface Properties

Daftar tabel berikut adalah semua properti antarmuka pengguna (user interface):

Properti Deskripsi
box-sizing Memperbolehkan kamu untuk memasukan padding dan border dalam sebuah total lebar dan tinggi elemen
Nilai:
  • border-box
  • content-box
  • inherit
  • initial

Nilai:
  • auto
  • id
  • target-name
  • inherit
  • initial
nav-index Menentukan susunan tab untuk sebuah element
Nilai:
  • auto
  • number
  • inherit
  • initial
nav-left Menentukan dimana posisi untuk navigasi ketika menggunakan arrow-left navigation key (kunci navigasi arah kiri)
Nilai:
  • auto
  • id
  • target-name
  • inherit
  • initial
nav-right Menentukan dimana posisi untuk navigasi ketika menggunakan arrow-right navigation key (kunci navigasi arah kanan)
Nilai:
  • auto
  • id
  • target-name
  • inherit
  • initial
nav-up Menentukan dimana posisi untuk navigasi ketika menggunakan arrow-up navigation key (kunci navigasi arah atas)
Nilai:
  • auto
  • id
  • target-name
  • inherit
  • initial
outline-offset Menambahkan ruang diantara sebuah outline dan tepi atau border dari sebuah element
Nilai:
  • length
  • inherit
  • initial

Nilai:
  • vertical
  • horizontal
  • both
  • none
  • inherit
  • Initial
