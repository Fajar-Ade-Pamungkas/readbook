CSS3 border-image Property

Properti border-image CSS3 memperbolehkan kamu untuk menentukan sebuah image untuk digunakan pengganti dari border normal disekitar element.

Properti ini memiliki tiga bagian:

  • Image untuk digunakan sebagai border
  • Dimana harus memotong (slice) image
  • Mendefinisikan apakah bagian pertengahan harus dibolak-balik (repeated) atau direnggangkan (stretched)

Kita akan menggunakan image berikut (yang bernama "border.png"):

Properti border-image mengambil/membawa image dan memotongnya (slice) kedalam sembilan bagian, seperti tic-tac-toe board. ia kemudian menempatkan sudut pada sudut, dan bagian tengah dibolak-balik atau direnggangkan seperti yang kamu tentukan.

Catatan: Agar border-image bekerja, element juga memerlukan properti border yang diatur!

Disini, middle section (bagian pertengahan) dari image dibolak-balik untuk membuat border:

Image sebagai border!

Disini adalah kodenya:

Contoh:

#borderimg {
border: 10px solid transparent;
padding: 15px;
-webkit-border-image: url(border.png) 30 round; /* Safari 3.1-5 */
-o-border-image: url(border.png) 30 round; /* Opera 11-12.1 */
border-image: url(border.png) 30 round;
}

CSS3 border-image - Different Slice Values

Nilai pemotong yang berbeda seutuhnya mengubah tampilan dari border:

Contoh 1:

border-image: url(border.png) 50 round;

Contoh 2:

border-image: url(border.png) 20% round;

Contoh 3:

border-image: url(border.png) 30% round;

CSS3 Border Properties

Properti Deskripsi
border-image Sebuah shorthand property untuk mengatur semua border image properti dalam satu deklarasi
border-image-source Menentukan path (jejak) image untuk digunakan sebagai sebuah border
border-image-slice Menentukan bagaimana untuk memotong (slice) border image
border-image-width Menentukan lebar dari border image
border-image-outset Menentukan jumlah dengan yang mana border image area sulit memperluas border box (specifies the amount by which the border image area extends beyond the border box).
border-image-repeat Menentukan apakah border image harus dibolak-balik
