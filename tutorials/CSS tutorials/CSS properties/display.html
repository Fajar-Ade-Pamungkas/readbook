Definisi dan Penggunaan

Properti display menentukan tipe dari box yang digunakan untuk sebuah element HTML.

Default value: inline
Inherited: no
Animatable: no. Read about animatable
Version CSS1
JavaScript syntax: object.style.display="none" Try it

Browser Support

Nomor dalam tabel menentukan browser versi pertama yang mendukung properti secara penuh.

Nomor yang diikuti oleh -webkit-, -moz- atau -o- menentukan versi pertama yang bekerja dengan awalan kata.

Catatan: Nilai "inline-table", "table", "table-caption", "table-cell", "table-column", "table-column-group", "table-row", dan "table-row-group" tidak didukung di IE7 dan versi sebelumnya. IE8 diperlukan !DOCTYPE. IE9 mendukung nilai-nilai tersebut.

Catatan: Nilai "flex" dan "inline-flex" memerlukan sebuah awalan kata untuk berjalan di Safari. Untuk "flex" gunakanlah "display: -webkit-flex", untuk "inline-flex" gunakanlah "display: -webkit-inline-flex;".

Nilai Properti

Nilai Deskripsi Mainkan
inline Nilai default. Menampilkan sebuah element sebagai inline element atau element dalam baris (seperti <span) Klik disini
block Menampilkan sebuah element sebagai block element atau element blok (seperti <p>) Klik disini
flex Menampilkan sebuah element sebagai sebuah block-level flex container. Baru di CSS3 Klik disini
inline-block Menampilkan sebuah element sebagai inline-level block container. Didalam blok ini diformat sebagai block-level box, dan element itu sendiri diformat sebagai inline-level box Klik disini
inline-flex Menampilkan sebuah element sebagai sebuah inline-level flex container. Baru di CSS3 Klik disini
inline-table Element ditampilkan sebagai sebuah inline-level table Klik disini
list-item Membiarkan element berkelakuan/berjalan seperti element <li> Klik disini
run-in Menampilkan sebuah element sebagai sebuah salah satu dari block atau inline, tergantung pada context atau keadaan Klik disini
table Membiarkan element berkelakuan/berjalan seperti element <table> Klik disini
table-caption Membiarkan element berkelakuan/berjalan seperti element <caption> Klik disini
table-column-group Membiarkan element berkelakuan/berjalan seperti element <colgroup> Klik disini
table-header-group Membiarkan element berkelakuan/berjalan seperti element <thead> Klik disini
table-footer-group Membiarkan element berkelakuan/berjalan seperti element <tfoot> Klik disini
table-row-group Membiarkan element berkelakuan/berjalan seperti element <tr> Klik disini
none Element tidak akan ditampilkan semua (tidak memiliki efek pada layout) Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
