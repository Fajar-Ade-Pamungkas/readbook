Definisi dan Penggunaan

Properti border-bottom-left-radius mendefinisikan bentuk dari border tentang sudut kanan-bawah.

Tip: Properti ini memperbolehkan kamu untuk menambahkan rounded border untuk element!

Default value: 0
Inherited: no
Animatable: Yes. Read about animatable Try it
Version CSS3
JavaScript syntax: object.style.borderBottomRightRadius="25px" Try it

Sintaksis CSS

border-bottom-left-radius: length|% [length]|initial|inherit

Catatan: Dua nilai length (panjang) atau % (percent) dari properti border-bottom-left-radius mendefinisikan radii dari seperempat ellipse (oval) yang mendefinisikan bentuk sudut dari sisi bagian luar border. Nilai pertama adalah radius horizontal, nilai yang kedua adalah raidus vertikal. Jkka nilai kedua dihilangkan ia akan disalin dari nilai pertama. Jika salah satu dari length (panjang) adalah 0, sudut adalah kotak, bukan tumpul (rounded). Persentage untuk radius horizontal mengacu pada width (lebar) dari border box, sedangkan persentage untuk radius vertikal mengarah pada height (tinggi) dari border box.

Nilai Properti

Nilai Deskripsi Mainkan
% Mendefinisikan bentuk dari sudut kiri bawah dalam satuan % Klik disini
length Mendefinisikan bentuk dari sudut kiri bawah. Nilai defaultnya adalah 0 Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
