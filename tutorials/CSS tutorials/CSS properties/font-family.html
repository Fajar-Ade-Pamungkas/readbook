Definisi dan Penggunaan

Properti font-family menentukan font (tipe tulisan) untuk element.

Properti font-family bisa menggenggam beberapa nama font sebagai sistem "fallback". Sistem "fallback" merupakan sistem yang mana jika browser tidak mendukung font pertama yang diinginkan, lalu browser akan mencoba pada font berikutnya.

Disini ada dua tipe nama font family:

  • font-name - Nama dari sebuah font-family, seperti "times", "courier", "times", dan lain sebagainya.
  • generic-family - Nama dari sebuah generic-family, seperti "serif", "sans-serif", "cursive", "fantasy", "monospace".

Mulailah dengan font yang kamu inginkan, dan selalu akhiri dengan sebuah generic family, untuk mengizinkan browser mengangkut font yang mirip dalam generic family, jika tidak ada font lainnyanyang tersedia.

Catatan: Pisahkanlah setiap nilai dengan sebuah koma.

Catatan: Jika sebuah nama font berisikan white-space (spasi), ia harus diberikan single quotes ('...') dan harus digunakan ketika attribute "style" dalam HTML.

Default value: Tergantung pada browser
Inherited: Yes
Animatable: no. Read about animatable
