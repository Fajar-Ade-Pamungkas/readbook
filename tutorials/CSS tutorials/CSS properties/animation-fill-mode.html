Definisi dan Penggunaan

Properti animation-fill-mode menentukan sebuah style untuk element ketika animasi tidak berjalan (ketika animasi diselesaikan, atau ketika animasi memiliki penundaan).

Dengan default, CSS animation tidak berpengaruh pada element sampai keyframe pertama "dijalankan", dan kemudian berhenti sekali keyframe terakhir telah lengkap. Properti animation-fill-mode bisa mengesampingkan perilaku ini.

Default value: none
Inherited: no
Animatable: no. Read about animatable
Version CSS3
JavaScript syntax: object.style.animationFillMode="both" Try it

Sintaksis CSS

animation-fill-mode: none|forwards|backwards|both|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
none Nilai default. Animation tidak akan menerapkan style untuk element target sebelum atau sesudah ia berjalan Klik disini
forward Setelah animation berakhir (ditegaskan oleh animation-iteration-count), animation akan menerapkan nilai properti untuk time animation berakhir Klik disini
backward Animation akan menerapkan nilai properti yang dikenal dalam keyframe yang akan memulai iteration pertama dari animation, selama periode didefinisikan oleh animation-delay. Ini adalah seriap nilai dari keyframe (ketika animation-direction adalah "normal" atau "alternate") atau itu dari keyframe (ketika animation-direction adalah "reverse" atau "alternate-reverse") Klik disini
both Animation akan mengikuti perintah/garis forward dan backward. Ia akan memperpanjang properti animation di kedua arah Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
