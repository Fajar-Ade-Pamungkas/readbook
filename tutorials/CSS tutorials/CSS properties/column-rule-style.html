CSS3 column-rule-style Property

Contoh

Contoh:

Menentukan style (gaya) dari garis diantara kolom-kolom:

div {
-webkit-column-rule-style: dotted; /* Chrome, Safari, Opera */
-moz-column-rule-style: dotted; /* Firefox */
column-rule-style: dotted;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
</style>
</head>
<body>

</body>
</html>

Definisi dan Penggunaan

Properti column-rule-style menentukan style atau gaya dari garis diantara kolom-kolom.

Default value: none
Inherited: no
Animatable: No, Read about animatable
Version CSS3
JavaScript syntax: object.style.columnRuleStyle="dotted" Try it

Browser Support

Nomor dalam tabel menentukan browser versi pertama yang mendukung properti secara penuh.

Nomor yang diikuti oleh -webkit-, -moz- atau -o- menentukan versi pertama yang bekerja dengan awalan kata.

Sintaksis CSS

column-rule-style: none|hidden|dotted|dashed|solid|double|groove|ridge|inset|outset|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
dotted Mendefinisikan rule/garis titik-titik Klik disini
dashed Mendefinisikan rule/garis putus-putus Klik disini
solid Mendefinisikan rule/garis lurus Klik disini
double Mendefinisikan rule/garis ganda Klik disini
groove Mendefinisikan rule/garis beralur (groove) 3D. efek tergantung pada nilai lebar dan warna Klik disini
ridged Mendefinisikan 3D ridged rule. Efek tergantung pada nilai lebar dan warna Klik disini
inset Mendefinisikan 3D inset rule. Efek tergantung pada nilai lebar dan warna Klik disini
outset Mendefinisikan 3D outset rule. efek tergantung pada nilai lebar dan warna Klik disini
none Mendefinisikan tidak ada rule/garis Klik disini
hidden Mendefinisikan border tersembunyi Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial

-moz-column-count: 3; /* Firefox */

column-gap

Membagi teks dalam element <div> kedalam 3 kolom dan menentukan celah (gap) 40px diantara kolom:

div {
-webkit-column-count: 3; /* Chrome, Safari, Opera */
-moz-column-count: 3; /* Firefox */
column-count: 3;

-webkit-column-gap: 40px; /* Chrome, Safari, Opera */
-moz-column-gap: 40px; /* Firefox */
column-gap: 40px;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
</style>
</head>
<body>

</body>
</html>

column-rule

Menentukan width (lebar), style (gaya), color (warna) dari garis diantara kolom.

div {
-webkit-column-count: 3; /* Chrome, Safari, Opera */
-moz-column-count: 3; /* Firefox */
column-count: 3;

-webkit-column-gap: 40px; /* Chrome, Safari, Opera */
-moz-column-gap: 40px; /* Firefox */
column-gap: 40px;

-webkit-column-rule: 4px outset #ff00ff; /* Chrome, Safari, Opera */
-moz-column-rule: 4px outset #ff00ff; /* Firefox */
column-rule: 4px outset #ff00ff;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
</style>
</head>
<body>

</body>
</html>
