Definisi dan Penggunaan

Properti align-content memodifikasi perilaku dari properti flex-wrap. Properti ini mirip dengan align-item, tapi pengganti dari perataan flex items, align-content meratakan flex lines.

Tip: Gunakanlah properti justify-content utuk meratakan item-item pada main-axis atau poros utama (secara horizontal).

Catatan: Ada harus menjadi multiple lines dari item-item utuk ptoperti ini tuk memiliki efek.

Default value: stretch
Inherited: no
Animatable: no. Read about animatable
Version CSS3
JavaScript syntax: object.style.alignContent="center" Klik disini

Sintaksis CSS

align-content: stretch|center|flex-start|flex-end|space-between|space-around|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
stretch Nilai default. Baris-baris merenggang untuk mengambil ruang yang tersisa Klik disini
center Baris-baris dikemas ke (arah) tengah dari flex container Klik disini
flex-start Baris-baris dikemas ke (arah) awal dari flex container Klik disini
flex-end Baris-baris dikemas ke (arah) akhir dari flex container Klik disini
space-between Baris-baris dasar didistribusikan di flex container Klik disini
space-around Baris-baris dasar didistribusikan di flex container, dengan setengah ukuran ruang di setiap akhirnya Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
