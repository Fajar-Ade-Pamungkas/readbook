Definisi dan Penggunaan

Properti animation-timing-function menentukan kurva kecepatan dari animasi.

Kurva kecepatan mendefinisikan TIME (waktu) sebuah animasi menggunakan untuk menggubah dari satu pengaturan CSS style ke pengaturan CSS style lainnya.

Kurva kecepatan digunakan untuk membuat perubahan secara perlahan.

Default value: ease
Inherited: no
Animatable: no. Read about animatable
Version CSS3
JavaScript syntax: object.style.animationTimingFunction="linear" Try it

Sintaksis CSS

animation-timing-function: linear|ease|ease-in|ease-out|ease-in-out|step-start|step-end|steps(int,start|end)|cubic-bezier(n,n,n,n)|initial|inherit;

animation-timing-function menggunakan fungsi matematika, yang disebut kurva Cubic Bezier, untuk membuat kurva kecepatan. Kamu bisa menggunakan nilai kamu sendiri dalam fungsi ini atau menggunakan salah satu dari nilai yang telah ditentukan:

Nilai Properti

Nilai Deskripsi Mainkan
linear Animasi memiliki kecepatan yang sama dari awal sampai akhir Klik disini
ease Nikai default. Animasi memiliki awalan pelan, lalu depat, sebelum animasi berakhir secara perlahan Klik disini
ease-in Animasi memiliki awalan pelan Klik disini
ease-out Animasi memiliki akhiran pelan Klik disini
ease-in-out Animasi memiliki keduanya awalan pelan dan akhiran pelan Klik disini
step-start Equivalent (padankata) dengan steps(1, start) Klik disini
step-end Equivalent (padankata) dengan steps(1, end) Klik disini
steps(int, start|end) Menentukan stepping function. dengan dua parameter. Parameter pertama menentukan angka interval pada fungsi. Ini harus bilangan positif (lebih besar dari 0). Parameter kedua, yang mana adalah optional, apakah salah satu dari nilai "start" atau "end", dan menentukan titik yang mana mengubah nilai yang terjadi dengan interval. Jika parameter kedua dihilangkan, ia diberikan nilai "end" Klik disini
cubic-bezier(n,n,n,n) Mendefinisikan nilai kamu sendiri dalam fungsi cubic-bezier Klik disini
posibble Nilai-nilai ini merupakan nilai numerik dari 0 sampai dengan 1 Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial

Catatan: Cobalah nilai yang berbeda dalam bagian contoh-contoh dibawah ini.

Contoh:

Untuk memahami lebih baik nilai timing function yang berbeda;

Disini ada lima element <div> yang berbeda dengan lima nilai yany berbeda:

/* For Chrome, Safari and Opera */
#div1 {-webkit-animation-timing-function: linear;}
#div2 {-webkit-animation-timing-function: ease;}
#div3 {-webkit-animation-timing-function: ease-in;}
#div4 {-webkit-animation-timing-function: ease-out;}
#div5 {-webkit-animation-timing-function: ease-in-out;}

/* Standard syntax */
#div1 {animation-timing-function: linear;}
#div2 {animation-timing-function: ease;}
#div3 {animation-timing-function: ease-in;}
#div4 {animation-timing-function: ease-out;}
#div5 {animation-timing-function: ease-in-out;}
