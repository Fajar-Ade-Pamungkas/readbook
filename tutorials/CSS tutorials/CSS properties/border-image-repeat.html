Definisi dan Penggunaan

Properti border-image-repeat menentukan apakah border gambar harus diulang-ulang (repeat), ditumpilkan (rounded) atau direnggangkan (stretch).

Tip: Lihatlah juga properti border-image (sebuah shorthand property untuk mengatur semua properti border-image-*).

Default value: stretch
Inherited: no
Animatable: no. Read about animatable
Version CSS3
JavaScript syntax: object.style.borderImageRepeat="round" Try it

Sintaksis CSS

border-image-repeat: stretch|repeat|round|initial|inherit;

Catatan: Properti ini menentukan bagaimana gambar untuk sisi-sisi dan bagian tengah daei border gambar diskala dan tiled. Jadi, kamu bisa menentukan dua nilai disini. Jika nilai kedua dihilangkan, nilai kedua diasumsikan sama seperti nilai pertama.

Nilai Properti

Nilai Deskripsi Mainkan
stretch Nilai default. Gambar direnggangkan untuk mengisi area Klik disini
repeat Gambar diulang-ulang (repeat) untuk mengisi area Klik disini
round The image is tiled (repeated) to fill the area. If it does not fill the area with a whole number of tiles, the image is rescaled so it fits Klik disini
space The image is tiled (repeated) to fill the area. If it does not fill the area with a whole number of tiles, the extra space is distributed around the tiles
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
