CSS3 column-rule Property

Contoh

column-rule

Menentukan width (lebar), style (gaya), color (warna) dari garis diantara kolom.

div {
-webkit-column-rule: 4px outset #ff00ff; /* Chrome, Safari, Opera */
-moz-column-rule: 4px outset #ff00ff; /* Firefox */
column-rule: 4px outset #ff00ff;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
</style>
</head>
<body>

</body>
</html>

Definisi dan Penggunaan

Properti column-rule merupakan sebuah shorthand property untuk pengaturan semua properti column-rule-*.

Properti column-rule mengatur width (lebar), style (gaya), dan color (warna) dari garis diantara kolom.

Default value: medium none color
Inherited: no
Animatable: Yes. see individual porperties Read about animatable Try it
Version CSS3
JavaScript syntax: object.style.columnRule="3px outset blue" Try it

Sintaksis CSS

column-rule: column-rule-width column-rule-style column-rule-color|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
column-rule-width Mengatur width (lebar) dari garis diantara kolom-kolom. Nilai defaultnya adalah medium
column-rule-style Mengatur style (gaya) dari garis diantara kolom-kolom. Nilai defaultnya adalah none
column-rule-color Mengatur color (warna) dari garis diantara kolom-kolom. Nilai defaultnya adalah color
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
