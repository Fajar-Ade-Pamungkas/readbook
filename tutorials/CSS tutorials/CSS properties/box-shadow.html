Definisi dan Penggunaan

Properti box-shadow melampirkan/menambahkan satu atau lebih bayangan ke sebuah element.

Default value: none
Inherited: no
Animatable: Yes. Read about animatable Try it
Version CSS3
JavaScript syntax: object.style.boxShadow="10px 20px 30px blue" Try it

Sintaksis CSS

box-shadow: none|h-shadow v-shadow blur spread color |inset|initial|inherit;

Catatan: Properti box-shadow melekatkan satu atau lebih bayangan untuk sebuah element. Properti menggunakan koma untuk memisahkan penggunaan bayang lebih dari satu, masing-masing dengan panjang nilai 2-4 sebuah optional color, dan sebuah optional inset keyword. Omitted lengths are 0.

Nilai Properti

Nilai Deskripsi Mainkan
none Nilai default. Tidak ada bayangan yang akan ditampilkan Klik disini
h-shadow Diperlukan. Posisi bayangan horizontal. Nilai negatif diizinkan Klik disini
v-shadow Diperlukan. Posisi bayangan vertikal. Nilai negatif diizinkan Klik disini
blur Optional. Jarak blur Klik disini
spread Optional. Ukuran bayangan. Nilai negatif diizinkan Klik disini
color Diperlukan.Warna bayangan. Nilai defaultnya adalah black. Lihatlah di CSS Color Values untuk daftar lengkap tentang nilai warna yang diperbolehkan.
Catatan: Dalam Safari (di PC) parameter warna diperlukan. Jika kamu tidak menentukan warna, bayangan tidak ditampilkan. 		Klik disini
inset Optional. Mengubah bayangan dari bayangan bagian luar (outset) ke bayangan bagian dalam Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
