Definisi dan Penggunaan

Properti singkat background mengatur semua properti-properti latar belakang dalam satu deklarasi.

Properti-properti yang bisa diatur adalah: background-color, background-image, background-position, background-size, background-repeat, background-origin, background-clip, and background-attachment.

Itu tidakpenting jika salah satu nilai diatas dihilangkan, seperti contoh background:#ff0000 url(smiley.gif); diizinkan.

Default value: Lihatlah individual properties
Inherited: no
Animatable: Yes, see individual properties Read about animatable
Version CSS1 + properti-properti baru di CSS3
JavaScript syntax: object.style.background="red url(smiley.gif) top left no-repeat" Try it

Sintaksis CSS

background: color image position/size repeat origin clip attachment initial|inherit;

Jika salah satu properti dapam shorthand declaration adalah properti background-size, kamu harus gunakan / (tanda garis miring) untuk memisahkan ia dari properti properti background-position, sebagai contoh background:url(smiley.gif) 10px 20px/50px 50px; will result in a background image, positioned 10 pixels from the left, 20 pixels from the top, and the size of the image will be 50 pixels wide and 50 pixels high.

Nilai Properti

Nilai Deskripsi CSS
background-color Menentukan background warna untuk digunakan pada element 1
background-image Menentukan SATU atau LEBIH background gambar untuk digunakan pada element 3
background-position Menentukan posisi dari background gambar
background-size Menentukan ukuran dari background gambar 3
background-repeat Menentukan bagaimana untuk mengulang-ulang background gambar 1
background-origin Menentukan area penempatan (positioning area) dari background gambar 3
background-clip Menentukan area pewarnaan (painting area) dari background gambar 3
background-attacment Menentukan apakah background gambar ditetap atau dapat di-scroll dengan sisa dari halaman 3
initial Mengatur properti ini untuk nilai defaultnya. Read about initial 3
inherit Mewarisi properti ini dari nilai element induknya. Read about initial 2
