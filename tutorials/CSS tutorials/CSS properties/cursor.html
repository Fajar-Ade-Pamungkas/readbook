Definisi dan Penggunaan

Properti cursor menentukan tipe dari cursor untukditampilkan ketika pointing (penunjukan) dalam sebuah element.

Default value: auto
Inherited: Yes
Animatable: no. Read about animatable
Version CSS2
JavaScript syntax: object.style.cursor="crosshair" Try it

Nilai Properti

Nilai Deskripsi Mainkan
alias Cursor mengindikasikan sebuah nama sebutan tentang sesuatu yang sedang dibuat Klik disini
all-scroll Cursor mengindikasikan bahwa sesuatu bisa di-scroll dalam arah (direction) apapun Klik disini
auto nilai default. Browser yang mengatur cursor Klik disini
cell Cursor mengindikasikan bahwa sebuah cell (atau mengatur cell) boleh diseleksi Klik disini
context-menu Cursor mengindikasikan bahwa sebuah context-menu tersedia Klik disini
col-resize Cursor mengindikasika bahwa kolom bisa diukur ulang secara horizontal Klik disini
copy Cursor mengindikasikan sesuatu yang disalin Klik disini
crosshair The cursor render as a crosshair Klik disini
default Default cursor Klik disini
e-resize Cursor mengindikasikan bahwa sebuah tepi dari sebuah box yang sedang digerakan ke kanan (east/timur) Klik disini
ew-resize Mengindikasikan sebuah bidirectional resize cursor Klik disini
grab Cursor mengindikasikan bahwa sesuatu bisa diangkut Klik disini
grabbing Cursor mengindikasikan bahwa sesuatu bisa diangkut Klik disini
help Cursor mengindikasikan bahwa bantuan tersedia Klik disini
move Cursor mengindikasikan sesuatu yang sedang digerakan Klik disini
n-resize Cursor mengindikasikan bawah tepi dsri sebuah bix sedang digerakan keatas (north/utara) Klik disini
ne-resize Cursor mengindikasikan bahwa tepi box sedang digerakan keatas dan kekanan (north/east) Klik disini
nesw-resize Mengindikasikan sebuah bidirectional resize cursor Klik disini
ns-resize Mengindikasikan sebuah bidirectional resize cursor Klik disini
nw-resize Cursor mengindikasikan tepu box sedang digerkan keatas dan kiri (north/west) Klik disini
nwse-resize Mengindikasikan sebuah bidirectional resize cursor Klik disini
no-drop Cursor mengindikasikan bahwa dragged item tidak bisa ditaruh/diturunkan disini Klik disini
none Tidak ada cursor yang diberikan untuk element Klik disini
not-allowed Cursor mengindikasikan bahwa requested action tidak akan dieksekusi/dijalankan Klik disini

progress Cursor mengindikasikan bahwa program sedang sibuk (dalam progress) Klik disini

URL Sebuah comma separated list dari URL-URL untuk custom cursor. Catatan: Tetaplah tentukan generic cursor di akhir list, dalam kasus none tentang URL cursor yang didefinisi bisa digunakan Klik disini
vertical-text Cursor mengindikasikan teks vertikal (menurun) yang bokeh diseleksi Klik disini
w-resize Cursor mengindikasikan bahwa sebuah tepi box sedang digerakan ke kiri (west) Klik disini
wait Cursor mengindikasikan bahwa program sedang sibuk Klik disini
zoom-in Cursor mengindikasikan bahwa sesuatu bisa di-zoom in (dilihat lebih besar) Klik disini
zoom-out Cursor mengindikasikan bahwa sesuatu bisa di-zoom out (dilihat lebih kecil) Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
