Contoh:

Menentukan warna dari garis diantara kolom-kolom:

Definisi dan Penggunaan

Properti column-rule-color menentukan warna dari garis diantara kolom-kolom.

Default value: Warna element saat ini
Inherited: no
Animatable: Yes. Read about animatable Try it
Version CSS3
JavaScript syntax: object.style.columnRuleColor="0000ff" Try it

Browser Support

Nomor dalam tabel menentukan browser versi pertama yang mendukung properti secara penuh.

Nomor yang diikuti oleh -webkit-, -moz- atau -o- menentukan versi pertama yang bekerja dengan awalan kata.

Sintaksis CSS

column-rule-color: color|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
color Menentukan warna dari garus. Lihatlah ke CSS Color Values untuk daftar lengkap tentang nilai warna yang diperbolehkan Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial

column-rule

Menentukan width (lebar), style (gaya), color (warna) dari garis diantara kolom.

div {
-webkit-column-count: 3; /* Chrome, Safari, Opera */
-moz-column-count: 3; /* Firefox */
column-count: 3;

-webkit-column-gap: 40px; /* Chrome, Safari, Opera */
-moz-column-gap: 40px; /* Firefox */
column-gap: 40px;

-webkit-column-rule: 4px outset #ff00ff; /* Chrome, Safari, Opera */
-moz-column-rule: 4px outset #ff00ff; /* Firefox */
column-rule: 4px outset #ff00ff;
}
