Definisi dan Penggunaan

Properti background-color mengatur warna latar belakang sebuah element.

Latar belakang element adalah ukuran total dari element, termasuk padding dan border (tapi bukan termasuk margin).

Tip: Gunakanlah background warna dan teks berwarna untuk membuat teks mudsh tuk dibaca.

Default value: transparent
Inherited: no
Animatable: Iya, lihat di Read about animatable
Version CSS1
JavaScript syntax: object.style.backgroundColor="#00ff00" Try it

Sintaksis CSS

background-color: color|transparent|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
color Menentukan background warna. Lihatlah di CSS Color Values daftar nilai warna yang diperbolehkan. Klik disini
transparent Menentukan bahwa background warna harus transparan. Ini adalah default Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
