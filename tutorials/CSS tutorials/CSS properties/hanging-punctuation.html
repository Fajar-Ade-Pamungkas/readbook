Definisi dan Penggunaan

Properti hanging-punctuation menentukan apakah sebuah pemberian tanda baca (punctuation mark) boleh ditempatkan diluar line box di awal atau akhir baris penuh teks.

Default value: none
Inherited: Yes
Animatable: no. Read about animatable
Version CSS3
JavaScript syntax: object.style.hangingPunctuation="first" Try it

Browser Support

Tidak ada mayor browser yang mendukung properti hanging-punctuation.

Sintaksis CSS

hanging-punctuation: none|first|last|allow-end|force-end|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
none Tidak ada pemberian tanda baca (punctuation mark) yang boleh ditempatkan diluar line box di awal ataupun diakhir dari full line of text Klik disini
first Tanda baca boleh ditempatkan diluar dari awal tepi baris pertama Klik disini
last Tanda baca (punctuation) boleh ditempatkan diluar dari akhir tepi baris terakhir Klik disini
allow-end Tanda baca (punctuation) boleh ditempatkan diluar dari tepi akhir dari semua baris. Jika tanda baca tidak diharapkan fit prior to justification Klik disini
force-end Tanda baca (punctuation) boleh ditempatkan diluar dari tepi akhir dari semua baris. Jika justification memungkinkan di baris ini, lalu ia memaksskan punctuation untuk ditempatkan Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
