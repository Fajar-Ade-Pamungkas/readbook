Definisi dan Penggunaan

Perintah @keyframes menentukan kode animasi.

Selama animation, kamu bisa mengubah dari satu pengaturan CSS style ke pengaturan CSS style lainnya.

Ditentukan ketika style mengubah yang akan terjadi dalam persen, atau dengan kata kunci "from" dan "to", yang mana adalah sama seperti 0% dan 100%. 0% adalah awalan dari animasi, 100% adalah ketika animasi telah selesai atau lengkap.

Tip: Untuk browser support terbaik, kamu harus selalu definisikan selektor untuk 0% dan 100%.

Catatan: Gunakanlah properti animation untuk mengontrol rupa animasi, dan jiga untuk mengikatkan animasi kepada selektor.

Sintaksis CSS

@keyframes animationname {keyframes-selector {css-styles;}}

Nilai Properti

Nilai Deskripsi Mainkan
animationname Diperlukan. Mendefinisikan nama dari animasi Klik disini
keyframes-selector Diperlukan. Persentase tentang durasi animasi. Legal values:
0-100%
from (sama seperti 0%)
to (sama seperti 100%)
Catatan: Kamu bisa memiliki banyak keyframes-selector dalam satu deklarasi animation. 		Klik disini
css-style Diperlukan. Satu atau lebih properti CSS style yang legal Klik disini
