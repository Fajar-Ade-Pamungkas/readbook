Definisi dan Penggunaan

Dalam properti font-variant, semua huruf kecil dikonversikan menjadi huruf besar. Akan tetapi, huruf besar yang dikonversikan tampak dalam ukuran font yang lebih kecil dari huruf besar aslinya pada teks.

Properti font-variant menentukan apakah iya atau tidak teks harus ditampilkan dalam sebuah font kecil-kapital.

Default value: normal
Inherited: Yes
Animatable: no. Read about animatable
Version CSS1
JavaScript syntax: object.style.fontVariant="small-caps" Try it

Sintaksis CSS

font-variant: normal|small-caps|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
normal Browser menampilkan sebuah font normal. Ini adalah nilai default Klik disini
small-caps Browser menampilkan sebuah font (huruf) besar-kecil Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
