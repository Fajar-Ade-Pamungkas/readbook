Definisi dan Penggunaan

Properti columns merupakan sebuah shorthand property untuk mengatur column-width dan column-width.

Default value: auto auto
Inherited: no
Animatable: Yes. see individual properties. Read about animatable Try it
Version CSS3
JavaScript syntax: object.style.columns="100px 3" Try it

Sintaksis CSS

columns: auto|column-width column-count|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
auto Nilai default. Mengatur kedua properti column-count menjadi "auto" dan column-width menjadi auto Klik disini
column-width Mengatur lebar dari kolom
column-count Menentukan number (berapa jumlah) dari kolom
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial

column-gap

Membagi teks dalam element <div> kedalam 3 kolom dan menentukan celah (gap) 40px diantara kolom:

div {
-webkit-column-count: 3; /* Chrome, Safari, Opera */
-moz-column-count: 3; /* Firefox */
column-count: 3;

-webkit-column-gap: 40px; /* Chrome, Safari, Opera */
-moz-column-gap: 40px; /* Firefox */
column-gap: 40px;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
</style>
</head>
<body>

</body>
</html>

column-rule

Menentukan width (lebar), style (gaya), color (warna) dari garis diantara kolom.

div {
-webkit-column-count: 3; /* Chrome, Safari, Opera */
-moz-column-count: 3; /* Firefox */
column-count: 3;

-webkit-column-gap: 40px; /* Chrome, Safari, Opera */
-moz-column-gap: 40px; /* Firefox */
column-gap: 40px;

-webkit-column-rule: 4px outset #ff00ff; /* Chrome, Safari, Opera */
-moz-column-rule: 4px outset #ff00ff; /* Firefox */
column-rule: 4px outset #ff00ff;
}

<html>
