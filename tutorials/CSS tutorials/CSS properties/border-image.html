Definisi dan Penggunaan

Properti border-image memperbolehkan kamu untuk mengubah sebuah gambar untuk dogunakan sebagai pengganti dari border normal yang mengelilingi sebuah element.

Properti border-image merupakan shorhand property untuk pengaturan border-image-source, border-image-slice, border-image-width, border-image-outset dan border-image-repeat.

Menghapus nilai akan diatur menjadi nilai-nilai defaultnya.

Default value: none 100% 1 0 stretch
Inherited: no
Animatable: no. Read about animatable
Version CSS3
JavaScript syntax: object.style.borderImage="url(border.png) 30 round" Try it

Sintaksis CSS

border-image: source slice width outset repeat|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
border-image-source Path gambar untuk digunakan sebagai border
border-image-slice Bagaimana untuk memotong border gambar
border-image-width Menentukan lebar dari border gambar
border-image-outset Jumlah yang mana area border gambar memperpanjang melebihi border box
border-image-repeat Menentukan apakah border gambar harus diulang-ulang (repeated), dibulat (rounded) atau direnggangkan (stretched)
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial

Contoh Lebih

Contoh:

Different slice values completely changes the look of the border:

#borderimg1 {
border: 10px solid transparent;
padding: 15px;
-webkit-border-image: url(border.png) 50 round; /* Safari 3.1-5 */
-o-border-image: url(border.png) 50 round; /* Opera 11-12.1 */
border-image: url(border.png) 50 round;
}

#borderimg2 {
border: 10px solid transparent;
padding: 15px;
-webkit-border-image: url(border.png) 20% round; /* Safari 3.1-5 */
-o-border-image: url(border.png) 20% round; /* Opera 11-12.1 */
border-image: url(border.png) 20% round;
}

#borderimg3 {
border: 10px solid transparent;
padding: 15px;
-webkit-border-image: url(border.png) 30% round; /* Safari 3.1-5 */
-o-border-image: url(border.png) 30% round; /* Opera 11-12.1 */
border-image: url(border.png) 30% round;
}
