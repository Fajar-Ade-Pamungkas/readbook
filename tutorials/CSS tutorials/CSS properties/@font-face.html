Definisi dan Penggunaan

Dengan @font-face rule. web designers tidak lebih jauh harus menggunakan salah satu dari "web-safe" fonts.

Dalam @font-face rule baru kamu pertama kali harus mendefinisikan nama untuk font (contoh. myFirstFont), dan kemudian point untuk font file.

Tip: Gunakanlah lowercase letter untuk font URL. Uppercase letters bisa memberikan hasil yang diluar perkiraan dalam IE!

untuk menggunakan font untuk sebuah element HTML, mengarah pada nama font (myFirstFont) selama properti font-family:

div {
font-family: myFirstFont;
}

Contoh Lebih

Contoh:

Kmau harus mrnambahkan perintah @font-face lainnya yang berisikan pendeskripsi untuk teks tebal:

@font-face {
    font-family: myFirstFont;
    src: url(sansation_bold.woff);
font-weight: bold;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
</style>
</head>
<body>

</body>
</html>

File "sansation_bold.woff" merupakan file font lainnya, yang berisi karakter tebal untuk Sansation font.

Browser akan menggunakan ini kapanpun sebuah potongan dari teks dengan font-family "myFirstFont" harus diubah sebagai bold (tebal).

Dengan cara ini kamu bisa memiliki banyak perintah @font-face untuk font yang sama.
