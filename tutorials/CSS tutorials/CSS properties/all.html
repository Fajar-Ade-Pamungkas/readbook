CSS all Property

Contoh:

Mengubah semua properti yang diterapkan untuk element atau induk element untuk nilai initial mereka:

div {
background-color: yellow;
color: red;
all: initial;
}

</style>

</body>
</html>

Definisi dan Penggunaan

Properti all mereset semua properti, terlepas dari unicode-bidi dan direction, untuk nilah initial atau inherited mereka

Default value: No (tidak ada)
Inherited: no
Animatable: no. Read about animatable
Version CSS3
JavaScript syntax: object.style.all="initial" Try it

Nomor dalam tabel menentukan browser versi pertama yang mendukung properti secara penuh.

Sintaksis CSS

all: initial|inherit|unset;

Nilai Properti

Nilai Deskripsi Mainkan
initial Mengubah semua properti yang diterapkan untuk element atau induk element menjadi nilai initial mereka
inherit Mengubah semua properti yang diterapkan untuk element atau induk element menjadi nilai induk mereka
unset Mengubah semua properti yang diterapkan untuk element atau induk element menjadi nilai induk mereka jika mereka dapat diwariskan atau untuk nilai initial mereka jika tidak
