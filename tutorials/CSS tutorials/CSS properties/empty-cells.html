Definisi dan Penggunaan

Properti empty-cells mengatur apakah menampilkan birder dan background pada cell kosong dalam sebuah tabel atau tidak (hanya untuk mode "separated borders").

Default value: show
Inherited: Yes
Animatable: no. Read about animatable
Version CSS2
JavaScript syntax: object.style.emptyCells="hide" Try it

Sintaksis CSS

empty-cells: show|hide|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
show Background dan border ditampilkan pada cell kosong . Ini adalah nilai default Klik disini
hide Tidak ada bsckground dan border yang ditampilkan pada cell kosong Klik disini
Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
