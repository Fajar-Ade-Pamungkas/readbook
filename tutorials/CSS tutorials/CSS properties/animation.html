Definisi dan Penggunaan

Properti animation merupakan shorthand property untuk delapan properti animation:

  • animation-name
  • animation-duration
  • animation-timing-function
  • animation-delay
  • animation-iteration-count
  • animation-fill-mode
  • animation-play-state

Catatan: Selalu tentukan properti animation-duration, jika tidak durasinya adalah 0 (detik), dan tidak akan dimainkan.

Default value: none 0 1 normal none running
Inherited: no
Animatable: no. Read about animatable
Version CSS3
JavaScript syntax: object.style.animation="mymove 5s infinite" Try it

Sintaksis CSS

animation: name duration timing-function delay iteration-count direction fill-mode play-state;

Nilai Properti

Nilai Deskripsi
animation-name Menentukan nama dari keyframe yang kamu mau ikatkan dengan selector
animation-duration Menentukan berapa lama (dalam satuan) detik atau milidetik agar animasi selesai dimainkan
animation-timing-function Menentukan kurva kecepatan animasi
animation-delay Menentukan sebuah jeda (delay) sebelum animasi akan dimulai
animation-iteration-count Menentukan berapa banyak/kali animasi harus dimainkan
animation-direction Menentukan apakah animasi harus berputar/main di melawan arah jarum jam atau tidak
animation-fill-mode Menentukan apakah nilai diterapkan oleh animation diluar waktu/saat ia sedangn jalankan
animation-play-state Menentukan apakah animation sedang berjalan atau di-pause
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
