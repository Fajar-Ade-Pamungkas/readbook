Definisi dan Penggunaan

Properti list-style-type menentukan tipe dari penanda list-item dalam sebuah daftar atau list.

Default value: disc
Inherited: Yes
Animatable: no. Read about animatable
Version CSS1
JavaScript syntax: object.style.listStyleType="square" Try it

Browser Support

Nomor dalam tabel menentukan browser versi pertama yang mendukung properti secara penuh.

Nomor yang diikuti oleh -webkit-, -moz- atau -o- menentukan versi pertama yang bekerja dengan awalan kata.

BROWSER SUPPORT
Properti
list-style-type 1.0 12.0 4.0 1.0 1.0 3.5

Catatan: Internet Explorer dan Opera 12 dan versi sebelumnya tidak mendukung nilai: cjk-ideographic, hebrew, hiragana, hiragana-iroha, katana, dan katakana-iroha.

Catatan: IE8, dan versi sebelumnya hanya mendukung nilai properti: decimal-leading-zero, lower-greek, lowet-latin, upper-latin, armenian, georgian, dan inherit jika DOCTYPE dicantumkan.

Sintaksis CSS

list-style-type: values;

Nilai Properti

Nilai Deskripsi Mainkan

lower-latin Penanda adalah lower-latin (a, b, c, d, dsb.) Klik disini

ul {
list-style: none; /* Remove list bullets */
padding: 0;
margin: 0;
}

li {
padding-left: 16px;
}

li:before {
content: "•"; /* Insert content that looks like bullets */
padding-right: 8px;
color: blue; /* Or a color you prefer */
}
