Definisi dan Penggunaan

Properti align-item menentukan perataan default untuk item-item didalam flexible container.

Tip: Gunakanlah properti align-self setiap item untuk mengesampinhkan properti align-items.

Default value: stretch
Inherited: no
Animatable: no. Read about animatable
Version CSS3
JavaScript syntax: object.style.alignItems="center" Klik disini

Sintaksis CSS

align-items: stretch|center|flex-start|flex-end|baseline|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
stretch Default. Item-item direnggangkan untuk membenarkan (fit) kontainer Klik disini
center Item-item diposisikan di tengah kontainer Klik disini
flex-start Item-item diposisikan di awalan kontainer Klik disini
flex-end Item-item diposisikan di akhir kontainer Klik disini
baseline Item-item diposisikan di baris dasar kontainer Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
