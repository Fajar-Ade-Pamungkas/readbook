Definisi dan Penggunaan

Properti float menentukan apakah iya atau tidak sebuah box (sebuah element) harus mengapung (float).

Catatan: Element dengan posisi absolute akan mengabaikan properti float!.

Default value: none
Inherited: no
Animatable: no. Read about animatable
Version CSS1
JavaScript syntax: object.style.cssFloat="left" Try it

Browser Support

Nomor dalam tabel menentukan browser versi pertama yang mendukung properti secara penuh.

Nomor yang diikuti oleh -webkit-, -moz- atau -o- menentukan versi pertama yang bekerja dengan awalan kata.

Sintaksis CSS

float: none|left|right|initial|inherit

Nilai Properti

Nilai Deskripsi Mainkan
none Element tidak diapungkan (float), dannakan ditampilkan dimana dapat terdapat dalam teks. Ini adalag jilai default. Klik disini
left Element mengapung (float) ke kiri Klik disini
right Element mengapung (float) ke kanan Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
