Definisi dan Penggunaan

Properti background-position mengatur posisi awal dari sebuah background gambar.

Tip: Dengan default, background-image ditempatkan di sudut kiri atas dari sebuah element, dan mengulang-ulang (repeated) secara vertikal dan secara horizontal.

Default value: 0% 0%
Inherited: no
Animatable: Yes. Read about animatable
Version CSS1
JavaScript syntax: object.style.backgroundPosition="center" Try it
