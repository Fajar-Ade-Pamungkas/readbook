Definisi dan Penggunaan

Properti filter mendefinisikan efek visual (seperti blur dan saturation) untuk sebuah element (seringkali untuk <img>).

Default value: none
Inherited: no
Animatable: Yes. Read about animatable
Version CSS3
JavaScript syntax: object.style.WebkitFilter="grayscale(100%)" Try it

Browser Support

Nomor dalam tabel menentukan browser versi pertama yang mendukung properti secara penuh.

Nomor yang diikuti oleh -webkit-, -moz- atau -o- menentukan versi pertama yang bekerja dengan awalan kata.

BROWSER SUPPORT
Properti
filter 18.0 -webkit- Not supported 3.5 6.0 -webkit- 15.0 -webkit-

Catatan: Versi lama dari Internet Explorer (4.0 sampai 8.0) mendukung sebuah properti filter non-standar yang diprotes. Ini paling sering digunakan untuk opacity ketika memerlukan dukungan dari versi E8 kebawah.

Sintaksis CSS

filter: none | blur() | brightness() | contrast() | drop-shadow() | grayscale() | hue-rotate() | invert() | opacity() | saturate() | sepia() | url();

Tip: Untuk menggunakan multiple filter, memisahkan masing-masing filter dengan sebuah spaci (lihatlah "Contoh lebih" dibawah ini).

Fungsi-Fungsi Filter

Catatan: Filter yang menggunakan nilai percen (seperti contoh 75%), juga menerima nilai seperti decimal (contoh. 0.75).

Nilai Properti

Nilai Deskripsi Mainkan
none Nilai default. Menentukan tidak ada efek Klik disini
blur(px) Menerapkan sebuah efek blur pada gambar. Semakin besar nilai akan membuatnya lebih blur.
Jika tidak ada nilai yang ditentukan. 0 akan digunakan 		Klik disini
brightness(%) Menyesuaikan kecemerlangan (brightness) gambar.
0% akan membuat gambar (menjadi) hitam secara keseluruhan.
100% (1) adalah default dan merepresentasikan gambar aslinya.
Nilai lebih dari 100% akan menyediakan hasil yang lebih cemerlang (brighter). 		Klik disini
drop-shadow(h-shadow v-shadow blur spread color) Menerapkan sebuah efek bayangan untuk gambar. Nilai yang diperbolehkan:
h-shadow - Diperlukan. Menentukan sebuah nilai pixel untuk bayangan horizontal. Nilai negatif menempatkan bayangan ke kiri dari gambar.
v-shadow - Diperlukan. Menentukan sebuah nilai pixel untuk bayangan vertikal. Nilai negatif menempatkan bayangan ke atas dari gambar.
blur - Optional. Ini adalah nilai ketiga, dan harus dalam satuan pixel. Menambahkan sebuah efek blur untuk bayangan. Semakin besar nilai akan membuat semakin blur (bayanhan menjadi lebih besar dan lebih cerah). Nilai negatif diperbolehkan. jika tidak ada nilai. 0 akan digunakan (tepi bayangan runcing).
spread - Optional. Ini adalah nilai keempat. dan harus dalam satuan pixel. Nilai positif akan mengakibatkan bayangan menjadi meluas dan tumbuh lebih besar, dan nilai negatid mengakibatkan bayangan menyusut. Jika tidak ditentukan, ia akan menjadi 0 (bayangan akan berukuran sama seperti element).
Catatan: Chrome, Safari, dan Opera dan mungkin browser lainnya tidak mendukung panjang keempat; ini tidak akan terjemahkan jika ditambahkan.
color - Optional. Menambahkan warna pada bayangan. Jika tidak ditentukan, warna ditentukan oleh browser (biasanya hitam).
Sebuah contoh pembuata bayangan warna merah, yang mana besarnya 8px secara horizontal dan vertikal, dengan sebuah efek blur 10px:
filter: drop-shadow(8px 8px 10px red);
Tip: Filter ini mirip dengan properti box-shadow. 		Klik disini
grayscale(%) Mengkonversikan atau mengubah gambar menjadi grayscale.
0% (0) adalah default dan merepresentasikan gambar aslinya.
100% akan membuat gambar menjadi abu-abu secara keseluruhan (digunakan untuk gambar hitam dan putih).
Catatan: Nilai negatif tidak diperbolehkan. 		Klik disini
opacity(%) Mengatur level sifat tembus cahaya untuk gambar. opacity-level mendeskripsikan transparency-level, dimana:
0% adalah transparan penuh.
100% (1) adalah nilai default dan merepresentasikan gambar aslinya (tidak ada transparency).
Catatan: Nilai negatid tidak diperbolehkan.
Tip: Filter ini mirip dengan properti opacity.
Klik disini
sepia(%) Mengkonversikan/mengubah gambar menjadi sepia (warna coklat tua).
0% (0) adalah default dan merepresentasikan gambar aslinya.
100% akan membuat gambar menjadi sepia seutuhnya.
Catatan: Nilai negatif tidak diperbolehkan 		Klik disini
url() Fungsi url() mengambil lokasi dari sebuah file XML yang menentukan filter SVG, dan boleh memasukan sebuah anchor untuk element filter tertentu. Contoh:
filter: url(svg-url#element-id); 		Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
