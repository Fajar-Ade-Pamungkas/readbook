Definisi dan Penggunaan

Properti singkat border-top mengatur semua properti border bawah dalam satu deklarasi.

Properti-properti yang bisa diatur adalah (dalam border): border-top-width, border-top-style, and border-top-color.

Ini tidak penting jika salah satu dari nilai diatas dihilangkan, sebagai contoh. border-top:solid #ff0000; diperbolehkan.

Default value: medium none color
Inherited: no
Animatable: Yes, See individual properties Read about animatable Try it
Version CSS1
JavaScript syntax: object.style.borderTop="5px red solid" Try it

Sintaksis CSS

border-top: border-width border-style border-color|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
border-top-width Menentukan tebal/lebar border (bagian) bawah. Nilai defaultnya adalah "medium"
border-top-style Menentukan style/tipe border (bagian) bawah. Nilai defaultnya adalah "none"
border-top-color Menentukan warna border (bagian) bawah. Nilai defaultnya adalah warna element itu sendiri
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
