Definisi dan Penggunaan

Properti border-bottom-color mengatur warna dari border (bagian) bawah element.

Catatan: Tetaplah mendeklarasikan properti border-stule sebelum properti border-bottom-color. Sebuah element harus memiliki border sebelum kamu mengubah warnanya.

Default value: Warna element saat ini
Inherited: no
Animatable: Yes, Read about animatable Try it
Version CSS1
JavaScript syntax: object.style.borderBottomColor="blue" Try it

Sintaksis CSS

border-bottom-color: color|transparent|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
color Menentukan warna untuk border (bagian) bawah, lihatlah di CSS color values nilai-nilai warna diperbolehkan Klik disini
transparent Mengatur warna border (bagian) bawah menjadi transparan Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
