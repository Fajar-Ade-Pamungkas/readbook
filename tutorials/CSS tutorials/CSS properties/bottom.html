Definisi dan Penggunaan

Untuk element berposisi secara absolute, properti bottom mengatur tepi bawah dari suatu element untuk sebuah satuan diatas/dibawah dari sudut bawah tentang anchestor berposisi paling dekatnya. Catatan: Jika suatu element berposisi absolute tidak memiliki anchestor, ia menggunakan body element dan menggerakannya sepanjang page scrolling. Catatan: Element "berposisi" merupakan satu posisi yang memiliki apapun kecuali static.

Untuk element berposisi secara relative, properti bottom mengatur tepi bawah dari element untuk unit diatas/dibawah posisi normalnya.

Catatan: Jika "position:static;", properti bottom tidak memiliki efek.

Default value: auto
Inherited: no

Sintaksis CSS

bottom: auto|length|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
auto Membiarkan browser mengkalkulasikan posisi tepu bawah. Ini adalah nilai default Klik disini
length Mengatur posisi tepi bawah dalam (satuan) px, cm dan sebagainya. Nilai negatif diperbolehkan Klik disini
% Mengatur posisi tepi bawah dalam (satuan) % dari element pengisi. Nilai negatif diperbolehkan Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
