Definisi dan Penggunaan

Properti background-repeat mengatur jika/bagaimana sebuah background gambar akan diulang-ulang (repeat).

Dengan default, background-image diulang-ulang secara vertikal dan secara horizontal.

Tip: Background gambar ditempatkan berdasarkan properti background-position. Jika background-position tidak ditentukan, gambar selalu ditempatkan di sudut kiri atas element.

Default value: repeat
Inherited: no
Animatable: no. Read about animatable
Version CSS1
JavaScript syntax: object.style.backgroundRepeat="repeat-x" Try it

Catatan: IE8 dan versi sebelumnya tidak mendukung multiple background images pada satu eleement.

Sintaksis CSS

background-repeat: repeat|repeat-x|repeat-y|no-repeat|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
repeat Background gambar akan diulang-ulang secara vertikal dan secara horizontal. Ini adalah nilai default Klik disini
repeat-x Background gambar akan diulang-ulang secara horizontal saja Klik disini
repeat-y Background gambar akan diulang-ulang secara vertikal saja Klik disini
no-repeat Background gambar tidak akan diulang-ulang secara vertikal maupun horizontal Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
