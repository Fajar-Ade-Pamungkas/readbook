Definisi dan Penggunaan

Properti background-blend-mode mendefinisikan mode campuran dari setiap lapisan background (warna dan/atau gambar).

Default value: normal
Inherited: no
Animatable: no. Read about animatable
Version CSS3
JavaScript syntax: object.style.backgroundBlendMode="screen" Try it

Sintaksis CSS

background-blend-mode: normal|multiply|screen|overlay|darken|lighten|color-dodge|saturation|color|luminosity;

Nilai Properti

Nilai Deskripsi Mainkan
normal Nilai default. Mengatur mode campuran untuk/menjadi normal Klik disini
multiply Mengatur mode campuran untuk/menjadi multiply (berlipat ganda) Klik disini
screen Mengatur mode campuran untuk/menjadi screen (layar) Klik disini
overlay Mengatur mode campuran untuk/menjadi melapisi (overlay) Klik disini
darken Mengatur mode campuran untuk/menjadi menggelapkan (darken) Klik disini
lighten Mengatur mode campuran untuk/menjadi menerangi (lighten) Klik disini
color-dodge Mengatur mode campuran untuk/menjadi menghindari warna (color-dodge) Klik disini
saturation Mengatur mode campuran untuk/menjadi kejenuhan (saturation) Klik disini
color Mengatur mode campuran untuk/menjadi warna Klik disini
luminosity Mengatur mode campuran untuk/menjadi kilauan (luminosity) Klik disini
