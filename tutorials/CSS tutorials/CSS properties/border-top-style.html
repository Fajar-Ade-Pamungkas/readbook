Definisi dan Penggunaan

Properti border-top-style mengatur style/tipe dari border (bagian) atas element.

Default value: none
Inherited: no
Animatable: No, Read about animatable
Version CSS1
JavaScript syntax: object.style.borderTopStyle="dotted" Try it

Sintaksis CSS

border-top-color: none|hidden|outset|inset|ridge|double|solid|dotted|dashed|groove||transparent|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
dotted Mendefinisikan border titik-titik untuk style/tipe border (bagian) bawah Klik disini
dashed Mendefinisikan border garis putus-putus untuk style/tipe border (bagian) bawah Klik disini
solid Mendefinisikan border garis untuk style/tipe border (bagian) bawah Klik disini
double Mendefinisikan border garis ganda untuk style/tipe border (bagian) bawah Klik disini
groove Mendefinisikan border beralur (groove) 3D untuk style/tipe border (bagian) bawah, efek tergantung pada properti border-color Klik disini
ridged Mendefinisikan 3D ridged border untuk style/tipe border (bagian) bawah, efek tergantung pada properti border-color Klik disini
inset Mendefinisikan 3D inset border untuk style/tipe border (bagian) bawah, efek tergantung pada properti border-color Klik disini
outset Mendefinisikan 3D outset border untuk style/tipe border (bagian) bawah, efek tergantung pada properti border-color Klik disini
none Mendefinisikan tidak ada border untuk style/tipe border (bagian) bawah, Klik disini
hidden Mendefinisikan border tersembunyi untuk style/tipe border (bagian) bawah, Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
