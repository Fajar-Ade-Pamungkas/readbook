Definisi dan Penggunaan

Properti border-image-width menentukan ketebalan dari border gambar.

Tip: Lihatlah juga properti border-image (sebuah shorthand property untuk mengatur semua properti border-image-*).

Default value: 1
Inherited: no
Animatable: no. Read about animatable
Version CSS3
JavaScript syntax: object.style.borderImageWidth="30px" Try it

Sintaksis CSS

border-image-width: number|%|auto|initial|inherit;

Catatan: Properti border-image-width bisa mengambil satu dari empat nilai (sisi atas, kanan, bawah, dan kiri). Jika keempat nilai dihilangkan, maka nilai keempat sama seperti yang nilai kedua. Jika yang nilai ketiga dihilangkan juga, maka nilai ketiga akan sama dengan nilai pertama, jika nilai kedua dihilangkan juga, maka nilai kedua akan sama dengan nilai pertama.

Nilai Properti

Nilai Deskripsi Mainkan
length Sebuah satuan panjang (px) yang menentukan ukuran dari border-width Klik disini
number Nilai default adalah 1. Merepresentasikan multiple yang memiliki hubungan dengan border-width Klik disini
% Mengarah ke ukuran dari area border gambar: Lebar area untuk horizontal offsets, tinggi area untuk vertical offset Klik disini
auto Jika ditentukan, lebar merupakan tinggi atau lebar intrinsik dari image slice yang berhubungan Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
