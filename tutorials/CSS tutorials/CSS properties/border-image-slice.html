Definisi dan Penggunaan

Properti border-image-slice menentukan bagaimana untuk memotong gambar yang ditentu dengan border-image-source. Gambar harus dipotong kedalam sembilan bagian: empat sudut, 4 tepi, dan 1 tengah (middle);

Bagian "middle" (tengah) diperlakukan sebagai fully transparenr, tanpa kata kunci fill yang diatur.

Tip: Lihatlah juga properti border-image (sebuah shorthand property untuk mengatur semua properti border-image-*).

Default value: 100%
Inherited: no
Animatable: no. Read about animatable
Version CSS3
JavaScript syntax: object.style.borderImageSlice="30%" Try it

Sintaksis CSS

border-image-slice: number|%|fill|initial|inherit;

Catatan: Properti border-image-slice bisa mengambil satu dari empat nilai (sisi atas, kanan, bawah, dan kiri). Jika keempat nilai dihilangkan, maka nilai keempat sama seperti yang nilai kedua. Jika yang nilai ketiga dihilangkan juga, maka nilai ketiga akan sama dengan nilai pertama, jika nilai kedua dihilangkan juga, maka nilai kedua akan sama dengan nilai pertama.

Nilai Properti

Nilai Deskripsi Mainkan
number Nomor-nomor merepresentasikan pixel untuk pola scan (raster) gambar atau koodinat untuk gambar vektor Klik disini
% Percentages atau persen mengarah pada tinggi dan lebar dari gambar. Nilai defaultnya adalah 100% Klik disini
fill Mengakibatkan bagian tengah gambar untuk ditampilkan Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
