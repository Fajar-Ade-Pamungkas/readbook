Definisi dan Penggunaan

Properti border-collapse mengatur apakah border-border tabel dilekatkan (collapse) kedalam single border atau dilepaskan seperti dalam standar HTML.

Default value: separate
Inherited: Yes
Animatable: no. Read about animatable
Version CSS2
JavaScript syntax: object.style.borderCollapse="collapse" Try it

Catatan: Jika !DOCTYPE Tidak ditetapkan, properti border-collapse bisa memproduksi hasil yang tak diperkirakan.

Sintaksis CSS

border-collapse: separate|collapse|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
separate Border-border dilepaskan/dipisahkan (properti border-spacing dan empty-cells tidak akan diabaikan). Ini adalah nilai default Klik disini
collapse Border-border dilipat (collapse) kedalam single border ketika memungkinkan (properti border-spacing dan empty-cells akan diabaikan) Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
