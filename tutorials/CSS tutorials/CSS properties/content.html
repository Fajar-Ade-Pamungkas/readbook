Definisi dan Penggunaan

Properti content digunakan dengan pseudo-element :before dan :after, untik menyisipkan konten yang dihasilkan.

Default value: normal
Inherited: no
Animatable: no. Read about animatable
Version CSS2
JavaScript syntax: You can't give an element a pseudo-class by using JavaScript, but there are other ways to get the same result: Try it

Nomor dalam tabel menentukan browser versi pertama yang mendukung properti secara penuh.

Nomor yang diikuti oleh -webkit-, -moz- atau -o- menentukan versi pertama yang bekerja dengan awalan kata.

Properti

Catatan: IE8 hanya mendukung properti content jika !DOCTYPE ditentukan.

Sintaksis CSS

content: normal|none|counter|attr|string|open-quote|close-quote|no-open-quote|no-close-quote|url|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
normal Nilai default. Mengatur konten, jika ditentukan, menjadi nirmal, yang mana default adalah "none" (yang mana tak ada apapun) Klik disini
none Mengatur konten, jika ditentukan, menjadi tidak ada apapun (nothing) Klik disini
url(url) Mengatur konten menjadi beberapa macam media (sebuah gambar, suara, video, dan sebagainya) Klik disini
open-quote Mengatur konten untuk menjadi sebuah opening quote Klik disini
close-quote Mengatur konten untuk menjadi sebuah closint quote Klik disini
no-open-quote Mencopot opening quote dari konten. Jika ditentukan Klik disini
no-close-quote Mencopot closint quote dari konten. Jika ditentukan Klik disini
string Mengatur konten untuk teks yang kamu tentukan Klik disini
counter Mengatur konten sebagai counter (penghitung) Klik disini
attr(attribute) Mengatur konten sebagai salah satu dari selector's attribute Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
