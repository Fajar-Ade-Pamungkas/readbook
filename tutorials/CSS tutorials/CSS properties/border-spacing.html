Sintaksis CSS

border-spacing: length|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
length
length 		Menentukan jarak antara border-border dari berbatasan cell dalam (satuan) px, cm dan sebagainya. Nilai negatif tidak diperbolehkan
  • Jika (hanya) satu nilai length ditetapkan. Ia menentukan spacing secara vertikal dan horizontal
  • Jika dua nilai length ditetapkan. Nilai menentukan spacing secara horizontal. Nilai kedua menentukan spacing secara vertikal
Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
