Definisi dan Penggunaan

Properti border-image-outsetmenentukan jumlah dari area border gambar yang mana yang memperpanjang melebihi border box.

Tip: Lihatlah juga properti border-image (sebuah shorthand property untuk mengatur semua properti border-image-*).

Default value: 0
Inherited: no
Animatable: no. Read about animatable
Version CSS3
JavaScript syntax: object.style.borderImageOutset="10px" Try it

Sintaksis CSS

border-image-outset: length|number|initial|inherit;

Catatan: Properti border-image-outset bisa mengambil satu dari empat nilai (sisi atas, kanan, bawah, dan kiri). Jika keempat nilai dihilangkan, maka nilai keempat sama seperti yang nilai kedua. Jika yang nilai ketiga dihilangkan juga, maka nilai ketiga akan sama dengan nilai pertama, jika nilai kedua dihilangkan juga, maka nilai kedua akan sama dengan nilai pertama.

Nilai Properti

Nilai Deskripsi Mainkan
length Suatu satuan panjang yang menentukan berapa jauh dari tepi border-image akan ditampakkan. Nilai defaultnya adalah 0 Klik disini
number Merepresentasikan multiple dari yang memiliki hubungan dengan border-width Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
