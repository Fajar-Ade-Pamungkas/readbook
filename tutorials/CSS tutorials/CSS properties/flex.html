Definisi dan Penggunaan

Properti flex menentukan panjang dari item, relatif ke sisa dari flexible items didalam kontainer yang sama.

Properti flex merupakan sebuah shorthand property untuk flex-grow, flex-shrink, dan flex-basis.

Catatan: Jika element bukan sebuah flexible item, properti flex tidak memiliki pengaruh.

Default value: 0 1 auto
Inherited: no
Animatable: Yes. Read about animatable Try it
Version CSS3
JavaScript syntax: object.style.flex="1" Try it

Sintaksis CSS

flex: flex-grow flex-shrink flex-basis|auto|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
flex-grow Sebuah nomor yang menentukan berapa banyak item yang akan tumbuh relatif ke sisa dari flexible items
flex-shrink Sebuah nomor yang menentukan berapa banyak item yang akan merenggang relatif ke sisa dari flexible items
flex-basis Panjang item. Nilai legal: "auto", "inherit", atau nomor dengan satuan "%", "px", "em", atau satuan panjang lainnya
auto Sama seperti 1 1 auto
none Sama seperti 0 0 auto
initial Sama seperti 0 1 auto. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
