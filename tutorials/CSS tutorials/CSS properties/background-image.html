Definisi dan Penggunaan

Properti background-image mengatur satu atau lebih background gambar untuk sebuah element.

Background dari sebuah element adalah ukuran total dari element, termasuk padding dan border (tpi tidak termasuk margin).

Dengan default, sebuah background-image ditempatkan di sudut kiri atas dari element, dan diulang-ulang secara vertikal dan secara horizontal.

Tip: Jangan lupa untuk selalu mengatur background-color untuk digunakan jika gambar tidak tersedia.

Default value: none
Inherited: no
Animatable: no. Read about animatable
Version CSS + Nilai-nilai baru di CSS3
JavaScript syntax: object.style.backgroundImage="url(smiley.gif)" Try it

Sintaksis CSS

background-image: url|none|initial|inherit;

Nilai Properti

Nilai Deskripsi Mainkan
url('URL') URL untuk mencari gambar pada directory. Untuk menentukan lebih dari satu gambar, memisahkan URL-URL dengan koma Klik disini
none Tidak ada background gambar yang akan ditampilkan. Ini adalah nilai default Klik disini
initial Mengatur properti ini untuk nilai defaultnya. Read about initial
inherit Mewarisi properti ini dari nilai element induknya. Read about initial
