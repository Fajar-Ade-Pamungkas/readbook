Definisi dan Penggunaan

Properti fot-size-adjust memberikan kamu kontrol lebih baik tentang ukuran font ketika font terseleksi pertama tidak tersedia.

The font-size-adjust property gives you better control of the font size when the first selected font is not available.

Ketika sebuah font tidak tersedia, browser menggunakan font yang ditentukan kedua. Ini dapat hasil dalam sebuah perubahan besar untuk ujuran font. Untuk mengatasi ini, gunakanlah ptoperti font-size-adjust.

When a font is not available, the browser uses the second specified font. This could result in a big change for the font size. To prevent this, use the font-size-adjust property.

Semua font memiliki sebuah "aspect value" yang mana merupakan size-dofference antara huruf kecil "x" dan huruf besar "X".

All fonts have an "aspect value" which is the size-difference between the lowercase letter "x" and the uppercase letter "X".

Ketika browser mengetahui "aspect value" untuk font yang diseleksi pertama, browser bisa membentuk apa ukuran font untuk digunakan ketika tampilan teks dengan font pikihan kedua

When the browser knows the "aspect value" for the first selected font, the browser can figure out what font-size to use when displaying text with the second choice font.

Default value: none
Inherited: Yes
Animatable: Yes. Read about animatable
Version CSS3

Nilai Deskripsi Mainkan
