CSS Box Model

The CSS Box Model

Semua elemen HTML bisa diperhitungkan sebagai box. Di CSS, istilah "box model" digunakan ketika berbicara tentang design dan layout.

CSS Model Box secara pokok merupakan kotak yang membungkus sekeliling setiap elemen HTML. Box Model terdiri dari: margin, border, padding dan konten aktual.

Gambar dibawah ini mengilustrasikan Box Model:

margin
border
padding
content

Penjelasan dari bagian - bagian yang berbeda:

  • Content - Konten (isi) box, dimana teks dan gambar diperlihatkan
  • Padding - Membereskan area disekeliling konten. Padding ialah transparan (tidak berwarna)
  • Border - Sebuah garis atau border yang mengelilingi padding dan konten
  • Margin - Membereskan area diluar border. Margin juga transparan (tidak berwarna)

Box modek mengizinkan kita untuk menambahkan border disekitar element dan untuk mendefinisikan space (ruang) antara element

width: 300px;

Width and Height of an Element

Width dan Height dari suatu Element

Pada urutan untuk mengatur width (lebar) dan height (tinggi) suatu element dengan benar dalam semua browser, kamu perlu mengetahui bagaimana box model bekerja.

PENTING: Ketika kamu mengatur properti width dan height dari sebuah element dengan CSS, kamu juga mengatur width (lebar) dan height (tinggi) area konten. Untuk mengkalkulasi (menghitung) ukuran penuh dari suatu element, kamu juga harus menambahkan padding dan margin.

Mengharap kita ingin men-style element <div> untuk memiliki total lebar dari 350px:

Contoh:

Ketika layar (browser window) menjadi lebih kecil daripada 768px, setiap kolom harus memiliki width: 100%

div {
width: 320px;
padding: 10px;
border: 5px solid gray;
margin: 0;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div {
width: 320px;
padding: 10px;
border: 5px solid gray;
margin: 0;
} </style>
</head>
<body>

<h2>Calculate the total width:</h2>

<img src="klematis4_big.jpg" width="350" height="263" alt="Klematis">
<div>The picture above is 350px wide. The total width of this element is also 350px.</div>

</body>
</html>

Calculate the total width:

The picture above is 350px wide. The total width of this element is also 350px.

Disini adala perhutungannya:

320px (width)
+ 20px (padding kanan + kiri)
+ 10px (border kanan + kiri)
+ 0px (margin kanan + kiri)
= 350px

Total width dari suatu element harus dikalkulasikan seperti:

Lebar total element = width (atau lebar) + padding kiri + padding kanan + border kiri + border kanan + margin kiri + margin kanan

Total height dari suatu element harus dikalkulasikan seperti:

Tinggi total element = hight (atau tinggi) + padding atas + padding bawah + border atas + border bawah + margin atas + margin bawah

Catatan untuk IE lama: Internet Explorer 8 atau versi sebelumnya, memasukkan padding dan border dalam properti width. Untuk memperbaiki masalah ini, tambahkanlah <!DOCTYPE html> ke halaman HTML.
