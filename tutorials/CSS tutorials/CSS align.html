CSS Layout - Horizontal Align

Dalam CSS, bebebrapa properti bisa digunakan untuk perataan (teks) element secara horizontal.

Center Align - Using margin

Perataan (teks) Tengah - Menggunakan (properti) margin

Pengaturan lebar dari block-level element akan mencegahnya dari peregangan (stretching) ke tepi dari penampungnya (container). Gunakanlah margin: auto; untuk secara horizontal menengahkan suatu elemen dengan kontainernya.

Element inu akan mengambil lebar tertentu dan sisa dari ruang akan di-split (membelah) secara sama antara dua margin (kiri dan kanan):

Contoh:

.center {
    margin: auto;
    width: 60%;
    border: 3px solid #73AD21;
    padding: 10px;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
.center {
    margin: auto;
    width: 60%;
    border: 3px solid #73AD21;
    padding: 10px;
}
</style>
</head>
<body>

<div class="center">
<p><b>Note: </b>Using margin:auto will not work in IE8, unless a !DOCTYPE is declared.</p>
</div>

</body>
</html>

Note: Using margin:auto will not work in IE8, unless a !DOCTYPE is declared.

Tip: Perataan tengah (center) tidak memiliki efek jika properti width tidak diatur (atau mengaturnya menjadi 100%).

Tip: Perataan teks, lihatlah pada chapter CSS Text.

Left and Right Align - Using position

Perataan Kiri dan Kanan - Menggunakan (properti) position

Satu metode untuk perataan element ialah dengan mengggunakan properti position: absolute; :

Contoh:

.right {
    position: absolute;
    right: 0px;
    width: 300px;
    border: 3px solid #73AD21;
    padding: 10px;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
.right {
    position: absolute;
    right: 0px;
    width: 300px;
    border: 3px solid #73AD21;
    padding: 10px;
}
</style>
</head>
<body>

<div class="right">
<p>In my younger and more vulnerable years my father gave me some advice that I've been turning over in my mind ever since.</p>
</div>

</body>
</html>

In my younger and more vulnerable years my father gave me some advice that I've been turning over in my mind ever since.

Catatan: Element berposisi absolute dicopot dari aliran normal (normal flow) dan bisa saling meliputi/melengkapi (overlap) element.

Tip: Ketika perataan element dengan properti position, selalu definisikan properti margin dan padding untuk element <body>. Ini uhtuk menghindari perbedaan visual dalam browser yang berbeda.

Ada juga satu masalah dengan IE8 dan versi sebelumnya, ketika menggunakan properti position. Jika container element atau element penampung (dalam kasus kita <div class="container">) memiliki lebar tertentu, dan deklarasi !DOCTYPE dihilangkan, IE8 dan versi sebelumnya akan menambahkan margin 17px pada sisi kanan. Ini terlihat menjadi space reserved (pemesanan ruang) untuk scrollbar. Jadi, selalu atur deklarasi !DOCTYPE ketika menggunakan properti position:

Contoh:

body {
    margin: 0;
    padding: 0;
}

.container {
    position: relative;
    width: 100%;
}

.right {
    position: absolute;
    right: 0px;
    width: 300px;
    background-color: #b0e0e6;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
body {
    margin: 0;
    padding: 0;
}

.container {
    position: relative;
    width: 100%;
}

.right {
    position: absolute;
    right: 0px;
    width: 300px;
    background-color: #b0e0e6;
}
</style>
</head>
<body>

<div class="container">
<div class="right">
<p><b>Note: </b>When aligning using the position property, always include the !DOCTYPE declaration! If missing, it can produce strange results in IE browsers.</p>
</div>
</div>

</body>
</html>

Note: When aligning using the position property, always include the !DOCTYPE declaration! If missing, it can produce strange results in IE browsers.
