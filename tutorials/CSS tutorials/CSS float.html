CSS Layout - Float and Clear Properties

Properti float menentukan apakah iya atau tidak suatu element harus float (mengapung).

Properti clear digunakan untuk mengontrol tingkah laku dari floating element.

The clear Property

Properti clear digunakan untuk mengontrol tingkah laku dari floating element (element apung).

Element setelah sebuah floating element akan mengalir (flow) disekitarnya. Untuk menghindari ini, gunakanlah properti clear.

Properti clear menentukan pada sisi yang mana dari suatu element, floating element tidak diperbolehkan untutuk mrngapung (float):

Contoh:

div {
    clear: left;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
.div1 {
float: left;
width: 100px;
height: 50px;
margin: 10px;
border: 3px solid #73AD21;
}

.div2 {
border: 1px solid red;
}

.div3 {
float: left;
width: 100px;
height: 50px;
margin: 10px;
border: 3px solid #73AD21;
}

.div4 {
border: 1px solid red;
clear: left;
}
</style>
</head>
<body>

<h2>Without clear</h2>
<div class="div1">div1</div>

<h2>Using clear</h2>
<div class="div3">div3</div>
<div class="div4">div4 - Using clear moves div4 down below the floated div3. The value "left" clears elements floated to the left. You can also clear "right" and "both".</div>

The clearfix Hack - overflow: auto;

Jika suatu element tinggi atau ramping daripada penampung element tersebut. dan element itu diapungkan (floated), element akan meluap (overflow) keluar dari penampungnya (container).

Kemudian kita bisa menambahkan overflow: auto; untuk penampung element untuk memperbaiki masalah ini:

Contoh:

.clearfix {
overflow: auto;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div {
border: 3px solid #73AD21;
}

.img1 {
float: right;
}

.clearfix {
overflow: auto;
}

.img2 {
float: right;
}
</style>
</head>
<body>

<p>In this example, the image is taller than the element containing it, and it is floated, so it overflows outside of its container:</p>

All CSS Float Properties

Properti Deskripsi
clear Menentukan pada sisi yang mana dari sebuah element dimana floating element tidak diizinkan untuk mengapung (float)
Nilai:
  • none - Default, Mengizinkan floating element pada kedua sisi
  • left - Tidak ada floating element yang diperbolehkan pada sisi kiri
  • right - Tidak ada floating element yang diperbolehkan pada sisi kanan
  • both - Tidak ada floating element yang diperbolehkan pada salah satu dsri kedua sisi kiri dan kanan
  • inherit - Mewariskan (nilai) properti ini dari parent element (elemen induk)
  • initial - Mengatur (agar) properti ini menjadi nilai default.
float Menentukan apakah iya atau tidak suatu element harus mengapung (float)
Nilai:
  • none - Element tidak diapungkan (not floated), dan akan ditampilkan hanya dimana element terjadi dalsm teks. Ini merupakan nilai default
  • left - Element mengapung ke kiri
  • right - Element mengapung ke kanan
  • inherit - Mewariskan (nilai) properti ini dari parent element (elemen induk)
  • initial - Mengatur (agar) properti ini menjadi nilai default.
overflow Menentukan apa yang terjadi jika konten melebihi batas (overflow) box element
Nilai:
  • visible - nilai default. Konten tidak di-clip dan ini mungkin menyumbang keluar content box
  • hidden - Konten di-clip dan tidak ada scrolling mechanim yang disediakan
  • scroll - Konten di-clip dan ada scrolling mechanism yang disediakan
  • auto - Harus meyebabkan ada scrolling mechanism yang disediakan
  • inherit - Mewariskan (nilai) properti ini dari parent element (elemen induk)
  • initial - Mengatur (agar) properti ini menjadi nilai default.
overflow-x Menentukan apa yang dilakukan dengan tepi kiri/kanan dari kontent jika ia melebihi batas area konten element
Nilai:
  • visible - nilai default. Konten tidak di-clip dan ini mungkin menyumbang keluar content box
  • hidden - Konten di-clip dan tidak ada scrolling mechanim yang disediakan
  • scroll - Konten di-clip dan ada scrolling mechanism yang disediakan
  • auto - Harus meyebabkan ada scrolling mechanism yang disediakan
  • inherit - Mewariskan (nilai) properti ini dari parent element (elemen induk)
  • initial - Mengatur (agar) properti ini menjadi nilai default.
overflow-y Menentukan apa yang dilakukan dengan tepi atas/bawah dari kontent jika ia melebihi batas area konten element
Nilai:
  • visible - nilai default. Konten tidak di-clip dan ini mungkin menyumbang keluar content box
  • hidden - Konten di-clip dan tidak ada scrolling mechanim yang disediakan
  • scroll - Konten di-clip dan ada scrolling mechanism yang disediakan
  • auto - Harus meyebabkan ada scrolling mechanism yang disediakan
  • inherit - Mewariskan (nilai) properti ini dari parent element (elemen induk)
  • initial - Mengatur (agar) properti ini menjadi nilai default.
