CSS3 3D Transforms

Pada chapter ini kamu akaj mempelajari tentang metode-metode transformasi 3 Dimensi berikut:

  • rotateX()
  • rotateY()
  • rotateZ()

The rotateX() Method

Metode rotateX() digunakan untuk merotasi/memutar sebuah element disekitar sumbu-X (nya) pada nilai derajat yang diberikan.

Contoh:

div {
-webkit-transform: rotateX(150deg); /* Safari */
transform: rotateX(150deg);
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div {
width: 300px;
height: 100px;
background-color: yellow;
border: 1px solid black;
}

div#myDiv {
-webkit-transform: rotateX(150deg); /* Safari */
transform: rotateX(150deg); /* Standard syntax */
}
</style>
</head>
<body>


This a normal div element.



The rotateX() method rotates an element around its X-axis at a given degree. This div element is rotated 150 degrees.


Note: Internet Explorer 9 (and earlier versions) does not support the rotateX() method.



</body>
</html>

CSS3 Transform Properties

Daftar tabel berikut adalah semua properti transformasi 3D:

Properti Deskripsi
transform Menerapkan transformasi 2D atau 3D untuk sebuah element
transform-origin Memperbolehkan kamu untuk mengubah posisi pada element bertransformasi
transform-style Menentukan bagaimana nested element dibuat (render) pada ruang 3 Dimensi
perspective Menentukan perspektif (pemandangan) pada bagaimana element 3 Dimensi ditunjukan
perspective-origin Menentukan posisi bawah dari element 3 Dimensi
backface-visibility Mendefinisikan apakah (iya) atau tidak element harus diperlihatkan ketika bukan lapisan luar layar (not facing the screen)

3D Transform Methods

Fungsi Deskripsi
matrix3d(n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n) Mendefinisikan transformasi 3D, menggunakan 4x4 matrix dari 16 nilai
translate3d(x,y,z) Mendefinisikan translasi 3D
translateX(x) Mendefinisikan translasi 3D, hanya menggunakan nilai untuk sumbu-X
translateY(y) Mendefinisikan translasi 3D, hanya menggunakan nilai untuk sumbu-Y
translateZ(z) Mendefinisikan translasi 3D, hanya menggunakan nilai untuk sumbu-Z
scale3d(x,y,z) Mendefinisikan transformasi skala 3D
scaleX(x) Mendefinisikan transformasi skala 3D dengan memberikan sebuah nilai untuk sumbu-X
scaleY(y) Mendefinisikan transformasi skala 3D dengan memberikan sebuah nilai untuk sumbu-Y
scaleZ(z) Mendefinisikan transformasi skala 3D dengan memberikan sebuah nilai untuk sumbu-Z
rotate3d(x,y,z,angle) Mendefinisikan rotasi 3 Dimensi
rotateX(angle) Mendefinisikan rotasi 3 Dimensi sepanjang sumbu-X
rotateY(angle) Mendefinisikan rotasi 3 Dimensi sepanjang sumbu-Y
rotateZ(angle) Mendefinisikan rotasi 3 Dimensi sepanjang sumbu-Z
perspective(n) Mendefinisikan sebuah perspective view (tampilan pemandangan) untuk element bertransformasi 3 Dimensi
