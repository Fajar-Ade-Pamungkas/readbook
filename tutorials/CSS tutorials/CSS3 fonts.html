CSS3 Web Fonts

CSS3 Web Fonts - The @font-face Rule

Web font mengizinkan para pendisain web utuk menggunakan font yang tidak diistall pada komputer pengguna.

Ketika kamu telah menemukan/membeli font yang kamu inginkan untuk digunakan, hanya masukan font file pada web server kalian, dan font tersebut akan secara otomatis didownload untuk pengguna ketika dibutuhkan:

Font "milik" kamu didefinisikan dengan aturan @font-face CSS3.

Different Font Formats

Perbedaan format font

TrueType Fonts (TTF)

TrueType merupakan standar font yang dikembangkan pada akhir 1980-an, oleh Apple dan Miscrosoft. TrueType merupakan format font paling umum untuk Mac OS dan Microsoft Windows operating systems.

OpenType Fonts (OTF)

OpenType adalah sebuah format untuk scalable comouter fonts. OpenType dibangun pada TruType, dan merupakan registered trademark of Microsoft (merk yang terdaftar milik microsoft). OpenType font biasanya digunakan saat ini pada major computer platforms.

Web Open Font Format (WOFF)

WOFF adalah sebuah format font untuk digunakan dalam web page, WOFF dikembangkan pada tahun 2009, dan sekarang merupakan rekomendasi W3C. WOFF merupakan hal pokok OpenType dan TruType dengan peringkasan dan penambahan metadata. Tujuannya adalah untuk mendukung pendistribusian font dari server kepada client lewat network dengan bandwidth constraints (batasan bandwidth).

Web Open Font Format (WOFF 2.0)

TrueType/OpenType font yang menyediakan peringkasan (compression) lebih baik daripada WOFF 1.0.

SVG Fonts/Shapes

SVG font mengizinkan SVG untuk digunakan sebagai glyphs (figur simbolis) ketika menampilksn teks. Spesifikasi SVG 1.1 mendefinisikan sebuah modul font yang mengizinkan penciptaan font dengan sebuah dokumen SVG. Kamu juga bisa menerapkan CSS untuk dokumen SVG, dengan aturan @font-face bisa diterapkan untuk teks dalam dokumen SVG.

Embedded OpenType Fonts (EOT)

EOT font merupakan sebuah compact form dari OpenType font yang didesign oleh Microsoft untuk menggunakan font yang ditanam pada web pages.

Using The Font You Want

Penggunan Font yang Kamu Mau

Pada aturan @font-face CSS3 kamu pertama kali harus mendefinisikan sebuah nama kepada font (sebagai contoh: myFirstFont), dan kemudian point untuk font file.

Tip: Selalu gunakan huruf kecil untuk font URL. hpHuruf besar bisa memberikan hasil ekspektasi yang keliru dalam IE.

Untuk emnggunakan font kepada sebuah element HTML, arahkanlah pada nama dari font (myFirstFont) sepanjang properti font-family:

Contoh:

@font-face {
font-family: myFirstFont;
src: url(sansation_light.woff);
}

div {
font-family: myFirstFont;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
@font-face {
font-family: myFirstFont;
src: url(sansation_light.woff);
}

div {
font-family: myFirstFont;
}
</style>
</head>
<body>

<div>
With CSS3, websites can finally use fonts other than the pre-selected "web-safe" fonts.
</div>

<p><b>Note:</b> Internet Explorer 8 and earlier, do not support the @font-face rule.</p>

</body>
</html>

Using Bold Text

Penggunaan Teks Tebal

Kamu harus menambahkan aturan @font-face lainnya termasuk descriptor untuk bold text:

Contoh:

@font-face {
font-family: myFirstFont;
src: url(sansation_bold.woff);
font-weight: bold;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
@font-face {
font-family: myFirstFont;
src: url(sansation_light.woff);
}

@font-face {
font-family: myFirstFont;
src: url(sansation_bold.woff);
font-weight: bold;
}

div {
font-family: myFirstFont;
}
</style>
</head>
<body>

<div>
With CSS3, websites can <b>finally</b> use fonts other than the pre-selected "web-safe" fonts.
</div>

<p><b>Note:</b> Internet Explorer 8 and earlier, do not support the @font-face rule.</p>

</body>
</html>
With CSS3, websites can finally use fonts other than the pre-selected "web-safe" fonts.

Note: Internet Explorer 8 and earlier, do not support the @font-face rule.

File "sansation_bold.woff" adalah font file lainnya, yang berisikan karakter tebal untuk Sansation font.

Browser akan menggunakan ini kapanpun sebuah bagian dari teks dengan font-family "myFirstFont" harus lebih memberi seperti bold.

Cara ini ksmu bisa memiliki banyakn @font-face rule untuk font yang sama.
