CSS Units

CSS memiliki beberapa units (satuan-satuan) yang berbeda untuk pengekspresian/pengukuran panjang.

Banyak properti CSS mengambil nikai units (satuan-satuan) "panjang", seperti width, margin, padding, font-size, border-width, dan sebagainya.

Panjang merupakan nomor yang diikuti oleh satuan panjang, seperti 10px, 2em, dan sebagainya.

Whitespace (spasi) tidak diperlihatkan diantara nomor dan unit. Akan tetapi, jika nilainya 0, unit bisa dihilangkan.

Untuk beberapa properti CSS, (nilai) panjang negatif tidak diperbolehkan.

Ada dua tipe dari satuan panjang: relative dan absolute.

Relative Lengths

Satuan panjang Relatif menentukan panjang relatif untuk properti panjang lainnya. Satuan panjang relatif menskala lebih baik diantara rendering mediums yang berbeeda.

Unit Deskripsi
em Relatif (cenderung) untuk font-size dari element (2em berarti 2 times ukuran dari font saat ini) Klik disini
ex Relatif (cenderung) x-height dari font saat ini (rarely used) Klik disini
ch Relatif (cenderung) untuk width (lebar) dari "0" (nol)
rem Relatif (cenderung) untuk font-size dari root element
vw Relatif (cenderung) untuk 1% dari width (lebar) viewport* Klik disini
vh Relatif (cenderung) untuk 1% dari height (tinggi) viewport* Klik disini
vmin Relatif (cenderung) untuk 1% dari viewport's* smaller dimension (dimensi viewport yang lebih kecil) Klik disini
vmax Relatif (cenderung) untuk 1% dari viewport's* larger dimension (dimensi viewport yang lebih besar) Klik disini
%

Tips: Satuan em dan rem merupakan latihan dalam pembuatan perfectly scalable layout!

* Viewport = Ukuran window browser. Jika viewport adalah 50cm lebar, 1vw = 0.5 cm.

Absolute Lengths

Satuan-satuan panjang absolut (mutlak) diperbaiki dan sebuah panjang yang diekpresikan dalam segala dari ini akan diperlihatkan sebagai ukuran tepat.

Satuan-satuan panjang absokut tidak direkomendasi untuk digunakan pada screen (layar), karena ukuran layar yang bervariasi. Akan tetapi, itu bisa digunakan jika output medium yang diketahui, seperti untuk print layout.

Unit Deksripsi
cm Centimeter Klik disini
mm milimeters Klik disini
in inches (1in = 96px = 2.54cm) Klik disini
px* Pixels (1px = 1/96th dari 1in) Klik disini
pt points (1pt = 1/72 dari 1in) Klik disini
pc picas (1pc = 12pt) Klik disini

* Pixels (px) merupakan (satuan panjang) relatif untuk perangkat tampilan. Untuk low-dpi devixpces, 1px merupakan one device pixel (dot) dari display. Untuk printer dan layar resolusi tinggi 1px menyatakan multiple device pixels.
