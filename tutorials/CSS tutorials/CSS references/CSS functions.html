CSS Functions Reference

CSS Functions

Kolom "CSS" mengindikasikan di CSS versi yang mana fungsi didefinisikan (CSS1, CSS2, CSS3).

Fungsi Deskripsi CSS
attr() Mengembalikan nilai dari sebuah attribute dari element yang terseleksi 2
calc() Memperbolehkan kamu untuk menampilkan kalkulasi untuk memutuskan nilai properti CSS 3
cubic-bezier() Menentukan rumus bezier untuk element 3
hsl() Menentukan rumus pencampuran warna dengan metode hue (corak warna), saturation (kejenuhan), lighting (pencahayaan) 3
hsla() Menentukan rumus pencampuran warna dengan metode hue (corak warna), saturation (kejenuhan), lighting (pencahayaan) dan alpha (sifat tembus cahaya) 3
linear-gradient() Membuat sebuah "image" yang mana mempresentasikan gradien linier warna 3
radial-gradient() Membuat sebuah "image" yang mana mempresentasikan sebuah gradient warna radial (seperti gelombang sinyal) dari pusat gradient 3
repeating-linear-gradient() Membuat sebuah "image" terdiri dari gradient berulang-ulang 3
repeating-radial-gradient() Mirip dengan radial-gradient(), tapi ia mengulang secara otomatis color stop dengan tidak terbatas di kedua arahnya 3
rgb() Menentukan rumus pencampuran warna red, green, blue untuk element 2
rgba() Menentukan rumus pencampuran warna red, green, blue dengan penambahan alpha (sifat tembus cahaya) untuk element 3

Contoh 2: Fungsi calc()

Menggunakan calc() untuk mengkalkulasikan lebar dari element <div>:

position: absolute;
left: 50px;
width: calc(100% - 100px);
border: 1px solid black;
background-color: yellow;
padding: 5px;
text-align: center;
}

<head>

left: 50px;
width: calc(100% - 100px);

}
</style>
</head>
