CSS Legal Color Values

CSS Colors

Warna-warna dalam CSS bisa ditentukan dengan metode-metode berikut:

  • Hexadecimal colors
  • RGB colors
  • RGBA colors
  • HSL colors
  • HSLA colors
  • Predefined/Color-browser color names

Hexadecimal Colors

Nilai warna Hexadecimal didukung di semua mayor browser.

Sebuah warna hexadecimal ditentukan dengan #RRGGBB, dimana RR (red), GG (green), BB (blue). bilangan hexadecimal menentukan komponen-komponen warna. Semua nilai hsrus berada diantara 00 dan FF.

Untuk contoh, nilai #0000ff diterjemahkan sebagai biru, karena komponen biru diatur menjadi nilai tertingginya (ff) dan yang lainnya diatur menjadi 00.

Contoh:

Mendefinisikan warna HEX yang berbeda:

#p1 {background-color: #ff0000;}   /* red */
#p2 {background-color: #00ff00;}   /* green */
#p3 {background-color: #0000ff;}   /* blue */
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
#p1 {background-color:#ff0000;}
#p2 {background-color:#00ff00;}
#p3 {background-color:#0000ff;}
#p4 {background-color:#ffff00;}
#p5 {background-color:#ff00ff;}
</style>
</head>
<body>

<p>RGB colors:</p>

RGB Colors

Nilai warna RGB didukung di semua mayor browser.

Sebuah nilai warna RGB ditentukan dengan rgb(red, green, blue). Etiap parameter (red, green, blue) mendefinsikan intensitas warna dan bisa menjadi bilangan antara 0 sampai 255 atau nilai persen (dari 0% sampai 100%).

Untuk contoh, nilai rgb(0,0,255) diterjemahkan sebagai biru, karena parameter blue diataur menjadi lebih tinggi dan yang lainnya diatur manjadi 0.

Juga, niali berikut mendefinisikan warna yanh sama rgb(0,0,255) dan rgb(0%,0%,100%).

Contoh:

Mendefinisikan warna RGB yang berbeda:

#p1 {background-color: rgb(255,0,0);}   /* red */
#p2 {background-color: rgb(0,255,0);}   /* green */
#p3 {background-color: rgb(0,0,255);}   /* blue */
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
#p1 {background-color:rgb(255,0,0);}
#p2 {background-color:rgb(0,255,0);}
#p3 {background-color:rgb(0,0,255);}
#p4 {background-color:rgb(192,192,192);}
#p5 {background-color:rgb(255,255,0);}
#p6 {background-color:rgb(255,0,255);}
</style>
</head>

RGBA Colors

Nilai warna RGBA didukung di IE9+, Firefox 3+, Chrome, Safari, dan di Opera 10+.

Nilai warna RGBA merupakan sebuah ekstensi dari nilai warna RGB dengan sebuah Alpha channel - Yang mana menentukan opacity (sifat tembus cahaya) dari object.

Sebuah nikai warna RGBA ditentukan dengan: rgba(red, green, blue, alpha). Parameter alpha adalah sebuah nomor diantara 0.0 (transparan secara penuh) dan 1.0 (tidak transparan).

Contoh:

Mendefinisikan warna RGB yang berbeda dengan opacity:

#p1 {background-color: rgba(255, 0 , 0, 0.3);}   /* red with opacity */
#p2 {background-color: rgba(0, 255 , 0, 0.3);}   /* green with opacity */
#p3 {background-color: rgba(0,0,255, 0,3);}   /* blue with opacity */
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
#p1 {background-color:rgba(255, 0, 0, 0.3);}
#p2 {background-color:rgba(0, 255, 0, 0.3);}
#p3 {background-color:rgba(0, 0, 255, 0.3);}
#p4 {background-color:rgba(192, 192, 192, 0.3);}
#p5 {background-color:rgba(255, 255, 0, 0.3);}
#p6 {background-color:rgba(255, 0, 255, .3);}
</style>
</head>
<body>

<p>RGBA colors:</p>

HSL Colors

Nilai warna HSL didukung di IE9+, Firefox, Chrome, Safari daj Opera 10+.

HSL merepresentasikan hue (corak warna), saturation (kejenuhan), lightness (tingkat keterangan) - dan merepresentasikan sebuah cylindrical-coordinate representation of colors.

Sebuah nilai warna HSL ditentukan dengan: hsl(hue, saturation, lightness).

Hue adalah sebuah derajat di roda warna (dari 0 sampai 360) - 0 (atau 360) adalah merah, 120 adalah hijau, 240 adalah biru. Saturation adalah nilai persen: 0% berarti sebuah bentuk dari gray (abu-abu) dan 100% adalah full color. Lightness juga adalah persen: 0% adalah hitam, 100% adalah putih.

Contoh:

Mendefinisikan warna HSL yang berbeda:

#p1 {background-color: hsl(120, 100%, 50%);}   /* green */
#p2 {background-color: hsl(120, 100%, 75%);}   /* light green */
#p3 {background-color: hsl(120, 100%, 25%);}   /* dark green */
#p4 {background-color: hsl(120, 60%, 70%);}    /* pastel green */
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
#p1 {background-color:hsl(120,100%,50%);}
#p2 {background-color:hsl(120,100%,75%);}
#p3 {background-color:hsl(120,100%,25%);}
#p4 {background-color:hsl(120,60%,70%);}
#p5 {background-color:hsl(290,100%,50%);}
#p6 {background-color:hsl(290,60%,70%);}
</style>
</head>
<body>

<p>HSL colors:</p>

HSLA Colors

Nilai warna HSLA didukung di IE9+, Firefox 3+, Chrome, Safari, dan di Opera 10+.

Nilai warna HSLA merupakan ekstensi dari nilai warna HSL dengan Alpha channel - Yang mana mengindikasikan opacity (sifat tembus cahaya) dari object.

Sebuah nilai HSLA ditentukan dengan: hsla(hue, saturation, lightness, alpha), dimana parameter alpha mendefinisikan opacity. Parameter alpha merupakan sebuah nomor antara 0.0 (transparan secara penuh) dan 1.0 (tidak transparan).

Contoh:

Mendefinisikan warna HSL yang berbeda dengan opacity:

#p1 {background-color: hsla(120, 100%, 50%, 0.3);}   /* green with opacity*/
#p2 {background-color: hsla(120, 100%, 75%, 0.3);}   /* light green with opacity */
#p3 {background-color: hsla(120, 100%, 25%, 0.3);}   /* dark green with opacity */
#p4 {background-color: hsla(120, 60%, 70% 0.3);}    /* pastel green with opacity */
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
#p1 {background-color:hsla(120,100%,50%,0.3);}
#p2 {background-color:hsla(120,100%,75%,0.3);}
#p3 {background-color:hsla(120,100%,25%,0.3);}
#p4 {background-color:hsla(120,60%,70%,0.3);}
#p5 {background-color:hsla(290,100%,50%,0.3);}
#p6 {background-color:hsla(290,60%,70%,0.3);}
</style>
</head>
