CSS Selector Reference

CSS Selectors

Dalam CSS, Selektor adalah pola yang digunakan untuk menyeleksi element(s) yang kamu mau untuk di-style.

Gunakanlah CSS Selector Tester untuk mendemonstrasikan selekto-selektor yang berbeda.

Kolom "CSS" mengindikasikan di CSS versi yang mana propertu didefinisikan (CSS1, CSS2, CSS3).

Properti Contoh Deskripsi Contoh CSS

#id #firstname Menyeleksi element dengan id="firstname" 1
* * Menyeleksi semua element 2
element p Menyeleksi semua element <p> 1
