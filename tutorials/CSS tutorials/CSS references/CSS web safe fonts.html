Kombinasi Font yang umum digunakan

Properti font-family harus mengikat beberapa nama-nama font seperti sistem "fallback", untuk memastikan kompabilitas maksimal antara browser/sistem operasi. Jika browser tidak mendukung font pertama, Ini akan mencoba font selanjutnya.

Mulailah dengan font yang inginkan, dan akhiri dengan generic family sebagai sistem "fallback", untuk membiarkan browser mengambil font yang mirip dalam generic famiky, jika tidak ada font lainnya yang tersedia:

Contoh:

p {
font-family: "Times New Roman", Times, serif;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
p.serif {
font-family: "Times New Roman", Times, serif;
}

p.sansserif {
font-family: Arial, Helvetica, sans-serif;
}
</style>
</head>
<body>

<h1>CSS font-family</h1>
<p class="serif">This is a paragraph, shown in the Times New Roman font.</p>
<p class="sansserif">This is a paragraph, shown in the Arial font.</p>
</body>
</html>
