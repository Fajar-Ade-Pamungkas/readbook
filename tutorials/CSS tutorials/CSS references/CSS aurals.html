Aural Style Sheets

Aural style sheet menggunakan sebuah kombinasi dari perpaduan bicara dan efek suara untuk membuat pengguna mendengar informasi, pengganti dari informasi bacaan.

Penyajian aural bisa digunakan:

  • Oleh orang tuna netra
  • Untuk membantu para pengguna untuk belajar membaca
  • Untuk membantu para pengguna yang memikiki masalah membaca
  • Untuk home entertaiment
  • Dalam mobil
  • Oleh komunitas-komunitas print-impaired

Penyajian aural mengubah dokumen menjadi teks penjelasan dan umpan ini untuk screen reader (sebuah program yang membaca semua karakter pada layar).

Sebuah contoh dari Aural style sheet:

h1, h2, h3, h4{
voice-family: male;
richness: 80;
cue-before: url("beep.au");
}

Contoh diatas akan membuat speech synthesizer memutar suara, lalu membicarakan (mengeja) headers dalam setiap rich male voice.
