CSS Animatable

Definition and Usage

Penjelasan dan Penggunaan

Beberapa properti CSS merupakan animatable, berarti bahwa properti itu bisa digunakan dalam animation dan transition.

Properti (yang termasuk) animatable bisa berubah secara berangsung-angsur dari satu nilai ke nilai lainnya, seperti ukuran, nomor, persen dan warna.

Browser Support

Nomor dalam tabel menentukan browser versi pertama yang mendukung secara penuh CSS animation.

Nomor-nomor yang diikuti dengan -webkit-, -moz-, atau -o- menentukan versi pertama yang berkerja dengan awalan kata.

Example

Contoh:

Menganimasikan background-color dari merah menjadi biru:

/* Code for Chrome, Safari and Opera */
@-webkit-keyframes mymove {
from {background-color: red;}
to {background-color: blue;}
}

/* Standard syntax */
@keyframes mymove {
from {background-color: red;}
to {background-color: blue;}
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
#myDIV {
width: 300px;
height: 200px;
background: red;
-webkit-animation: mymove 5s infinite; /* Chrome, Safari, Opera */
animation: mymove 5s infinite;
}

/* Code for Chrome, Safari and Opera */
@-webkit-keyframes mymove {
from {background-color: red;}
to {background-color: blue;}
}

/* Standard syntax */
@keyframes mymove {
from {background-color: red;}
to {background-color: blue;}
}
</style>
</head>
<body>

<p>Gradually change the background-color from red, to blue:<p> <div id="myDIV"></div> <p>The background-position property is <em>animatable</em> in CSS.</p> <p><strong>Note:</strong> CSS Animations do not work in Internet Explorer 9 and earlier versions.</p> </body>
</html>
