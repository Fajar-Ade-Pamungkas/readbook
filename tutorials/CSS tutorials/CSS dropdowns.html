Example Explained

Contoh menerangkan

HTML → Gunakanlah element apapun untuk membuat konten dropdown, sebagai contoh element <span> atau <button>.

Gunakanlah sebuah element penampung (seperti <div>) untuk membuat konten dropdown dan tambahkanlah apapun yang ingin (masukan) kedalam <div> tersebut.

Bungkuslah atau lilitlah element <div> disekitar element-element untuk posisi konten dropdown secara benae dengan CSS.

CSS → class .dropdown Gunakanlah position: relative, yang mana diperlukan ketika kita ingin konten dropdown ditempatkan kanan bawah (dari) dropdown button (menggunakan position: absolute).

Class .dropdown-content menggenggam konten dropdown yang sebenarnya. .dropdown-content dihilangkan sebagai default, dan akan ditampilkan saat hover (lihatlah dibawah). Catatan min-width diatur menjadi 160px. Merasa bebas untuk mengubah ini. Tip: Jika kamu ingin lebar (width) dari konten dropdown menjadi seperti dropdown button, aturlah width menjadi 100% (dan overflow: auto; untuk dapat me-scroll pada small screen atau layar yang lebih kecil).

Pengganti dari oenggunaan border, kita sudah menggunakan CSS3 bix-shadow property untuk membuat dropdown menu tampak seperto sebuah "card".

Selector :hover digunakan untuk menampilkan dropdown menu ketika user menggerakan mouse ke dropdown button.

Dropdown Menu

Buatlah sebuah dropdown menu yang memperbolehkan user atau pengguna untuk memilih sebuah opsi dari daftar:

Contoh ini meirip dengan contoh sebelumnya, kecuali bahwa kita menambahkan link didalam dropdown box dan me-style mereka untuk memperbaiki dropdown button yang di-style:

Contoh:

<style>
/* Style The Dropdown Button */
.dropbtn {
    background-color: #4CAF50;
    color: white;
    padding: 16px;
    font-size: 16px;
    border: none;
    cursor: pointer;
}

/* The container <div> - needed to position the dropdown content */
.dropdown {
    position: relative;
    display: inline-block;
}

/* Dropdown Content (Hidden by Default) */
.dropdown-content {
    display: none;
    position: absolute;
    background-color: #f9f9f9;
    min-width: 160px;
    box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2);
}

/* Links inside the dropdown */
.dropdown-content a {
    color: black;
    padding: 12px 16px;
    text-decoration: none;
    display: block;
}

/* Change color of dropdown links on hover */
.dropdown-content a:hover {background-color: #f1f1f1}

/* Show the dropdown menu on hover */
.dropdown:hover .dropdown-content {
    display: block;
}

/* Change the background color of the dropdown button when the dropdown content is shown */
.dropdown:hover .dropbtn {
    background-color: #3e8e41;
}
</style>

<div class="dropdown">
  <button class="dropbtn">Dropdown</button>
  <div class="dropdown-content">
    <a href="#">Link 1</a>
    <a href="#">Link 2</a>
    <a href="#">Link 3</a>
  </div>
</div>
