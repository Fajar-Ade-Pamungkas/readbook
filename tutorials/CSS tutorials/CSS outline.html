CSS Outline Properties

Outline adalah garis yang digambar mengelilingi suatu elemen - diluar border. Ini busa digunakan untuk membuat element "stand out".

Properti outline CSS menentukan style, cokor, dan width dari outline.

Element ini memiliki garis hitam tipis dan double outline yang tebalnya 10px dan berwarna hijau.

CSS Outline

Otukine merupakan garis yang digambar mengelilingi element (diluar border) untuk membuat element "stand out".

Outline Style

Properti outline-style menentukan style dari outline.

properti outline-style bisa memiliki salah satu nilai berikut:

  • dotted - Mendefinisikan sebuah outline titik
  • dashed - Mendefinisikan sebuah outline garis terpisah - pisah
  • solid - Mendefinisikan sebuah outline garis
  • double - Mendefinisikan sebuah outline ganda
  • groove - Mendefinisikan sebuah outline beralur (grooved) 3D. Efek tergantung kepada properti outline-color
  • ridge - Mendefinisikan sebuah 3D ridged outline. Efek tergantung kepada properti outline-color
  • inset - Mendefinisikan sebuah 3D inset outline. Efek tergantung kepada properti outline-color
  • outset - Mendefinisikan sebuah 3D outset outline. Efek tergantung kepada properti outline-color
  • none - Mendefinisikan tidak ada outline
  • hidden - Mendefinisikan sebuah outline tersembunyi

Contoh pertama berikut mengatur border hitam lurus disekitar setiap element <p>, lalu contoh menampilkan perbedaan nilai outline-style:

Contoh:

p {
    border: 1px solid black;
    outline-color: red;
}

p.dotted {outline-style: dotted;}
p.dashed {outline-style: dashed;}
p.solid {outline-style: solid;}
p.double {outline-style: double;}
p.groove {outline-style: groove;}
p.ridge {outline-style: ridge;}
p.inset {outline-style: inset;}
p.outset {outline-style: outset;}

Outline Color

Properti outline-color digunakan untuk mengatur warna dari outline.

Warna bisa diatur dengan:

  • sebuah nama warna valid - seperti "red"
  • Sebuah nilai HEX - seperti "#ff0000"
  • Sebuah nilai RGB - seperti "rgb(255,0,0)"
  • invert - Menunjukkan color inversion (yang mana meyakinkan bahwa outkine ditampilkan, tanpa memperhatikan warna background)

Contoh:

p {
    border: 1px solid black;
    outline-style: double;
    outline-color: red;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
p {
    border: 1px solid blue;
    outline-style: double;
    outline-color: red;
}
</style>
</head>
<body>

<p> A colored outline.</p>
</body>
</html>

Outline Width

Properti outline-width menentukan lebar dari outline.

Lebar bisa diatur sebagai ukuran tertentu (dalam satuan px, pt, cm, em, dan sebagainya) atau menggunakan salah satu dari tiga nilai yang sudah dikenal seperti: thin, medium, atau thick.

Contoh:

p {border: 1px solid black;}

p.one {
    outline-style: double;
    outline-color: red;
    outline-width: thick;
}

p.two {
    outline-style: double;
    outline-color: green;
    outline-width: 3px;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
p {border: 1px solid blue;}

p.one {
    outline-style: double;
    outline-color: red;
    outline-width: thick;
}

p.two {
    outline-style: double;
    outline-color: green;
    outline-width: 3px;
}
</style>
</head>
<body>

<p class="one">A thick outline.</p>
<p class="two">A thinner outline.</p>

</body>
</html>

Outline - Shorthand property

Untuk mempersingkat kode, Ini mungkin untuk menentukan semua properti outline individual dalam satu properti.

properti outline merupakan shorthand property untuk properti outline individual berikut:

  • outline-width
  • outline-style (required)
  • outline-color

Contoh:

p {
    border: 1px solid black;
    outline: 5px dotted red;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
p {
    border: 1px solid blue;
    outline: 5px dotted red;
}
</style>
</head>
<body>

<p>An outline.</p>
</body>
</html>
