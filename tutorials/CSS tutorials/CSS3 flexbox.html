CSS3 Flexible Box

CSS3 Flexbox

Flexible box atau flexbox adalah sebuah mode tata ruang (layout) baru dalam CSS3.

Penggunaan dari flexbox memastikan bahwa element berjalan dengan dapat diperkirakan ketika page layout harus mengakomondasikan atau menyediakan perbedaan ukuran layar dan perbedaan tampilan perangkat.

Untuk aplikasi kebanyakan, flexible box model menyediakan sebuah kemajuan melebihi block model dalam hal itu, flexbox tidak menggunakan float, nor melakukan flex container's margin collapse dengan margin dari konten itu sendiri.

CSS3 Flexbox Concepts

Konsep Flexbox CSS3

Flexbox terdiri dari flex containers (flex penampung) dan flex item (flex pokok).

Sebuah flex container dideklarasikan dengan pengaturan properti display dari element ke salah satu flex (membuat seperti sebuah block) atau inline-flex (membuat seperti inline).

Difalam sebuah flex container ada satu atau lebih flex item.

Catatan: Segalanya diluar sebuah flex container dan didalam sebuah flex item dibuat seperti biasa. Flexbox mendefinisikan bagaimana flex item disusun didalam sebuah flex container.

Flex item ditempatkan didalam sebuah flex container sepanjang flex line (baris flex). Dengan default hanya ada satu flex line per flex container.

Contoh berikut menampilkan tiga flex item. Mereka diposisikan dengan default: sepanjang horizontal flex line, dari kiri ke kanan:

Contoh:

.flex-container {
display: -webkit-flex;
display: flex;
width: 400px;
height: 250px;
background-color: lightgrey;
}

.flex-item {
background-color: cornflowerblue;
width: 100px;
height: 100px;
margin: 10px;
}

Flex Direction

Properti flex-direction menentuksn arah dari flexible item didalam flex container. Nilai default dari flex-direction adalah row (left-to-right, top-to-bottom).

Nilai lainnya sebagai berikut:

  • row-reverse - Jika mode penulisan (arah) adalah kiri ke kanan, flex item akan disusun dari kanan ke kiri
  • column - Jika sisitem penulisan adalah horixontal, flex item akan disusun secara vertikal

The justify-content Property

Properti justify-content perataan secara horizontal item (dari) flexible container ketika tidak menggunakan semua ruang yang tersedia pada sumbu utama.

Memungkinkan nilai sebagai berikut:

  • flex-start - Nilai default. Item-item ditempatkan di awal dari container
  • flex-end - Item-item ditempatkan di akhir dari container
  • center - Item-item ditempatkan di tengah dari container
  • space-between - Item-item ditempatkan dengan ruang diantara baris
  • space-around - Item-item ditempatkan dengan ruang sebelum, diantara dan setelah baris

The align-items Property

Properti align-items perataan secara vertikal item (dari) flexible container ketika item-item tidak mengguankan semua ruang yang tersedia pada sumbu silang (cross-axis).

Nilai yang memungkinkan adalah sebagai berikut:

  • stretch - Nilai default. Item-item direnggangkan untuk memperbaiki (fit) container.
  • flex-start - Item-item diposisikan di atas dari container
  • flex-end - Item-item diposisikan di bawah dari container
  • center - Item-item diposisikan du tengah dari container (secara vertikal)
  • baseline - Item-item diposisikan di baseline (baris dasar) dari container

The flex-wrap Property

Properti flex-wrap menentukan apakah flex items harus diselimuti atau tidak, jika ada yang tidak cukup ruangan untuk mereka (item) pada satu baris flex.

Nilai yanh memungkinkan adalah sebagai berikut:

  • nowrap - Nilai default. Flexible items tidak akan diselubungi (no wrap)
  • wrap - Flexible items akan diselubungi jika diperlukan
  • wrap-reverse - Flexible items akan diselubungi, jika diperlukan, dalam susunan terbalik

The align-content Property

Properti align-content memodifikasi tingkah laku dari properti flex-wrap. Properti ini mirip dengan align-items, tapi sebagai pengganti dari perataan flex items, properti ini meratakan flex lines.

Nilai yang memungkinkan adalah sebagai berikut:

  • stretch - Nilai default. Baris direnggangkan untuk mengambil ruang yang tersisa.
  • flex-start - Baris diangkut kearah awal dari flex container
  • flex-end - Baris diangkut kearah akhri dari flex container
  • center - Baris diangkut kearah tengah dari flex container
  • space-between - Baris dasar didistribusikan pada flex container, dengan ukuran setengah ruang di salah satu akhiran

margin: 10px;

Margin

Pengaturan margin: auto; akan menyerap ruang ekstra. margin: auto; akan digunakan untuk mendorong flex item kedalam posisi yang berbeda.

Pada contoh berikut kita mengatur margin-right: auto; pada flex item pertama. Ini akan mengakibatkan semua ruang ekstra untuk diserap ke kanan dari element itu:

Contoh:

.flex-item {
background-color: cornflowerblue;
width: 75px;
height: 75px;
margin: 10px;
}

.flex-item:first-child {
margin-right: auto;
}

align-self

Contoh berikut mengatur nilai align-self berbeda untuk setiap flex item:

Contoh:

.flex-item {
background-color: cornflowerblue;
width: 60px;
min-height: 100px;
margin: 10px;
}

align-self: flex-start;
}

-webkit-align-self: center;
align-self: center;

flex

Properti flex menentukan panjang dari flex item, relatif ke sisa lainnya dari flex item didalam container yang sama.

Pada contoh berikut, flex item pertama akan menghabiskan 2/4 dari ruang kosong, dan dua flex item lainnya akan menghabiskan 1/4 dari setiap ruang kosong:

Contoh:

.flex-item {
background-color: cornflowerblue;
margin: 10px;
}

.item1 {
-webkit-flex: 2;
flex: 2;
}

.item2 {
-webkit-flex: 1;
flex: 1;
}

.item3 {
-webkit-flex: 1;
flex: 1;

CSS3 Flexbox Properties

Daftar tabel berikut adalah properti CSS yang digunakan dengan flexbox:

Properti Deskripsi
display Menentukan tipe dari box yanh digunakan untuk sebuah HTML element
flex-direction Menentukan arah dari flexible item didalam sebuah flex container
justify-content Perataan secara horizontal flex items ketika item tidak menggunakan semua ruang yang tersedia pada sumbu utama (main-axis)
align-items Perataan secara vertikal flex items ketika item tidak menggunakan semua ruang yang tersedia pada sumbu silang (cross-axis)
flex-wrap Menentukan apakah flex items harus diselubungi atau diselimuti atau tidak, jika tidak ada cukup ruangan untuk mereka (item) di satu flex line
align-content Memodifikasi tingkah laku dari properti flex-wrap. Properti ini mirip dengan align-items, tapi sebagai pengganti dari perataan flex items, properti ini meratakan flex lines
flex-flow Sebuah shorthand property untuk flex-direction dan flex-wrap
order Menentukan susunan dari sebuah flexible item relatif ke sisa lainnya dari flex items didalam container yang sama
align-self Digunakan pada flex items. Mengesampingkan properti align-items container
flex Menentukan panjang daru sebuah flex item, relatif ke sisa lainnya dari flex items didalam container yanh sama
