Styling Links

Link bisa digayakan (style) dengan dengan properti CSS apapun (contoh color, font-famiky, background dan sebagainya).

Contoh

a {
    color: hotpink;
}
CSS tutorial

Pada penambahan, Link bisa di-style secara berbeda tergantung pada inginan anda.

Empat keadaan link adalah:

  • a:hover - normal, link yang belum dikunjungi
  • a:visited - Link yang sudah pengguna kunjungi
  • a:hover - Link ketika pengguna menggerakan mouse pada link
  • a:active - Link saat link diklik

Contoh

/* unvisited link */
a:link {
    color: red;
}

/* visited link */
a:visited {
    color: green;
}

/* mouse over link */
a:hover {
    color: hotpink;
}

/* selected link */
a:active {
    color: blue;
}

This is a link

Note: a:hover MUST come after a:link and a:visited in the CSS definition in order to be effective.

Note: a:active MUST come after a:hover in the CSS definition in order to be effective.

Ketika pengaturan styke untuk beberapa keadaan link, ada beberapa urutan peraturan:

  • a:hover HARUS tiba setelah a:link dan a:visited
  • a:active HARUS tiba setelah a:hover

Text Decoration

Properti text-decoration paling banyak digunakan untuk menghapus garis bawah dari link:

Contoh

a:link {
    text-decoration: none;
}

a:visited {
    text-decoration: none;
}

a:hover {
    text-decoration: underline;
}

a:active {
    text-decoration: underline;
}

Note: a:hover MUST come after a:link and a:visited in the CSS definition in order to be effective.

Note: a:active MUST come after a:hover in the CSS definition in order to be effective.

Background Color

Propertibackground-color bisa digunakan untuk menentukan warna latar belakang untuk link:

Contoh

a:link {
    background-color: yellow;
}

a:visited {
    background-color: cyan;
}

a:hover {
    background-color: lightgreen;
}

a:active {
    background-color: hotpink;
}

Advanced - Link Buttons

Dikembangkan - Tombol link

Contoh ini mempertontonkan contoh kemajuan lebih dimana kita mengkombinasikan bebebrapa properti CSS untuk menampilkan link sebagau box/button:

Contoh

a:link, a:visited {
    background-color: #f44336;
    color: white;
    padding: 14px 25px;
    text-align: center;
    text-decoration: none;
    display: inline-block;
}

a:hover, a:active {
    background-color: red;
}
