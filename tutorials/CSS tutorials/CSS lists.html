HTML Lists and CSS List Properties

Pada HTML, ada dua tipe utama dari list (daftar):

  • unordered list (<ul>) - List item ditandai dengan bullet
  • ordered list (<ol>) - List item ditandai dengan nomor atau huruf

Properti list CSS memperbolehkan kamu untuk:

  • Mengatur penanda list item yang berbeda untuk ordered list
  • Mengatur penanda list item yang berbeda untuk unordered list
  • Mengatur sebuah image sebagau oenanda list item
  • Menambahkan background warna untuk list dan list item

Different List Item Markers

Penanda List Item yang berbeda

Properti list-style-type digunakan untuk menentukan tipe penanda list item.

Contoh berikut menampilkan beberapa penanda list item yang tersedia:

Contoh

ul.a {
    list-style-type: circle;
}

ul.b {
    list-style-type: square;
}

ol.c {
    list-style-type: upper-roman;
}

ol.d {
    list-style-type: lower-alpha;
}

Contoh

List - Shorthand property

Properti list-style merupakan shorthand property. Ini digunakan untuk mengatur semua properti list dalam satu deklarasi:

Contoh

ul {
    list-style: square inside url('sqpurple.gif');
}

  • Tea

Ketika menggunakan shorthand property, urutan dari nilai properti adalah:

  • list-style-type (jika sebuah list-style-image ditentukan, nilai dsri properti inu akan ditampilkan jika image untuk beberapa alasan tidak bisa ditampilkan)
  • list-style-position (menentukan apakah penanda list item harus tampak didalam atau dikuar content flow)
  • list-style-image (menentukan sebuah image sebagai penanda list item)

Jika salah satu dari nilai properti diatas dihilangkan, nilai default untuk properti yang hilang akan disisipkan, jika ada.

Styling List With Colors

Kita bisa juga menggayakan (style) list dengan warna, untuk memebuatnya terlihat sedikit lebih menarik.

Apapun yang ditambahlan ke tag <ol> atau <ul>, berefek untuk keseluruhan list, saat properti ditambahkan ke tag <ol> akan berefek kepada satu item list saja.

Contoh

ol {
    background: #ff9999;
    padding: 20px;
}

ul {
    background: #3399ff;
    padding: 20px;
}

ol li {
    background: #ffe5e5;
    padding: 5px;
    margin-left: 35px;
}

ul li {
    background: #cce5ff;
    margin: 5px;
}
