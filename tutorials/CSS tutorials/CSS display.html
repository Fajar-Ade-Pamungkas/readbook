CSS Layout - The Display Property

Properti display merupakan properti CSS paling penting untuk controlling layout.

The display Property

Ptoperti dispkay menentukan jika/bagaimana suatu element ditampilkan.

Setiap elemen HTML memiliki nilai display default tergantung pada tipe apa element itu, nilai display default untuk kebanyakan element adalah block atau inline.

Block-level Elements

Block-level element selalu dimulai di baris baru dan mengambil penuh lebar yang tersedia (merenggang sampai kiri dan kanan sejauh yang ia bisa).

Element <div merupakan element block-level.

Berikut adalah contoh block-level elements:

  • <div>
  • <h1> - <h6>
  • <p>
  • <form>
  • <header>
  • <footer>
  • <section>

Inline Elements

Inline element tidak memulai pada garis baru dan hanya mengambil selebar yang diperlukan.

Ini merupakan inline <span> element didalam paragraf.

Berikut adalah contoh inline elements (element pada baris):

  • <span>
  • <a>
  • <img>

Display: none;

Properti display: none; secara umum digunakan dengan JavaSfript utuk menyembunyikan dan menampilkan element tanpa penghapusan atau pembuatan ulang elemen. Coba lihatlah di contoh terakhir kita pada halaman ini jika kamu ingin tahu ini bisa dicapai.

Element <script> menggunakan display: none; sebagau defaultnya.

Override The Default Display Value

Mengesampingkan Nilai Display Default

Seperti yang telah disebutkan, setiap element memiliki nilai display default. Akan tetapu, kamu bisa mengabaikan ini.

Pengubahan suatu inline element untuk block element atau vice versa, bisa berguna untuk pembuatan halaman agar terlihat spesifik, dan masil mengikuti standar web.

Contoh umum membuat inline <li> element untuk horizontal menu:

Contoh:

li {
    display: inline;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
li {
    display: inline;
}
</style>
</head>

Catatan: Pengaturan properti display dari suatu element hanya mengubah bagaimana element ditampilkan, BUKAN apa jenis dari elementnya. Jadi, suatu inline element dengan display: block; tidak diperbolehkan untuk memiliki block element lainnya didalam element tersebut.

Contoh berikut menampilkan element <span> sebagai block element:

Contoh:

span {
    display: block;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
span {
    display: block;
}
</style>
</head>
<body>

Hide an Element - display:none or visibility:hidden?

Menyembunyikan suatu element (dengan) display: none; atau visibility: hidden; ?

Penghilangan suatu element busa dilakukan dengan pengatran properti display menjadi none. Element akan disembunyikan, dan halaman akan disembunyikan seperti jika element tidak ada:

Contoh:

h1.hidden {
    display: none;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
h1.hidden {
    display: none;
}
</style>
</head>
<body>

<h1>This is a visible heading</h1>
<h1 class="hidden">This is a hidden heading</h1>
<p>Notice that the h1 element with display: none; does not take up any space.</p>

</body>
</html>

CSS Display/Visibility Properties

Properti Deskripsi
display Menentukan bagaimana element harus ditampilkan
visibility Menentukan apakah iya atau tidak element harus ditampilkan (visible)
Nilai:
  • visible - element ditampilkan
  • hidden - element disembunyikan
  • inherit - element mewariskan nilai properti yang sama dari parent element
  • initial - (nilai) properti dijadikan sebagai nilai default
