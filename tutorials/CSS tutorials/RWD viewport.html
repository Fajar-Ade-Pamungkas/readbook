Responsive Web Design - The Viewport

What is The Viewport?

Apa itu Viewport ?

Viewport merupakan area yang dapat dilihat oleh pengguna dari halaman web.

Viewport berbeda - beda dengan perangkat, dan menjadi lebih kecil pada perangkat mobile (ponsel) daripada di sebuah layar komputer.

Sebelum tablet dan perangkat mobile, halaman web didesign hanya untuk layar komputer, dan viewport umum untuk halaman web yang memiliki design statis dan fixed size (ukuran yang tetap).

Lalu, ketika kita mulai menjelajah internet menggunakan tablet dan perangkat mobile, halaman web dengan ukuran tetap menjadi terlalu besar untuk viewport yang layak, browser pada skala perangkat itu down (turun) untuk memuat halaman web sepenuhnya untuk layar.

Ini tidak sempurna !! Tapi cepat memperbaiki.

Setting The Viewport

Pengaturan Viewport

HTML memperkenalkan sebuah metode untuk membiarkan web designer untuk mengambil alih viweport, sepanjang tag <meta>.

Kamu harus memasukan element viewport <meta> berikut dalam semua halaman web kamu.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Element viewport <meta> memberikan browser intruction untuk bagaimana mengontrol page's dimensions dan scaling.

Bagian width=device-width mengatur lebar halaman untuk menuruti screen-width perangkat (yang mana akan bervariasi tergantung pada perangkat tersebut).

Bagian initial-scale=1.0 mengatur initial zoom level ketika halaman dimuat pertamakali oleh browser.

Size Content to The Viewport

Users digunakan untuk men-scroll (menggulung) website secara vertical di dekstop dan perangkat mobile - tapi tidak sexara horizontal!

Jadi, jika user dipaksakan untuk men-scroll secara horizontal, atau zoom out (memperbesar), untuk mengerti seluruh web oage ia menghasikan dalam sebuah user experience yang kecil (poor)

Beberapa aturan - aturan tambahan berikut:

  1. JANGAN gunakan large fixed width (lebar ukuran yang besarnya tetap) - Untuk contoh, jika suatu gambar ditampilkan pada lebar yang lebih luas dari viewport ini bisa menyebabkan viewport men-scroll secara horizontal. Ingatlah untuk menyesuaikan konten ini menjadi lebih layak dengan lebar (width) dari viewport.
  2. JANGAN biarkan konten mengandalkan pada suatu keterangan viewport width untuk menyumbang dengan baik. - Sejak itu dimensi layar dan width dalam CSS pixels memiliki variasi luas antara perangkat lainnya, konten tidak harus mengandalkan suatu keterangan levar viewport untuk membuatnya bagus (render well)
  3. Gunakanlah CSS media queries untuk menerapkan perbedaan styling kepada layar besar atau kecil - Pengaturan large absolut CSS widths untuk page elements, akan mengakibatkan element menjadi terlalu luas untuk viewport pada perangkat yang lebih kecil. Sebagai pengganti, pertimbangkanlah penggunaan relative width value, semacam itu seperti width: 100%;, Berhati - hatilah menggunakan large absolute positioning values, Ini mungkin menyebabkan element jatuh keluar viewport pada perangkat yang kecil.
