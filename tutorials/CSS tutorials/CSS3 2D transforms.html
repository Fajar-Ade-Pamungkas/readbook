CSS3 2D Transforms

CSS3 Transforms

CSS3 transform memperbokehkan kamu untuk perpindahan (translate), perputaran (rotasi), memperbesar & memperkecil (scale) dan mencondong/memiringkan (skew) element.

Transformasi adalah sebuah efek yang membiarkan sebuah element berubah benruj, ukuran dan posisi.

CSS3 mendukung transformasi 2D dan 3D.

Mouse over the elements below to see the difference between a 2D and a 3D transformation:

CSS3 2D Transforms

Pada chapter ini kamu akan mempelajari tentang metode-metode transformasi 2D berikut:

  • translate()
  • rotate()
  • scale()
  • skewX()
  • skewY()
  • matrix()

Tip: Kamu akan mempelajari tentang 3D transforms pada chapter selanjutnya.

The translate() Method

Metode translate() mengerakan suatu element dari posisinya saat ini (menurut pada parameter yang diberikan untuk X-axis [sumbu-X] dan Y-axis [sumbu-Y]).

Contoh berikut memindahkan element <div> 50 pixel ke kanan, dan 100 pixel kebawah dari posisinya saat ini:

Contoh:

div {
-ms-transform: translate(50px,100px); /* IE 9 */
-webkit-transform: translate(50px,100px); /* Safari */
transform: translate(50px,100px);
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div {
width: 300px;
height: 100px;
background-color: yellow;
border: 1px solid black;
-ms-transform: translate(50px,100px); /* IE 9 */
-webkit-transform: translate(50px,100px); /* Safari */
transform: translate(50px,100px); /* Standard syntax */
}
</style>
</head>
<body>

<div>
The translate() method moves an element from its current position. This div element is moved 50 pixels to the right, and 100 pixels down from its current position.
</div>

</body>
</html>

The rotate() Method

Metode rotate() digunakan untuk merotasi/memutar suatu element searah jarum jam (clockwise) atau melawan arah jarum jam (counter-clockwise) menurut pada derajat yang diberikan.

Contoh berikut merotasi/memutar element <div> searah jarum jam dengan 20 derajat:

Contoh:

div {
-ms-transform: rotate(20deg); /* IE 9 */
-webkit-transform: rotate(20deg); /* Safari */
transform: rotate(20deg);
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div {
width: 300px;
height: 100px;
background-color: yellow;
border: 1px solid black;
}

div.rotasiPositif {
-ms-transform: rotate(20deg); /* IE 9 */
-webkit-transform: rotate(20deg); /* Safari */
transform: rotate(20deg); /* Standard syntax */
}
</style>
</head>

The scale() Method

Metode scale() digunakan untuk meningkatkan (increment) atau menurunkan (decrement) ukuran dari suatu element (berdasarkan pada parameter yang diberikan untuk lebar dam tingginya).

Contoh berikut meningkatkan element <div> untuk 2 time dari lebar aslinya dan 3 time dari tinggi aslinya:

Contoh:

div {
-ms-transform: scale(2,3); /* IE 9 */
-webkit-transform: scale(2,3); /* Safari */
transform: scale(2,3);
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div.scaleIncrement {
margin: 150px;
width: 200px;
height: 100px;
background-color: yellow;
border: 1px solid black;
border: 1px solid black;
-ms-transform: scale(2,3); /* IE 9 */
-webkit-transform: scale(2,3); /* Safari */
transform: scale(2,3); /* Standard syntax */
}
</style>
</head>

The matrix() Method

Metode matrix() digunakan untuk mengkombinasikan semua metode transformasi 2D menjadi satu:

Metode matrix() mengambil 6 parameter, termasuk fungsi matematik, yang mana mengizinkan kamu untuk me-rotasi, me-scale, me-translate dan me-skew element.

Parameter sebagai berikut:

matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY())

Contoh:

div {
-ms-transform: matrix(1, -0.3, 0, 1, 0, 0); /* IE 9 */
-webkit-transform: matrix(1, -0.3, 0, 1, 0, 0); /* Safari */
transform: matrix(1, -0.3, 0, 1, 0, 0);
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div {
width: 300px;
height: 100px;
background-color: yellow;
border: 1px solid black;
}

div.myMatrix1 {
-ms-transform: matrix(1, -0.3, 0, 1, 0, 0); /* IE 9 */
-webkit-transform: matrix(1, -0.3, 0, 1, 0, 0); /* Safari */
transform: matrix(1, -0.3, 0, 1, 0, 0); /* Standard syntax */
}

div.myMatrix2 {
-ms-transform: matrix(1, 0, 0.5, 1, 150, 0); /* IE 9 */
-webkit-transform: matrix(1, 0, 0.5, 1, 150, 0); /* Safari */
transform: matrix(1, 0, 0.5, 1, 150, 0); /* Standard syntax */
}
</style>
</head>
<body>

CSS3 Transform Properties

Daftar tabel berikut adalah semua properti 2D transform:

Properti Deskripsi
transform Menerapkan transformasi (perubahan) 2D dan 3D untuk sebuah element
transform-origin Memperbolehkan kamu untuk mengubah posisi pada element yang ditransformasi

2D Transform Methods

Fungsi Deskripsi
matrix(n,n,n,n,n,n) Mendefinisikan 2D transformation, menggunakn sebuah matrix dari enam nilai
translate(x,y) Mendefinisikan 2D translation, menggerakan element sepanjang sumbu-X dan sumbu-Y
translateX(n) Mendefinisikan 2D translation, menggerakan element sepanjang sumbu-X
translateY(n) Mendefinisikan 2D translation, menggerakan element sepanjang sumbu-Y
scale(x,y) Mendefinisikan 2D scale transformation, mengubah lebar dan tinggi element
scaleX(n) Mendefinisikan 2D scale transformation, mengubah lebar element
scaleY(n) Mendefinisikan 2D scale transformation, mengubah tinggi element
skew(sudut-x, sudut-y) Mendefinisikan 2D skew transformation sepanjang sudut-X dan sudut-Y
skewX(sudut) Mendefinisikan 2D skew transformation sepanjang sudut-X
skewY(sudut) Mendefinisikan 2D skew transformation sepanjang sudut-Y
rotate() Mendefinisikan 2D rotation, sudut/angle ditentukan dalam parameter
