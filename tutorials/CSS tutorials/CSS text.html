TEXT FORMATTING

This text is styled with some of the text formatting properties. The heading uses the text-align, text-transform, and color properties. The paragraph is indented, aligned, and the space between characters is specified. The underline is removed from this colored "Try it yourself" link.

Text Color

Properti color digunakan untuk mengatur warna teks:

Dengan CSS, warna sangat sering ditentukan dengan:

  • a color name (nama warna) - seperti "red"
  • a HEX value (nilai HEX) - seperti "#ff0000"
  • an RGB value (nilai RGB) - seperti "rgb(255,0,0)"

Lihatlah di CSS Color Values untuk daftar lengkao yang nilai warna mungkin (diperbolehkan).

Warna teks default utuk halaman didefinisikan dalam selektor body.

Input

body {
    color: blue;
}

h1 {
    color: green;
}

This is heading 1

This is an ordinary paragraph. Notice that this text is blue. The default text color for a page is defined in the body selector.

Catatan: Untuk W3C mengalah pada CSS: Jika kamu mendefinisikan properti color, kamu juga harus mendefinisikan properti background-color.

Text Alignment

Properti text-align digunakan untuk mengatur perataan horizontal dsri suatu teks.

Teks bisa diratakan kiri (left) atau kanan (right), tengah (center) atau (kiri-kanan).

Contoh berikut menampilkan perataan tengah, dan kiri dan kanan teks (perataan kiri merupkan defualt jika direksi teks adalah left-to-right dan perataan kanan merupakan default jika direksi teks adalah right-to-left):

Input

h1 {
    text-align: center;
}

h2 {
    text-align: left;
}

h3 {
    text-align: right;
}

Heading 1 (center)

Heading 2 (left)

Heading 3 (right)

The three headings above are aligned center, left and right.

Ketika properti text-align diatur menjadi "justify", setiap baris direnggangkan jadi setiap baris memiliki lebar yang sama, dan margin kanan dan kiri luruh (seperti dalam majalah dan koran):

Input

div {
    text-align: justify;
}

Example text-align: justify;

The text-align: justify; value stretches the lines so that each line has equal width (like in newspapers and magazines).

Text Decoration

Properti text-decoration digunakan untuk mengatur atau mencopot dekorssi dari text.

Nilai text-decoration: none, seringkali digunakan untuk menghilangkan underline (garis bawah) dari link:

Input

a {
    text-decoration: none;
}

A link with no underline: Readbook.com

Nilai text-decoration lainnya digunakan untuk mendekorasi text:

Input

h1 {
    text-decoration: overline;
}

h2 {
    text-decoration: line-through;
}

h3 {
    text-decoration: underline;
}

Text Transformation

Properti text-transform digunakan untuk menentuksn huruf besar atau kecil dalam teks.

Itu bisa digunakan untuk mengganti teks apapun kedalam huruf besar atau kecil atau capitalize.

Input

p.uppercase {
    text-transform: uppercase;
}

p.lowercase {
    text-transform: lowercase;
}

p.capitalize {
    text-transform: capitalize;

All CSS Text Properties

Properti Deskripsi
color Mengatur warna teks

vertical-align Mengatur perataan secara vertikal suatu element
white-space Menentukan bagaimana white-space didalam suatu element dikendalikan
word-spacing Menambahkan atau mengurangi spasi antara kata dalam suatu teks
