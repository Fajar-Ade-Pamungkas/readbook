CSS3 Backgrounds

CSS3 Backgrounds

CSS3 berisikan beberapa properti background baru, yang mana memperbolehkan kontrol besar tentang element background.

Pada chapter ini kamu akan mempelajari bagaimana untuk menambahkan multiple background image ke satu element.

Kamu juga akan mempelajari tentang properti-properti baru CSS3 berikut:

  • background-size
  • background-origin
  • background-clip

Dua nilai lainyya memungkinkan untuk background-size: contain; dan background-size: cover;

Kata kunci contain men-skala background image menjadi sebesar mungkin (tapi keduanya lebar dan tinggi harus fit didalam area konten). Seperti itu, tergantung pada proporsi dari background image dsn area penempatan background, Mungkin ada beberapa area dari background yang mana tidak di-cover oleh background image.

Kata kunci cover men-skala background image jadi area konten di-cover secara penuh oleh background image (keduanya lebar dan tinggu sama atau melebihi area konten). Seperti itu, beberapa bagian dari background image mungkin tidak diperlihatkan dalam area penempatan background.

Full Size Background Image

Background Image Ukuran Penuh

Sekarang kita ingin background image pada sebuah website yang meng-cover keseluruhan browser window pada semua waktu.

The requirements are as follows (Syarat-syarat yang harus diikuti):

  • Isilah keseluruhan halaman dengan image (tidaka da white space)
  • Me-skala image diperlukan
  • Center image pada halaman
  • Tidak mengakibatkan scrollbar

Contoh berikut menampilkan bagaimana untuk melakukan itu: Gunakanlah element html (elemen html selalu paling kurang tinggi dari browser window). Kemudian aturlah sebuah fixed dan centered background padanya, Lalu sesuaikan ukurannya dengan properti background-size:

Contoh:

html {
background: url(img_flower.jpg) no-repeat center fixed;
background-size: cover;
}

<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>

CSS3 background-origin Property

Properti background-origin CSS3 menentukan dimana backgroun image diposisikan/ditempatkan.

Properti menambil 3 nilai berbeda, yaitu:

  • border-box - background image dimulai dari sudut kiri atas dari border
  • padding-box - (default) background image dimulai dari sudut kiri atas dari tepi padding
  • content-box - background image dimulai dari sudut kiri atas dari konten
