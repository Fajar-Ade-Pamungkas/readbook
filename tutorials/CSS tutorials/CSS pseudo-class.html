CSS Pseudo-classes

What are Pseudo-classes?

Apa itu Pseudo-class ?

Pseudo-class digunakan untuk mendefinisikan keadaan istimewa dari suatu element.

Untuk contoh, pseudo-class bisa digunakan untuk:

  • Me-style suatu element ketika pengguna atau user menggerak mouse pada element tersebut
  • Me-style link yang telah dikunjungi atau belum dikunjungi secara berbeda
  • Me-style suatu element ketika element dapat memusatkan atau focus

Syntax

Sintaksis daei pseudo-class:

selector:pseudo-class {
    property:value;
}

Anchor Pseudo-classes

Link bisa ditampilkan dalam cara yang berbeda-beda:

Contoh:

/* unvisited link */
a:link {
    color: #FF0000;
}

a:visited {
    color: #00FF00;
}

/* mouse over link */
a:hover {
    color: #FF00FF;
}

CSS - The :lang Pseudo-class

Pseudo-class :lang mengizinkan kamu untuk mendefinisikan peraturan untuk language (bahasa) yang berbeda-beda:

Pada contoh berikut, :lang mendefinisikan tanda kutipan (quottation) untuk element <q> dengan lang="no":

Contoh:

q:lang(no) {
    quotes: "~" "~";
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
q:lang(no) {
    quotes: "~" "~";
}
</style>
</head>
<body>

<p> Some text <q lang="no">A quote in a paragraph</q> Some teks</p>.

</body>
</html>
