CSS3 Media Queries - Examples

CSS3 Media Queries - More Examples

Memperbolehkan kita melihat ke beberapa contoh lebih dari penggunaan media queries.

Kita akan memulai dengan sebuah daftar nama yang mana fungsi seperti email links. HTML nya adalah:

Contoh 1:

<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
ul {
list-style-type: none;
}

ul li a {
color: green;
text-decoration: none;
padding: 3px;
display: block;
}
</style>
</head>
<body>

<ul>
<li><a data-email="johndoe@example.com" href="mailto:johndoe@example.com">John Doe</a></li>
<li><a data-email="marymoe@example.com" href="mailto:marymoe@example.com">Mary Moe</a></li>
<li><a data-email="amandapanda@example.com" href="mailto:amandapanda@example.com">Amanda Panda</a></li>
</ul>

</body>
</html>

Perhatikanlah attribute data-email. Pada HTML5, kita bisa menggunakan attribute dengan awalan data- untuk menyimpan informasi. Kita akan menggunakan attribute data- nanti.

Width from 520 to 699px - Apply an email icon to each link

Lebar dari 250px sampai 699px - Terapkan sebuah email icon untuk setiap link

Ketika lebar browser berada diantara 520px sampai 699px, kita akan menerapkan sebuah email icon untuk setiap email link:

Contoh 2:

@media screen and (max-width: 699px) and (min-width: 520px) {
ul li a {
padding-left: 30px;
background: url(email-icon.png) left center no-repeat;
}
}
<!DOCTYPE html>
<html>
<head>

<style>

list-style-type: none;
}

color: green;
text-decoration: none;
padding: 3px;
display: block;
}

@media screen and (max-width: 699px) and (min-width: 520px) {
ul li a {
padding-left: 30px;
background: url(email-icon.png) left center no-repeat;
}
}
</style>
</head>

Width from 700 to 1000px - Preface the links with a text

Lebar dari 700px sampai 1000px - pendahuluan (preface) link dengan teks

Ketika lebar browser berada pada 700px sampai 1000px, kita akan mendahului setiap email link dengan "Email: " teks:

Contoh 3:

@media screen and (max-width: 1000px) and (min-width: 700px) {
ul li a:before {
content: "Email: ";
font-style: italic;
color: #666666;
}
}

Width above 1001px - Apply email address to links

Lebar diatas 1001 pixel - Terapkan email address untuk link

Ketika lebar browser berada diatas 1001 pixel, kita akan melampirkan/menambahkan email address untuk link.

Kita akan menggunakan nilai dari attribute data- untuk menambahkan email address setelah nama personal:

Contoh 4:

@media screen and (min-width: 1001px) {
ul li a:after {
content: " (" attr(data-email) ")";
font-size: 12px;
font-style: italic;
color: #666666;
}
}

ul li a {
padding-left: 30px;
background: url(email-icon.png) left center no-repeat;
}
}

@media screen and (max-width: 1000px) and (min-width: 700px) {
ul li a:before {
content: "Email: ";
font-style: italic;
color: #666666;
}
}

@media screen and (min-width: 1001px) {

font-style: italic;

}
</style>
</head>

Width above 1151px - Add icon as we used before

Untuk browser melebar diatas 1151 pixel, kita akam menambahkan lagi ikon seperti yang kita gunakan sebelumnya.

Disini, kita tidak harus menulis sebuah blok media query tambahan, kita hanya bisa melampirkan media query tambahan untuk sesuatu yang sudah ada menggunakan sebuah koma (ini akan berjalan seperti operator OR):

Contoh 5:

@media screen and (max-width: 699px) and (min-width: 520px), (min-width: 1151px) {
ul li a {
padding-left: 30px;
background: url(email-icon.png) left center no-repeat;
}
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
ul {
list-style-type: none;
}

ul li a {
color: green;
text-decoration: none;
padding: 3px;
display: block;
}

@media screen and (max-width: 699px) and (min-width: 520px), (min-width: 1151px) {
ul li a {
padding-left: 30px;
background: url(email-icon.png) left center no-repeat;
}
}

@media screen and (max-width: 1000px) and (min-width: 700px) {
ul li a:before {
content: "Email: ";
font-style: italic;
color: #666666;
}
}

@media screen and (min-width: 1001px) {
ul li a:after {
content: " (" attr(data-email) ")";
font-size: 12px;
font-style: italic;
color: #666666;
}
}
</style>
</head>
<body>

Menggunakan daftar email link pada sidebar dalam web page

Contoh ini meletakan daftar email link kedalam sidebar kiri dari webpage.

Contoh 6:

#nav {
list-style-type: none;
}

#nav li a {
color: green;
text-decoration: none;
padding: 3px;
display: block;
}

#nav {
width: 35%;
float:left;
}

@media screen and (max-width: 699px) and (min-width: 520px), (min-width: 1151px) {
#nav li a {
padding-left: 30px;
background: url(email-icon.png) left center no-repeat;
}
}

#nav li a:before {
content: "Email: ";
font-style: italic;
color: #666666;
}

#nav li a:after {
content: " (" attr(data-email) ")";
