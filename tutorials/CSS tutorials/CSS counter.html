CSS Counters

Using CSS Counters

Penggunaan CSS Counter

CSS counter ialah seperti "variabel". Nilai variabel bisa dinaikan dengan peraturan CSS (yang mana akan meninggalkan jejak (track) berapa banyak waktu digunakan).

Agar bekerja dengan CSS counter kita akan menggunakan properti berikut:

  • counter-reset - Memebuat atau me-reset sebuah nilai counter
  • counter-increment - Meningkatkan atau menaikan sebuah nilai counter
  • content - Menyisipkan content yang dihasilkan
  • counter() atau fungsi counter() - Menambahkan nilai dari suatu counter untuk sebuah element

Untuk mengggunakan CSS counter, counter pertama kali harus dibuat dengan counter-reset.

Contoh berikut membuat sebuah counter untuk halaman (dalam selector body), kemudian menaikan nilai counter untuk setiap element <h2> dan menambahkan "Section <value of the counter>:" untuk permulaan dari setiap element <h2>:

Contoh:

body {
counter-reset: section;
}

h2::before {
counter-increment: section;
content: "Section " counter(section) ": ";
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
body {
counter-reset: section;
}

h2::before {
counter-increment: section;
content: "Section " counter(section) ": ";
}
</style>
</head>
<body>

Nesting Counters

Contoh berikut membuat satu counter untuk (section) halaman dan satu counter untuk seriap element <h1> (sebsection). Counter "section" akan dihitung untuk setiap element <h1> dengan "Section <value of section counter>.", dan "subsection" counter akan dihitung untuk setiap element <h2> dengan "<value of section counter>.<value of subsection counter>":

Contoh:

body {
counter-reset: section;
}

h1 {
counter-reset: subsection;
}

h1::before {
counter-increment: section;
content: "Section " counter(section) ". ";
}

h2::before {
counter-increment: subsection;
content: counter(section) "." counter(subsection) " ";
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
body {
counter-reset: section;
}

h1 {
counter-reset: subsection;
}

h1::before {
counter-increment: section;
content: "Section " counter(section) ". ";
}

h2::before {
counter-increment: subsection;
content: counter(section) "." counter(subsection) " ";
}
</style>
</head>
<body>

Sebuah counter bisa juga dimanfaatkan untuk membuat outlined list karena hal baru dari sebuah counter yang dibuat secara otomatis pada child element. Disini kita menggunakan fungsi counter() untuk menyisipkan string diantara tingkat yang berbeda dari nested counter:

Contoh:

ol {
counter-reset: section;
list-style-type: none;
}

li::before {
counter-increment: section;
content: counters(section,".") " ";
}
