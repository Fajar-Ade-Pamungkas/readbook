Responsive Web Design - Images

Using The width Property

Jika properti width diatur menjadi 100%, image akan menjadi responsive dan scale up and down:

Contoh

img {
    width: 100%;
    height: auto;
}
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
img {
    width: 100%;
    height: auto;
}
</style>
</head>
<body>

<img src="../lamborghini.jpeg">
<p>Resize the browser window to see how the image will scale.</p>

</body>
</html>

Resize the browser window to see how the image wil scale.

Perhatikanlah bahwa pada contoh diatas, image akan di-scale up menjadi lebih besar daripada ukuran alsinya. Solusi yang lebih baik, pada banyak kasus, akan digunakan properti max-width sebagai pengganti.

Using The max-width Property

Jika properti max-width diatur menjadi 100%, image akan me-scale down jika itu tidak harus, tapi jangan pernah me-scale up menjadi lebih besar daripada ukuran asliny:

Contoh

img {
    max-width: 100%;
    height: auto;
}
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
img {
    width: 100%;
    height: auto;
} </style>
</head>
<body>

<img src="../lamborghini.jpeg">
<p>Resize the browser window to see how the image wil scale when the width is less than 460px.</p>

</body>
</html>

Background Images

Background image juga bisa merespon untuk resizing (pengukuran kembali) dan scaling (penimbangan).

Disini kita akan menampilkan 3 metode yang berbeda:

1. Jika properti background-size diatur menjadi "contain", background image akan me-scale, dan mencoba untuk memperbaiki area konten. Akan tetapi, image akan tetap pada rasio aspeknya (hubungan sebanding antara lebar dan tinggi gambar):

Disini adalah kode CSS:

Contoh:

div {
    width: 100%;
    height: 400px;
    background-image: url('img_flowers.jpg');
    background-repeat: no-repeat;
    background-size: contain;
    border: 1px solid red;
}
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
.metode01 {
    width: 100%;
    height: 400px;
    background-image: url('img_flowers.jpg');
    background-repeat: no-repeat;
    background-size: contain;
    border: 1px solid red;
}

</style>
</head>
<body>

<div class="metode01">
</div>
</body>
</html>

2. Jika properti background-size diatur menjadi "100% 100%", background image akan melebar menjadi cover dari area konten sepenuh:

Disini adalah kode CSS:

Contoh:

.metode02 {
    width: 100%;
    height: 400px;
    background-image: url('img_flowers.jpg');
    background-size: 100% 100%;
    border: 1px solid red;
}
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
.metode02 {
    width: 100%;
    height: 400px;
    background-image: url('img_flowers.jpg');
    background-size: 100% 100%;
    border: 1px solid red;
}

</style>
</head>
<body>

<div class="metode02">
</div>
</body>
</html>

3. Jika properti background-size diatur menjadi "cover", background image akan me-scale menjadi sampul area konten sepenuhnya. Inglah bahwa nilai "cover" menjaga aspect ratio, dan beberapa bagian daru background image mungkin akan di-clip:

Disini adalah kode CSS:

Contoh:

.metode03 {
    width: 100%;
    height: 400px;
    background-image: url('img_flowers.jpg');
    background-size: cover;
    border: 1px solid red;
}
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
.metode03 {
    width: 100%;
    height: 400px;
    background-image: url('img_flowers.jpg');
    background-size: cover;
    border: 1px solid red;
}

</style>
</head>
<body>

<div class="metode03">
</div>
</body>
</html>

Different Images for Different Devices

Gambar yang Berbeda untuk Perangkat yang Berbeda

Suatu gambar besar bisa disempurnakan di layar komputer besar, tapi tidak berguna di perangkat kecil. Kenapa memuat sebuah large image ketika kamu harus me-scale down background image bagaimanapun? Untuk mengurangi muatan atau untuk alasan lain, kamu bisa gunakan media query untuknpmenampilkan (ukuran) gambar yang berbeda di perangkat yang berbeda.

Disini adalah satu large image (gambar besar) dan satu smaller image (gambar yang lebih kecil) yang akan ditampilkan pada perangkat yang berbeda:

Contoh:

/* For width 400px and larger: */
@media only screen and (min-width: 400px) {
body {
background-image: url('img_flowers.jpg');
}
}
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
/* For width 400px and larger: */
@media only screen and (min-width: 400px) {
body {
background-image: url('img_flowers.jpg');
}
}
</style>
</head>
<body>

<p margin-top="360px"> Resize the browser width and the background image will change at 400px. </div>
</body>
</html>

Kamu bisa menggunakan media query min-device-width, sebagai pengganti dari min-width, pengganti dari browser width. Kemudian image tidak akan berubah ketika kamu me-resize browser window:

Contoh:

/* For device width smaller than 400px: */
body {
background-repeat: no-repeat;
background-image: url('img_smallflower.jpg');
}

/* For device width 400px and larger: */
@media only screen and (min-device-width: 400px) {
body {
background-image: url('img_flowers.jpg');
}
}

<meta name="viewport" content="width=device-width, initial-scale=1.0">

}

/* For device width 400px and larger: */

body {
background-image: url('img_flowers.jpg');
}
}
</style>
</head>

HTML5 <picture> Element

HTML5 memperkenalkan element <picture>, yang mana membiarkan kamu mendefinisikan lebih dari satu image.

Browser Support

BROWSER SUPPORT
Element
<picture> Not Supported 38.0 38.0 Not Supported 25.0

Element <picture> bekerja mirip element <video> dan <audio>. Kamu harus mengatur sumber yang berbeda, dan sumber pertama yang saran image yang digunakan pertama:

Contoh:

<picture>
  <source srcset="img_smallflower.jpg" media="(max-width: 400px)">
  <source srcset="img_flowers.jpg">
  <img src="img_flowers.jpg" alt="Flowers">
</picture>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<body>

<picture>
  <source srcset="img_smallflower.jpg" media="(max-width: 400px)">
  <source srcset="img_flowers.jpg">
  <img src="img_flowers.jpg" alt="Flowers">
</picture> </body>
</html>
     

Attribute srcset diperlukan dan mendefinisikan source dari image.

Attrihute media adalah opsi, menyetujui media query yang kamu temukan atau cari di CSS @media rule.

Kamu juga harus mendefinisikan sebuah element <img> untuk browser yang tidak men-support element <picture>.
