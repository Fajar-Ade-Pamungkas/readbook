CSS3 Media Queries

CSS2 Introduced Media Types

Perintah @media, diperkenalkan dalam CSS2, membuat media query memungkinkan untuk mendefinisikan perintah style yang berbeda untuk ripe media yang berbeda.

Contoh: Kamu bisa memiliki satu set dari perintah style untuk layar komputer, satu untuk printer, satu untuk handheld devices, satu untuk perangkat tipe TV dan sebagainya.

Sayangnya tipe media tipe ini tidak pernah mendapat banyak support oleh devices, lainnya daripada tipe media print.

CSS3 Introduces Media Queries

Media queries bisa digunakan untuk memeriksa banyak hal, seperti:

  • Width (lebar) dan height (tinggi) dari viewport
  • Width (lebar) dan height (tinggi) dari perangkat
  • Orientasi (adalah tablet/phone dalam mode lanscapd atau portrait?)
  • Resolusi

Penggunaan media queries merupakan sebuah teknik populer untuk pengiriman sebuah style sheet yang dikhususkan untuk tablet, iPhone dan Android.

Media Query Syntax

Sintaksis Media Queries

Sebuah media query terdiri dari sebuah tipe media dan bisa berisikan satu atau lebih expressions, yang mana memecahkan untuk salah satu dari true atau false.

@media not|only mediatype and (expressions) {
    CSS-Code;
}

Hasil dari query adalah true jika tipe media tertentu cocok dengan tipe dari perangkat, dokumen sedang ditampilkan aktif dan semua expressions dalam media query adalah true (benar). Ketika sebuha media query itu true, style sheet yang tersambung atau perintah style diterapkan, berikut perintah cascading (css) normal.

Tanpa kamu menggunakan operator not atau only, tipe media merupakan optional dan semua tipe akan disirat/dimaksud.

Kamu juga bisa memiliki stylesheet berbeda untuk media berbeda:

<link rel="stylesheet" media="mediatype and|not|only (expressions)" href="print.css">

CSS3 Media Types

Nilai Deskripsi

screen Digunakan untuk layar komputer, tablet, smartphones dan sebagainya

Media Queries Simple Examples

Satu cara untuk menggunakan media queries ialah untuk memiliki sebuah bagian CSS alternatif yang benar didalam style sheet kamu.

Contoh berikut mengubah background-color menjadi lightgreen jika (lebar) viewport nya adalah 480 pixel atau lebih lebar (Jika viewport lebih kecil diabndingkan 480 pixel, menu akan berada diatas konten):

Contoh:

@media screen and (min-width: 480px) {
body {
background-color: lightgreen;
}
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
body {
background-color: pink;
}

@media screen and (min-width: 480px) {
body {
background-color: lightgreen;
}
}
</style>
</head>
<body>

<h1>Resize the browser window to see the effect!</h1>
<p>The media query will only apply if the media type is screen and the viewport is 480px wide or wider.</p>

</body>
</html>

Contoh berikut menampilkan sebuah menu yang akan mengapung (float) ke kiri halaman jika (lebar) viewport 480 pixel atau lebih lebar (jika viewport lebih kecil dibandingkan 480 pixel, menu akan berada diatas konten):

Contoh:

@media screen and (min-width: 480px) {
#leftsidebar {width: 200px; float: left;}
#main {margin-left:216px;}
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
.wrapper {overflow:auto;}

#main {margin-left: 4px;}
#leftsidebar {float: none;width: auto;}
#menulist {margin:0;padding:0;}

.menuitem {
background:#CDF0F6;
border:1px solid #d4d4d4;
border-radius:4px;
list-style-type:none;
margin:4px;
padding:2px;
}

@media screen and (min-width: 480px) {
#leftsidebar {width:200px;float:left;}
#main {margin-left:216px;}
}
</style>
</head>
<body>

<div class="wrapper">
<div id="leftsidebar">
<ul id="menulist">
<li class="menuitem">Menu-item 1</li>
<li class="menuitem">Menu-item 2</li>
<li class="menuitem">Menu-item 3</li>
<li class="menuitem">Menu-item 4</li>
<li class="menuitem">Menu-item 5</li>
</ul>
</div>
<div id="main">
<h1>Resize the browser window to see the effect!</h1>
<p>This example shows a menu that will float to the left of the page if the viewport is 480 pixels wide or wider. If the viewport is less than 480 pixels, the menu will be on top of the content.</p>
</div>
</div>

</body>
</html>
