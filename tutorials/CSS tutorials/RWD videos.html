Responsive Web Design - Videos

Using The width Property

Jika properti width diatur menjadi 100%, video akan menjadi responsive dan scale up and down:

Contoh

video {
    width: 100%;
    height: auto;
}
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
video {
    width: 100%;
    height: auto;
}
</style>
</head>
<body>

<video controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
Your browser does not support the video tag.
</video>
<p>Resize the browser window to see how the video will scale.</p>

</body>
</html>

Resize the browser window to see how the image wil scale.

Perhatikanlah bahwa pada contoh diatas, image akan di-scale up menjadi lebih besar daripada ukuran alsinya. Solusi yang lebih baik, pada banyak kasus, akan digunakan properti max-width sebagai pengganti.

Using The max-width Property

Jika properti max-width diatur menjadi 100%, video akan me-scale down jika itu tidak harus, tapi jangan pernah me-scale up menjadi lebih besar daripada ukuran asliny:

Contoh

video {
    max-width: 100%;
    height: auto;
}
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
video {
    max-width: 100%;
    height: auto;
}
</style>
</head>
<body>

<video width="400" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
Your browser does not support the video tag.
</video>
<p>Resize the browser window to see how the video wil scale when the width is less than 460px.</p>

</body>
</html>
