Hoverable Buttons

Tombol yang dapat di-"hover"

Tip: Guanaknalh properti transition-duration untuk memutuskan kecepatan dari efek "hover":

Contoh:

.button {
-webkit-transition-duration: 0.4s; /* Safari */
transition-duration: 0.4s;
}

.button:hover {
background-color: #4CAF50; /* Green */
color: white;
}

Disabled Buttons

Tombol Dilumpuhkan

Guanaknlah properti opacity untuk menambahkan efek tembus cahaya untuk button (membuat sebuah "disabled" look).

Tip: Kamu juga bisa menambahkan properti cursor dengan sebuah nilai "not-allowed", yang mana akan menampilkan sebuah "no parking sign" ketika kamu menggerakan mouse pada button:

Contoh:

.disabled {
opacity: 0.6;
cursor: not-allowed;
}

}

.button2 {width: 50%;}
.button3 {width: 100%;}
