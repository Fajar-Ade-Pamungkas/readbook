CSS Attribute Selectors

Style HTML Elements With Specific Attributes

Me-Style Elemen-elemen HTML Dengan Attribut Tertentu

Attribute selector memungkinkan untuk me-style elemen-elemen HTML yang memiliki attribut tertentu atau nilai attribut tertentu.

CSS [attribute] Selector

Selector [attribute] digunakan untuk menyeleksi elemen-elemen dengan attribute tertentu.

Contoh berikut menyeleksi semua element <a> dengan attribute target:

Contoh:

a[target] {
background-color: yellow;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
a[target] {
background-color: yellow;
}
</style>
</head>
<body>

<p>The links with a target attribute gets a yellow background:</p>

<a href="../index.html">w3schools.com</a>
<a href="http://www.disney.com/" target="_blank">disney.com</a>
<a href="http://www.wikipedia.org/" target="_top">wikipedia.org</a>

<p><b>Note:</b> For [<i>attribute</i>] to work in IE8 and earlier, a DOCTYPE must be declared.</p>

</body>
</html>

The links with a target attribute gets a yellow background:

w3schools.com disney.com wikipedia.org

CSS [attribute="value"] Selector

Selector [attribute="value"] digunakan untuk menyeleksi elemen-elemen dengan attribute dan nilai tertentu.

Contoh berikut menyeleksi semua element <a> dengan attribute target="_blank":

Contoh:

a[target="_blank"] {
background-color: yellow;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
a[target="_blank"] {
background-color: yellow;
}
</style>
</head>
<body>

<p>The links with a target="_blank" gets a yellow background:</p>

<a href="../index.html">w3schools.com</a>
<a href="http://www.disney.com/" target="_blank">disney.com</a>
<a href="http://www.wikipedia.org/" target="_top">wikipedia.org</a>

<p><b>Note:</b> For [<i>attribute</i>] to work in IE8 and earlier, a DOCTYPE must be declared.</p>

</body>
</html>

The link with target="_blank" gets a yellow background:

w3schools.com disney.com wikipedia.org

Note: For [attribute] to work in IE8 and earlier, a DOCTYPE must be declared.

CSS [attribute~="value"] Selector

Selector [attribute~="value"] digunakan untuk menyeleksi elemen-elemen dengan sebuah nilai attribute yang berisikan kata tertentu.

Contoh berikut menyeleksi semua element dengan attribute title yang berisikan space-separated list of words, salah satu yang mana merupakan "flower":

Contoh:

[title~="flower"] {
border: 5px solid yellow;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
[title~="flower"] {
border: 5px solid yellow;
}
</style>
</head>
<body>

<p>All images with the title attribute containing the word "flower" get a yellow border.</p>

<img src="klematis.jpg" title="klematis flower" width="150" height="113">
<img src="img_flwr.gif" title="flower" width="224" height="162">
<img title="tree" width="200" height="358">

<p><b>Note:</b> For [<i>attribute</i>~=<i>value</i>] to work in IE8 and earlier, a DOCTYPE must be declared.</p>

</body>
</html>

All images with the title attribute containing the word "flower" get a yellow border.

Note: For [attribute~=value] to work in IE8 and earlier, a DOCTYPE must be declared.

Contoh diatas akan mencocokan elemen-elemen dengan title="flower", title="summer flower", dan title="flower new", tapi tidak title="my-flower" atau title="flowers".

CSS [attribute|="value"] Selector

Selector [attribute |="value"] digunakan untuk menyeleksi elemen-elemen dengan attribute tertentu yang diawali dengan nilai tertentu.

Contoh berikut menyeleksi semua element dengan nilai attribute class yang dimulai dengan "top":

Catatan: Nilai tidak harus keseluruhan kata, salah satu pun bisa, seperticlass="top", atau yang diikuti dengan sebuah hyphen( - ), seperti class="top-text" !

Contoh:

[class|="top"] {
background: yellow;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
[class|="top"] {
background: yellow;
}
</style>
</head>
<body>

<h1 class="top-header">Welcome</h1>
<p class="top-text">Hello world!</p>
<p class="topcontent">Are you learning CSS?</p> <p><b>Note: For [<i>attribute</i>|=<i>value</i>] to work in IE8 and earlier, a DOCTYPE must be declared.</p>

</body>
</html>

Styling Forms

Selector attribute bisa digunakan untuk styling form tanpa class atau ID:

Contoh:

input[type="text"] {
width: 150px;
display: block;
margin-bottom: 10px;
background-color: yellow;
}

input[type="button"] {
width: 120px;
margin-left: 35px;
display: block;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
input[type="text"] {
width: 150px;
display: block;
margin-bottom: 10px;
background-color: yellow;
}

input[type="button"] {
width: 120px;
margin-left: 35px;
display: block;
}
</style>
</head>
