CSS Background

Properti background CSS digunakan untuk mendefinisikan efek latar belakang kepada element.

Properti background CSS:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

Background Color

Warna Latar Belakang

Properti background-color menentukan warna latar belakang dsei suatu element.

Warna latar belakang dsri halaman bisa diatur seperti ini :

Input

body {
background-color:blue;
}

Dengan CSS, sebuah warna adalah yang paling sering ditentukan dengan :

  • sebuah nama warna valid - seperti "red"
  • Sebuah nilai HEX - seperti "#ff0000"
  • Sebuah nilai RGB - seperti "rgb(255,0,0)"

Lihatlah di CSS Color Values untuk daftar lengkap tentang nilai color yang memungkinkan.

Pada contoh dibawah ini, element <h1>, <p> dan <div> memiliki warna latar belakang yang berbeda

Input

div {
background-color:blue;
}
h1 {
background-color:red;
}
p {
background-color:green;
}

Background Image

Gambar Latar Belakang

Properti background-image menentukan sebuah gambar untuk digunakan sebagai latar belakang suatu element.

Dengan defualt, Gambar akan di-repeat (dicetak banyak) jadi ia akan menyampul element sepenuhnya.

Gambar latar belakang untuk sebuah halaman bisa diatur seperti ini :

Input

body {,
background-image:url("Rain Fall.png");
}

Background Image - Repeat Horizontally or Vertically

Dengan default, properti background-image me-repeat sebuah gambar background secara horizontal dan vertikal.

Beberapa gambar harus di-repeat (ulang) secara horizontal atau vertikal saja, atau merek akan terlihat aneh atau tidak baik, seperti ini:

Background Image - Fixed position

Untuk menentukan bahwa background inage harus di perbaikai (fixed) (tidak akan di-scroll dengan rest of the page), gunakanlah properti background-attachment:

Input

body {
background-image:url("Rain Fall.png");
     background-repeat: no-repeat;
    background-position: right top;

Background - Shorthand property

Untuk mempersingkat kode, Shorthand properti (background) memungkinkan untuk menentukan semua properti background dalam properti tunggal. Ini dipanggil dengan shorthand porperti

Shorthand properti untuk background merupakan semua background:

Input

body {
    background: #ff0000 url("Rain Fall.png") no-repeat right top;
}
