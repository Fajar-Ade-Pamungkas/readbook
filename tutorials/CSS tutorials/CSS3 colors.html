CSS3

CSS3 Colors

CSS mendukung color names (nama dari warna), hexadecimal dan RGB colors.

Dalam penambahan, CSS3 juga memperkenalkan:

  • RGBA colors
  • HSL colors
  • HSLA colors
  • opacity

Browser Support

RGBA Colors

Nilai RGB color merupakan ektensi (perluasan) dari nilai RGB color dengan sebuah alpha channel - yang mana menentukan opacity (sifat tak tempus cahaya) untuk warna.

Sebuah nilai RGB color ditentukan dengan: rgba(red, green, blue, alpha). Parameter alpha merupakan sebuah nomor antara 0.0 (transparan secara penuh) dan 1.0 (tak tembus cahaya).

rgba(255, 0, 0, 0.2);

rgba(255, 0, 0, 0.4);

rgba(255, 0, 0, 0.6);

rgba(255, 0, 0, 0.8);

Contoh berikut mendefinisikan perbedaan RGBA color:

Contoh:

#p1 {background-color: rgba(255, 0, 0, 0.3);}  /* red with opacity */
#p2 {background-color: rgba(0, 255, 0, 0.3);}  /* green with opacity */
#p3 {background-color: rgba(0, 0, 255, 0.3);}  /* blue with opacity */
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
#p1 {background-color:rgba(255,0,0,0.3);}
#p2 {background-color:rgba(0,255,0,0.3);}
#p3 {background-color:rgba(0,0,255,0.3);}
#p4 {background-color:rgba(192,192,192,0.3);}
#p5 {background-color:rgba(255,255,0,0.3);}
#p6 {background-color:rgba(255,0,255,0.3);}
</style>

HSL Colors

HSL didirikan untuk Hue (corak warna), Saturation (kejenuhan) dan Lightness (pencahayaan).

Swbuah nilai HSL color ditentukan dengan: hsl(hue, saturation, lightness).

Hue merupakan derajat pada color wheel atau roda warna (dari 0 sampai 360):

  • 0 (atau 360) adalah merah
  • 120 adalah hijau
  • 240 adalah biru

Saturation adalah sebuah nilai persen: 100% adalah full color.

Lightness juga merupakan sebuah nilai persen: 0% adalah dark (black) and 100% adalah white.

hsl(0, 100%, 30%);

hsl(0, 100%, 50%);

hsl(0, 100%, 70%);

hsl(0, 100%, 90%);

Contoh berikut mendefinisikan perbedaan HSL color

Contoh:

#p4 {background-color: hsl(120, 60%, 70%);}   /* pastel green */

HSLA Colors

Nilai HSLA color merupakan sebuah ekstensi (perluasan) dari nilai HSL color:

Sebuah nilai HSLA color ditentukan dengan: hsla(hue, saturation, lightness, alpha), dimana parameter alpha mendefinisikan opacity, Parameter alpha merupakan sebuah nomor antara 0.0 (transparan secara penuh) dan 1.0 (tak transparan sepenuhnya).

hsla(0, 100%, 30%, 0.3);

hsla(0, 100%, 50%, 0.3);

hsla(0, 100%, 70%, 0.3);

hsla(0, 100%, 90%, 0.3);

Opacity

Properti opacity CSS3 mengatur sifat tembus cahaya atau tembus pandang untuk sebuah nilai RGB tertentu.

Nilai properti opacity harus sebuah nomor antara 0.0 (transparan secara penuh) dan 1.0 (tak transparan sepenuhnya).

rgb(255, 0, 0);opacity:0.2;

rgb(255, 0, 0);opacity:0.4;

rgb(255, 0, 0);opacity:0.6;

rgb(255, 0, 0);opacity:0.8;

Catatlah bahwa teks diatas juga akan menjadi (lebih) tidak tembus pandang.

Contoh berikut menampilkan perbedaan nilai RGB dengan opacity:

Contoh:

#p1 {background-color:rgb(255,0,0);opacity:0.6;}  /* red with opacity */
#p2 {background-color:rgb(0,255,0);opacity:0.6;}  /* green with opacity */
#p3 {background-color:rgb(0,0,255);opacity:0.6;}  /* blue with opacity */
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
#p1 {background-color:rgb(255,0,0);opacity:0.6;}
#p2 {background-color:rgb(0,255,0);opacity:0.6;}
#p3 {background-color:rgb(0,0,255);opacity:0.6;}
#p4 {background-color:rgb(192,192,192);opacity:0.6;}
#p5 {background-color:rgb(255,255,0);opacity:0.6;}
#p6 {background-color:rgb(255,0,255);opacity:0.6;}
</style>
</head>
