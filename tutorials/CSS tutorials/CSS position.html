CSS Layout - The position Properti

Properti position menentukan tipe metode peletakan atau posisi untuk suatu element (bisa staic, relative, fixed atau absolute).

The position Property

Properti position

Properti position menentukan tipe metode peletakan atau posisi yang digunakan kepada element.

Ada empat perbedaan nilai posisi, yaitu:

  • static
  • relative
  • fixed
  • absolute

Element kemudian diposisikan menggunakan properti top, bottom , left, dan right. Akan tetapi, properti ini tidak akan bekerja tanpa properti position diatur terlebih dahulu. Mereka juga bekerja secara beda tergantung pada nilai posisinya.

position: static;

HTML element bisa diposisi static (tetap) dengan default.

Element berposisi static tidak terpengaruh oleh properti top, bottom, left, right.

Suatu element dengan position:static; tidak ditempatkan pada cara spesial apapun; Ini selalu berposisi menurut pada normal flow (aliran normal) dari halaman:

Element <div> ini memiliki position: static;

Disini adalah CSS yang digunakan:

Contoh:

div.static {
    position: static;
    border: 3px solid #73AD21;
}

<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>

    position: static;
    border: 3px solid #73AD21;
}
</style>
</head>

position: relative;

Suatu element dengan position: relative; diposisikan relatif pada posisi normal.

Pengaturan properti top, right, bottom, dan left dari suatu element berposisi relatif akan mengakibatkan element bisa menyesuaikan diri untuk berpindah dari posisi normalnya. Konten lainnya tidak akan disesuaikan menjadi fit kedalam celah (gap) kiri oleh element itu.

Element <div> ini memeiliki posisi: relative;

Disini adalah CSS yang digunakan:

Contoh:

div.relative {
    position: relative;
    left: 30px;
    border: 3px solid #73AD21;
}

position: fixed;

Suatu element dengan position: fixed; diposisikan relatif (tergantung) pada viewport, Yang mana artinya ini selalu stay (tinggal) di tempat yang sama bahkan jika halaman di-scroll. Properti rop, right, bottom, dan left digunakan untuk posisi element.

Suatu fixed element tidak meninggalkan celah pads halaman dimana ia yang secara normal telah dilokasikan.

Perhatikanlah pada elemen (berposisi) fixed di sudut kanan halaman. Disini adalah CSS yang digunakan:

Contoh:

div.fixed {
    position: fixed;
    bottom: 0;
    right: 0;
    width: 300px;
    border: 3px solid #73AD21;
}

position: absolute;

Suatu elemebt dengan position: absolute; diposisikan relatif (tergantung) pada diposisi terdekat dengan ancestor (pengganti dari posisi relatif/tergantung pada viewport, seperti fixed).

Akan tetapi; Jika element berposisi absolute belum diposisi ancestor, element menggunakan body dokumen, dan bergerak beserta page scrolling.

Catatan: Suatu "positioned" element merupakan satu yang punya posisi apapun kecuali static.

Overlapping Elements

Saling melingkupi element

Ketika element diposisikan, element bisa saling melingkupi element lainnya.

Properti z-index menentukan penyusunan urutan (stack order) suatu element (yang mana element harus ditempatkan di depean dari atau dibelakang element lainnya).

Suatu element bisa memiliki (nilai) stack order positip atau negatif.

Ini merupakan suatu headingBecause image yang memiliki z-index: -1;, ini akan ditempatkan dibelakang teks.

Ini adalah heading

Karena Image memiliki z-index: -1;, maka image akan ditempatkan dibelakang teks.

Contoh:

img {
    position: absolute;
    left: 0px;
    top: 0px;
    z-index: -1;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
img {
    position: absolute;
    left: 0px;
    top: 0px;
    z-index: -1;
}
</style>
</head>
<body>

<p style="font-size:larger;"> Ini adalah heading</p>
<img src="../logos/Chrome.png" width="100" height="140">
<p> Karena Image memiliki z-index: -1;, maka image akan ditempatkan dibelakang teks. </p>

</body>
</html>

Ini adalah heading

Karena Image memiliki z-index: -1;, maka image akan ditempatkan dibelakang teks.

Suatu ekement dengan stack order lebih besar selalu berada didepan dari suatu elemen dengan stack order yang lebih kecil.

Catatan: Jika kedua posisi element saling melingkupi tanpa ditentukan z-index. Element diposisi terakhir dalam kode HTML akan ditampilkan diatas.

All CSS Positioning Properties

Properti Deskripsi
bottom Mengatur margin bagian bawah untuk posisi box
clip Menjepit suatu element yang diposisikan secara absolute
cursor Menentukan tipe cursor untuk ditampilkan
left Mengatur margin bagian kiri untuk posisi box
overflow Menentukan apa yang terjadi jika konten melebihi batas (overflow) box elemen
overflow-x Menentukan apa yang dilakukan dengan tepi kiri/kanan dari kontent jika ia melebihi batas area konten elemen
overflow-y Menentukan apa yang dilakukan dengan tepi atas/bawah dari kontent jika ia melebihi batas area konten elemen
position Menentukan tipe penempatan suatu element
right Mengatur margin bagian kanan untuk posisi box
top Mengatur margin bagian atas untuk posisi box
z-index Mengatur stack order (menyusun urutan) dari suatu element
