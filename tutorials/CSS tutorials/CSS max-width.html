CSS Layout - width and max-width

Penggunaan width, max-width dan margin: auto;

Seperti yang sudah disebutkan di chapter sebelumnya, block-level element selalu mengambil penuh lebar yang tersedia (merengkangkan kiri dan kanan sepanjang yang ia bisa).

Pengaturan width dari block-level element akan mencegahnya dsri peregangan ke tepi kontainernya. Lalu, kamu buss mengatur margin menjadi auti, untuk posisi tengah ekement dengan kontainernya. Element akan mrhgambil lebar tertentu dan sisa ruang akan di-split (potong) dengan ssma antara dua margin:

Element <div> memiliki width: 500px; dan margin diatur menjadi auto.

Catatan: Masalah dengan <div> diatas terjadi ketika browser window lebih kecil daripada lebar element. browser lalu menambahkan horizontal scrollbar untuk halaman.

Penggunaan max-width pengganti, pada situasi ini, akan meningkatkan browser's handling untuk small window. Ini penting ketika pembuatan situs yang dapat berguna di small device (perangkat kecil).

Element <div> ini memiliki max-width:500px; dan margin diatur menjadi auto.

Tip Ukuran ulanglah browser window menjadi lebih kecil daripada lebar 500px, untuj melihat perbedaan antara dua div!

Disini adalah contoh tentang dua div diatas:

Contoh:

div.ex1 {
    width: 500px;
    margin: auto;
    border: 3px solid #73AD21;
}

div.ex2 {
    max-width: 500px;
    margin: auto;
    border: 3px solid #73AD21;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div.ex1 {
    width: 500px;
    margin: auto;
    border: 3px solid #73AD21;
}

div.ex2 {
    max-width: 500px;
    margin: auto;
    border: 3px solid #73AD21;
}
</style>
</head>
<body>

<div class="ex1">
<p> Element <div> memiliki <mark>width: 500px;</mark> dan margin diatur menjadi auto. </p>
</div>

<div class="ex2">
<p> Element <div> ini memiliki <mark>max-width:500px;</mark> dan margin diatur menjadi auto. </p>
</div>
</body>
</html>
