CSS3 border-radius - Specify Each Corner

CSS3 border-radius - Menentukan Setiap Sudutnya

Jika kamu hanya menentukan satu nilai untuk properti border-radius, radius ini akan diterapkan ke semua (keempat) sudut.

Akan tetapi, kamu bisa menentukan setiap sudut secara terpisah jika kamu ingin. Disini adalah aturannya:

  • Empat nilai - nilai pertama menentukan top-left corner, nilai kedua menerapkan top-right corner, nilai ketiga menerapkan bottom-right corner, dan nilai keempat menerapkan bottom-left corner
  • Tiga nilai - nilai pertama menentukan top-left corner, nilai kedua menerapkan top-right corner dan bottom-right corner, dan nilai ketiga menerapkan bottom-left corner
  • Dua nilai - nilai pertama menentukan top-left corner dan bottom-right, nilai kedua menerapkan top-right corner dan bottom-right corner
  • Satu nilai - Semua (keempat) sudut ditumpulkan secara sama

Disini adalah 3 contoh:

1. Empat nilai - border-radius: 15px 50px 30px 5px

2. Tiga nilai - border-radius: 15px 50px 30px:

3. Dua nilai - border-radius: 15px 50px:

Disini adalah kodenya:

CSS3 Rounded Corners Properties

Properti Deskripsi
border-radius Sebuah shorthand property untuk mengatur semua (keempat) border radius dalam datu deklarasi
border-top-left-radius Mendefinisikan bentuk border dari sudut kiri atas
border-top-right-radius Mendefinisikan bentuk border dari sudut kanan atas
border-bottom-right-radius Mendefinisikan bentuk border dari sudut kanan bawah
border-bottom-left-radius Mendefinisikan bentuk border dari sudut kiri bawah
