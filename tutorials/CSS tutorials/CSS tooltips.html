CSS Tooltip

Membuat tooltip (alat saran) dengan CSS.

Demo: Tooltip Examples

Sebuah tooltips seringkali digunakan untuk menentukan informasi ekstra tentang sesuatu ketika pengguna atau user menggerakan pointer mouse ke sebuah element:

Basic Tooltip

Tooptip Dasar

Buatlah tooltip yang terlihat ketika ppuser mengerakan mouse ke sebuah element:

Contoh:

<style>
/* Tooltip container atau Penampung Tooltip */
.tooltip {
    position: relative;
    display: inline-block;
    border-bottom: 1px dotted black; /* Jika kamu ingin garis titik dibawah hoverable text */ }

/* Tooltip text */
.tooltip .tooltiptext {
    visibility: hidden;
    width: 120px;
    background-color: black;
    color: #fff;
    text-align: center;
    padding: 5px 0;
    border-radius: 6px;
 
    /* Posisi dsri teks tooltip - lihatlah contoh dibawah! */
    position: absolute;
    z-index: 1;
}

/* Menampilkan teks tooltip ketika kamu menggerakan mouse pada tooltip container */
.tooltip:hover .tooltiptext {
    visibility: visible;
}
</style>

Contoh Menerangkan

HTML → Gunakanlah element penamoung (seperti <div>) dan tambahkanlah class "tooltip" untuk itu. Ketika user menggerakan mouse pada <div> ini, ia akan menampilkan teks tooltip.

Teks tooltip ditempatkan didalam sebuah inline element (seperti <span>) dengan class="tooltiptext".

CSS → Class tooltip menggunakan position: relative; yang mana diperlukan untuk posisi teks tooltip (position: absolute;). Catatan: Lihatlah contih dibawah tentang bagaimana untuk posisi tooltip.

class tooltiptext menggenggam teks tooltip yang sebenernya. Class tooltiptext disembunyikan (hidden) sebagai default, dan akan ditampilkan saat hover (lihatlah dibawah). Kita telah menambahkan beberapa style dasar untuknya: 120px width, black background color, white text color, centered text, dan 5px top and bottom padding.

Properti CSS3 border-radius digunakan untuk menambahkan sudut tumpul pada teks tooltip.

selector :hover digunakan untuk menampilkan teks tooltip ketika pengguna menggerakan mouse pada <div> dengan class="tooltip".

Positioning Tooltips

Penempatan Tooltips

Pada contoh ini, tooltip ditempatkan ke kanan (left: 105%;) dari "hoverable" text (<div>). perhatikanlah juga top: -5px; digunakan untuk menempatkan ia pada pertengahan dari element penampungnya. Kita gunakan nomor 5 karena teks tooltio memiliki top dan bottom padding 5px. Jika kamu meningkatkan paddingnya, tingkatkan juga nilai dari properti top untuk memastikan bahwa ia (tooltip) tetap dalam pertengahan (jika ini merupakan sesuatu yang kamu inginkan). menerapkan hal yang sama jika kamu ingin tooltip ditempatkan ke kiri.

Right Tooltip:

.tooltip .tooltiptext {
    top: -5px;
    left: 105%;
}

Tooltip Arrows

Untuk membuat sebuah arrow (arah panah) yang harus diperlihatkan dari sisi tertentu dari tooltip, tambahkanlah kontent "kosong" (content:" ";) setrlah tooltip. dengan pseudo-element class ::after dengan properti content. Arrow itu sendiri dibuat menggunakan borders. Ini akan membuat tooltip terlihat seperti speech bubble.

Contoh ini menampilkan bagaimana untuk menambahkan sebuah arrow (arah oanah) ke bawah dari tooltip:

Top Arrow:

.tooltip .tooltiptext::after {
    content: " ";
    position: absolute;
    top: 100%; /* Berada atas dari tooltip */
    left: 50%;
    margin-left: -5px;
    border-width: 5px;
    border-style: solid;
    border-color: black transparent transparent transparent;
}

<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
.tooltip {
position: relative;
display: inline-block;
border-bottom: 1px dotted black;
}

.tooltip .tooltiptext {
visibility: hidden;
width: 120px;
background-color: black;
color: #fff;
text-align: center;
border-radius: 6px;
padding: 5px 0;
position: absolute;
z-index: 1;
bottom: 150%;
left: 50%;
margin-left: -60px;
}

.tooltip .tooltiptext::after {

Posisi arrow didalam tooltip: top: 100%; akan menempatkan arrow dibawah tooltip. left: 50%; akan menempatkan arrow ditengah.

Catatan: properti border-width menentukan ukuran dsri arrow. Jika kamu mengubahnya, ubah juga nilai margin-left menjadi sama dengan border-width. Ini akan menjaga arrow berada ditengah.

border-color digunakan untuk transformasi konten kedalam sebuah arrow. Kita mengatur border top menjadi hitam, dan mengistirahatkan menjadi transparan. Jika semua sisi adalah hitam, kamu harus mengakhiri lagi dengan sebuah black square box.

Fade In Tooltips (Animation)

Memudarkan Tooltip (Animasi)

Jika kamu ingin memudarkan teks tooltip ketika ia ditampilkan, kamu bisa menggunakan properti transitiin CSS3 bersamaan dengan properti opacity, dan menjalankan dari ketidak terlihatan secara lengkap menjadi 100% terlihat, pada nomor dari detik yang ditentukan (1 detik dalam setiap contoh).

Bottom Arrow:

.tooltip .tooltiptext {
    opacity: 0;
    transition: opacity 1s;
}

.tooltip:hover .tooltiptext {
    opacity: 1;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
.tooltip {
position: relative;
display: inline-block;
border-bottom: 1px dotted black;
}

.tooltip .tooltiptext {
visibility: hidden;
width: 120px;
background-color: black;
color: #fff;
text-align: center;
border-radius: 6px;
padding: 5px 0;
position: absolute;
z-index: 1;
bottom: 100%;
left: 50%;
margin-left: -60px;

/* Fade in tooltip - takes 1 second to go from 0% to 100% opac: */
opacity: 0;
transition: opacity 1s;
}

.tooltip:hover .tooltiptext {
visibility: visible;
opacity: 1;
}
</style>
</head>
<body>

<div class="tooltip">Hover over me
<span class="tooltiptext">Tooltip text</span>
</div>

</body>
