CSS Height and Width Dimension

CSS Dimension Properties

Properti dimensi CSS mengizinkan kamu untuk mengendalikan tinggi dan lebar sebuah element.

Element ini memiliki width (lebar) 100%.

Setting height and width

Pengaturan height dan width

Properti height dan width digunakan untuk mengatur tinggi dan lebar suatu element.

height dan width bisa diatur menjadi auto (ini merupakan default, Mengartikan bahwa browser mengkalkulasikan tinggi dan lebar secara otomatis), atau ditentukan dalam nilai seperti px, cm, atau sebagainya, atau dalam persen (%) dari blok pengisi.

Element ini memiliki tinggi 100 pixel dan lebar 500 pixel.

Catatan: properti height dan width tidak termasuk padding, border atau margin; mereka mengatur tinggi dan lebar dari area didslsm padding, border, dan margin element!

Contoh berikut menampilkan element <div> dengan tinggi 100px dan lebar 500px:

Input

div.p02 {
    width: 500px;
    height: 100px;
    border: 3px solid #73AD21;
}

Element ini memiliki tinggi 100 pixel dan lebar 500 pixel.

Setting max-width

Pengaturan maxz-width

Properti max-width digunakan untuk mengatur lebar maksimum dari suatu element.

max-width bisa ditentukan dalam nilai (satuan) panjang , seperti px, cm, dan sebagainya, atau dalam persen (%) dari blok pengisi, atau mengatur menjadi none (ini adalah default. Artinya bahwa tidak ada (nilai) lebar maksimum)

Masalah dengan <div> diatas terjadi ketika browser window lebih kecil daripada lebar element (500px). Browser kemudian menambahkan sebuah horizontal scrollbar untuk halaman.

Penggunaan max-width sebagai pengganti, dalam situasi ini, akan meningkatkan browser's handling untuk small window.

Tip: Drag-lah (geserlah) browser window menjadi lebih kecil daripada 500px lebar, untuk melihat perbedaan antara dua div!

Element <div> ini memiliki tinggi (height) 100 pixel dan lebar maksimal (max-width) 500 pixel.

Catatan Nilai dari max-width mengesampingkan lebar (width) element.

Fontih berikut menampilkan sebuah element <div> dengan height 100px dan lebar max-width 500px:

Input

div.p03 {
    max-width: 500px;
    height: 100px;
    border: 3px solid #73AD21;
}

Element <div> ini memiliki tinggi (height) 100 pixel dan lebar maksimal (max-width) 500 pixel.
