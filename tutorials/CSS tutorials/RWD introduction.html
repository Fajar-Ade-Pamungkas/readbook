Responsive Web Design - Introduction

What is Responsive Web Design?

Apa itu Responsive Web Design?

Responsive web design membuat web page kamu terlihat bagus dia semua perangkat.

Responsive web design hanya menggunakan HTML dan CSS.

Responsive web design bukan merupakan sebuah program atau sebuah JavaScript.

Designing For The Best Experience For All Users

Perancangan Untuk Pengalaman Terbaik untuk Semua Pengguna.

Web page bisa ditampilkan dalam banyak perangkat yang berbeda: desktops, tablets, dan phones. Web page kamu harus terlihat bagus dsn mudsh untuk digunakan, tanpa memperhatikan perangkat yang digunakan.

Web page harusnya tidak meninggalkan informasi ke perangkat yang lebih kecil, tapi lebih baik menyesuaikan konten web page tersebut ke perangkat apapun dengan baik:

Ini disebut dengan Responsive Web Design ketika kamu menggunakan CSS dan HTML untuk resize (mengukur kembali), hide (sembunyikan), shrink (menyusutkan), enlarge (memperbesar), atau move (memindahkan) konten agar membuatnya terlihat lebih bagus pada layar apapun.
