CSS Combinators

CSS Combinators

Combinator adalag sesuatu yang menjelaskan hubungan antara selector.

CSS selector bisa berisikan lebih dari satu selector. Diantara selector, kita bisa memasukan sebuah combinator.

Ada empat combinator yang berbeda-beda dalam CSS3:

  • descendant selector (space)
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)

Descendant Selector

Descendant selector mencocokan semua element yang merupakan descendant (keturunan) dari suatu element tertentu.

Contoh berikut menyeleksi semua eleemnt <p> didalam element <div>:

Contoh:

div p {
    background-color: yellow;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div p {
    background-color: yellow;
}
</style>
</head>
<body>

<div>
<p>Paragraph 1 in the div.</p>
<p>Paragraph 2 in the div.</p>
<span><p>Paragraph 3 in the div.</p></span>
</div>

<p>Paragraph 4. Not in a div.</p>
<p>Paragraph 5. Not in a div.</p>

</body>
</html>

Paragraph 1 in the div.

Paragraph 2 in the div.

Paragraph 3 in the div.

Paragraph 4. Not in a div.

Paragraph 5. Not in a div.

Child Selector

Child selector menyeleksi semua element yang merupakan anak yang dekat (immediate children) dari suatu element tertentu:

Contoh berikut menyeleksi semua element <p> yang merupakan anak yang dekat (immediate children) dari sebuah element <div>:

Contoh:

div > p {
    background-color: yellow;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div > p {
    background-color: yellow;
}
</style>
</head>
<body>

<div>
<p>Paragraph 1 in the div.</p>
<p>Paragraph 2 in the div.</p>
<span><p>Paragraph 3 in the div.</p></span> <!-- not Child but Descendant-->
</div>

<p>Paragraph 4. Not in a div.</p>
<p>Paragraph 5. Not in a div.</p>

</body>
</html>

Paragraph 1 in the div.

Paragraph 2 in the div.

Paragraph 3 in the div.

Paragraph 4. Not in a div.

Paragraph 5. Not in a div.

Adjacent Sibling Selector

Adjacent sibling selector menyeleksi semua element yang merupakan kakak/adik yang berdekatan (adjacent sibling) dari suatu element tertentu.

Sibling element harus memiliki element induk (parent element) yang sama dan "adjacent" (berdekatan) berarti "immediately following" (berikut dengan segera).

Contoh berikut menyeleksi semua element <p> yang ditempatkan dengan segera setelah element <div>:

Contoh:

div + p {
    background-color: yellow;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div + p {
    background-color: yellow;
}
</style>
</head>
<body>

<div>
<p>Paragraph 1 in the div.</p>
<p>Paragraph 2 in the div.</p>
</div>

<p>Paragraph 5. Not in a div.</p>
<p>Paragraph 4. Not in a div.</p>

</body>
</html>

Paragraph 1 in the div.

Paragraph 2 in the div.

Paragraph 3. Not in a div.

Paragraph 4. Not in a div.

General Sibling Selector

General sibling selector menyeleksi semua element yang merupakan kakak/adik (sibling) dari suatu element tertentu.

Contoh berikut menyeleksi semua element <p> yang merupakan kakak/adik (sibling) dari element <div>:

Contoh:

div ~ p {
    background-color: yellow;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div ~ p {
    background-color: yellow;
}
</style>
</head>
<body>

<div>
<p>Paragraph 1 in the div.</p>
<p>Paragraph 2 in the div.</p>
</div>

<p>Paragraph 3. Not in a div.</p>
<p>Paragraph 4. Not in a div.</p>

</body>
</html>

Paragraph 1 in the div.

Paragraph 2 in the div.

Paragraph 3. Not in a div.

Paragraph 4. Not in a div.
