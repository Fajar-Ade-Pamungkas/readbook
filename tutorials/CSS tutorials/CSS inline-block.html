CSS Layout - inline-block

The inline-block Value

Inline-block sudah memungkinkan untuk waktu yang lama untuk membuat sebuah kotak grid (grid of boxes) yang mengisi lebar dan melilit (wrap) browser dengan bagus (ketika browser diukur ulang), dengan menggunakan properti float.

Akan tetapi, nilai inline-block dari properti display membuat ini bahkan lebih mudah.

Element inline-block seperti element dalam baris (inline) tapi merrka bisa memiliki sebuah lebar (width) dan tinggi (height).

Examples

Cara lama - Pengunaan float (perhatikanlah bahwa kita juga perlu memnentukan properti clear untuk element setelah floating boxes atau kotak mengapung):

Contoh:

.floating-box {
    float: left;
    width: 150px;
    height: 75px;
    margin: 10px;
    border: 3px solid #73AD21;
}

.after-box {
    clear: left;
border: 3px solid red;
}

Efek yang sama bisa dicapai dengan penggunaan nilai inline-block dari properti display (perhatikanlah bahwa tidak ada properti clear yang diperlukan):

Contoh:

.floating-box {
    display: inline-block;
    width: 150px;
    height: 75px;
    margin: 10px;
    border: 3px solid #73AD21;
}
.after-box {
border: 3px solid red;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
.floating-box {
    display: inline-block;
    width: 150px;
    height: 75px;
    margin: 10px;
    border: 3px solid #73AD21;
}
.after-box {
border: 3px solid red;
}
</style>
</head>
<body>
