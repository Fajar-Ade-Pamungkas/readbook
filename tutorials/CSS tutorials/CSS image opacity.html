CSS Image Opacity / Transparency

Oembuatan image transparan menggunakan CSS dengan mudah.

Properti opacity CSS merupakan bagian dari rekomendasi CSS3.

Example 1 - Creating a Transparent Image

Contoh 1 - Pembuatan sebuah Image/Foto Transparan

Properti CSS3 untuk tembus pandang adalah opacity.

Pertama-tama kita akan menampilkan pada kamu, bagaimana untuk membuat sebuah image transparan dengan CSS.

Regular image:
image with transparency:

Lihatlah CSS berikut:

Contoh:

img {
    opacity: 0.4;
    filter: alpha(opacity=40); /* For IE8 and earlier */
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
img {
    opacity: 0.4;
    filter: alpha(opacity=40); /* For IE8 and earlier */
}
</style>
</head>
<body>

<h1>Image Transparency</h1>
<img src="klematis.jpg" width="150" height="113" alt="klematis">

</body>
</html>

Image Transparency

Properti opacity bisa menambil nilai daei 0.0 - 1.0. Lebih kecil nilainya maka lebih transparan.

IE8 dan versi sebelumnya menggunakan filter:alpha(opacity=x).x bisa menambil nilai dari 0 - 100. Nilai lebih kecil membuat element lebih transparan.

Example 2 - Image Transparency - Hover Effect

Contoh 2 - Image Transparan (dengan) Efek Hover

Mouse over the images:

CSS terlihat seperti ini:

Contoh:

img {
    opacity: 0.4;
    filter: alpha(opacity=40); /* For IE8 and earlier */
}

img:hover {
    opacity: 1.0;
    filter: alpha(opacity=100); /* For IE8 and earlier */
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
img {
    opacity: 0.4;
    filter: alpha(opacity=40); /* For IE8 and earlier */
}

img:hover {
    opacity: 1.0;
    filter: alpha(opacity=100); /* For IE8 and earlier */
}
</style>
</head>
<body>

<h1>Image Transparency</h1> <img src="klematis.jpg" width="150" height="113" alt="klematis">
<img src="klematis2.jpg" width="150" height="113" alt="klematis">

<p><b>Note:</b> In IE, a !DOCTYPE must be added for the :hover selector to work on other elements than the a element.</p>

</body>
</html>

Image Transparency

Note: In IE, a !DOCTYPE must be added for the :hover selector to work on other elements than the a element.

Blok CSS pertama mirip dengan kode pada Example 1. Pada penambahan, kita telah menambahkan apa yang harus terjadi ketika user menggerakan mouse pada image. Pada kasus ini kita ingin image TIDAK ditransparansikan ketika user meng-hover image. CSS untuk ini adalah opacity: 1;.

Ketika Penunjuk mouse bergerak dari image, image akan ditransparansikan lagi.

Example 3 - Text in Transparent Box

Contoh 3 - Teks dalam Kotak Transparan

Ini merupakan beberapa teks yang ditempatkan pada kotak transparan.

Source code terlihat seperti ini:

 
   

This is some text that is placed in the transparent box.

 

Contoh:

div.background {
    background: url(klematis.jpg) repeat;
    border: 2px solid black;
}

div.transbox {
    margin: 30px;
    background-color: #ffffff;
    border: 1px solid black;
    opacity: 0.6;
    filter: alpha(opacity=60); /* For IE8 and earlier */
}

div.transbox p {
    margin: 5%;
    font-weight: bold;
    color: #000000;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div.background {
    background: url(klematis.jpg) repeat;
    border: 2px solid black;
}

div.transbox {
    margin: 30px;
    background-color: #ffffff;
    border: 1px solid black;
    opacity: 0.6;
    filter: alpha(opacity=60); /* For IE8 and earlier */
}

div.transbox p {
    margin: 5%;
    font-weight: bold;
    color: #000000;
}
</style>
</head>
<body>

<div class="background">
  <div class="transbox">
    <p>This is some text that is placed in the transparent box.</p>
  </div>
</div>

</body>
</html>
 
   

This is some text that is placed in the transparent box.

 

Pertama-tama, kita membuat sebuah element <div> (dengan class="background") dengan sebuah background inage, dan sebuah border. Kemudian kita membuat sebuah <div> lainnya (dengan class="transbox") didalam <div> pertama. <div class="transbox"> memiliki sebuah background color.

dan sebuah border - div adalah transparan. Didalam <div> transparan, kita tambahkan beberapa teks didalam element <p>.
