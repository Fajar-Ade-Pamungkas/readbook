CSS Margins

Properti margin CSS mengatur ukuran white space DILUAR border.

Catatan: Margin merupakan ruang transparan atau completely transparent - dan tidak bisa memiliki sebuah background-color (warna)!

Dengan CSS, kamu memiliki kendali penuh pada margin. Ada properti - properti CSS untuk pengaturan margun untuk setiap sisi dari element (top, right, bottom, and left).

Margin - Individual Sides

CSS memiliki properti untuk penentuan margin utuk setiap sisi sebuah element:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

    • Semua properti margin bisa memiliki nilai berikut:

    • auto - Browser mengkalkulasi margin secara otomatis
    • length - Menentukan margin dalam satuan px, pt, cm, dan sebagainya.
    • % - Menentukan margin dalam satuan persen (%) width atau lebar dari element pengisi
    • inherit - Menentukan bahwa margin harus diwariskan (inherited) dari parent element (elemen induk)

    Catatan: Margin juga memungkinkan untuk menggunakan nilai negatif untuk margin; untuk overlap content (konten saling melingkupi).

    Contoh berikut mengatur perbedaan margin untuk semua keempat sisi dari element </p>:

    Input

    p {
        margin-top: 100px;
        margin-bottom: 100px;
        margin-right: 150px;
        margin-left: 80px;
    }

    This paragraph has a top and bottom margin of 100px, a left margin of 80px, and a right margin of 150px.

    Contoh berikut memperbolehkan marginkiri diwariskan dari parent element:

    Input

    div.container01 {
        border: 1px solid red;
        margin-left: 100px;
    }

    p.p1 {
        margin-left: inherit;
    }

Margin - Shorthand Property

Untuk mempersingkat kode, ini memungkinkan untuk menentukan semua properti margin dalam satu deklarasi.

Properti margin merupakan sebuah shorthand properti untuk properti margin individual sebagai berikut:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Input

p.p2 {
margin:100px 150px 100px 80px;
}

This paragraph has a top and bottom margin of 100px, a left margin of 80px, and a right margin of 150px.

Jadi, Disini adalah penjelasan bagaimana itu bekerja:

Jika properti margin memiliki empat properti:

  • margin: 25px 50px 75px 100px
    1. top margin is 25px
    2. right margin is 50px
    3. bottom margin is 75px
    4. left margin is 100px

Jika properti margin memiliki tiga nilai

  • margin: 25px 50px 75px;
    1. top margin is 25px
    2. right and left margins are 50px
    3. bottom margin is 75px

Jika properti margin memiliki dua nilai:

  • margin: 25px 50px;
    1. top and bottom margins are 25px
    2. right and left margins are 50px

Jika properti margin memiliki satu nilai

  • margin: 25px;
    1. all four margins are 25px

Use of The auto Value

Gunakanlah Nilai auto

Kamu bisa mengatur ptoperti margin menjadi auto untuk (posisi) pusat mendatar dengan kontainernya.

Element lalu akan mengambil kembali lebar yang ditentukan dan sisa spasinya akan di-split (dirobek) secara sama antara kiri dan kanan margin:

Input

div.p4 {
    width: 300px;
    margin: auto;
    border: 1px solid red;
}
