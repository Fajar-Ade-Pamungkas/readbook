How to Use CSS3 Transitions?

Bagaimana untuk Menggunakan CSS3 Transitions?

Untuk membuat sebuah efek transisi, kamu harus menentukan dua hal, yaitu:

  • Properti CSS yang ingin ditambahkan untuk sebuah efek
  • Durasi dari efek tersebut

Catatan: Jika bagian durasi tidak ditentukan, transisi tidak akan memiliki efek, karena nilai default (nya) adalah 0.

Contoh berikut menampilkan element <div> dengan 100px (lebar) * 100px (tinggi) berwarna merah. Element <div> juga telah ditentukan sebuah efek transisi untuk properti width, dengan durasi 2 detik:

Contoh:

div {
width: 100px;
height: 100px;
background: red;
-webkit-transition: width 2s; /* Safari */
transition: width 2s;
}

Efek transisi akan dimulai ketika properti CSS yang ditentukan (yaitu width) berubah nilai.

Sekarang, ayo tentukan sebuah nilai baru untuk properti width ketika user mendekatkan mouse ke element <div> tersebut:

Contoh:

div:hover {
width: 300px;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div {
width: 100px;
height: 100px;
background: red;
-webkit-transition: width 2s; /* For Safari 3.1 to 6.0 */
transition: width 2s;
}

div:hover {
width: 300px;
}
</style>
</head>

Specify the Speed Curve of the Transition

Properti transition-timing-function menentukan kecepatan kurva dari efek transisi:

Properti transition-timing-function bisa memikiki nilai-nilai berikut:

  • ease - Menentukan efek transisi dengan diawali pelan, lalu cepat, lalu diakhiri pelan (ini adalah default)
  • linear - Menentukan efek transisi dengan kecepatan yang sama dari awal sampai akhir
  • ease-in - Menentukan efek transisi dengan awalan pelan
  • ease-out - Menentukan efek transisi dengan akhiran pelan
  • ease-in-out - Menentukan efek transisi dengan awalan dan akhiran pelan
  • cubic-bezier(n,n,n,n) - Membiarkan kamu mendefinisikan nilai kamu sendiri dalam fungsi cubic-bezier

Contoh berikut menampilkan beberapa perbedaan kecepatan kurva yang bisa digunakan:

Contoh:

#div1 {transition-timing-function: linear;}
#div2 {transition-timing-function: ease;}
#div3 {transition-timing-function: ease-in;}
#div4 {transition-timing-function: ease-out;}
#div5 {transition-timing-function: ease-in-out;}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div {
width: 100px;
height: 100px;
background: red;
-webkit-transition: width 2s; /* Safari */
transition: width 2s;
}

/* For Safari 3.1 to 6.0 */
#div1 {-webkit-transition-timing-function: linear;}
#div2 {-webkit-transition-timing-function: ease;}
#div3 {-webkit-transition-timing-function: ease-in;}
#div4 {-webkit-transition-timing-function: ease-out;}
#div5 {-webkit-transition-timing-function: ease-in-out;}

/* Standard syntax */
#div1 {transition-timing-function: linear;}
#div2 {transition-timing-function: ease;}
#div3 {transition-timing-function: ease-in;}
#div4 {transition-timing-function: ease-out;}
#div5 {transition-timing-function: ease-in-out;}

div:hover {
width: 300px;
}
</style>
</head>

Delay the Transition Effect

Penundaan Efek Transisi

Properti transition-delay menentukan sebuah penundaan (alam satuan detik) untuk efek transisi.

Contoh berikut memiliki 1 menit penundaan (delay) sebelum dimulai:

Contoh:

div {
    -webkit-transition-delay: 1s; /* Safari */
    transition-delay: 1s;
}

-webkit-transition: width 3s; /* Safari */
-webkit-transition-delay: 1s; /* Safari */
transition: width 3s;
transition-delay: 1s;

CSS3 Transition Properties

Properti Deskripsi
transition Sebuah shorthand property untuk mengatur empat properti transisi kedalam satu properti
transition-delay Menentukan sebuah penundaan (dalam satuan detik) untuk efek transisi
transition-duration Menentukan seberapa detik atau milidetik sebuah efek transisi sehingga lengkap
transition-property Menentukan nama dari properti CSS yang berefek transisi untuk (element)
transition-timing-function Menentukan kecepatan kurva dari efek transisi
