CSS Border

CSS Border Properties

Properti boder CSS membolehkan kamu untuk menentukan style, width, color dari sebuah element's border.

Element ini memiliki sebuah groove border, tebal 15px dan warna gold.

Border Style

Properti border-style menentukan apa macam - macam border untuk tampilan.

Nilai - nilai berikut diperbolehkan:

  • dotted - Mendefinisikan sebuah border titik
  • dashed - Mendefinisikan sebuah border garis terpisah - pisah
  • solid - Mendefinisikan sebuah border garis
  • double - Mendefinisikan sebuah border ganda
  • groove - Mendefinisikan sebuah border beralur (grooved) 3D. Efek tergantung kepada properti border-color
  • ridge - Mendefinisikan sebuah 3D ridged border. Efek tergantung kepada properti border-color
  • inset - Mendefinisikan sebuah 3D inset border. Efek tergantung kepada properti border-color
  • outset - Mendefinisikan sebuah 3D outset border. Efek tergantung kepada properti border-color
  • none - Mendefinisikan tidak ada border
  • hidden - Mendefinisikan sebuah border tersembunyi

Properti border-style bisa memiliki dari satu untuk 4 nilai (for the top border, right border, bottom border, and the left border).

Input

body {
p.dotted {border-style: dotted;}
p.dashed {border-style: dashed;}
p.solid {border-style: solid;}
p.double {border-style: double;}
p.groove {border-style: groove;}
p.ridge {border-style: ridge;}
p.inset {border-style: inset;}
p.outset {border-style: outset;}
p.none {border-style: none;}
p.hidden {border-style: hidden;}
p.mix {border-style: dotted dashed solid double;}
}

Border Width

Properti border-width menentukan lebar dari keempat border.

width bisa diatur seperti ukuran (satuan) tertentu (seperti px, pt, cm, em, dan lain sebagainya) atau dengan penggunaan salah satu dari nilai yang dikenal: thin, medium, atau thick.

properti border-width bisa memiliki nilai masing-masing untuk setiap sisi bordersatu untuk keempat (for the top border, right border, bottom border, and the left border).

Input

p.one {    
border-style: solid;
    border-width: 5px;
}

p.two {    
border-style: solid;
    border-width: medium;
}

p.three {    
border-style: solid;
    border-width: 2px 10px 4px 20px;
}

Border Color

Properti border-color digunakan untuk mengatur warna dari keempat border.

Warna bisa diatur dengan:

  • sebuah nama warna valid - seperti "red"
  • Sebuah nilai HEX - seperti "#ff0000"
  • Sebuah nilai RGB - seperti "rgb(255,0,0)"
  • transparent

Properti border-color bisa memiliki nilai masing - masing untuk setiap sisi border (for the top border, right border, bottom border, and the left border). 

Jika border-color tidak diatur, border mewariskan (inherits) warna tentang element.

Input

p.one {    
border-style: solid;
    border-color: red;
}

p.two {    
border-style: solid;
    border-color: green;
}

p.three {    
border-style: solid;
    border-color: red green blue yellow;
}

Border - Individual Sides

Dari contoh diatas kamu telah melihat bahwa border memungkinkan untuk menentukan border yang berbeda untuk setiap sisi border.

Di CSS, Juga ada properti untuk penentuan setiap sisi border (top, right, bottom, and left):

Input

p {
border-top-style: dotted;
border-right-style: solid;
border-bottom-style: dotted;
border-left-style: solid;
}

Jadi, disini adalah bagaimana border bekerja:

Jika properti border-style memiliki empat nilai:

  • border-style: dotted solid double dashed;
    1. border atas adalah dotted
    2. border kanan adalah solid
    3. border bawah adalah double
    4. border kiri adalah dashed

Jika properti border-style memikiki tiga nilai:

  • border-style: dotted solid double;
    1. border atas adalah dotted
    2. borders kanan dan kiri adalah solid
    3. border bawah adalah double

Jika properti border-style memikiki dua nilai:

  • border-style: dotted solid;
    1. border atas dan bawah adalah dotted
    2. borders kanan dan kiri adalah solid

Jika properti border-style memikiki satu nilai:

  • border-style: dotted;
    1. Semua border adalah dotted

Properti border-style digunakan pada contoh diatas. Akan tetapi, border-style juga bekerja dengan border-width dan border-color.

Border - Shorthand Property

Seperti yang bisa kamu lihat, Ada banyak properti untuk pertimbangan ketika menyetujui sebuah border.

Untuk shorten kode, Border juga memingkinkan untuk menentukan semua properti border masing - masing dalam satu properti.

Properti border merupakan shorthand property untuk properti boder individual berikut:

  • border-width
  • border-style (required)
  • border-color

Input

p {
border: 5px solid red;
}
