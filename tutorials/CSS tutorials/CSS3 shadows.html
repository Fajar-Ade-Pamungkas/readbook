CSS3 Shadow Effects

Dengan CSS3 kamu busa menambahkan shadow (bayangan) ke teks dan element.

Pada chapter ini kamu akan mempelajari tentang properti-properti berikut:

  • text-shadow
  • box-shadow

Browser Support

Nomor pada tabel menentukan browser versi pertama yang mendukung secara penuh properti ini.

Nomor yang diikut dengan -webkit-, -moz- atau -o- menentukan versi pertama yang bekerja dengan awalan properti.

CSS3 Text Shadow

Properti text-shadow menerapkan bayangan untuk teks.

Pada penggunaan paling sederhana, kamu hanya menentukan horizontal shadow (2px) dan vertical shadow (2px):

Text shadow effect!

Contoh:

h1 {
text-shadow: 2px 2px;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
h1 {
text-shadow: 2px 2px;
}
</style>
</head>
<body>

<h1>Text-shadow effect!</h1>

<p><b>Note:</b> Internet Explorer 9 and earlier versions, do not support the text-shadow property.</p>

</body>
</html>

CSS3 box-shadow Property

Properti box-shadow CSS3 menerapkan bayangan untuk element.

Pada penggunaan paling sederhana, kamu hanya menentukan horizontal dan vertical shadow:

Ini adalah element <div> kuning dengan box-shadow warna hitam

Contoh:

div {
box-shadow: 10px 10px;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div {
width: 300px;
height: 100px;
padding: 15px;
background-color: yellow;
box-shadow: 10px 10px;
}
</style>
</head>
<body>

<div>This is a div element with a box-shadow</div>
</body>
</html>
This is a div element with a box-shadow

Ini adalah element <div> kuning dengan box-shadow warna silver

Kamu juga busa menambahkan bayangan untuk pseudo-class ::before dan ::after, untuk membuat efek yang menarik:

Contoh:

#boxshadow {
position: relative;
-moz-box-shadow: 1px 2px 4px rgba(0, 0, 0,0.5);
-webkit-box-shadow: 1px 2px 4px rgba(0, 0, 0, .5);
box-shadow: 1px 2px 4px rgba(0, 0, 0, .5);
padding: 10px;
background: white;
}

/* Make the image fit the box */
#boxshadow img {
width: 100%;
border: 1px solid #8a4419;
border-style: inset;
}

#boxshadow::after {
content: '';
position: absolute;
z-index: -1; /* hide shadow behind image */
-webkit-box-shadow: 0 15px 20px rgba(0, 0, 0, 0.3);
-moz-box-shadow: 0 15px 20px rgba(0, 0, 0, 0.3);
box-shadow: 0 15px 20px rgba(0, 0, 0, 0.3);
width: 70%;
left: 15%; /* one half of the remaining 30% */
height: 100px;
bottom: 0;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
#boxshadow {
position: relative;
-moz-box-shadow: 1px 2px 4px rgba(0, 0, 0,0.5);
-webkit-box-shadow: 1px 2px 4px rgba(0, 0, 0, .5);
box-shadow: 1px 2px 4px rgba(0, 0, 0, .5);
padding: 10px;
background: white;
}

/* Make the image fit the box */
#boxshadow img {
width: 100%;
border: 1px solid #8a4419;
border-style: inset;
}

box-shadow: 0 15px 20px rgba(0, 0, 0, 0.3);
width: 70%;
left: 15%; /* one half of the remaining 30% */
height: 100px;
bottom: 0;
}
</style>
</head>

Contoh:

div.card {
width: 250px;
box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.2), 0 6px 20px 0 rgba(0, 0, 0, 0.19);
text-align: center;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
/* UNTUK NUMBER */
div.card {
width: 250px;
box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.2), 0 6px 20px 0 rgba(0, 0, 0, 0.19);
text-align: center;
}

div.header {
background-color: #4CAF50;
color: white;
padding: 10px;
font-size: 40px;
}

div.container {
padding: 10px;
}


/* UNTUK IMAGE */
div.polaroid {
width: 250px;
box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.2), 0 6px 20px 0 rgba(0, 0, 0, 0.19);
text-align: center;
}

div.container {
padding: 10px;
}
</style>
</head>
