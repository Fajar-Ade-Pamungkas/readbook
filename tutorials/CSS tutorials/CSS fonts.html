CSS Font Families

Pada CSS, ada dua tipe nama font family:

  • generic family - Suatu kelompok dari font family dengan penampilan yang mirip (seperti "Serif" atau "Monispace")
  • font family - Sebuah font family tertentu (seperti "Time new roman" atau "Arial")

Font Family

Font family dari suatu teks diatur dengan properti font-family.

Properti font-family harus menggenggam beberapa nama font sebagai sistem "fallback". Jika browser tidak men-suppport font pertama, mencoba font selanjutnya dan seterusnya.

Awali dengan font yang kamu inginkan, dan akhiri dengan generic family untuj membiarkan browser mengangkut font yang mirip dalam gereric family, jika tidak font lain disediakan.

Catatan: Jika nama dari font family lebih daru satu kata, harus ditandai kutip, seperti "Times New Roman".

Lebih dari satu font family ditentukan dalam koma pemisah daftar:

Contoh

p.serif {
    font-family: "Times New Roman", Times, serif;
}

p.sansserif {
    font-family: Arial, Helvetica, sans-serif;
}

Font Style

Properti font-family paling banyak digunakan untuk menentukan teks miring.

Propertiini memiliki tiga nilai:

  • normal - teks ditampilkan secara normal
  • italic - teks ditamoilkan secara italic (miring)
  • oblique - teks is "learning" (oblique sangat mirip dengan italic, tapi disupport kecil)

Contoh

p.normal {
    font-style: normal;
}

p.italic {
    font-style: italic;
}

p.oblique {
    font-style: oblique;
}

Font Size

Properti font-size mengatur ukuran dari suatu teks.

Bisa me-manage ukuran teks penting dalam web design.

Selalu gunakan properti HTML tag, seperti <h1> - <h6> untuk heading dan <p> untuk paragraf.

Nilai font-size bisa menjadi absolute atau relative.

Nilai Absolute:

  • Mengatur teks menjadi ukuran tertentu
  • Jangan izinkan pengguna untuk mengubah ukuran teks dalam semua browser (itu buruk untuk alasan accessibility)
  • Absolute size berguna ketika ukuran fisik dari output yang diketahui.

Relative Size:

  • Mengatur ukuran relative untuk melingkupi element
  • Izinkan pengguna untuk mengubah ukuran teks pada browser

Catatan: Jika kamu tidak menentukan font-size (ukuran font), ukuran defualt untuk teks normal seperti paragraf adalah 16px (16px = 1em)

Set Font Size With Pixels

Mengatur Ukuran Font Dengan Pixel

Pengaturan ukuran teks dengan pixel memberikan kamu kontrol penuh ukuran teks:

Contoh

h1 {
    font-size: 40px;
}

h2 {
    font-size: 30px;
}

p {
    font-size: 14px;
}

Set Font Size With Em

Mengatur Ukuran Font Dengan Em

Untuk mengizinkan para pengguna untuk me-resize teks (dalam browser menu), banyak pengembang menggunakan em sebagai pengganti dari pixel.

Satuan ukur em direkomendasikan oleh W3C.

1em sama dengan font saat ini. ukuran teks default dalam browser adalah 16px. Jadi, ukuran default dari 1em adalah 16px.

Ukuran bisa dikalkulasikan dari pixel menjadi em menggunakan formula: pixels/16=em

Contoh

h1 {
    font-size: 2.5em; /* 40px/16=2.5em */
}

h2 {
    font-size: 1.875em; /* 30px/16=1.875em */
}

p {
    font-size: 0.875em; /* 14px/16=0.875em */
}

Font Variant

Properti font-variant menentukan apakah Ya atau Tidak teks harus ditampilkan dalam small-caps font (huruf besar atau kecil).

Pada small-cap fot, Semua lowercase (huruf kecil) diubah menjadi uppercase letter (huruf besar). Akan tetapi, mengubah huruf besar tampak dslsm ukuran font yang kebih kecil daripada huruf besar aslinya dalam teks.

Contoh

p.normal {
    font-variant: normal;
}

p.small {
    font-variant: small-caps;
}
