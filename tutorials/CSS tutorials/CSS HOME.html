CSS Tutorials

CSS adalah bahasa stylesheet (sehelai gaya) yang mendeskripsikan presentasi tentang element HTML (atau XML).

CSS mendeskipsikan bagaimana element harus diterjemahkan pada layar, pada kertas atau pada media lainnya.

Tutorial ini akan mengajari kamu trntang CSS dari dasar menjadi lebih maju.

Examples in Each Chapter

Contoh di Setiap Chapter

Tutorial CSS ini berisikan banyak contoh CSS.

Kamu bisa mencoabnya dengan menyalin kode lalu terapkan pada dokumen kamu.

Contoh CSS

<!DOCTYPE html>
<html>
<head>
<title> Judul Halaman </title>
<style>
body {    
background-color: #d0e4fe;
}

h1 {
    color: orange;
    text-align: center;
}

p {
    font-family: "Times New Roman";
    font-size: 20px;
}
</style>
</head>
<body>
<h1> Ini heading pertamaku </h1>
<p> Ini paragraf pertamaku</p>

</body>
</html>
