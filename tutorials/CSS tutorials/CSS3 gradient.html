CSS3 Gradients

Gradient Background

CSS3 gradient membolehkan kamu untuk menampilkan smooth transitions antara dua atau lebih warna tertentu.

Versi dulu, kamu harus menggunakan image untuk efek ini. Akan tetapi, dengan menggunakan CSS3 gradient kamu bisa mengurangi download time (waktu unduh) dan bandwitdh usage (pemakaian bandwidth). Dalam penambahannya, element dengan gradient terlihat lebih baik ketika di-zoom, karena gradient dihasilkan oleh browser.

CSS3 mendefinisikan dua tipe gradient:

  1. Linear Gradients (goes down/up/left/right/diagonally)
  2. Radial Gradients (defined by their center)

CSS3 Linear Gradients

Untuk membuat sebuah linear gradient kamu harus mendefinisikan dalam paling kurang dua color stop. Color stops adalah warna yang ingin kamu berikan smooth transition. Kamu juga bisa mengatur titik mulai dan arah (atau sudut) bersamaan dengan efek gradient.

Sintaksis

background: linear-gradient(direction, color-stop1, color-stop2, ...);

Linear Gradient - Top to Bottom (this is default)

Contoh berikut menampilkan sebuah linear gradient yang dimulai dari atas. Ini dimulai dengan warna merah, berubah menjadi kuning:

Contoh:

#grad {
background: red; /* For browsers that do not support gradients */
background: -webkit-linear-gradient(red, yellow); /* For Safari 5.1 to 6.0 */
background: -o-linear-gradient(red, yellow); /* For Opera 11.1 to 12.0 */
background: -moz-linear-gradient(red, yellow); /* For Firefox 3.6 to 15 */
background: linear-gradient(red, yellow); /* Standard syntax */
}

Sintaksis

background: linear-gradient(angle, color-stop1, color-stop2);

Using Transparency

CSS3 gradient juga mendukung transparan, yang mana bisa digunakan untuk membuat fading effect.

Untuk menambahkan transparency, kita menggunakan fungsi rgba() untuk mendefinisikan color stops. Parameter terakhir pada fungsi rgba() bisa sebuah nomor dari 0 sampai 1 dan fungsi ini mendefinisikan transparency dari warna 0 mengindikasikan full transparency, 1 mengindikasikan full color (tidak ada transparency).

Contoh berikut menampilkan sebuah kinear gradient yang dimulai dari kiri. Ini dimulai dengan transparan secara penuh, pergeseran menjadi warna merah penuh:

Contoh:

#grad {
background: red; /* For browsers that do not support gradients */
background: -webkit-linear-gradient(left,rgba(255,0,0,0),rgba(255,0,0,1)); /*Safari 5.1-6*/
background: -o-linear-gradient(right,rgba(255,0,0,0),rgba(255,0,0,1)); /*Opera 11.1-12*/
background: -moz-linear-gradient(right,rgba(255,0,0,0),rgba(255,0,0,1)); /*Fx 3.6-15*/
background: linear-gradient(to right, rgba(255,0,0,0), rgba(255,0,0,1)); /*Standard*/
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
#grad1 {
height: 200px;
background: -webkit-linear-gradient(left, rgba(255,0,0,0), rgba(255,0,0,1)); /* For Safari 5.1 to 6.0 */
background: -o-linear-gradient(right, rgba(255,0,0,0), rgba(255,0,0,1)); /* For Opera 11.1 to 12.0 */
background: -moz-linear-gradient(right, rgba(255,0,0,0), rgba(255,0,0,1)); /* For Firefox 3.6 to 15 */
background: linear-gradient(to right, rgba(255,0,0,0), rgba(255,0,0,1)); /* Standard syntax (must be last) */
}
</style>
</head>

CSS3 Radial Gradients

Sebuah radial gradient didefinisikan oleh pusatnya.

Untuk membuat radial gradient kamu juga harus mendefinisikan paling kurang dua color stops.

Sintaksis

background: radial-gradient(shape size at position, start-color, ..., last-color);

Dengan default, bentuknya adalah oval, ukurannya adalah farthest-cornee, dan posisinya di tengah.
