CSS Pseudo-elements

What are Pseudo-Elements?

Apa itu psudo-element?

CSS pseudo-elemnt digunakan untuk me-style bagian-bagian tertentu dari suatu element.

untuk contoh, pseudo-element bisa digunakan untuk:

  • Me-style huruf pertama atau baris pertama, dari sebuah element
  • Menyisipkan konten sebelum (after) atau sesudah (after) konten dari element

Syntax

Sintaksis

Contoh

Sintaksis (cara penulisan) dari pseudo-element:

selector::pseudo-element {
    property:value;
}

Perhatikanlah notasi titik dua - ::first-line versus :first-line

double colon menggantikan notasi single-colon untuj pseudo-element pada CSS3. Ini merupakan sebuah percobaan (attempt) dari W3C untuk membedakan (distinguish) antara pseudo-class dan pseudo-element.

Sintaksis single-colon digunakan untuk kedua pseudo-class dan pseudo-element dalan CSS2 dan CSS1.

Untuk kesesuaian yang terbalik, sintaksis single-colon dapat diterima untuk pseudo-element CSS2 dan CSS1.

The ::first-line Pseudo-element

Pseudo-element ::first-line digunakan untjk menambahkan special style untuk baris pertama dari sebuah teks.

Contoh berikut adalah format baris pertama dari teks pada semua element <p>:

Contoh:

p::first-line {
    color: #ff0000;
    font-variant: small-caps;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
p::first-line {
    color: #ff0000;
    font-variant: small-caps;
}
</style>
</head>
<body>

<p>You can use the ::first-letter pseudo-element to add a special effect to the first character of a text!</p>

</body>
</html>

You can use the ::first-letter pseudo-element to add a special effect to the first character of a text!

Catatan: pseudo-element first-line hanya bisa diterapkan untuk block-level elements.

Properti berikut menerapkan untuk pseudo-element ::first-line:

  • font properties
  • color properties
  • background properties
  • word-spacing
  • letter-spacing
  • text-decoration
  • vertical-align
  • text-transform
  • line-height
  • clear

Pseudo-elements and CSS Classes

Pseudo-element bisa dikombinasikan dengan CSS classes:

Contoh:

p.intro::first-letter {
    color: #ff0000;
    font-size:200%;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
p.intro::first-letter {
    color: #ff0000;
    font-size:200%;
}
</style>
</head>
<body>

<p class="intro">This is an introduction.</p>
<p>This is a paragraph with some text. A bit more text even.</p>

</body>
</html>

This is an introduction.

This is a paragraph with some text. A bit more text even.

Contoh diatas skan menampilkan huruf pertama dsru paragraf dengan class="intro", dalam warna merah dan ukuran larger (lebih besar).

Multiple Pseudo-elements

Beberapa pseudo-element juga busa dikombinasikan.

Pada Contoh berikut, huruf pertama dari paragraf akan menjadi merah, ukuran font menjadi xx-large. Sisa-sisa lainnya dari baris pertama akan menjadi biru, dan dalam small-caps. Sisa-sisa lainnya dari paragraf akan menjadi ukuran font dan warna defautl:

Contoh:

p::first-letter {
    color: #ff0000;
    font-size: xx-large;
}

p::first-line {
    color: #0000ff;
    font-variant: small-caps;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
p::first-letter {
    color: #ff0000;
    font-size: xx-large;
}

p::first-line {
    color: #0000ff;
    font-variant: small-caps;
}
</style>
</head>
<body>

<p>You can combine the ::first-letter and ::first-line pseudo-elements to add a special effect to the first letter and the first line of a text!</p>

</body>
</html>

CSS - The ::before Pseudo-element

Pseudo-element ::before bisa digunakan untuk menyisipkan beberapa konten sebelum konten (yang dimaksud) dari sebuah element.

Contoh berikut menyisipkan sebuah gambar sebelum kontan dari setiap element <h1>:

Contoh:

h1::before {
    content: url(smiley.gif);
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
h1::before {
    content: url(smiley.gif);
}
</style>
</head>
<body>

<h1>This is a heading</h1>
<p>The ::before pseudo-element inserts content before the content of an element.

<h1>This is a heading</h1>
<p><b>Note:</b> IE8 supports the content property only if a !DOCTYPE is specified.</p>

</body>
</html>

CSS - The ::selection Pseudo-element

Pseudo-element ::selection mencocokan/membandingkan bagian dari sebuah element yang diseleksi oleh user.

Properti-properti CSS berikut bisa diterapkan untuk ::selection: color, background, cursor, dan outline.

Contoh berikut membuat teks merah yang diseleksi pada background kuning:

Contoh:

::selection {
    color: red;
    background: yellow;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
::-moz-selection { /* Code for Firefox */
color: red;
background: yellow;
}

::selection {
    color: red;
    background: yellow;
}
</style>
</head>
<body>

<h1>Select some text on this page:</h1>

<p>This is a paragraph.</p> <div>This is some text in a div element.</div>

<p><strong>Note:</strong> ::selection is not supported in Internet Explorer 8 and earlier versions.</p>
<p><strong>Note:</strong> Firefox supports an alternative, the ::-moz-selection property.</p>

</body>
</html>
