CSS3 Introduction

CSS3 adalah standar terakhir untuk CSS.

CSS3 merupakan completely backward-compatible dengan versi sebelumnya dari CSS.

Bagian ini mengajarkan kamu tentang semua feature baru dalam CSS3.

CSS3 Modules

CSS3 telah dibagi kedalam "modules". CSS3 berisikan "spesifikasi CSS dulu" (yang mana telah dipecahkan kedalam bagian yang lebih kecil). Pada penambahan, modul-modul baru ditambahkan.

Beberapa dari modul CSS3 yang paling penting adalah:

  • Selectors
  • Box Model
  • Backgrounds and Borders
  • Image Values and Replaced Content
  • Text Effects
  • 2D/3D Transformations
  • Animations
  • Multiple Column Layout
  • User Interface

Kebanyakan dari properti-properti baru CSS3 yang baru diimplementasikan pada modern browsers.
