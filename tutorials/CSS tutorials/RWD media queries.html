Responsive Web Design - Media Query

What is a Media Query?

Apa itu Media Query ?

Media queri adalah suatu teknik CSS yang diperkenalkan di CSS3.

Media query menggunakan aturan @media untuk memasukan sebuah blok properti CSS hanya jika suatu kondisi tertentu itu benar.

Contoh

Jika browser window itu lebih kecil dari 500px, background akan berubah menjadi lightblue

@media only screen and (max-width: 500px) {
    body {
        background-color: lightblue;
    }
}

Add a Breakpoint

Di chapter sebelumnya, kita membuat sebuah web oage dengan rows (baris) dan columns (kolom), dan itu merespon, tapi itu tidak terlihat bagus di small screen (layar kecil).

Media query bisa membantu dengan itu. Kita bisa menambahkan sebuah breakpoint dimana bagian tertentu dari design akan berprilaku dengan berbeda pada setiap sisi dari breakpoint.

Always Design for Mobile First

Selalu Design untuk Mobile pertama kali.

Mobile First berarti perancangan untuk mobile sebelum perancangan untuk dekstop dan perangkat lainnya (Ini akan membuat halaman menampilkan lebih xepat di browser yang lebih kecil).

Ini berarti bahwa kita harus membuat beberapa perubahaan dalam CSS kita.

Sebagai pengganti dari pengubahan style ketika width pada layar lebih kecil dibandingkan 768px, kita harus mengubah design ketika width menjadi lebih besar dibandingkan 768px. Ini akan membuat design mobile kita pertama kali dimuat:

Contoh HTML

Untuk dekstop:

Sesi pertama dan ketika akan men-span (merentangkan) 3 kolom masing-masing, sesi bagian tengah akan me-span 6 kolom.

Untuk tablet:

Sesi pertama akan me-span 3 kolom, sesi kedua akan me-span 9 dan sesi ketiga akan ditampilkan dibawah dua sesi (tersebut) pertama, dan ia akan me-span 12 kolom:

<div class="row">
<div class="col-3 col-m-3">...</div>

<div class="col-3 col-m-12">...</div>
</div>

Orientation: Portrait / Landscape

Media query bisa juga digunakan untuk mengubah layout dari halaman tergantung pada orientasi dari browser.

Kamu bisa memiliki sebuah set dari properti CSS yang hanya akan diterapkan ketika borwser window lebih luas (lebar) daripada tingginya, yang disebut juga "Landscape" orientation:

Contoh:

Web page akan memiliki background warna lightblue jika orientasi berada dalam mode landscape:

@media only screen and (orientation: landscape) {
    body {
        background-color: lightblue;
    }
}
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>

body {
background-color: lightgreen;
}

@media only screen and (orientation: landscape) {
    body {
        background-color: lightblue;
    }
}

</style>
</head>
<body>

<p>Resize the browser window. When the width of this document is larger than the height, the background-color is "lightblue", otherwise it is "lightgreen".</p>
</body>
</html>
