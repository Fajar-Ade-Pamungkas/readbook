CSS Padding

Properti padding mendefinisikan white space (ruang putih) konten element dan border element.

Padding membersihkan area di sekeliling konten (didalam border) dari suatu element.

Catatan: padding berefek dengan background warna dari element!

Dengan CSS, kamu memiliki kendali penuh pada padding. Ada properti - properri CSS untuk pengaturan padding untuk setiap sisi dari element (top, right, bottom, and left).

Padding - Individual Sides

CSS memiliki properti untuk penentuan padding untuk setiap sisi sebuah element:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Semua properti padding bisa memiliki nilai berikut:

  • length - Menentukan margin dalam satuan px, pt, cm, dan sebagainya.
  • % - Menentukan margin dalam satuan persen (%) width atau lebar dari element pengisi
  • inherit - Menentukan bahwa margin harus diwariskan (inherited) dari parent element (elemen induk)

Contoh berikut mengatur perbedaan margin untuk semua keempat sisi dari element </p>:

Input

p {
    padding-top: 50px;
    padding-bottom: 30px;
    padding-right: 50px;
    padding-left: 80px;
}

This paragraph has a top and bottom margin of 100px, a left margin of 80px, and a right margin of 150px.

Padding - Shorthand Property

Untuk mempersingkat kode, ini memungkinkan untuk menentukan semua properti padding dalam satu deklarasi.

Properti padding merupakan sebuah shorthand properti untuk properti padding individual sebagai berikut:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Input

p.p2 {
padding:20px 50px 10px 30px;
}

This paragraph has a top and bottom margin of 100px, a left margin of 80px, and a right margin of 150px.

Jadi, Disini adalah penjelasan bagaimana itu bekerja:

Jika properti padding memiliki empat properti:

  • padding: 25px 50px 75px 100px
    1. top padding is 25px
    2. right padding is 50px
    3. bottom padding is 75px
    4. left padding is 100px

Jika properti padding memiliki tiga nilai

  • padding: 25px 50px 75px;
    1. top padding is 25px
    2. right and left padding are 50px
    3. bottom padding is 75px

Jika properti padding memiliki dua nilai:

  • padding: 25px 50px;
    1. top and bottom padding are 25px
    2. right and left padding are 50px

Jika properti padding memiliki satu nilai

  • padding: 25px;
    1. all four padding are 25px
