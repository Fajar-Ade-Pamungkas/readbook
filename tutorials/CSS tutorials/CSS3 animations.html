What are CSS3 Animations?

Apa itu CSS3 Animations?

Sebuah animasi membiarkan element secara berangsur-angsur beeubah dari satu style menjadi style lainnya.

Kamu bisa mengubah sebanyak properti CSS yang kamu mau, sebanyak waktu yang kamu mau.

Untuk menggunakan CSS3 animation, kamu pertama kali harus menentukan beberapa keyframes (kerangka kunci) untuk animasi.

Keyframe menggenggam style, element akan memiliki pada waktu yang pasti.

The @keyframes Rule

Ketika kamu menentukan CSS style kedalam aturan @keyframes, animasi akan secara berangsur-angsur berubah daei style saat ini menjadi style baru di waktu yang pasti.

Untuk dapat animasi bekerja, kamu harus melingkari (bind) animasi ke sebuah element.

Contoh berikut melingkari animasi "example" untuk element <div>

Contoh:

/* The animation code */
@keyframes example1 {
from {background-color: red;}
to {background-color: yellow;}
}

/* The element to apply the animation to */
div {
width: 100px;
height: 100px;
background-color: red;
animation-name: example1;
animation-duration: 4s;
}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div {
width: 100px;
height: 100px;
background-color: red;
-webkit-animation-name: example1; /* Chrome, Safari, Opera */
-webkit-animation-duration: 4s; /* Chrome, Safari, Opera */
animation-name: example1;
animation-duration: 4s;
}

/* Chrome, Safari, Opera */
@-webkit-keyframes example1 {
from {background-color: red;}
to {background-color: yellow;}
}

/* Standard syntax */
@keyframes example1 {
from {background-color: red;}
to {background-color: yellow;}
}
</style>
</head>

Catatan: Jika properti animation-duration tidak ditentukan, animasi tidak akan memiliki efek, karena default (nya) adalah 0.

Properti animation-duration juga memungkinkan untuk menggunakan persen. Dengan menggunakan persen, kamu bisa menambah sebanyak style mengubah seperti yang kamu suka.

Specify the Speed Curve of the Animation

Properti animation-timing-function menentukan kecepatan kurva dari animasi.

Ptoperti animation-timing-function bisa memiliki nilai-nilai berikut:

  • ease - Menentukan efek transisi dengan diawali pelan, lalu cepat, lalu diakhiri pelan (ini adalah default)
  • linear - Menentukan efek transisi dengan kecepatan yang sama dari awal sampai akhir
  • ease-in - Menentukan efek transisi dengan awalan pelan
  • ease-out - Menentukan efek transisi dengan akhiran pelan
  • ease-in-out - Menentukan efek transisi dengan awalan dan akhiran pelan
  • cubic-bezier(n,n,n,n) - Membiarkan kamu mendefinisikan nilai kamu sendiri dalam fungsi cubic-bezier cubic-bezier

Contoh berikut menampil beberapa tentang kurva kecepatan yang beda yang bisa digunakan:

Contoh:

#div1 {animation-timing-function: linear;}
#div2 {animation-timing-function: ease;}
#div3 {animation-timing-function: ease-in;}
#div4 {animation-timing-function: ease-out;}
#div5 {animation-timing-function: ease-in-out;}
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
div {
width: 100px;
height: 50px;
background-color: red;
font-weight: bold;
position: relative;
-webkit-animation: mymove9 5s infinite; /* Chrome, Safari, Opera */
animation: mymove9 5s infinite;
}

/* Chrome, Safari, Opera */
#div1 {-webkit-animation-timing-function: linear;}
#div2 {-webkit-animation-timing-function: ease;}
#div3 {-webkit-animation-timing-function: ease-in;}
#div4 {-webkit-animation-timing-function: ease-out;}
#div5 {-webkit-animation-timing-function: ease-in-out;}

/* Standard syntax */
#div1 {animation-timing-function: linear;}
#div2 {animation-timing-function: ease;}
#div3 {animation-timing-function: ease-in;}
#div4 {animation-timing-function: ease-out;}
#div5 {animation-timing-function: ease-in-out;}

/* Chrome, Safari, Opera */
@-webkit-keyframes mymove9 {
from {left: 0px;}
to {left: 300px;}
}

/* Standard syntax */
@keyframes mymove9 {
from {left: 0px;}
to {left: 300px;}
}
</style>
</head>

<p><strong>Note:</strong> The animation-timing-funtion property is not supported in Internet Explorer 9 and earlier versions.</p>

CSS3 Animation Properties

Daftar tabel berikut aturan @keyframes dan semua properti animation:

Properti Deskripsi
@keyframes Menentukan animation code
animation Sebuah shorthand property untuk mengatur semua properi animation
animation-delay Menentukan sebuah penundaan untuk memulai sebuah animasi
animation-direction Menentukan apakah sebuah animation harus dimainkan dalam arah berbalik (reverse direction) atau bundaran pengganti (alternate cycles)
animation-duration Menentukan berapa lama (dalam satuan) detik atau milidetik sebuah animasi untuk selesai dalam satu putaran
animation-fill-mode Menentukan sebuah style untuk elemen ketika animasi tidak dimainkan (ketika animasi diakhiri, atau ketika animasi telah tertunda)
animation-iteration-count Menentukan nomor tentang waktu sebuah animasi harus dimainkan
animation-name Menentukan nama dari @keyframes animation
animation-play-state Menentukan apakah animasi sedang berjalan atau terhenti
animation-timing-function Menentukan kurva kecepatan dari animasi
