JavaScript /jQuery DOM Selectors

jQuery vs JavaScript

jQuery diciptakan pada tahun 2006 oleh John Resig. jQuery didesign untuk menangani Browser Incompatiblities (browser yang tidak sesuai) dan untuk mempermudah HTML DOM Manifulation, Event Handling, Animation dan Ajax.

Untuk lebih dari 10 tahun, jQuery telah menjadi JavaScript library yang paling populer di dunia.

Akan tetapi, setelah JavaScript versi 5 (pada tahun 2009), Kebanyakan jQuery utilities bisa dipecahkan dengan beberapa garis tentang standar JavaScript:
