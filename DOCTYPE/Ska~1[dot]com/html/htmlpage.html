Apa itu HTML

HTML adalah bahasa yang pembangun sebuah halaman web dengan menggunakan pengkodean atau bahasa yang sudah di tetapkan oleh badan internasional ISO. ISO singkatan dari International Standarization of Organization adalah badan internasional yang menaungi segala hal tentang Dunia IT atau computing.


pop

Beberapa istilah dalam HTML

HTML merupakan bahasa paling dasar yang berperan dalam pembangunan webpage, dan html mempunyai beberapa bahasa pengkodean dasar yang disebut Element Tag , yang mana tag tersebut memiliki fungsi yang penting dalam pengisian konten apa yang ingin kita masukan dalam web page. Didalam sebuah tag terdapat kode pelengkap yang disebut juga dengan Attribute , attribute disini berfungsi sebagai pengatur element tag yang kita masukan kedalam web page,dan bila attribute tidak diikut sertakan pada elemen tag maka element tersebut akan ditampilkan secara default atau pengaturan yang semestinya. Kita ambil contoh apabila kita memasukan sebuah gambar pada web page tanpa tanpa kita sertakan attribute didalamnya maka gambar akan tampil berdasarkan ukuran gambar tersebut. Dan berhubung attribute termasuk pengkodean CSS maka kita akan membahasnya di CSS lesson.

Single Tag (jomblo) and Double Tag (berpasangan)

Single Tag atau bisa disebut tag jomblo disini ialah tag yang hanya memiliki satu tag saja untuk dapat menampilkan tampilan pada halaman web. Sedangkan, Double Tag atau tag berpasangan merupakan tag element yang harus memiliki dua tag untuk dapat menampilkan hasil masukan, double tag ini dimulai dengan (opening tag) lalu masukan yang biasanya berisi kata-kata lalu diakhiri dengan (/closing tag).

Berikut adalah contoh single tag dan element tag

(p) ini Paragraph(/p)

Dokumen HTML

Semua dokumen HTML harus dimulai dengan tipe pendeklarasi (!DOCTYPE html). Dokumen HTML itu dimulai dengan opening tag (html). dan diakhiri dengan closing tag (/html). Untuk bagian tampilannya berada diantara tag (body). dan (/body).
