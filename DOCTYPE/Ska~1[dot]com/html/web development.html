Di website ini saya akan jelaskan beberapa istilah Web Development seperti : HTML, CSS, javaScript, PHP, w3.css, SQL dan lain sebagainya. Akan tetapi sebelum itu saya akan sedikit menjelaskan sedikit tentang Web Development

Apa itu Web Development ?

Sebelumnya apakah ada yang mengetahui apa itu Web Development?. Mungkin bagi sebagian prang yang belum mengerti soal IT tidak tau istilah di atas. WEB DEVELOPMENT adalah ilmu komputer yang bergerak dalam hal mengembangan suatu website atau situs web dengan menggunakan 3 main coding languages atau 3 bahasa kode utama yaitu HTML, CSS dan JavaScript.



Salah satu contoh Website paling populer saat ini bergerak di bidang media sosial yaitu Facebook , WhatsApp , Line , Youtube , Twitter dan lain sebagainya. Adapula dalam hal transportasi seperti gojek dan grab, dalam hal Perdagangan seperti Tokopedia , Bukalapak , Shopee . Dan dalam hal reservation atau booking (pemesanan)suatu pelayan seperti hotel atau tiket penerbangan.


Pada awalnya semua itu berawal dari sebuah website yang dikembangkan kembali hingga dikemas dalam bentuk aplikasi di smartphone agar memudahkan pembawaan dan pengaksesannya (portable).


Menjadi seorang web developer itu susah - susah gampang, tergantung sejauh kemauan dan keinginan kita untuk menguasai dunia IT salah satunya membuat website. Di dunia yang semakin modern ini dunia IT merupakan investasi yang paling menguntung bila dilihat dari segi zaman, apalagi segala segi bidang kehidupan sekarang berkenaan dengan dunia teknologi. So if you want to be a success person you should tried it, it's fun guys!.
